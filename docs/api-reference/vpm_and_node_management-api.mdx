Lifecycle control, fleet configuration, and deployment policies for distributed node management.

- **Category**: Platform
- **Complexity**: simple
- **Paths**: 1 | **Schemas**: 2
- **Tier**: Standard
- **Related domains**: `sites`, `system`

## Use Cases

- Manage Virtual Private Mesh (VPM) configuration
- Configure node lifecycle and management
- Monitor VPM and node status

## Primary Resources

- **node_config**: Node configuration for edge device settings
- **vpm_config**: VPM configuration for virtual platform management

|--------|------|-------------|
| GET | `/api/data/namespaces/system/upgrade_status` | Upgrade Status. |
