بيانات وصفية للإثراء مرتبطة بمخططات فحص السلامة. راجع [امتدادات OpenAPI](../development/#openapi-extensions) للاطلاع على تعريفات الامتدادات.

## المخططات المُثرَاة

### النمط

جميع المخططات المطابقة لـ `healthcheck.*SpecType` تتلقى إثراءات:

- `healthcheckCreateSpecType`
- `healthcheckReplaceSpecType`
- `healthcheckGetSpecType`

بالإضافة إلى ذلك، يحتوي المخطط المتداخل `healthcheckHttpHealthCheck` على إثراءات لتكوين فحص السلامة عبر HTTP.

## القيم الافتراضية المُطبَّقة من الخادم

الحقول المعلَّمة بـ `x-f5xc-server-default: true` يتم تطبيق قيمتها الافتراضية (`default`) بواسطة خادم F5 XC API عند حذفها من الطلبات.

### حقول المستوى الأعلى

| الحقل | القيمة الافتراضية | النوع | الوصف |
|-------|-------------------|-------|-------|
| `jitter` | `0` | عدد صحيح | قيمة الارتعاش المطلقة لعشوائية التوقيت |
| `jitter_percent` | `0` | عدد صحيح | الارتعاش المبني على النسبة المئوية لعشوائية التوقيت |

### مخطط healthcheckHttpHealthCheck

| الحقل | القيمة الافتراضية | النوع | الوصف |
|-------|-------------------|-------|-------|
| `use_origin_server_name` | `{}` | كائن | اسم خادم الأصل لترويسة Host |
| `headers` | `{}` | كائن | ترويسات مخصصة |
| `request_headers_to_remove` | `[]` | مصفوفة | الترويسات المراد إزالتها من الطلبات |
| `use_http2` | `false` | قيمة منطقية | دعم HTTP/2 |
| `expected_status_codes` | `[]` | مصفوفة | رموز الحالة المقبولة (فارغة = 200-299) |

## القيم الموصى بها

الحقول المعلَّمة بـ `x-f5xc-recommended-value` تشير إلى القيم التي تملأها وحدة تحكم الويب F5 XC مسبقاً عند إنشاء موارد جديدة.

### حقول المستوى الأعلى

| الحقل | القيمة الموصى بها | النوع | الوصف |
|-------|-------------------|-------|-------|
| `timeout` | `3` | عدد صحيح | مهلة فحص السلامة بالثواني |
| `interval` | `15` | عدد صحيح | الفاصل الزمني بين فحوصات السلامة بالثواني |
| `unhealthy_threshold` | `1` | عدد صحيح | الإخفاقات المتتالية قبل التعليم كغير سليم |
| `healthy_threshold` | `3` | عدد صحيح | النجاحات المتتالية قبل التعليم كسليم |
| `jitter_percent` | `30` | عدد صحيح | نسبة الارتعاش للاستخدام في بيئة الإنتاج |

### مخطط healthcheckHttpHealthCheck

| الحقل | القيمة الموصى بها | النوع | الوصف |
|-------|-------------------|-------|-------|
| `path` | `"/"` | سلسلة نصية | مسار نقطة نهاية فحص السلامة |
| `use_http2` | `false` | قيمة منطقية | إعداد دعم HTTP/2 |
| `expected_status_codes` | `["200"]` | مصفوفة | رموز الحالة التي تشير إلى سلامة الأصل |
| `use_origin_server_name` | `{}` | كائن | اسم خادم الأصل لترويسة Host |

## توصيات متغيرات OneOf

المخططات التي تحتوي على مجموعات حقول متنافية (OneOf) تتضمن `x-f5xc-recommended-oneof-variant` للإشارة إلى المتغير الأكثر استخداماً. تحدد هذه البيانات الوصفية الخيار النموذجي عند وجود خيارات متعددة، بناءً على الإعدادات الافتراضية لوحدة تحكم F5 XC وأنماط الاستخدام الشائعة.

### مجموعات OneOf في المستوى الأعلى

| المخطط | مجموعة OneOf | المتغير الموصى به | الوصف |
|--------|-------------|-------------------|-------|
| `healthcheckCreateSpecType` | `health_check` | `http_health_check` | نوع فحص السلامة عبر HTTP |
| `healthcheckReplaceSpecType` | `health_check` | `http_health_check` | نوع فحص السلامة عبر HTTP |

### مجموعات OneOf في healthcheckHttpHealthCheck

| المخطط | مجموعة OneOf | المتغير الموصى به | الوصف |
|--------|-------------|-------------------|-------|
| `healthcheckHttpHealthCheck` | `host_header_choice` | `use_origin_server_name` | ترويسة Host لطلبات فحص السلامة |

#### خيار ترويسة Host

تتحكم مجموعة OneOf المسماة `host_header_choice` في كيفية تحديد ترويسة Host في طلبات HTTP لفحص السلامة.

| المتغير | النوع | الوصف |
|---------|-------|-------|
| `use_origin_server_name` | `object` (فارغ) | استخدام اسم خادم الأصل كترويسة Host. هذا هو الاختيار الافتراضي في وحدة تحكم F5 XC. |
| `host_header` | `string` | تحديد قيمة مخصصة لترويسة Host لطلبات فحص السلامة. |

**مرجع مخطط API**: `x-ves-oneof-field-host_header_choice: ["host_header", "use_origin_server_name"]`

## مرجع امتدادات OpenAPI

تُضاف هذه الامتدادات الخاصة بالمورِّد إلى مخطط OpenAPI القياسي لنقل السلوك الافتراضي الخاص بـ F5 XC.

### x-f5xc-server-default

**النوع**: `boolean`

عندما تكون القيمة `true`، يشير ذلك إلى أن القيمة الافتراضية (`default`) المصاحبة يفرضها خادم F5 XC API. يمكن حذف الحقول التي تحمل هذا الامتداد بأمان من طلبات API—حيث يطبق الخادم القيمة الافتراضية تلقائياً.

```yaml
use_http2:
  type: boolean
  default: false
  x-f5xc-server-default: true
```

### x-f5xc-recommended-value

**النوع**: `any` (يطابق نوع الحقل)

يحدد قيمة تستخدمها وحدة تحكم الويب F5 XC كقيمة افتراضية مملوءة مسبقاً. هذه القيمة لا يفرضها الخادم ولكنها تمثل التكوين الأولي النموذجي للموارد الجديدة المنشأة عبر وحدة التحكم.

```yaml
timeout:
  type: integer
  x-f5xc-recommended-value: 3
```

### x-f5xc-recommended-oneof-variant

**النوع**: `object` (خريطة من اسم المجموعة إلى اسم المتغير)

للمخططات التي تحتوي على مجموعات حقول متنافية، يحدد المتغير الافتراضي أو الأكثر شيوعاً. المفتاح هو اسم مجموعة OneOf والقيمة هي اسم حقل المتغير الموصى به.

```yaml
healthcheckCreateSpecType:
  type: object
  x-f5xc-recommended-oneof-variant:
    health_check: "http_health_check"
```
## الوصول إلى البيانات

### مواصفات OpenAPI

| الملف | المحتوى |
|-------|---------|
| `docs/specifications/api/virtual.json` | `healthcheckHttpHealthCheck`، `healthcheckCreateSpecType`، `healthcheckReplaceSpecType`، `healthcheckGetSpecType` |
| `docs/specifications/api/openapi.json` | المواصفات المدمجة مع جميع المخططات |

### هيكل validation.json

```text
defaults.resources.healthcheck
├── server_applied      # الحقول ذات x-f5xc-server-default: true
├── recommended         # الحقول ذات x-f5xc-recommended-value
├── oneof_recommended   # توصيات متغيرات OneOf
└── nested_recommended  # القيم الموصى بها للمخططات المتداخلة
```
