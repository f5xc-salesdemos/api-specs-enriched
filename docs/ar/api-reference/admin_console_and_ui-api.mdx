العناصر المرئية ذات النطاق المحدود بمساحة الاسم مع إدارة الإصدارات. نشر الأدوات المخصصة وإدارة الكتالوج لأسطح البوابة.

- **الفئة**: المنصة
- **التعقيد**: بسيط
- **المسارات**: 2 | **المخططات**: 14
- **المستوى**: قياسي
- **المجالات ذات الصلة**: `admin`، `system`

## حالات الاستخدام

- إدارة مكونات واجهة المستخدم الثابتة لوحدة تحكم المشرف
- نشر واسترداد أصول واجهة المستخدم ضمن مساحات الأسماء
- تكوين عناصر واجهة وحدة التحكم
- إدارة بيانات تعريف مكونات واجهة المستخدم المخصصة

## الموارد الأساسية

- **ui_component**: مكون واجهة مستخدم لتخصيص وحدة التحكم
- **static_asset**: أصل ثابت لموارد واجهة المستخدم

## نقاط النهاية

| الطريقة | المسار | الوصف |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | سرد مكون واجهة المستخدم الثابت. |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | الحصول على مكون واجهة المستخدم الثابت. |
