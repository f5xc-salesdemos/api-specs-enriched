عناصر مرئية محددة النطاق مع إدارة الإصدارات. نشر الأدوات المخصصة وإدارة الكتالوج لأسطح البوابة.

- **الفئة**: المنصة
- **التعقيد**: بسيط
- **المسارات**: 2 | **المخططات**: 14
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `admin`، `system`

## حالات الاستخدام

- إدارة مكونات واجهة المستخدم الثابتة لوحدة تحكم المسؤول
- نشر واسترجاع أصول واجهة المستخدم ضمن مساحات الأسماء
- تكوين عناصر واجهة وحدة التحكم
- إدارة البيانات الوصفية لمكونات واجهة المستخدم المخصصة

## الموارد الأساسية

- **ui_component**: مكون واجهة المستخدم لتخصيص وحدة التحكم
- **static_asset**: أصل ثابت لموارد واجهة المستخدم

## نقاط النهاية

| الطريقة | المسار | الوصف |
|---------|--------|-------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | عرض قائمة المكونات الثابتة لواجهة المستخدم. |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | الحصول على مكون ثابت لواجهة المستخدم. |
