معالجة اللغة الطبيعية مع إشارات الجودة ومراقبة الشذوذ. مصادقة الرموز للاشتراك في تدفقات البيانات.

- **الفئة**: AI
- **التعقيد**: بسيط
- **المسارات**: 11 | **المخططات**: 66
- **المستوى**: متقدم


## حالات الاستخدام

- الوصول إلى الميزات المدعومة بالذكاء الاصطناعي
- تكوين سياسات مساعد الذكاء الاصطناعي
- تمكين اكتشاف شذوذ التدفق
- إدارة جمع بيانات الذكاء الاصطناعي

## الموارد الأساسية

- **ai_policy**: سياسة الذكاء الاصطناعي للتحكم في حركة مرور الذكاء الاصطناعي التوليدي
- **ai_gateway**: بوابة الذكاء الاصطناعي لإدارة واجهات برمجة تطبيقات LLM
