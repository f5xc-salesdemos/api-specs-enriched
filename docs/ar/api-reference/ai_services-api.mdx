معالجة اللغة الطبيعية مع إشارات الجودة ومراقبة الحالات الشاذة. مصادقة الرموز للاشتراك في تدفقات البيانات.

- **الفئة**: ذكاء اصطناعي
- **التعقيد**: بسيط
- **المسارات**: 11 | **المخططات**: 66
- **المستوى**: متقدم


## حالات الاستخدام

- الوصول إلى الميزات المدعومة بالذكاء الاصطناعي
- تكوين سياسات مساعد الذكاء الاصطناعي
- تمكين اكتشاف حالات التدفق الشاذة
- إدارة جمع بيانات الذكاء الاصطناعي

## الموارد الأساسية

- **ai_policy**: سياسة الذكاء الاصطناعي للتحكم في حركة بيانات الذكاء الاصطناعي التوليدي
- **ai_gateway**: بوابة الذكاء الاصطناعي لإدارة واجهات برمجة تطبيقات نماذج اللغة الكبيرة
