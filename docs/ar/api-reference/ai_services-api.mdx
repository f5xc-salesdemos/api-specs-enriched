معالجة اللغة الطبيعية مع إشارات الجودة ومراقبة الشذوذ. مصادقة الرمز المميز لاشتراكات تدفق البيانات.

- **الفئة**: الذكاء الاصطناعي
- **التعقيد**: بسيط
- **المسارات**: 11 | **المخططات**: 66
- **المستوى**: متقدم


## حالات الاستخدام

- الوصول إلى الميزات المدعومة بالذكاء الاصطناعي
- تهيئة سياسات المساعد الذكي
- تمكين كشف شذوذات التدفق
- إدارة جمع بيانات الذكاء الاصطناعي

## الموارد الأساسية

- **ai_policy**: سياسة الذكاء الاصطناعي للتحكم في حركة مرور الذكاء الاصطناعي التوليدي
- **ai_gateway**: بوابة الذكاء الاصطناعي لإدارة واجهة برمجة تطبيقات LLM
