فهرسة الموارد مع التصنيف التلقائي وتحديد ملفات الأمان. التسلسلات الهرمية التنظيمية تقسّم الوصول حسب الوظيفة أو سياسة الحماية.

- **الفئة**: الأمان
- **التعقيد**: متقدم
- **المسارات**: 45 | **المخططات**: 279
- **المستوى**: متقدم
- **المجالات ذات الصلة**: `virtual`، `network_security`

## حالات الاستخدام

- اكتشاف وفهرسة واجهات API
- اختبار أمان وسلوك واجهات API
- إدارة بيانات اعتماد واجهات API
- تعريف مجموعات API وسياسات الاختبار

## الموارد الأساسية

- **api_definition**: تعريف مخطط API للاكتشاف والحماية
- **api_endpoint**: تكوين وحماية نقطة نهاية API فردية
- **api_rate_limit**: تكوين تحديد معدل API للتحكم في حركة المرور
