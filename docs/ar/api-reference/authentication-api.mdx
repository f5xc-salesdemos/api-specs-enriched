إدارة الهوية مع تكامل المزودين وسياسات الوصول والتحكم في دورة حياة بيانات الاعتماد.

- **الفئة**: المنصة
- **التعقيد**: بسيط
- **المسارات**: 14 | **المخططات**: 38
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `system`، `users`

## حالات الاستخدام

- تكوين آليات المصادقة
- إدارة مزودي OIDC وOAuth
- تكوين توفير المستخدمين عبر SCIM
- إدارة بيانات اعتماد API والوصول
- تكوين سياسات تسجيل الحسابات

## الموارد الأساسية

- **authentication_policy**: سياسة المصادقة لوصول المستخدمين وAPI
- **token**: رمز API للوصول البرمجي
- **api_credential**: بيانات اعتماد API لمصادقة الخدمات
