تنسيق الأجهزة القديمة باستخدام نصوص iRule ومزامنة مجموعات البيانات. ربط الخوادم الافتراضية وجمع المقاييس.

- **الفئة**: المنصة
- **التعقيد**: متقدم
- **المسارات**: 29 | **المخططات**: 238
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `marketplace`

## حالات الاستخدام

- إدارة أجهزة BigIP F5
- تكوين نصوص iRule
- إدارة مجموعات البيانات
- تكامل BigIP CNE

## الموارد الأساسية

- **bigip_pool**: مجمع BIG-IP لتكامل موازنة الأحمال
- **bigip_device**: تسجيل وإدارة أجهزة BIG-IP
