تنسيق الأجهزة القديمة باستخدام نصوص iRule ومزامنة مجموعات البيانات. ربط الخوادم الافتراضية وجمع المقاييس.

- **الفئة**: المنصة
- **التعقيد**: متقدم
- **المسارات**: 29 | **المخططات**: 238
- **المستوى**: متقدم
- **المجالات ذات الصلة**: `marketplace`

## حالات الاستخدام

- إدارة أجهزة BigIP F5
- تكوين نصوص iRule
- إدارة مجموعات البيانات
- دمج BigIP CNE

## الموارد الأساسية

- **bigip_pool**: مجموعة BIG-IP لتكامل موازنة التحميل
- **bigip_device**: تسجيل وإدارة جهاز BIG-IP
