تصنيف التهديدات من خلال التحليل السلوكي ومطابقة التوقيعات. الحظر التلقائي لأنماط حركة المرور الضارة.

- **الفئة**: الأمان
- **التعقيد**: متوسط
- **المسارات**: 19 | **المخططات**: 74
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `bot_defense`, `shape`, `virtual`

## حالات الاستخدام

- تكوين مثيلات الدفاع ضد الروبوتات لكل مساحة أسماء
- إدارة فئات تهديدات TPM للتصنيف
- توفير مفاتيح API لأنظمة الدفاع الآلية
- دمج خدمات استخبارات التهديدات

## الموارد الأساسية

- **bot_defense_instance**: مثيل الدفاع ضد الروبوتات للنشر
- **threat_category**: فئة التهديد للتصنيف
