تصنيف التهديدات مع التحليل السلوكي ومطابقة التوقيعات. الحجب الآلي لأنماط حركة المرور الضارة.

- **الفئة**: الأمان
- **التعقيد**: متوسط
- **المسارات**: 19 | **المخططات**: 74
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `bot_defense`, `shape`, `virtual`

## حالات الاستخدام

- تكوين حالات دفاع Bot لكل نطاق
- إدارة فئات التهديدات TPM للتصنيف
- توفير مفاتيح API لأنظمة الدفاع الآلي
- دمج خدمات استخبارات التهديدات

## الموارد الأساسية

- **bot_defense_instance**: حالة دفاع Bot للنشر
- **threat_category**: فئة التهديد للتصنيف
