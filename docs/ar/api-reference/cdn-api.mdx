التوزيع العالمي مع قواعد التخزين المؤقت وعمليات المسح. مراقبة الأداء والتحليلات.

- **الفئة**: الشبكات
- **التعقيد**: متقدم
- **المسارات**: 29 | **المخططات**: 592
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `virtual`

## حالات الاستخدام

- تكوين موازنة حمل CDN
- إدارة خدمات شبكة توصيل المحتوى
- تكوين سياسات التخزين المؤقت
- إدارة توصيل البيانات وتوزيعها

## الموارد الرئيسية

- **cdn_loadbalancer**: موازن حمل شبكة توصيل المحتوى للتخزين المؤقت على الحافة
- **cdn_origin_pool**: مجموعة خوادم المصدر لاستقاء محتوى CDN
