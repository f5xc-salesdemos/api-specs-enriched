التوزيع العالمي مع قواعد التخزين المؤقت وعمليات التطهير. مراقبة الأداء والتحليلات.

- **الفئة**: الشبكات
- **التعقيد**: متقدم
- **المسارات**: 29 | **المخططات**: 592
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `virtual`

## حالات الاستخدام

- تكوين موازنة أحمال CDN
- إدارة خدمات شبكة توصيل المحتوى
- تكوين سياسات التخزين المؤقت
- إدارة توصيل البيانات وتوزيعها

## الموارد الأساسية

- **cdn_loadbalancer**: موازن أحمال شبكة توصيل المحتوى للتخزين المؤقت على الحافة
- **cdn_origin_pool**: مجموعة خوادم المصدر لتوفير محتوى CDN
