التوفير القائم على الرموز المميزة مع تنزيل الصور والتحقق المسبق قبل الترقية. يتيح تجميع الأساطيل إجراء عمليات جماعية عبر المواقع الموزعة.

- **الفئة**: البنية التحتية
- **التعقيد**: متوسط
- **المسارات**: 28 | **المخططات**: 231
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `customer_edge`، `sites`

## حالات الاستخدام

- إدارة دورة حياة موقع حافة العميل
- تكوين واجهات الشبكة وإعدادات الأسطول
- التعامل مع تسجيل المواقع وسير عمل الرموز المميزة
- تنفيذ ترقيات المواقع مع فحوصات ما قبل الترقية

## الموارد الأساسية

- **site_config**: تكوين الموقع لإعدادات عقدة الحافة
- **fleet_config**: تكوين الأسطول لإدارة المواقع المتعددة
- **registration_token**: رمز التسجيل لإلحاق المواقع
