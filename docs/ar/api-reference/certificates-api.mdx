سلاسل الشهادات وحزم CA الموثوقة. إدارة قوائم الإلغاء وتهيئة البيانات للعمليات المتعلقة بـ PKI.

- **الفئة**: الأمان
- **مستوى التعقيد**: متوسط
- **المسارات**: 16 | **المخططات**: 74
- **المستوى**: Standard
- **المجالات ذات الصلة**: `blindfold`، `system`

## حالات الاستخدام

- إدارة شهادات SSL/TLS
- تهيئة الجهات الموثوقة لإصدار الشهادات (CAs)
- إدارة قوائم إلغاء الشهادات (CRL)
- تهيئة بيانات الشهادات

## الموارد الأساسية

- **certificate**: شهادة TLS للاتصالات HTTPS الآمنة
- **ca_certificate**: شهادة الجهة المصدِّرة للشهادات لسلسلة الثقة
- **certificate_chain**: سلسلة الشهادات للتحقق الكامل من الثقة
