سلاسل الشهادات وحزم جهات الإصدار الموثوقة (CA). إدارة قوائم الإبطال وتكوين البيانات الوصفية لعمليات البنية التحتية للمفاتيح العامة (PKI).

- **الفئة**: الأمان
- **التعقيد**: متوسط
- **المسارات**: 16 | **المخططات**: 74
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `blindfold`، `system`

## حالات الاستخدام

- إدارة شهادات SSL/TLS
- تكوين جهات الإصدار الموثوقة (CAs)
- إدارة قوائم إبطال الشهادات (CRL)
- تكوين بيانات الشهادات الوصفية

## الموارد الأساسية

- **certificate**: شهادة TLS للاتصالات الآمنة عبر HTTPS
- **ca_certificate**: شهادة جهة الإصدار لسلسلة الثقة
- **certificate_chain**: سلسلة الشهادات للتحقق الكامل من الثقة
