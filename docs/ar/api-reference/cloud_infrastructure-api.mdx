اتصالات موفري السحابة المتعددة مع إقران البوابات وتكوين مسار الشبكة. تكامل خزينة بيانات الاعتماد وتعداد الشبكات الفرعية.

- **الفئة**: البنية التحتية
- **التعقيد**: متوسط
- **المسارات**: 31 | **المخططات**: 208
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `sites`, `customer_edge`

## حالات الاستخدام

- الاتصال بموفري السحابة (AWS وAzure وGCP)
- إدارة بيانات اعتماد السحابة والمصادقة
- تكوين الاتصال بالسحابة والتوفير المرن
- ربط مناطق السحابة وإدارتها

## الموارد الأساسية

- **aws_vpc_site**: نشر موقع AWS VPC مع توفير عقدة الحافة
- **azure_vnet_site**: نشر موقع Azure VNet مع توفير عقدة الحافة
- **gcp_vpc_site**: نشر موقع Google Cloud VPC مع توفير عقدة الحافة
- **cloud_credentials**: بيانات اعتماد مصادقة موفر السحابة لنشر الموقع
