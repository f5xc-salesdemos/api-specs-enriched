اتصالات مزودي الخدمات السحابية المتعددة مع ربط البوابات وتكوين مسارات الشبكة. تكامل خزنة بيانات الاعتماد وتعداد الشبكات الفرعية.

- **الفئة**: البنية التحتية
- **التعقيد**: متوسط
- **المسارات**: 31 | **المخططات**: 208
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `sites`، `customer_edge`

## حالات الاستخدام

- الاتصال بمزودي الخدمات السحابية (AWS، Azure، GCP)
- إدارة بيانات الاعتماد والمصادقة السحابية
- تكوين الاتصال السحابي والتوفير المرن
- ربط وإدارة المناطق السحابية

## الموارد الأساسية

- **aws_vpc_site**: نشر موقع AWS VPC مع توفير عقدة الحافة
- **azure_vnet_site**: نشر موقع Azure VNet مع توفير عقدة الحافة
- **gcp_vpc_site**: نشر موقع Google Cloud VPC مع توفير عقدة الحافة
- **cloud_credentials**: بيانات اعتماد مصادقة مزود الخدمة السحابية لنشر المواقع
