تنسيق الحاويات (Pod) دون تعقيدات المجموعة الكاملة. التنفيذ على مواقع الحافة، وفرض الحصص، وملفات تعريف الحوسبة الموحدة للتطبيقات الموزعة.

- **الفئة**: البنية التحتية
- **التعقيد**: متوسط
- **المسارات**: 13 | **المخططات**: 159
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `managed_kubernetes`، `sites`، `service_mesh`

## حالات الاستخدام

- نشر فضاءات أسماء XCCS (خدمات الحاويات) لأحمال العمل متعددة المستأجرين
- إدارة أحمال عمل الحاويات مع تنسيق مبسط
- تكوين عمليات نشر حاويات الحافة الموزعة
- تشغيل التطبيقات المحتواة دون تعقيدات K8s الكاملة

## الموارد الأساسية

- **virtual_k8s**: فضاء أسماء Kubernetes افتراضي لأحمال عمل الحاويات
- **workload**: تكوين نشر أحمال عمل الحاويات
- **pod_security_policy**: سياسة أمان الحاويات لقيود وقت تشغيل الحاويات
