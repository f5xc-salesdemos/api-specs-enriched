تنسيق الحاويات (Pod) دون تعقيد المجموعة الكاملة. تنفيذ على مواقع الحافة، وتطبيق الحصص، وملفات تعريف الحوسبة الموحّدة للتطبيقات الموزعة.

- **الفئة**: البنية التحتية
- **التعقيد**: متوسط
- **المسارات**: 13 | **المخططات**: 159
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `managed_kubernetes`، `sites`، `service_mesh`

## حالات الاستخدام

- نشر نطاقات XCCS (خدمات الحاويات) لأحمال العمل متعددة المستأجرين
- إدارة أحمال عمل الحاويات بتنسيق مبسّط
- تكوين عمليات نشر حاويات الحافة الموزعة
- تشغيل التطبيقات المعبأة في حاويات دون تعقيد K8s الكامل

## الموارد الأساسية

- **virtual_k8s**: نطاق Kubernetes الافتراضي لأحمال عمل الحاويات
- **workload**: تكوين نشر حمل عمل الحاوية
- **pod_security_policy**: سياسة أمان الحاوية (Pod) لقيود وقت تشغيل الحاوية
