سياسات البيانات الحساسة مع قواعد تصنيف مخصصة. تكوين منطقة LMA وضوابط الامتثال الجغرافية.

- **الفئة**: الأمان
- **التعقيد**: بسيط
- **المسارات**: 11 | **المخططات**: 65
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `blindfold`، `client_side_defense`

## حالات الاستخدام

- تكوين سياسات اكتشاف البيانات الحساسة
- تحديد أنواع بيانات مخصصة لتصنيف المعلومات الشخصية القابلة للتحديد (PII)
- إدارة تكوينات منطقة LMA
- دمج التكوينات الجغرافية للامتثال

## الموارد الأساسية

- **sensitive_data_policy**: سياسة البيانات الحساسة لحماية المعلومات الشخصية القابلة للتحديد (PII)
- **data_classification**: تصنيف البيانات لتصنيف المحتوى
