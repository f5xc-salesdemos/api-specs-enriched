سياسات البيانات الحساسة مع قواعد تصنيف مخصصة. تكوين منطقة LMA وعناصر التحكم في الامتثال القائمة على الموقع الجغرافي.

- **الفئة**: الأمان
- **التعقيد**: بسيط
- **المسارات**: 11 | **المخططات**: 65
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `blindfold`، `client_side_defense`

## حالات الاستخدام

- تكوين سياسات كشف البيانات الحساسة
- تعريف أنواع بيانات مخصصة لتصنيف معلومات التعريف الشخصية
- إدارة تكوينات منطقة LMA
- دمج التكوينات الجغرافية للامتثال

## الموارد الأساسية

- **sensitive_data_policy**: سياسة البيانات الحساسة لحماية معلومات التعريف الشخصية
- **data_classification**: تصنيف البيانات لتصنيف المحتوى
