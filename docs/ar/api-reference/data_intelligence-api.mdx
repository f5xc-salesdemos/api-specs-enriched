قواعد التصنيف وسياسات الموارد وتوليد الرؤى لسير عمل تحليل البيانات.

- **الفئة**: العمليات
- **التعقيد**: بسيط
- **المسارات**: 15 | **المخططات**: 73
- **المستوى**: قياسي
- **المجالات ذات الصلة**: `statistics`, `observability`

## حالات الاستخدام

- تحليل بيانات الأمان وحركة المرور
- توليد رؤى ذكية من السجلات
- تهيئة سياسات تحليلات البيانات

## الموارد الأساسية

- **analytics_query**: استعلام تحليلي لتحليل البيانات
- **data_export**: تهيئة تصدير البيانات للتقارير
