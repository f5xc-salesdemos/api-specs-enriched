قوائم الحظر، ومجموعات قواعد جدار الحماية، والحماية المبنية على الأنفاق. تحديد المعدل وتحليل الأنماط لأمان محيط الشبكة.

- **الفئة**: الأمان
- **التعقيد**: متقدم
- **المسارات**: 60 | **المخططات**: 246
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `network_security`, `virtual`

## حالات الاستخدام

- تكوين سياسات الحماية من هجمات DDoS
- مراقبة وتحليل تهديدات DDoS
- تكوين حماية البنية التحتية

## الموارد الأساسية

- **ddos_protection**: تكوين الحماية من DDoS لتنقية حركة المرور
- **ddos_mitigation_rule**: قاعدة التخفيف من DDoS للاستجابة للهجمات
