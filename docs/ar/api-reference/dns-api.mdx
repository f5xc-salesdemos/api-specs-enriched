تحليل الأسماء مع عمليات نقل المناطق وفحوصات السلامة. دعم أنواع السجلات والتفويض.

- **الفئة**: الشبكات
- **التعقيد**: متقدم
- **المسارات**: 49 | **المخططات**: 285
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `virtual`، `network`

## حالات الاستخدام

- تكوين موازنة حمل DNS
- إدارة مناطق ونطاقات DNS
- تكوين سياسات امتثال DNS
- إدارة مجموعات سجلات الموارد (RRSets)

## الموارد الرئيسية

- **dns_zone**: منطقة DNS موثوقة مع إمكانيات إدارة السجلات
- **dns_domain**: تفويض نطاق DNS وإعدادات التكوين
- **dns_load_balancer**: توزيع حركة المرور المبني على DNS مع التوجيه الجغرافي وتجاوز الفشل
