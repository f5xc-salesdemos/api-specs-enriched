تحليل الأسماء مع عمليات نقل المناطق وفحوصات الصحة. دعم أنواع السجلات والتفويض.

- **الفئة**: الشبكات
- **التعقيد**: متقدم
- **المسارات**: 49 | **المخططات**: 285
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `virtual`، `network`

## حالات الاستخدام

- تكوين موازنة أحمال DNS
- إدارة مناطق ونطاقات DNS
- تكوين سياسات امتثال DNS
- إدارة مجموعات سجلات الموارد (RRSets)

## الموارد الأساسية

- **dns_zone**: منطقة DNS موثوقة مع إمكانيات إدارة السجلات
- **dns_domain**: تفويض نطاق DNS وإعدادات التكوين
- **dns_load_balancer**: توزيع حركة المرور المستند إلى DNS مع التوجيه الجغرافي والتجاوز عند الفشل
