دقة الاسم مع نقل المناطق وفحوصات الصحة. أنواع السجلات ودعم التفويض.

- **الفئة**: الشبكات
- **التعقيد**: متقدم
- **المسارات**: 49 | **المخططات**: 285
- **المستوى**: قياسي
- **المجالات ذات الصلة**: `virtual`، `network`

## حالات الاستخدام

- تكوين موازنة تحميل DNS
- إدارة مناطق DNS والنطاقات
- تكوين سياسات الامتثال لـ DNS
- إدارة مجموعات سجلات الموارد (RRSets)

## الموارد الأساسية

- **dns_zone**: منطقة DNS موثوقة مع قدرات إدارة السجلات
- **dns_domain**: إعدادات تفويض نطاق DNS والتكوين
- **dns_load_balancer**: توزيع حركة المرور المستند إلى DNS مع التوجيه الجغرافي والتعافي من الأعطال
