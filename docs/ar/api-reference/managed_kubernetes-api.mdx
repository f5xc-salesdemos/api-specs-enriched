ربط أدوار Kubernetes وسياسات القبول. تكامل السجلات لأحمال عمل EKS وAKS وGKE.

- **الفئة**: البنية التحتية
- **التعقيد**: متوسط
- **المسارات**: 20 | **المخططات**: 95
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `container_services`، `sites`، `service_mesh`

## حالات الاستخدام

- إدارة التحكم في الوصول المستند إلى الأدوار والأمان لمجموعات XCKS (Kubernetes المُدارة)
- تكوين سياسات أمان الحاويات ووحدات التحكم في القبول
- إدارة سجلات الحاويات لعمليات النشر المؤسسية
- التكامل مع مجموعات Kubernetes الخارجية (EKS، AKS، GKE)

## الموارد الأساسية

- **mk8s_cluster**: تكوين مجموعة Kubernetes المُدارة
- **k8s_cluster_role**: تكوين دور التحكم في الوصول المستند إلى الأدوار لمجموعة Kubernetes
- **container_registry**: سجل صور الحاويات لصور أحمال العمل
