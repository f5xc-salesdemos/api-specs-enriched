ارتباطات الأدوار وسياسات القبول في Kubernetes. تكامل السجلات لأحمال عمل EKS وAKS وGKE.

- **الفئة**: البنية التحتية
- **التعقيد**: متوسط
- **المسارات**: 20 | **المخططات**: 95
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `container_services`, `sites`, `service_mesh`

## حالات الاستخدام

- إدارة RBAC للعنقود وأمانه في XCKS (Kubernetes المُدار)
- تهيئة سياسات أمان الـ pod ووحدات التحكم في القبول
- إدارة سجلات الحاويات لعمليات النشر المؤسسية
- التكامل مع عناقيد Kubernetes الخارجية (EKS وAKS وGKE)

## الموارد الأساسية

- **mk8s_cluster**: تهيئة عنقود Kubernetes المُدار
- **k8s_cluster_role**: تهيئة دور RBAC لعنقود Kubernetes
- **container_registry**: سجل صور الحاويات لصور أحمال العمل
