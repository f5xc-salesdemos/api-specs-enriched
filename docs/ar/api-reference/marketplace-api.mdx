توفير أنفاق GRE و IPSec من جهات خارجية مع مؤقتات DPD. تخصيص الموارد المشتركة عبر مساحات الأسماء مع عناصر التحكم في وضع البلاطات.

- **الفئة**: المنصة
- **التعقيد**: متوسط
- **المسارات**: 36 | **المخططات**: 183
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `bigip`، `admin`

## حالات الاستخدام

- الوصول إلى تكاملات الجهات الخارجية والإضافات
- إدارة إضافات السوق
- تكوين Terraform والتكاملات الخارجية
- إدارة سياسات TPM

## الموارد الأساسية

- **marketplace_item**: عنصر السوق لكتالوج الخدمات
- **subscription**: الاشتراك لاستحقاقات الخدمة
