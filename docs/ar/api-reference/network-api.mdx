بروتوكول بوابة الحدود مع إدارة ASN وعلاقات الأنظمة المستقلة. شبكة VPN من موقع إلى موقع تربط مراكز البيانات عبر قنوات مشفرة.

- **الفئة**: الشبكات
- **التعقيد**: متقدم
- **المسارات**: 81 | **المخططات**: 449
- **المستوى**: متقدم
- **المجالات ذات الصلة**: `virtual`، `network_security`، `dns`

## حالات الاستخدام

- تهيئة توجيه BGP وإدارة ASN
- إدارة أنفاق IPsec ومراحل IKE
- تهيئة موصلات الشبكة والمسارات
- إدارة SRv6 والشبكات الفرعية
- تعريف اتصالات الشرائح والسياسات
- تهيئة مجموعات بادئات IP

## الموارد الرئيسية

- **virtual_network**: شبكة افتراضية لاتصال المواقع والتقسيم
- **network_connector**: موصل شبكة للاتصال من موقع إلى موقع أو الاتصال السحابي
- **site_mesh_group**: تهيئة الاتصال الشبكي بين مواقع متعددة
