قواعد جدار الحماية مع قرارات التوجيه استناداً إلى المصدر أو الوجهة أو البروتوكول. يعمل التجزئة على عزل أحمال العمل بينما تتحكم وكلاء الوصول الخارجي في الوصول.

- **الفئة**: الأمان
- **التعقيد**: متقدم
- **المسارات**: 62 | **المخططات**: 386
- **المستوى**: متقدم
- **المجالات ذات الصلة**: `virtual`، `api`، `network`

## حالات الاستخدام

- تكوين سياسات جدار حماية الشبكة وقوائم التحكم بالوصول (ACL)
- إدارة سياسات NAT وإعادة توجيه المنافذ
- تكوين التوجيه المستند إلى السياسة
- تعريف شرائح الشبكة والسياسات
- تكوين سياسات وكيل التوجيه الأمامي

## الموارد الأساسية

- **network_policy**: سياسة أمان الشبكة لتصفية حركة المرور
- **forward_proxy_policy**: سياسة وكيل التوجيه الأمامي للتحكم في حركة المرور الصادرة
- **network_firewall**: جدار حماية الشبكة لحماية حركة المرور على الطبقتين 3/4
