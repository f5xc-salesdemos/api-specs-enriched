قواعد جدار الحماية مع قرارات التوجيه بناءً على المصدر أو الوجهة أو البروتوكول. يعمل التقسيم على عزل أحمال العمل بينما تتحكم الوكلاء الصادرون في الوصول.

- **الفئة**: الأمان
- **التعقيد**: متقدم
- **المسارات**: 62 | **المخططات**: 386
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `virtual`، `api`، `network`

## حالات الاستخدام

- تكوين سياسات جدار حماية الشبكة وقوائم التحكم في الوصول (ACL)
- إدارة سياسات NAT وإعادة توجيه المنافذ
- تكوين التوجيه المبني على السياسات
- تعريف شرائح الشبكة والسياسات
- تكوين سياسات الوكيل الأمامي

## الموارد الأساسية

- **network_policy**: سياسة أمان الشبكة لتصفية حركة المرور
- **forward_proxy_policy**: سياسة الوكيل الأمامي للتحكم في حركة المرور الصادرة
- **network_firewall**: جدار حماية الشبكة لحماية حركة المرور في الطبقة 3/4
