اكتشاف المثيلات، تكامل WAF، والاتصال بشبكة الخدمات. دورة حياة الاشتراك ومزامنة التكوين.

- **الفئة**: منصة
- **التعقيد**: بسيط
- **المسارات**: 13 | **المخططات**: 59
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `marketplace`

## حالات الاستخدام

- إدارة تكاملات منصة NGINX One
- تكوين مثيلات NGINX Plus
- دمج إدارة تكوين NGINX

## الموارد الأساسية

- **nginx_config**: تكوين NGINX لإعدادات الوكيل
- **nginx_upstream**: NGINX upstream لمجموعات الخوادم الخلفية
