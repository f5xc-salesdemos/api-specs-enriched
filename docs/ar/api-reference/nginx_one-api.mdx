اكتشاف المثيلات، وتكامل WAF، واتصال شبكة الخدمات. دورة حياة الاشتراك ومزامنة التكوين.

- **الفئة**: المنصة
- **التعقيد**: بسيط
- **المسارات**: 13 | **المخططات**: 59
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `marketplace`

## حالات الاستخدام

- إدارة تكاملات منصة NGINX One
- تكوين مثيلات NGINX Plus
- دمج إدارة تكوين NGINX

## الموارد الأساسية

- **nginx_config**: تكوين NGINX لإعدادات البروكسي
- **nginx_upstream**: upstream الخاص بـ NGINX لمجموعات خوادم الخلفية
