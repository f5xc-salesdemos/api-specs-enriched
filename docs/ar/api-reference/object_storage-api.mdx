توزيع المكتبة ذات الإصدارات لمدمجي تطبيقات الجوال. تتيح عناوين URL الموقّعة مسبقاً التنزيلات الآمنة مع إصدارات خاصة بكل نظام تشغيل لـ iOS وAndroid.

- **الفئة**: المنصة
- **التعقيد**: بسيط
- **المسارات**: 9 | **المخططات**: 17
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `marketplace`

## حالات الاستخدام

- إدارة خدمات تخزين الكائنات
- تهيئة الكائنات والحاويات المخزّنة
- إدارة سياسات التخزين

## الموارد الأساسية

- **object_store**: مخزن الكائنات لإدارة تخزين البيانات الثنائية الكبيرة
- **bucket**: حاوية التخزين لتخزين الكائنات
