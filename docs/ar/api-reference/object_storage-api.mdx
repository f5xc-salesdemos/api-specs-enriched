توزيع المكتبات المُصدَّرة لمُدمجي تطبيقات الجوال. تتيح عناوين URL الموقعة مسبقاً التنزيلات الآمنة مع بنيات خاصة بنظام التشغيل لكل من iOS وAndroid.

- **الفئة**: المنصة
- **التعقيد**: بسيط
- **المسارات**: 9 | **المخططات**: 17
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `marketplace`

## حالات الاستخدام

- إدارة خدمات تخزين الكائنات
- تكوين الكائنات المخزنة والحاويات
- إدارة سياسات التخزين

## الموارد الأساسية

- **object_store**: مخزن الكائنات لإدارة تخزين البيانات الثنائية الكبيرة
- **bucket**: حاوية تخزين لتخزين الكائنات
