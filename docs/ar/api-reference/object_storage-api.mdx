توزيع المكتبات المُصدَّرة لمُدمجي تطبيقات الهاتف المحمول. تُمكّن عناوين URL المُوقَّعة مسبقاً التنزيلات الآمنة مع بنيات خاصة بنظام التشغيل لكل من iOS وAndroid.

- **الفئة**: المنصة
- **التعقيد**: بسيط
- **المسارات**: 9 | **المخططات**: 17
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `marketplace`

## حالات الاستخدام

- إدارة خدمات تخزين الكائنات
- تكوين الكائنات والحاويات المُخزَّنة
- إدارة سياسات التخزين

## الموارد الأساسية

- **object_store**: مخزن الكائنات لإدارة تخزين البيانات الثنائية الكبيرة
- **bucket**: حاوية التخزين لتخزين الكائنات
