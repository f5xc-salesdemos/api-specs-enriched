توزيع مكتبات مُصدَّرة لمُدمجي تطبيقات الأجهزة المحمولة. تتيح عناوين URL الموقعة مسبقًا تنزيلات آمنة مع بنيات مخصصة لكل نظام تشغيل لـ iOS و Android.

- **الفئة**: المنصة
- **التعقيد**: بسيط
- **المسارات**: 9 | **المخططات**: 17
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `marketplace`

## حالات الاستخدام

- إدارة خدمات تخزين الكائنات
- تكوين الكائنات المُخزَّنة والحاويات
- إدارة سياسات التخزين

## الموارد الرئيسية

- **object_store**: مخزن الكائنات لإدارة تخزين البيانات الثنائية الكبيرة
- **bucket**: حاوية تخزين لتخزين الكائنات
