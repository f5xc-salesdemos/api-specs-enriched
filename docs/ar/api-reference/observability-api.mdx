فحوصات توافر HTTP مع قياس زمن الاستجابة. تنبيهات انتهاء صلاحية الشهادات ولوحات تحكم الحالة العالمية لصحة البنية التحتية.

- **الفئة**: العمليات
- **التعقيد**: متوسط
- **المسارات**: 56 | **المخططات**: 196
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `statistics`، `support`

## حالات الاستخدام

- تكوين المراقبة الاصطناعية
- تحديد سياسات المراقبة والاختبار
- إدارة لوحات تحكم قابلية المراقبة

## الموارد الأساسية

- **log_receiver**: مستقبل السجلات لجمع السجلات المركزي
- **metrics_receiver**: مستقبل المقاييس لجمع المقاييس المركزي
- **alert_policy**: سياسة التنبيه للمراقبة والإشعارات
