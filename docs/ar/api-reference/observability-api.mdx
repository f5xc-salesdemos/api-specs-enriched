فحوصات توفر HTTP مع قياس زمن الاستجابة. تنبيهات انتهاء صلاحية الشهادات ولوحات معلومات الحالة العامة لصحة البنية التحتية.

- **الفئة**: العمليات
- **التعقيد**: متوسط
- **المسارات**: 56 | **المخططات**: 196
- **المستوى**: قياسي
- **المجالات ذات الصلة**: `statistics`، `support`

## حالات الاستخدام

- تكوين المراقبة الاصطناعية
- تحديد سياسات المراقبة والاختبار
- إدارة لوحات معلومات المراقبة والرصد

## الموارد الأساسية

- **log_receiver**: مستقبل السجلات لجمع السجلات المركزي
- **metrics_receiver**: مستقبل المقاييس لجمع المقاييس المركزي
- **alert_policy**: سياسة التنبيهات للمراقبة والإشعارات
