تطبيق حصص قائمة على الوقت مع نوافذ قابلة للتهيئة بالساعات أو الدقائق أو الثواني. ضوابط خاصة بالبروتوكول لتشكيل حركة المرور.

- **الفئة**: الشبكات
- **التعقيد**: بسيط
- **المسارات**: 12 | **المخططات**: 70
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `virtual`، `network_security`

## حالات الاستخدام

- تهيئة سياسات محدد المعدل
- إدارة تكوينات التحكم في حركة المرور
- التحكم في تدفق حركة المرور والتسطير

## الموارد الأساسية

- **rate_limiter**: محدد معدل لتقييد حركة المرور والحماية
- **rate_limiter_policy**: سياسة محدد المعدل مع قواعد تقييد تفصيلية
- **rate_limit_threshold**: تهيئة حد معدل الحد للتحكم في حركة المرور
