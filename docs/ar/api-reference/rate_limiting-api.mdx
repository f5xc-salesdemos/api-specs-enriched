فرض الحصص المستندة إلى الوقت مع نوافذ قابلة للتكوين بالساعات أو الدقائق أو الثواني. عناصر تحكم خاصة بالبروتوكول لتشكيل حركة المرور.

- **الفئة**: الشبكات
- **التعقيد**: بسيط
- **المسارات**: 12 | **المخططات**: 70
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `virtual`، `network_security`

## حالات الاستخدام

- تكوين سياسات تحديد المعدل
- إدارة تكوينات المنظم
- التحكم في تدفق حركة المرور والطوابير

## الموارد الرئيسية

- **rate_limiter**: محدد المعدل لتقييد حركة المرور والحماية
- **rate_limiter_policy**: سياسة محدد المعدل مع قواعد تقييد مفصلة
- **rate_limit_threshold**: تكوين عتبة تحديد المعدل للتحكم في حركة المرور
