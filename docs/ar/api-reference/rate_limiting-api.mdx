فرض الحصص المبنية على الوقت مع نوافذ قابلة للتكوين بالساعات أو الدقائق أو الثواني. عناصر تحكم خاصة بالبروتوكول لتشكيل حركة المرور.

- **الفئة**: الشبكات
- **التعقيد**: بسيط
- **المسارات**: 12 | **المخططات**: 70
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `virtual`، `network_security`

## حالات الاستخدام

- تكوين سياسات محدد المعدل
- إدارة تكوينات المراقب
- التحكم في تدفق حركة المرور والانتظار

## الموارد الأساسية

- **rate_limiter**: محدد المعدل لتقييد حركة المرور والحماية
- **rate_limiter_policy**: سياسة محدد المعدل مع قواعد تقييد مفصلة
- **rate_limit_threshold**: تكوين حد تحديد المعدل للتحكم في حركة المرور
