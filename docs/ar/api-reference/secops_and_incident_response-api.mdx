التخفيف من المستخدمين الضارين مع تصنيف مستوى التهديد. إجراءات استجابة تلقائية لأنماط السلوك المشبوه.

- **الفئة**: الأمان
- **التعقيد**: بسيط
- **المسارات**: 4 | **المخططات**: 32
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `bot_defense`، `virtual`، `network_security`

## حالات الاستخدام

- تكوين سياسات التخفيف التلقائي من التهديدات
- تحديد قواعد الكشف عن المستخدمين الضارين
- إدارة سير عمل الاستجابة للحوادث
- تطبيق الحظر أو تحديد المعدل على التهديدات

## الموارد الرئيسية

- **mitigation_policy**: سياسة التخفيف للاستجابة للحوادث
- **malicious_user_rule**: قاعدة المستخدم الضار للتخفيف من التهديدات

| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations` | إنشاء تخفيف المستخدم الضار. |

| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations` | عرض قائمة تخفيف المستخدم الضار. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | الحصول على تخفيف المستخدم الضار. |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | حذف تخفيف المستخدم الضار. |
