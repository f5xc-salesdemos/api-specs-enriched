التعرف على التهديدات من خلال التحليل السلوكي وبصمة الجهاز. تكامل حزمة SDK للأجهزة المحمولة لحماية التطبيقات.

- **الفئة**: الأمان
- **التعقيد**: متقدم
- **المسارات**: 271 | **المخططات**: 813
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `bot_defense`، `virtual`

## حالات الاستخدام

- تكوين سياسات Shape Security
- إدارة منع البوتات والتهديدات
- تكوين سياسات SafeAP
- تفعيل التعرف على التهديدات

## الموارد الأساسية

- **shape_app_firewall**: جدار حماية تطبيقات Shape للحماية المتقدمة
- **shape_recognizer**: أداة التعرف من Shape لتحليل أنماط حركة المرور
