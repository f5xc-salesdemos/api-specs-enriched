التعرف على التهديدات من خلال التحليل السلوكي وبصمات الأجهزة. تكامل Mobile SDK لحماية التطبيقات.

- **الفئة**: الأمان
- **التعقيد**: متقدم
- **المسارات**: 271 | **المخططات**: 813
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `bot_defense`، `virtual`

## حالات الاستخدام

- تكوين سياسات Shape Security
- إدارة الروبوتات والحماية من التهديدات
- تكوين سياسات SafeAP
- تمكين التعرف على التهديدات

## الموارد الأساسية

- **shape_app_firewall**: جدار حماية تطبيقات Shape للحماية المتقدمة
- **shape_recognizer**: معرّف Shape لتحليل أنماط حركة المرور
