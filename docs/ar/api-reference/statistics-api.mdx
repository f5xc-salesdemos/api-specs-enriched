سياسات التنبيه مع تكاملات المستقبلات. تجميع السجلات وعروض الطوبولوجيا وتتبع صحة الموقع.

- **الفئة**: العمليات
- **التعقيد**: متقدم
- **المسارات**: 77 | **المخططات**: 575
- **المستوى**: قياسي
- **المجالات ذات الصلة**: `observability`, `support`

## حالات الاستخدام

- الوصول إلى إحصائيات التدفق والتحليلات
- إدارة التنبيهات وسياسات التنبيه
- عرض السجلات ومستقبلات السجلات
- إنشاء التقارير والرسوم البيانية
- تتبع الطوبولوجيا واكتشاف الخدمات
- مراقبة الحالة في المواقع

## الموارد الأساسية

- **dashboard**: لوحة التحكم لتصوير المقاييس
- **saved_query**: استعلام محفوظ لتحليل المقاييس والسجلات
