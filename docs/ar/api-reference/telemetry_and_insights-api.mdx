نقاط نهاية الجمع وتخزين المقاييس وتوليد الرؤى لسير عمل قابلية المراقبة.

- **الفئة**: العمليات
- **التعقيد**: متوسط
- **المسارات**: 27 | **المخططات**: 181
- **المستوى**: قياسي
- **المجالات ذات الصلة**: `observability`، `statistics`

## حالات الاستخدام

- جمع بيانات القياس عن بُعد وتحليلها
- توليد رؤى قابلة للتنفيذ من المقاييس
- تكوين سياسات جمع القياسات عن بُعد

## الموارد الأساسية

- **telemetry_receiver**: مستقبل القياس عن بُعد لجمع البيانات
- **insight_query**: استعلام الرؤى للتحليلات
