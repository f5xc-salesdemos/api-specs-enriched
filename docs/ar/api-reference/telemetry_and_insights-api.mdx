نقاط نهاية الجمع، وتخزين المقاييس، وتوليد الرؤى التحليلية لسير عمل المراقبة.

- **الفئة**: العمليات
- **التعقيد**: متوسط
- **المسارات**: 27 | **المخططات**: 181
- **المستوى**: قياسي
- **المجالات ذات الصلة**: `observability`، `statistics`

## حالات الاستخدام

- جمع وتحليل بيانات القياس عن بُعد
- توليد رؤى قابلة للتنفيذ من المقاييس
- تكوين سياسات جمع القياس عن بُعد

## الموارد الرئيسية

- **telemetry_receiver**: مستقبل القياس عن بُعد لجمع البيانات
- **insight_query**: استعلام الرؤى للتحليلات
