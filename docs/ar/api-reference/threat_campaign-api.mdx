سياسات الكشف وتتبع الهجمات وإنشاء قواعد التخفيف الآلي لعمليات الأمان.

- **الفئة**: الأمان
- **التعقيد**: متوسط
- **المسارات**: 1 | **المخططات**: 161
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `bot_defense`, `ddos`

## حالات الاستخدام

- تتبع حملات التهديدات وتحليلها
- مراقبة التهديدات النشطة وأنماط الهجوم
- تكوين تكامل معلومات التهديدات

## الموارد الأساسية

- **threat_campaign_policy**: سياسة اكتشاف حملة التهديدات والتخفيف منها

## نقاط النهاية

| الطريقة | المسار | الوصف |
|--------|------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | الحصول على حملة التهديدات بالمعرف. |
