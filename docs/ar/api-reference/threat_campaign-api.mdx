سياسات الاكتشاف وتتبع الهجمات وإنشاء قواعد التخفيف الآلية لعمليات الأمان.

- **الفئة**: الأمان
- **التعقيد**: متوسط
- **المسارات**: 1 | **المخططات**: 161
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `bot_defense`، `ddos`

## حالات الاستخدام

- تتبع وتحليل حملات التهديد
- مراقبة التهديدات النشطة وأنماط الهجمات
- تكوين تكامل معلومات التهديدات

## الموارد الأساسية

- **threat_campaign_policy**: سياسة اكتشاف حملات التهديد والتخفيف منها

## نقاط النهاية

| الطريقة | المسار | الوصف |
|---------|--------|-------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | الحصول على حملة التهديد حسب المعرّف. |
