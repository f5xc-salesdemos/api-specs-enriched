بيانات اعتماد تسجيل الموقع مع انتهاء صلاحية تلقائي. تحدد مفاتيح التصنيف التصنيفات المسموح بها بينما تُطبَّق العلامات المشتقة تلقائيًا بشكل ديناميكي.

- **الفئة**: المنصة
- **التعقيد**: بسيط
- **المسارات**: 13 | **المخططات**: 50
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `system`، `admin`

## حالات الاستخدام

- إدارة حسابات المستخدمين والرموز
- تكوين تعريف المستخدم
- إدارة إعدادات وتفضيلات المستخدم
- تكوين التصنيفات الضمنية والمعروفة

## الموارد الأساسية

- **user**: حساب مستخدم للوصول إلى المنصة
- **user_role**: دور المستخدم لإدارة الصلاحيات
- **namespace_role**: دور محدد النطاق للصلاحيات الدقيقة

## نقاط النهاية

| الطريقة | المسار | الوصف |
|---------|--------|-------|
| GET | `/api/config/namespaces/system/implicit_labels` | الحصول على التصنيفات الضمنية. |
| POST | `/api/config/namespaces/&#123;namespace&#125;/known_label/create` | إنشاء |
| POST | `/api/config/namespaces/&#123;namespace&#125;/known_label/delete` | حذف |
| GET | `/api/config/namespaces/&#123;namespace&#125;/known_labels` | الحصول |
