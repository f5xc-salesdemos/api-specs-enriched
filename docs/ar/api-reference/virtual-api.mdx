توزيع حركة المرور عبر المناطق مع قواعد التوجيه. فحوصات الصحة وسياسات تجاوز الفشل.

- **الفئة**: الشبكات
- **التعقيد**: متقدم
- **المسارات**: 133 | **المخططات**: 903
- **المستوى**: متقدم
- **النطاقات ذات الصلة**: `dns`, `service_policy`, `network`

## حالات الاستخدام

- تكوين موازنات تحميل HTTP/TCP/UDP
- إدارة مجموعات الأصل والخدمات
- تكوين المضيفين الافتراضيين والتوجيه
- تعريف محدد المعدل وسياسات الخدمة
- إدارة التوجيه المبني على الموقع الجغرافي
- تكوين سياسات الوكيل والتوجيه
- إدارة الحماية من البرمجيات الخبيثة وحملات التهديدات
- تكوين فحوصات الصحة ومراقبة نقاط النهاية

## الموارد الأساسية

- **http_loadbalancer**: موازن تحميل HTTP/HTTPS من الطبقة 7 لتوزيع حركة مرور التطبيقات
- **tcp_loadbalancer**: موازن تحميل TCP/UDP من الطبقة 4 لحركة مرور البروتوكولات غير HTTP
- **origin_pool**: مجموعة خوادم خلفية لتوزيع حركة مرور موازن التحميل
- **healthcheck**: تكوين مراقبة الصحة لتوافر خوادم الأصل
- **app_firewall**: سياسة جدار حماية تطبيقات الويب لحماية حركة مرور HTTP
- **service_policy**: قواعد التحكم في الوصول على مستوى الخدمة وإدارة حركة المرور
- **malicious_user_detection**: الكشف التلقائي عن سلوك المستخدم الضار والتخفيف منه
