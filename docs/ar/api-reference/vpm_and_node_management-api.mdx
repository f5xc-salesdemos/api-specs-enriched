التحكم في دورة الحياة، وتهيئة الأسطول، وسياسات النشر لإدارة العُقد الموزعة.

- **الفئة**: المنصة
- **التعقيد**: بسيط
- **المسارات**: 1 | **المخططات**: 2
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `sites`، `system`

## حالات الاستخدام

- إدارة تهيئة الشبكة الافتراضية الخاصة بالشبكة الخاصة (VPM)
- تهيئة دورة حياة العُقد وإدارتها
- مراقبة حالة VPM والعُقد

## الموارد الأساسية

- **node_config**: تهيئة العُقدة لإعدادات الأجهزة الطرفية
- **vpm_config**: تهيئة VPM لإدارة المنصة الافتراضية

## نقاط النهاية

| الطريقة | المسار | الوصف |
|--------|------|-------------|
| GET | `/api/data/namespaces/system/upgrade_status` | حالة الترقية. |
