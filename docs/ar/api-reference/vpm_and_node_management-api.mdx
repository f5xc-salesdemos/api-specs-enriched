التحكم في دورة الحياة، تكوين الأسطول، وسياسات النشر لإدارة العُقد الموزعة.

- **الفئة**: المنصة
- **التعقيد**: بسيط
- **المسارات**: 1 | **المخططات**: 2
- **المستوى**: قياسي
- **النطاقات ذات الصلة**: `sites`، `system`

## حالات الاستخدام

- إدارة تكوين الشبكة الخاصة الافتراضية (VPM)
- تكوين دورة حياة العُقد وإدارتها
- مراقبة حالة VPM والعُقد

## الموارد الأساسية

- **node_config**: تكوين العُقدة لإعدادات الأجهزة الطرفية
- **vpm_config**: تكوين VPM لإدارة المنصة الافتراضية

## نقاط النهاية

| الطريقة | المسار | الوصف |
|---------|--------|-------|
| GET | `/api/data/namespaces/system/upgrade_status` | حالة الترقية. |
