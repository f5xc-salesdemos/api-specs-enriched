Namespace-bezogene visuelle Elemente mit Versionierung. Bereitstellung benutzerdefinierter Widgets und Katalogverwaltung für Portal-Oberflächen.

- **Kategorie**: Plattform
- **Komplexität**: einfach
- **Pfade**: 2 | **Schemas**: 14
- **Stufe**: Standard
- **Verwandte Domänen**: `admin`, `system`

## Anwendungsfälle

- Verwaltung statischer UI-Komponenten für die Admin-Konsole
- Bereitstellung und Abruf von UI-Assets innerhalb von Namespaces
- Konfiguration von Konsolen-Schnittstellenelementen
- Verwaltung von Metadaten benutzerdefinierter UI-Komponenten

## Primäre Ressourcen

- **ui_component**: UI-Komponente zur Konsolenanpassung
- **static_asset**: Statisches Asset für UI-Ressourcen

## Endpunkte

| Methode | Pfad | Beschreibung |
|---------|------|--------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | Statische UI-Komponenten auflisten. |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | Statische UI-Komponente abrufen. |
