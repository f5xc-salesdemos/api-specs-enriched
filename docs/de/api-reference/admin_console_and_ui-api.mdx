Namespace-bezogene visuelle Elemente mit Versionierung. Benutzerdefinierte Widget-Bereitstellung und Katalogverwaltung für Portaloberflächen.

- **Kategorie**: Plattform
- **Komplexität**: einfach
- **Pfade**: 2 | **Schemata**: 14
- **Stufe**: Standard
- **Verwandte Domänen**: `admin`, `system`

## Anwendungsfälle

- Statische UI-Komponenten für die Admin-Konsole verwalten
- UI-Assets innerhalb von Namespaces bereitstellen und abrufen
- Konsolenoberflächen-Elemente konfigurieren
- Metadaten benutzerdefinierter UI-Komponenten verwalten

## Primäre Ressourcen

- **ui_component**: UI-Komponente zur Konsolenanpassung
- **static_asset**: Statisches Asset für UI-Ressourcen

## Endpunkte

| Methode | Pfad | Beschreibung |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | UI-Static-Komponente auflisten. |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | UI-Static-Komponente abrufen. |
