Verarbeitung natürlicher Sprache mit Qualitätssignalen und Anomaliemonitoring. Token-Authentifizierung für Datenstrom-Abonnements.

- **Kategorie**: KI
- **Komplexität**: einfach
- **Pfade**: 11 | **Schemata**: 66
- **Tier**: Erweitert


## Anwendungsfälle

- Zugriff auf KI-gestützte Funktionen
- KI-Assistentenrichtlinien konfigurieren
- Anomalieerkennung für Datenflüsse aktivieren
- KI-Datenerfassung verwalten

## Primäre Ressourcen

- **ai_policy**: KI-Richtlinie zur Steuerung von generativem KI-Datenverkehr
- **ai_gateway**: KI-Gateway für das LLM-API-Management
