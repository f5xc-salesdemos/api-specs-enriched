Verarbeitung natürlicher Sprache mit Qualitätssignalen und Anomalie-Überwachung. Token-Authentifizierung für Datenstrom-Abonnements.

- **Kategorie**: KI
- **Komplexität**: einfach
- **Pfade**: 11 | **Schemas**: 66
- **Stufe**: Erweitert


## Anwendungsfälle

- Zugriff auf KI-gestützte Funktionen
- KI-Assistenten-Richtlinien konfigurieren
- Flow-Anomalieerkennung aktivieren
- KI-Datenerfassung verwalten

## Primäre Ressourcen

- **ai_policy**: KI-Richtlinie zur Steuerung des generativen KI-Datenverkehrs
- **ai_gateway**: KI-Gateway zur Verwaltung von LLM-APIs
