Ressourcenkatalogisierung mit automatischer Klassifizierung und Sicherheitsprofilerstellung. Organisationshierarchien segmentieren den Zugriff nach Funktion oder Schutzrichtlinie.

- **Kategorie**: Sicherheit
- **Komplexität**: Fortgeschritten
- **Pfade**: 45 | **Schemas**: 279
- **Stufe**: Fortgeschritten
- **Verwandte Domänen**: `virtual`, `network_security`

## Anwendungsfälle

- APIs entdecken und katalogisieren
- API-Sicherheit und -Verhalten testen
- API-Anmeldeinformationen verwalten
- API-Gruppen und Testrichtlinien definieren

## Primäre Ressourcen

- **api_definition**: API-Schema-Definition für Erkennung und Schutz
- **api_endpoint**: Konfiguration und Schutz einzelner API-Endpunkte
- **api_rate_limit**: API-Ratenbegrenzungskonfiguration zur Verkehrssteuerung
