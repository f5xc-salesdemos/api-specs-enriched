Identitätsverwaltung mit Anbieterintegration, Zugriffsrichtlinien und Steuerung des Anmeldedaten-Lebenszyklus.

- **Kategorie**: Plattform
- **Komplexität**: einfach
- **Pfade**: 14 | **Schemas**: 38
- **Stufe**: Standard
- **Verwandte Domänen**: `system`, `users`

## Anwendungsfälle

- Authentifizierungsmechanismen konfigurieren
- OIDC- und OAuth-Anbieter verwalten
- SCIM-Benutzerbereitstellung konfigurieren
- API-Anmeldedaten und Zugriff verwalten
- Richtlinien für Kontoregistrierung konfigurieren

## Primäre Ressourcen

- **authentication_policy**: Authentifizierungsrichtlinie für Benutzer- und API-Zugriff
- **token**: API-Token für programmatischen Zugriff
- **api_credential**: API-Anmeldedaten für Service-Authentifizierung
