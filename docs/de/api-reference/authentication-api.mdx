Identitätsverwaltung mit Anbieterintegration, Zugriffsrichtlinien und Lebenszyklusverwaltung von Anmeldeinformationen.

- **Kategorie**: Plattform
- **Komplexität**: einfach
- **Pfade**: 14 | **Schemas**: 38
- **Stufe**: Standard
- **Verwandte Domänen**: `system`, `users`

## Anwendungsfälle

- Authentifizierungsmechanismen konfigurieren
- OIDC- und OAuth-Anbieter verwalten
- SCIM-Benutzerbereitstellung konfigurieren
- API-Anmeldeinformationen und Zugriff verwalten
- Kontoregistrierungsrichtlinien konfigurieren

## Primäre Ressourcen

- **authentication_policy**: Authentifizierungsrichtlinie für Benutzer- und API-Zugriff
- **token**: API-Token für programmatischen Zugriff
- **api_credential**: API-Anmeldeinformation für die Dienstauthentifizierung
