Legacy-Geräteorchestrierung mit iRule-Skripten und Datengruppen-Synchronisation. Virtual-Server-Bindungen und Metrik-Erfassung.

- **Kategorie**: Plattform
- **Komplexität**: Fortgeschritten
- **Pfade**: 29 | **Schemas**: 238
- **Stufe**: Fortgeschritten
- **Verwandte Domänen**: `marketplace`

## Anwendungsfälle

- BigIP F5-Appliances verwalten
- iRule-Skripte konfigurieren
- Datengruppen verwalten
- BigIP CNE integrieren

## Primäre Ressourcen

- **bigip_pool**: BIG-IP-Pool für Load-Balancing-Integration
- **bigip_device**: BIG-IP-Geräteregistrierung und -verwaltung
