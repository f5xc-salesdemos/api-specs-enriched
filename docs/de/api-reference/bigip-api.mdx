Legacy-Geräteorchestrierung mit iRule-Skripten und Datengruppensynchronisierung. Virtuelle Server-Bindungen und Metriken-Erfassung.

- **Kategorie**: Plattform
- **Komplexität**: fortgeschritten
- **Pfade**: 29 | **Schemas**: 238
- **Stufe**: Erweitert
- **Verwandte Domänen**: `marketplace`

## Anwendungsfälle

- BIG-IP F5-Appliances verwalten
- iRule-Skripte konfigurieren
- Datengruppen verwalten
- BIG-IP CNE integrieren

## Primäre Ressourcen

- **bigip_pool**: BIG-IP-Pool für die Lastverteilungsintegration
- **bigip_device**: BIG-IP-Geräteregistrierung und -verwaltung
