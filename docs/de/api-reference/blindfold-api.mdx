Verwaltung von Verschlüsselungsschlüsseln mit richtlinienbasierter Zugriffssteuerung. Prüfprotokollierung und sicherer Datenschutz.

- **Kategorie**: Sicherheit
- **Komplexität**: moderat
- **Pfade**: 27 | **Schemata**: 163
- **Stufe**: Erweitert
- **Verwandte Domänen**: `client_side_defense`, `certificates`

## Anwendungsfälle

- Geheimnisverwaltungsrichtlinien für die Verschlüsselung konfigurieren
- Verschlüsselung sensibler Daten verwalten
- Datenschutzrichtlinien durchsetzen

## Primäre Ressourcen

- **blindfold_secret**: Verschlüsseltes Geheimnis mit Blindfold-Verschlüsselung
- **secret_policy**: Konfiguration der Geheimnisverwaltungsrichtlinie
- **policy_document**: Richtliniendokument für Zugriffskontrolldefinitionen
