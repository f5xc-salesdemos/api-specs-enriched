Globale Verteilung mit Cache-Regeln und Löschoperationen. Leistungsüberwachung und Analysen.

- **Kategorie**: Netzwerke
- **Komplexität**: fortgeschritten
- **Pfade**: 29 | **Schemas**: 592
- **Stufe**: Fortgeschritten
- **Verwandte Domänen**: `virtual`

## Anwendungsfälle

- CDN-Lastverteilung konfigurieren
- Content-Delivery-Network-Dienste verwalten
- Caching-Richtlinien konfigurieren
- Datenbereitstellung und -verteilung verwalten

## Primäre Ressourcen

- **cdn_loadbalancer**: Content-Delivery-Network-Load-Balancer für Edge-Caching
- **cdn_origin_pool**: Origin-Server-Pool für CDN-Content-Sourcing
