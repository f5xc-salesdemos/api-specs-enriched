Token-basierte Bereitstellung mit Image-Downloads und Pre-Upgrade-Validierung. Flottengruppierung ermöglicht Massenoperationen über verteilte Standorte hinweg.

- **Kategorie**: Infrastruktur
- **Komplexität**: moderat
- **Pfade**: 28 | **Schemas**: 231
- **Stufe**: Standard
- **Verwandte Domänen**: `customer_edge`, `sites`

## Anwendungsfälle

- Verwaltung des Lebenszyklus von Customer-Edge-Standorten
- Konfiguration von Netzwerkschnittstellen und Flotteneinstellungen
- Handhabung der Standortregistrierung und Token-Workflows
- Durchführung von Standort-Upgrades mit Pre-Upgrade-Prüfungen

## Primäre Ressourcen

- **site_config**: Standortkonfiguration für Edge-Node-Einstellungen
- **fleet_config**: Flottenkonfiguration für die Multi-Standort-Verwaltung
- **registration_token**: Registrierungstoken für das Standort-Onboarding
