Zertifikatsketten und vertrauenswürdige Zertifizierungsstellen-Bundles. Verwaltung von Sperrlisten und Manifest-Konfiguration für PKI-Operationen.

- **Kategorie**: Sicherheit
- **Komplexität**: mittel
- **Pfade**: 16 | **Schemata**: 74
- **Stufe**: Standard
- **Verwandte Domänen**: `blindfold`, `system`

## Anwendungsfälle

- SSL/TLS-Zertifikate verwalten
- Vertrauenswürdige Zertifizierungsstellen konfigurieren
- Zertifikatssperrlisten (CRL) verwalten
- Zertifikatsmanifeste konfigurieren

## Primäre Ressourcen

- **certificate**: TLS-Zertifikat für sichere HTTPS-Verbindungen
- **ca_certificate**: Zertifizierungsstellen-Zertifikat für die Vertrauenskette
- **certificate_chain**: Zertifikatskette zur vollständigen Vertrauensverifizierung
