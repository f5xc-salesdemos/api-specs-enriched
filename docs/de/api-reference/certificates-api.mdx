Zertifikatketten und vertrauenswürdige CA-Bundles. Verwaltung von Sperrlisten und Manifestkonfiguration für PKI-Operationen.

- **Kategorie**: Sicherheit
- **Komplexität**: mittel
- **Pfade**: 16 | **Schemata**: 74
- **Tier**: Standard
- **Verwandte Domänen**: `blindfold`, `system`

## Anwendungsfälle

- SSL/TLS-Zertifikate verwalten
- Vertrauenswürdige Zertifizierungsstellen konfigurieren
- Zertifikatsperrlisten (CRL) verwalten
- Zertifikatmanifeste konfigurieren

## Primäre Ressourcen

- **certificate**: TLS-Zertifikat für sichere HTTPS-Verbindungen
- **ca_certificate**: Zertifizierungsstellenzertifikat für die Vertrauenskette
- **certificate_chain**: Zertifikatkette für die vollständige Vertrauensüberprüfung
