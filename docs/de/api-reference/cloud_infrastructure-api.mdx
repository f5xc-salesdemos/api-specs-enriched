Multi-Cloud-Provider-Verbindungen mit Gateway-Peering und Netzwerkpfad-Konfiguration. Integration von Credential-Vaults und Subnetz-Aufzählung.

- **Kategorie**: Infrastruktur
- **Komplexität**: moderat
- **Pfade**: 31 | **Schemas**: 208
- **Stufe**: Standard
- **Verwandte Domänen**: `sites`, `customer_edge`

## Anwendungsfälle

- Verbindung zu Cloud-Providern herstellen (AWS, Azure, GCP)
- Cloud-Anmeldeinformationen und Authentifizierung verwalten
- Cloud-Konnektivität und elastische Bereitstellung konfigurieren
- Cloud-Regionen verknüpfen und verwalten

## Primäre Ressourcen

- **aws_vpc_site**: AWS-VPC-Site-Bereitstellung mit Edge-Node-Provisionierung
- **azure_vnet_site**: Azure-VNet-Site-Bereitstellung mit Edge-Node-Provisionierung
- **gcp_vpc_site**: Google-Cloud-VPC-Site-Bereitstellung mit Edge-Node-Provisionierung
- **cloud_credentials**: Cloud-Provider-Authentifizierungsdaten für die Site-Bereitstellung
