Klassifikationsregeln, Ressourcenrichtlinien und Erkenntnisgewinnung für Datenanalyse-Workflows.

- **Kategorie**: Operationen
- **Komplexität**: einfach
- **Pfade**: 15 | **Schemas**: 73
- **Stufe**: Standard
- **Verwandte Domänen**: `statistics`, `observability`

## Anwendungsfälle

- Sicherheits- und Datenverkehrsdaten analysieren
- Intelligente Erkenntnisse aus Protokollen generieren
- Datenanalyserichtlinien konfigurieren

## Primäre Ressourcen

- **analytics_query**: Analyseabfrage für Datenanalysen
- **data_export**: Datenexportkonfiguration für Berichte
