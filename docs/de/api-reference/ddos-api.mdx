Sperrlisten, Firewallregelgruppen und tunnelbasierte Schutzmaßnahmen. Ratenbegrenzung und Mustererkennung für die Sicherheit des Netzwerkperimeters.

- **Kategorie**: Sicherheit
- **Komplexität**: Erweitert
- **Pfade**: 60 | **Schemata**: 246
- **Stufe**: Erweitert
- **Verwandte Domänen**: `network_security`, `virtual`

## Anwendungsfälle

- DDoS-Schutzrichtlinien konfigurieren
- DDoS-Bedrohungen überwachen und analysieren
- Infrastrukturschutz konfigurieren

## Primäre Ressourcen

- **ddos_protection**: DDoS-Schutzkonfiguration für die Datenverkehrsbereinigung
- **ddos_mitigation_rule**: DDoS-Abwehrregel für die Angriffsreaktion
