Deny-Listen, Firewall-Regelgruppen und tunnelbasierte Schutzmaßnahmen. Ratenbegrenzung und Musteranalyse für die Sicherheit des Netzwerkperimeters.

- **Kategorie**: Sicherheit
- **Komplexität**: Fortgeschritten
- **Pfade**: 60 | **Schemas**: 246
- **Stufe**: Erweitert
- **Verwandte Domänen**: `network_security`, `virtual`

## Anwendungsfälle

- DDoS-Schutzrichtlinien konfigurieren
- DDoS-Bedrohungen überwachen und analysieren
- Infrastrukturschutz konfigurieren

## Primäre Ressourcen

- **ddos_protection**: DDoS-Schutzkonfiguration für Traffic-Scrubbing
- **ddos_mitigation_rule**: DDoS-Abwehrregel für Angriffsreaktion
