Namensauflösung mit Zonentransfers und Gesundheitsprüfungen. Datensatztypen und Delegierungsunterstützung.

- **Kategorie**: Netzwerk
- **Komplexität**: Fortgeschritten
- **Pfade**: 49 | **Schemas**: 285
- **Stufe**: Standard
- **Verwandte Domänen**: `virtual`, `network`

## Anwendungsfälle

- DNS-Lastverteilung konfigurieren
- DNS-Zonen und Domänen verwalten
- DNS-Compliance-Richtlinien konfigurieren
- Ressourcendatensatzgruppen (RRSets) verwalten

## Primäre Ressourcen

- **dns_zone**: Autoritative DNS-Zone mit Datensatzverwaltungsfunktionen
- **dns_domain**: DNS-Domänendelegierung und Konfigurationseinstellungen
- **dns_load_balancer**: DNS-basierte Verkehrsverteilung mit geografischem Routing und Failover
