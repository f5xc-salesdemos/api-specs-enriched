Namensauflösung mit Zonenübertragungen und Integritätsprüfungen. Unterstützung für Eintragstypen und Delegierung.

- **Kategorie**: Netzwerk
- **Komplexität**: Erweitert
- **Pfade**: 49 | **Schemas**: 285
- **Stufe**: Standard
- **Verwandte Domänen**: `virtual`, `network`

## Anwendungsfälle

- DNS-Lastverteilung konfigurieren
- DNS-Zonen und Domänen verwalten
- DNS-Compliance-Richtlinien konfigurieren
- Ressourceneintrags-Sets (RRSets) verwalten

## Primäre Ressourcen

- **dns_zone**: Autoritative DNS-Zone mit Eintrags-Verwaltungsfunktionen
- **dns_domain**: DNS-Domänendelegierung und Konfigurationseinstellungen
- **dns_load_balancer**: DNS-basierte Datenverkehrsverteilung mit geografischem Routing und Failover
