GRE- und IPSec-Tunnel-Bereitstellung von Drittanbietern mit DPD-Timern. Gemeinsame Ressourcenzuweisung über Namespaces hinweg mit Kachel-Platzierungssteuerungen.

- **Kategorie**: Plattform
- **Komplexität**: moderat
- **Pfade**: 36 | **Schemas**: 183
- **Stufe**: Erweitert
- **Verwandte Domänen**: `bigip`, `admin`

## Anwendungsfälle

- Zugriff auf Drittanbieter-Integrationen und Add-ons
- Marketplace-Erweiterungen verwalten
- Terraform und externe Integrationen konfigurieren
- TPM-Richtlinien verwalten

## Primäre Ressourcen

- **marketplace_item**: Marketplace-Element für den Dienstkatalog
- **subscription**: Abonnement für Dienstberechtigungen
