Border-Gateway-Protokoll mit ASN-Verwaltung und Beziehungen zwischen autonomen Systemen. Site-to-Site-VPN zur Verbindung von Rechenzentren über verschlüsselte Kanäle.

- **Kategorie**: Netzwerk
- **Komplexität**: erweitert
- **Pfade**: 81 | **Schemas**: 449
- **Stufe**: Erweitert
- **Verwandte Domänen**: `virtual`, `network_security`, `dns`

## Anwendungsfälle

- BGP-Routing und ASN-Verwaltung konfigurieren
- IPsec-Tunnel und IKE-Phasen verwalten
- Netzwerkverbinder und Routen konfigurieren
- SRv6 und Subnetzbildung verwalten
- Segmentverbindungen und Richtlinien definieren
- IP-Präfixmengen konfigurieren

## Primäre Ressourcen

- **virtual_network**: Virtuelles Netzwerk für Site-Konnektivität und Segmentierung
- **network_connector**: Netzwerkverbinder für Site-to-Site- oder Cloud-Konnektivität
- **site_mesh_group**: Mesh-Konnektivitätskonfiguration zwischen mehreren Standorten
