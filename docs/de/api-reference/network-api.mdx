Border-Gateway-Protokoll mit ASN-Verwaltung und Beziehungen autonomer Systeme. Site-to-Site-VPN zur Verbindung von Rechenzentren über verschlüsselte Kanäle.

- **Kategorie**: Netzwerk
- **Komplexität**: Fortgeschritten
- **Pfade**: 81 | **Schemas**: 449
- **Stufe**: Fortgeschritten
- **Verwandte Domänen**: `virtual`, `network_security`, `dns`

## Anwendungsfälle

- BGP-Routing und ASN-Verwaltung konfigurieren
- IPsec-Tunnel und IKE-Phasen verwalten
- Netzwerk-Konnektoren und Routen konfigurieren
- SRv6 und Subnetzbildung verwalten
- Segmentverbindungen und -richtlinien definieren
- IP-Präfix-Sets konfigurieren

## Primäre Ressourcen

- **virtual_network**: Virtuelles Netzwerk für Standortkonnektivität und Segmentierung
- **network_connector**: Netzwerk-Konnektor für Site-to-Site- oder Cloud-Konnektivität
- **site_mesh_group**: Mesh-Konnektivitätskonfiguration zwischen mehreren Standorten
