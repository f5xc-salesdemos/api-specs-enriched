Instanzerkennung, WAF-Integration und Service-Mesh-Konnektivität. Abonnement-Lebenszyklus und Konfigurationssynchronisierung.

- **Kategorie**: Plattform
- **Komplexität**: einfach
- **Pfade**: 13 | **Schemas**: 59
- **Stufe**: Erweitert
- **Verwandte Domänen**: `marketplace`

## Anwendungsfälle

- NGINX One Plattform-Integrationen verwalten
- NGINX Plus-Instanzen konfigurieren
- NGINX-Konfigurationsverwaltung integrieren

## Primäre Ressourcen

- **nginx_config**: NGINX-Konfiguration für Proxy-Einstellungen
- **nginx_upstream**: NGINX Upstream für Backend-Server-Pools
