HTTP-Verfügbarkeitsprüfungen mit Latenzmessung. Zertifikatsablaufwarnungen und globale Statusdashboards für die Infrastrukturgesundheit.

- **Kategorie**: Betrieb
- **Komplexität**: moderat
- **Pfade**: 56 | **Schemata**: 196
- **Stufe**: Standard
- **Verwandte Domänen**: `statistics`, `support`

## Anwendungsfälle

- Synthetisches Monitoring konfigurieren
- Monitoring- und Testrichtlinien definieren
- Beobachtbarkeits-Dashboards verwalten

## Primäre Ressourcen

- **log_receiver**: Log-Empfänger für die zentrale Protokollerfassung
- **metrics_receiver**: Metrikempfänger für die zentrale Metrikerfassung
- **alert_policy**: Alarmrichtlinie für Monitoring und Benachrichtigung
