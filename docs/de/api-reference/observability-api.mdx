HTTP-Verfügbarkeitsprüfungen mit Latenzmessung. Zertifikatsablauf-Benachrichtigungen und globale Status-Dashboards für die Infrastrukturgesundheit.

- **Kategorie**: Betrieb
- **Komplexität**: moderat
- **Pfade**: 56 | **Schemas**: 196
- **Stufe**: Standard
- **Verwandte Domänen**: `statistics`, `support`

## Anwendungsfälle

- Synthetisches Monitoring konfigurieren
- Überwachungs- und Testrichtlinien definieren
- Observability-Dashboards verwalten

## Primäre Ressourcen

- **log_receiver**: Log-Empfänger für zentralisierte Log-Sammlung
- **metrics_receiver**: Metriken-Empfänger für zentralisierte Metrikensammlung
- **alert_policy**: Alarmrichtlinie für Überwachung und Benachrichtigung
