Zeitbasierte Kontingentdurchsetzung mit konfigurierbaren Zeitfenstern in Stunden, Minuten oder Sekunden. Protokollspezifische Steuerungen zur Verkehrsformung.

- **Kategorie**: Netzwerk
- **Komplexität**: einfach
- **Pfade**: 12 | **Schemas**: 70
- **Stufe**: Erweitert
- **Verwandte Domänen**: `virtual`, `network_security`

## Anwendungsfälle

- Ratenbegrenzungsrichtlinien konfigurieren
- Policer-Konfigurationen verwalten
- Datenverkehrsfluss und Warteschlangen steuern

## Primäre Ressourcen

- **rate_limiter**: Ratenbegrenzer zur Drosselung und zum Schutz des Datenverkehrs
- **rate_limiter_policy**: Ratenbegrenzungsrichtlinie mit detaillierten Drosselungsregeln
- **rate_limit_threshold**: Konfiguration des Ratenbegrenzungsschwellenwerts zur Datenverkehrssteuerung
