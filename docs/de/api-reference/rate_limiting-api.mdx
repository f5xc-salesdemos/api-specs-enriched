Zeitbasierte Kontingentverwaltung mit konfigurierbaren Fenstern in Stunden, Minuten oder Sekunden. Protokollspezifische Steuerungen für die Datenverkehrsformung.

- **Kategorie**: Netzwerk
- **Komplexität**: einfach
- **Pfade**: 12 | **Schemata**: 70
- **Stufe**: Erweitert
- **Verwandte Domänen**: `virtual`, `network_security`

## Anwendungsfälle

- Rate-Limiter-Richtlinien konfigurieren
- Policer-Konfigurationen verwalten
- Datenverkehrsfluss und Warteschlangen steuern

## Primäre Ressourcen

- **rate_limiter**: Rate Limiter für Datenverkehrsdrosselung und -schutz
- **rate_limiter_policy**: Rate-Limiter-Richtlinie mit detaillierten Drosselungsregeln
- **rate_limit_threshold**: Konfiguration des Rate-Limit-Schwellenwerts für die Datenverkehrssteuerung
