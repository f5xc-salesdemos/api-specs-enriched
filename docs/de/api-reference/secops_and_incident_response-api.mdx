Schadensbegrenzung für böswillige Benutzer mit Bedrohungsstufenklassifizierung. Automatisierte Reaktionsmaßnahmen für verdächtige Verhaltensmuster.

- **Kategorie**: Sicherheit
- **Komplexität**: einfach
- **Pfade**: 4 | **Schemata**: 32
- **Stufe**: Erweitert
- **Verwandte Domänen**: `bot_defense`, `virtual`, `network_security`

## Anwendungsfälle

- Automatisierte Richtlinien zur Bedrohungsminderung konfigurieren
- Regeln zur Erkennung böswilliger Benutzer definieren
- Workflows zur Incident-Response verwalten
- Sperren oder Ratenbegrenzungen auf Bedrohungen anwenden

## Primäre Ressourcen

- **mitigation_policy**: Eindämmungsrichtlinie für die Incident-Response
- **malicious_user_rule**: Regel für böswillige Benutzer zur Bedrohungsminderung

## Endpunkte

| Methode | Pfad | Beschreibung |
|--------|------|-------------|
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations` | Schadensbegrenzung für böswillige Benutzer erstellen. |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations/&#123;metadata.name&#125;` | Schadensbegrenzung für böswillige Benutzer ersetzen. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations` | Schadensbegrenzung für böswillige Benutzer auflisten. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | Schadensbegrenzung für böswillige Benutzer abrufen. |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | Schadensbegrenzung für böswillige Benutzer löschen. |
