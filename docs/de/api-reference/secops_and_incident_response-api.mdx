Schadensbegrenzung bei bösartigen Benutzern mit Bedrohungsstufenklassifizierung. Automatisierte Reaktionsmaßnahmen bei verdächtigen Verhaltensmustern.

- **Kategorie**: Sicherheit
- **Komplexität**: einfach
- **Pfade**: 4 | **Schemata**: 32
- **Stufe**: Erweitert
- **Verwandte Domänen**: `bot_defense`, `virtual`, `network_security`

## Anwendungsfälle

- Automatisierte Richtlinien zur Bedrohungsminderung konfigurieren
- Regeln zur Erkennung bösartiger Benutzer definieren
- Workflows zur Incident-Response verwalten
- Sperren oder Ratenbegrenzung auf Bedrohungen anwenden

## Primäre Ressourcen

- **mitigation_policy**: Minderungsrichtlinie für die Incident-Response
- **malicious_user_rule**: Regel für bösartige Benutzer zur Bedrohungsminderung

## Endpunkte

| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations` | Schadensbegrenzung für bösartige Benutzer erstellen. |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations/&#123;metadata.name&#125;` | Schadensbegrenzung für bösartige Benutzer ersetzen. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations` | Schadensbegrenzung für bösartige Benutzer auflisten. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | Schadensbegrenzung für bösartige Benutzer abrufen. |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | Schadensbegrenzung für bösartige Benutzer löschen. |
