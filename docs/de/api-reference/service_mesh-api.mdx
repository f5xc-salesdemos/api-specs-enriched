Anwendungstyp-Definitionen mit Erkennung und erlernter Schema-Analyse. Verkehrsmuster-Inferenz für intelligente Anfrageverarbeitung.

- **Kategorie**: Infrastruktur
- **Komplexität**: Fortgeschritten
- **Pfade**: 38 | **Schemas**: 247
- **Stufe**: Fortgeschritten
- **Verwandte Domänen**: `managed_kubernetes`, `container_services`, `virtual`

## Anwendungsfälle

- Service-Mesh-Konnektivität konfigurieren
- Endpoint-Erkennung und Routing verwalten
- NFV-Dienste konfigurieren
- Anwendungseinstellungen und -typen definieren

## Primäre Ressourcen

- **endpoint**: Service-Mesh-Endpunkt für Traffic-Routing
- **origin_pool**: Backend-Servergruppe für Load-Balancer-Verkehrsverteilung
- **service_discovery**: Service-Discovery-Konfiguration für dynamische Endpunkte
