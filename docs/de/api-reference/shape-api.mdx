Bedrohungserkennung mit Verhaltensanalyse und Gerätefingerabdruck. Mobile SDK-Integration für die Absicherung von Anwendungen.

- **Kategorie**: Sicherheit
- **Komplexität**: erweitert
- **Pfade**: 271 | **Schemas**: 813
- **Stufe**: Erweitert
- **Verwandte Domänen**: `bot_defense`, `virtual`

## Anwendungsfälle

- Shape-Sicherheitsrichtlinien konfigurieren
- Bot- und Bedrohungsabwehr verwalten
- SafeAP-Richtlinien konfigurieren
- Bedrohungserkennung aktivieren

## Primäre Ressourcen

- **shape_app_firewall**: Shape-Anwendungs-Firewall für erweiterten Schutz
- **shape_recognizer**: Shape-Recognizer zur Analyse von Datenverkehrsmustern
