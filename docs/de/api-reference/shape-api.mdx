Bedrohungserkennung mit Verhaltensanalyse und Geräte-Fingerprinting. Mobile SDK-Integration für Anwendungsschutz.

- **Kategorie**: Sicherheit
- **Komplexität**: Fortgeschritten
- **Pfade**: 271 | **Schemas**: 813
- **Stufe**: Erweitert
- **Verwandte Domänen**: `bot_defense`, `virtual`

## Anwendungsfälle

- Shape Security-Richtlinien konfigurieren
- Bot- und Bedrohungsabwehr verwalten
- SafeAP-Richtlinien konfigurieren
- Bedrohungserkennung aktivieren

## Primäre Ressourcen

- **shape_app_firewall**: Shape Application Firewall für erweiterten Schutz
- **shape_recognizer**: Shape Recognizer für die Analyse von Datenverkehrsmustern
