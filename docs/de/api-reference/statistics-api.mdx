Warnmeldungsrichtlinien mit Empfänger-Integrationen. Protokollaggregation, Topologieansichten und Standortüberwachung.

- **Kategorie**: Betrieb
- **Komplexität**: fortgeschritten
- **Pfade**: 77 | **Schemata**: 575
- **Stufe**: Standard
- **Verwandte Domänen**: `observability`, `support`

## Anwendungsfälle

- Auf Flussstatistiken und -analysen zugreifen
- Warnmeldungen und Warnmeldungsrichtlinien verwalten
- Protokolle und Protokollempfänger anzeigen
- Berichte und Grafiken erstellen
- Topologie und Diensterkennung verfolgen
- Status an Standorten überwachen

## Primäre Ressourcen

- **dashboard**: Dashboard zur Metrikvisualisierung
- **saved_query**: Gespeicherte Abfrage für Metriken und Protokollanalysen
