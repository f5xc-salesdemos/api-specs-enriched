Alarmierungsrichtlinien mit Empfänger-Integrationen. Protokollaggregation, Topologieansichten und Standort-Zustandsüberwachung.

- **Kategorie**: Betrieb
- **Komplexität**: Fortgeschritten
- **Pfade**: 77 | **Schemas**: 575
- **Stufe**: Standard
- **Verwandte Domänen**: `observability`, `support`

## Anwendungsfälle

- Zugriff auf Datenflussstatistiken und Analysen
- Verwaltung von Alarmen und Alarmierungsrichtlinien
- Anzeige von Protokollen und Protokollempfängern
- Erstellung von Berichten und Diagrammen
- Nachverfolgung von Topologie und Service-Erkennung
- Statusüberwachung an Standorten

## Primäre Ressourcen

- **dashboard**: Dashboard zur Metrik-Visualisierung
- **saved_query**: Gespeicherte Abfrage für Metrik- und Protokollanalyse
