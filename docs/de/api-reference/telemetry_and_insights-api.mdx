Erfassungsendpunkte, Metrikspeicherung und Insight-Generierung für Observability-Workflows.

- **Kategorie**: Betrieb
- **Komplexität**: mittel
- **Pfade**: 27 | **Schemas**: 181
- **Stufe**: Standard
- **Verwandte Domänen**: `observability`, `statistics`

## Anwendungsfälle

- Telemetriedaten erfassen und analysieren
- Umsetzbare Erkenntnisse aus Metriken generieren
- Richtlinien zur Telemetrieerfassung konfigurieren

## Primäre Ressourcen

- **telemetry_receiver**: Telemetrieempfänger zur Datenerfassung
- **insight_query**: Insight-Abfrage für Analysen
