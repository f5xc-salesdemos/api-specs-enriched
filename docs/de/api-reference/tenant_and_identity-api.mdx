Kontoansichtskonfigurationen und Admin-Benachrichtigungskanäle. Einmalpasswort-Zurücksetzungen, Bereitstellungsabläufe und aktive Verbindungsüberwachung.

- **Kategorie**: Plattform
- **Komplexität**: Fortgeschritten
- **Pfade**: 183 | **Schemas**: 511
- **Stufe**: Standard
- **Verwandte Domänen**: `users`, `authentication`, `system`

## Anwendungsfälle

- Benutzerprofile und Benachrichtigungseinstellungen verwalten
- Sitzungssteuerungen und OTP-Einstellungen konfigurieren
- Identitätsverwaltungsoperationen durchführen
- Erstmalige Benutzerzugriffsanfragen verarbeiten

## Primäre Ressourcen

- **user_profile**: Benutzerprofil mit persönlichen Einstellungen
- **session**: Benutzersitzung für den Authentifizierungsstatus
- **otp_policy**: OTP-Richtlinie für Multi-Faktor-Authentifizierung
