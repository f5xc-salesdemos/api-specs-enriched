Erkennungsrichtlinien, Angriffsnachverfolgung und automatisierte Generierung von Abwehrregeln für Sicherheitsoperationen.

- **Kategorie**: Sicherheit
- **Komplexität**: moderat
- **Pfade**: 1 | **Schemas**: 161
- **Stufe**: Standard
- **Verwandte Domänen**: `bot_defense`, `ddos`

## Anwendungsfälle

- Bedrohungskampagnen verfolgen und analysieren
- Aktive Bedrohungen und Angriffsmuster überwachen
- Integration von Bedrohungsinformationen konfigurieren

## Primäre Ressourcen

- **threat_campaign_policy**: Richtlinie zur Erkennung und Abwehr von Bedrohungskampagnen

## Endpunkte

| Methode | Pfad | Beschreibung |
|---------|------|--------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | Bedrohungskampagne nach ID abrufen. |
