Erkennungsrichtlinien, Angriffs-Tracking und automatisierte Mitigationsregelgenerierung für Sicherheitsoperationen.

- **Kategorie**: Sicherheit
- **Komplexität**: moderat
- **Pfade**: 1 | **Schemata**: 161
- **Stufe**: Standard
- **Zugehörige Domänen**: `bot_defense`, `ddos`

## Anwendungsfälle

- Bedrohungskampagnen verfolgen und analysieren
- Aktive Bedrohungen und Angriffsmuster überwachen
- Integration von Bedrohungsinformationen konfigurieren

## Primäre Ressourcen

- **threat_campaign_policy**: Erkennungs- und Mitigationsrichtlinie für Bedrohungskampagnen

## Endpunkte

| Methode | Pfad | Beschreibung |
|--------|------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | Bedrohungskampagne nach ID abrufen. |
