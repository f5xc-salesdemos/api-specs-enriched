Erkennungsrichtlinien, Angriffsverfolgung und automatisierte Abwehrregel-Generierung für Sicherheitsoperationen.

- **Kategorie**: Sicherheit
- **Komplexität**: mittel
- **Pfade**: 1 | **Schemas**: 161
- **Stufe**: Standard
- **Verwandte Domänen**: `bot_defense`, `ddos`

## Anwendungsfälle

- Bedrohungskampagnen verfolgen und analysieren
- Aktive Bedrohungen und Angriffsmuster überwachen
- Integration von Bedrohungsintelligenz konfigurieren

## Primäre Ressourcen

- **threat_campaign_policy**: Erkennungs- und Abwehrrichtlinie für Bedrohungskampagnen

## Endpunkte

| Methode | Pfad | Beschreibung |
|--------|------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | Bedrohungskampagne nach ID abrufen. |
