Anmeldedaten für die Standortregistrierung mit automatischem Ablauf. Taxonomieschlüssel definieren die zulässige Kategorisierung, während automatisch abgeleitete Tags dynamisch angewendet werden.

- **Kategorie**: Plattform
- **Komplexität**: einfach
- **Pfade**: 13 | **Schemas**: 50
- **Stufe**: Standard
- **Verwandte Domänen**: `system`, `admin`

## Anwendungsfälle

- Benutzerkonten und Tokens verwalten
- Benutzeridentifikation konfigurieren
- Benutzereinstellungen und Präferenzen verwalten
- Implizite und bekannte Labels konfigurieren

## Primäre Ressourcen

- **user**: Benutzerkonto für den Plattformzugriff
- **user_role**: Benutzerrolle für die Berechtigungsverwaltung
- **namespace_role**: Namespace-bezogene Rolle für feingranulare Berechtigungen
