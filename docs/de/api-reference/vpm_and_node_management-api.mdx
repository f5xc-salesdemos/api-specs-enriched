Lebenszyklussteuerung, Flottenkonfiguration und Bereitstellungsrichtlinien für die verteilte Knotenverwaltung.

- **Kategorie**: Plattform
- **Komplexität**: einfach
- **Pfade**: 1 | **Schemas**: 2
- **Stufe**: Standard
- **Verwandte Domänen**: `sites`, `system`

## Anwendungsfälle

- Konfiguration des Virtual Private Mesh (VPM) verwalten
- Knotenlebenszyklus und -verwaltung konfigurieren
- VPM- und Knotenstatus überwachen

## Primäre Ressourcen

- **node_config**: Knotenkonfiguration für Edge-Geräteeinstellungen
- **vpm_config**: VPM-Konfiguration für die virtuelle Plattformverwaltung

## Endpunkte

| Methode | Pfad | Beschreibung |
|---------|------|--------------|
| GET | `/api/data/namespaces/system/upgrade_status` | Upgrade-Status. |
