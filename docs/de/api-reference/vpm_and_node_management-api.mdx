Lebenszyklussteuerung, Fleet-Konfiguration und Bereitstellungsrichtlinien für das verteilte Node-Management.

- **Kategorie**: Plattform
- **Komplexität**: einfach
- **Pfade**: 1 | **Schemata**: 2
- **Stufe**: Standard
- **Verwandte Domänen**: `sites`, `system`

## Anwendungsfälle

- Virtual Private Mesh (VPM)-Konfiguration verwalten
- Node-Lebenszyklus und -Verwaltung konfigurieren
- VPM- und Node-Status überwachen

## Primäre Ressourcen

- **node_config**: Node-Konfiguration für Edge-Gerät-Einstellungen
- **vpm_config**: VPM-Konfiguration für das virtuelle Plattform-Management

## Endpunkte

| Methode | Pfad | Beschreibung |
|---------|------|--------------|
| GET | `/api/data/namespaces/system/upgrade_status` | Upgrade-Status. |
