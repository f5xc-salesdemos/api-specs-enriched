Lebenszyklussteuerung, Flottenkonfiguration und Bereitstellungsrichtlinien für verteiltes Node-Management.

- **Kategorie**: Plattform
- **Komplexität**: einfach
- **Pfade**: 1 | **Schemas**: 2
- **Stufe**: Standard
- **Verwandte Domänen**: `sites`, `system`

## Anwendungsfälle

- Virtual Private Mesh (VPM)-Konfiguration verwalten
- Node-Lebenszyklus und -Management konfigurieren
- VPM- und Node-Status überwachen

## Primäre Ressourcen

- **node_config**: Node-Konfiguration für Edge-Geräteeinstellungen
- **vpm_config**: VPM-Konfiguration für virtuelles Plattformmanagement

## Endpunkte

| Methode | Pfad | Beschreibung |
|---------|------|--------------|
| GET | `/api/data/namespaces/system/upgrade_status` | Upgrade-Status. |
