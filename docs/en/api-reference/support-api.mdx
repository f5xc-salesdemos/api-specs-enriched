Issue lifecycle with comments, severity changes, and resolution tracking. Packet capture for connection analysis.

- **Category**: Operations
- **Complexity**: moderate
- **Paths**: 53 | **Schemas**: 167
- **Tier**: Standard
- **Related domains**: `statistics`, `observability`

## Use Cases

- Submit and manage support tickets
- Track customer support requests
- Access operational support documentation

## Primary Resources

- **support_case**: Support case for issue tracking and resolution
- **alert**: System alert for operational notifications
- **audit_log**: Audit log for compliance and security tracking
