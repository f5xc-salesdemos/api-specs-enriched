Elementos visuales con ámbito de espacio de nombres y control de versiones. Implementación de widgets personalizados y gestión de catálogo para superficies de portal.

- **Categoría**: Plataforma
- **Complejidad**: simple
- **Rutas**: 2 | **Esquemas**: 14
- **Nivel**: Estándar
- **Dominios relacionados**: `admin`, `system`

## Casos de uso

- Gestionar componentes de UI estáticos para la consola de administración
- Implementar y recuperar recursos de UI dentro de espacios de nombres
- Configurar elementos de la interfaz de la consola
- Gestionar metadatos de componentes de UI personalizados

## Recursos principales

- **ui_component**: Componente de UI para personalización de la consola
- **static_asset**: Recurso estático para recursos de UI

## Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | Listar componente estático de UI. |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | OBTENER componente estático de UI. |
