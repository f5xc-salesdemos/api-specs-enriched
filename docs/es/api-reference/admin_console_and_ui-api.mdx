Elementos visuales con alcance de namespace y versionado. Despliegue de widgets personalizados y gestión de catálogo para superficies del portal.

- **Categoría**: Plataforma
- **Complejidad**: simple
- **Rutas**: 2 | **Esquemas**: 14
- **Nivel**: Estándar
- **Dominios relacionados**: `admin`, `system`

## Casos de Uso

- Gestionar componentes de UI estáticos para la consola de administración
- Desplegar y recuperar recursos de UI dentro de namespaces
- Configurar elementos de la interfaz de consola
- Gestionar metadatos de componentes de UI personalizados

## Recursos Principales

- **ui_component**: Componente de UI para personalización de consola
- **static_asset**: Recurso estático para recursos de UI

## Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | Listar componentes estáticos de UI. |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | Obtener componente estático de UI. |
