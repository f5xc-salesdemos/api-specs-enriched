Procesamiento de lenguaje natural con señales de calidad y monitoreo de anomalías. Autenticación por token para suscripciones a flujos de datos.

- **Categoría**: IA
- **Complejidad**: simple
- **Rutas**: 11 | **Esquemas**: 66
- **Nivel**: Avanzado


## Casos de Uso

- Acceder a funciones impulsadas por IA
- Configurar políticas del asistente de IA
- Habilitar detección de anomalías de flujo
- Gestionar la recopilación de datos de IA

## Recursos Principales

- **ai_policy**: Política de IA para el control de tráfico de IA generativa
- **ai_gateway**: Gateway de IA para la gestión de API de LLM

## Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | Evaluar consulta del Asistente de IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | Evaluar retroalimentación de consulta del Asistente de IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | Consulta del Asistente de IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | Retroalimentación de consulta del Asistente de IA. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | Habilitar función. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | Refrescar Token. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | Suscribirse al pipeline BFDP. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | Suscribirse al pipeline BFDP. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | Desuscribirse del pipeline BFDP. |
| POST | `/api/gia/gia/allocateip` | Asignar IP a través del Asignador Global de IP. |
| DELETE | `/api/gia/gia/deallocateip` | Desasignar IP a través del Asignador Global de IP. |
