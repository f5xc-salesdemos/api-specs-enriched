Procesamiento de lenguaje natural con señales de calidad y supervisión de anomalías. Autenticación mediante tokens para suscripciones a flujos de datos.

- **Categoría**: IA
- **Complejidad**: simple
- **Rutas**: 11 | **Esquemas**: 66
- **Nivel**: Avanzado


## Casos de uso

- Acceder a funcionalidades impulsadas por IA
- Configurar políticas del asistente de IA
- Habilitar la detección de anomalías en flujos
- Gestionar la recopilación de datos de IA

## Recursos principales

- **ai_policy**: Política de IA para el control de tráfico de IA generativa
- **ai_gateway**: Puerta de enlace de IA para la gestión de API LLM

## Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | Evaluar consulta del asistente de IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | Evaluar comentarios de la consulta del asistente de IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | Consulta del asistente de IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | Comentarios de la consulta del asistente de IA. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | Habilitar funcionalidad. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | Actualizar token. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | Suscribirse al pipeline BFDP. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | Suscribirse al pipeline BFDP. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | Cancelar suscripción al pipeline BFDP. |
| POST | `/api/gia/gia/allocateip` | Asignar IP mediante el asignador de IP global. |
| DELETE | `/api/gia/gia/deallocateip` | Desasignar IP mediante el asignador de IP global. |
