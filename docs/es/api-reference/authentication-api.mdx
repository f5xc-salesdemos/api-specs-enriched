Gestión de identidades con integración de proveedores, políticas de acceso y control del ciclo de vida de credenciales.

- **Categoría**: Plataforma
- **Complejidad**: simple
- **Rutas**: 14 | **Esquemas**: 38
- **Nivel**: Standard
- **Dominios relacionados**: `system`, `users`

## Casos de uso

- Configurar mecanismos de autenticación
- Gestionar proveedores OIDC y OAuth
- Configurar el aprovisionamiento de usuarios SCIM
- Gestionar credenciales de API y acceso
- Configurar políticas de registro de cuentas

## Recursos principales

- **authentication_policy**: Política de autenticación para acceso de usuarios y API
- **token**: Token de API para acceso programático
- **api_credential**: Credencial de API para autenticación de servicios

## Endpoints
