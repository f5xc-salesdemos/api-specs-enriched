Orquestación de dispositivos heredados con scripts iRule y sincronización de grupos de datos. Vinculación de servidores virtuales y recopilación de métricas.

- **Categoría**: Plataforma
- **Complejidad**: avanzada
- **Rutas**: 29 | **Esquemas**: 238
- **Nivel**: Avanzado
- **Dominios relacionados**: `marketplace`

## Casos de uso

- Gestionar dispositivos BigIP F5
- Configurar scripts iRule
- Gestionar grupos de datos
- Integrar BigIP CNE

## Recursos principales

- **bigip_pool**: Pool BIG-IP para integración de balanceo de carga
- **bigip_device**: Registro y gestión de dispositivos BIG-IP
