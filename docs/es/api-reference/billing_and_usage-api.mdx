Transiciones de planes, facturación y consumo de recursos. Límites de cuota a nivel de namespace y seguimiento de uso.

- **Categoría**: Plataforma
- **Complejidad**: moderada
- **Rutas**: 25 | **Esquemas**: 101
- **Nivel**: Estándar
- **Dominios relacionados**: `system`, `users`

## Casos de Uso

- Gestionar planes de suscripción y transiciones de facturación
- Configurar métodos de pago y facturas
- Rastrear el uso de cuotas de recursos entre namespaces
- Monitorear límites de uso y capacidad

## Recursos Principales

- **subscription**: Suscripción para derechos de servicio
- **quota**: Cuota de recursos para límites de uso
- **usage_report**: Informe de uso para seguimiento de consumo
