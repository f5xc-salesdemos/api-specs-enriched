Clasificación de amenazas con análisis de comportamiento y coincidencia de firmas. Bloqueo automatizado de patrones de tráfico malicioso.

- **Categoría**: Seguridad
- **Complejidad**: moderada
- **Rutas**: 19 | **Esquemas**: 74
- **Nivel**: Avanzado
- **Dominios relacionados**: `bot_defense`, `shape`, `virtual`

## Casos de uso

- Configurar instancias de defensa contra bots por espacio de nombres
- Gestionar categorías de amenazas TPM para clasificación
- Aprovisionar claves de API para sistemas de defensa automatizados
- Integrar servicios de inteligencia de amenazas

## Recursos principales

- **bot_defense_instance**: Instancia de defensa contra bots para despliegue
- **threat_category**: Categoría de amenaza para clasificación
