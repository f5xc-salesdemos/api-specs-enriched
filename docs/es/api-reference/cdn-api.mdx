Distribución global con reglas de caché y operaciones de purga. Monitoreo de rendimiento y análisis.

- **Categoría**: Redes
- **Complejidad**: avanzada
- **Rutas**: 29 | **Esquemas**: 592
- **Nivel**: Avanzado
- **Dominios relacionados**: `virtual`

## Casos de Uso

- Configurar balanceo de carga CDN
- Gestionar servicios de red de distribución de contenido
- Configurar políticas de caché
- Gestionar la entrega y distribución de datos

## Recursos Principales

- **cdn_loadbalancer**: Balanceador de carga de red de distribución de contenido para caché en el borde
- **cdn_origin_pool**: Grupo de servidores de origen para el aprovisionamiento de contenido CDN
