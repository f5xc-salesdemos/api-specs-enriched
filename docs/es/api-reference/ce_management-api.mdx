Aprovisionamiento basado en tokens con descargas de imágenes y validación previa a la actualización. La agrupación de flotas permite operaciones masivas en ubicaciones distribuidas.

- **Categoría**: Infraestructura
- **Complejidad**: moderada
- **Rutas**: 28 | **Esquemas**: 231
- **Nivel**: Estándar
- **Dominios relacionados**: `customer_edge`, `sites`

## Casos de Uso

- Gestionar el ciclo de vida de los sitios Customer Edge
- Configurar interfaces de red y ajustes de flotas
- Manejar el registro de sitios y los flujos de trabajo con tokens
- Ejecutar actualizaciones de sitios con verificaciones previas a la actualización

## Recursos Principales

- **site_config**: Configuración de sitio para ajustes del nodo edge
- **fleet_config**: Configuración de flota para gestión multi-sitio
- **registration_token**: Token de registro para la incorporación de sitios
