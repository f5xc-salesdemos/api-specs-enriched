Aprovisionamiento basado en tokens con descargas de imágenes y validación previa a la actualización. La agrupación en flotas permite operaciones masivas en ubicaciones distribuidas.

- **Categoría**: Infraestructura
- **Complejidad**: moderada
- **Rutas**: 28 | **Esquemas**: 231
- **Nivel**: Estándar
- **Dominios relacionados**: `customer_edge`, `sites`

## Casos de uso

- Gestionar el ciclo de vida del sitio Customer Edge
- Configurar interfaces de red y ajustes de flotas
- Gestionar el registro de sitios y los flujos de trabajo de tokens
- Ejecutar actualizaciones de sitios con verificaciones previas a la actualización

## Recursos principales

- **site_config**: Configuración del sitio para los ajustes del nodo perimetral
- **fleet_config**: Configuración de flota para la gestión multisitio
- **registration_token**: Token de registro para la incorporación de sitios
