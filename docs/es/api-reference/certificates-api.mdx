Cadenas de certificados y paquetes de CA de confianza. Gestión de listas de revocación y configuración de manifiestos para operaciones PKI.

- **Categoría**: Seguridad
- **Complejidad**: moderada
- **Rutas**: 16 | **Esquemas**: 74
- **Nivel**: Estándar
- **Dominios relacionados**: `blindfold`, `system`

## Casos de uso

- Gestionar certificados SSL/TLS
- Configurar CAs de confianza
- Gestionar listas de revocación de certificados (CRL)
- Configurar manifiestos de certificados

## Recursos principales

- **certificate**: Certificado TLS para conexiones HTTPS seguras
- **ca_certificate**: Certificado de autoridad de certificación para la cadena de confianza
- **certificate_chain**: Cadena de certificados para verificación completa de confianza
