Conexiones a múltiples proveedores de nube con emparejamiento de gateway y configuración de rutas de red. Integración con almacén de credenciales y enumeración de subredes.

- **Categoría**: Infraestructura
- **Complejidad**: moderada
- **Rutas**: 31 | **Esquemas**: 208
- **Nivel**: Estándar
- **Dominios relacionados**: `sites`, `customer_edge`

## Casos de Uso

- Conectar a proveedores de nube (AWS, Azure, GCP)
- Gestionar credenciales y autenticación en la nube
- Configurar conectividad en la nube y aprovisionamiento elástico
- Vincular y gestionar regiones en la nube

## Recursos Principales

- **aws_vpc_site**: Despliegue de sitio AWS VPC con aprovisionamiento de nodo edge
- **azure_vnet_site**: Despliegue de sitio Azure VNet con aprovisionamiento de nodo edge
- **gcp_vpc_site**: Despliegue de sitio Google Cloud VPC con aprovisionamiento de nodo edge
- **cloud_credentials**: Credenciales de autenticación del proveedor de nube para despliegue de sitios
