Reglas de clasificación, políticas de recursos y generación de análisis para flujos de trabajo de análisis de datos.

- **Categoría**: Operaciones
- **Complejidad**: simple
- **Rutas**: 15 | **Esquemas**: 73
- **Nivel**: Estándar
- **Dominios relacionados**: `statistics`, `observability`

## Casos de uso

- Analizar datos de seguridad y tráfico
- Generar análisis inteligentes a partir de registros
- Configurar políticas de análisis de datos

## Recursos principales

- **analytics_query**: Consulta de análisis para análisis de datos
- **data_export**: Configuración de exportación de datos para informes

## Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| POST | `/api/data-intelligence/namespaces/&#123;metadata.namespace&#125;/receivers` | Crear entrega de datos. |
| PUT | `/api/data-intelligence/namespaces/&#123;metadata.namespace&#125;/receivers/&#123;metadata.name&#125;` | Reemplazar entrega de datos. |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers` | Listar entrega de datos. |
