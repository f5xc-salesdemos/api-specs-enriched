Reglas de clasificación, políticas de recursos y generación de información para flujos de trabajo de análisis de datos.

- **Categoría**: Operaciones
- **Complejidad**: simple
- **Rutas**: 15 | **Esquemas**: 73
- **Nivel**: Estándar
- **Dominios relacionados**: `statistics`, `observability`

## Casos de Uso

- Analizar datos de seguridad y tráfico
- Generar información inteligente a partir de registros
- Configurar políticas de análisis de datos

## Recursos Principales

- **analytics_query**: Consulta analítica para análisis de datos
- **data_export**: Configuración de exportación de datos para informes
