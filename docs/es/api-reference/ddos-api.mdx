Listas de denegación, grupos de reglas de firewall y protecciones basadas en túneles. Limitación de velocidad y análisis de patrones para la seguridad del perímetro de red.

- **Categoría**: Seguridad
- **Complejidad**: avanzada
- **Rutas**: 60 | **Esquemas**: 246
- **Nivel**: Avanzado
- **Dominios relacionados**: `network_security`, `virtual`

## Casos de uso

- Configurar políticas de protección DDoS
- Monitorear y analizar amenazas DDoS
- Configurar protección de infraestructura

## Recursos principales

- **ddos_protection**: Configuración de protección DDoS para depuración de tráfico
- **ddos_mitigation_rule**: Regla de mitigación DDoS para respuesta a ataques
