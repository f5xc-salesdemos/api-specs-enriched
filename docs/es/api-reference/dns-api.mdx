Resolución de nombres con transferencias de zona y verificaciones de estado. Tipos de registros y soporte de delegación.

- **Categoría**: Redes
- **Complejidad**: avanzada
- **Rutas**: 49 | **Esquemas**: 285
- **Nivel**: Estándar
- **Dominios relacionados**: `virtual`, `network`

## Casos de Uso

- Configurar balanceo de carga DNS
- Gestionar zonas y dominios DNS
- Configurar políticas de cumplimiento DNS
- Gestionar conjuntos de registros de recursos (RRSets)

## Recursos Principales

- **dns_zone**: Zona DNS autoritativa con capacidades de gestión de registros
- **dns_domain**: Delegación de dominio DNS y configuración de ajustes
- **dns_load_balancer**: Distribución de tráfico basada en DNS con enrutamiento geográfico y conmutación por error
