Vinculaciones de roles de Kubernetes y políticas de admisión. Integración de registros para cargas de trabajo en EKS, AKS y GKE.

- **Categoría**: Infraestructura
- **Complejidad**: moderada
- **Rutas**: 20 | **Esquemas**: 95
- **Nivel**: Avanzado
- **Dominios relacionados**: `container_services`, `sites`, `service_mesh`

## Casos de Uso

- Gestionar RBAC y seguridad de clústeres XCKS (Kubernetes Gestionado)
- Configurar políticas de seguridad de pods y controladores de admisión
- Gestionar registros de contenedores para despliegues empresariales
- Integrar con clústeres Kubernetes externos (EKS, AKS, GKE)

## Recursos Principales

- **mk8s_cluster**: Configuración de clúster de Kubernetes Gestionado
- **k8s_cluster_role**: Configuración de roles RBAC de clúster de Kubernetes
- **container_registry**: Registro de imágenes de contenedor para imágenes de cargas de trabajo
