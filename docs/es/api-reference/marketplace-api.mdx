Aprovisionamiento de túneles GRE e IPSec de terceros con temporizadores DPD. Asignación de recursos compartidos entre namespaces con controles de colocación de mosaicos.

- **Categoría**: Plataforma
- **Complejidad**: moderada
- **Rutas**: 36 | **Esquemas**: 183
- **Nivel**: Avanzado
- **Dominios relacionados**: `bigip`, `admin`

## Casos de uso

- Acceder a integraciones y complementos de terceros
- Gestionar extensiones del marketplace
- Configurar Terraform e integraciones externas
- Gestionar políticas de TPM

## Recursos principales

- **marketplace_item**: Elemento del marketplace para el catálogo de servicios
- **subscription**: Suscripción para derechos de servicio
