Descubrimiento de instancias, integración WAF y conectividad de malla de servicios. Ciclo de vida de suscripción y sincronización de configuración.

- **Categoría**: Plataforma
- **Complejidad**: simple
- **Rutas**: 13 | **Esquemas**: 59
- **Nivel**: Avanzado
- **Dominios relacionados**: `marketplace`

## Casos de uso

- Gestionar integraciones de la Plataforma NGINX One
- Configurar instancias de NGINX Plus
- Integrar la gestión de configuración de NGINX

## Recursos principales

- **nginx_config**: Configuración de NGINX para ajustes de proxy
- **nginx_upstream**: NGINX upstream para grupos de servidores backend

## Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_csgs` | Listar la configuración del objeto CSG de NGINX One. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_csgs/&#123;name&#125;` | Solicitud GET. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_instances` | Listar la configuración del objeto de instancia de NGINX One. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_instances/&#123;name&#125;` | Solicitud GET. |
| POST | `/api/config/namespaces/&#123;namespace&#125;/nginx_dataplane_servers` | Obtener servidores de plano de datos de NGINX One. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_servers` | Listar las API de servidor de NGINX One. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_servers/&#123;name&#125;` | Solicitud GET. |
| POST | `/api/nginx/one/namespaces/system/n1/subscribe` | Suscribirse a NGINX One. |
| POST | `/api/nginx/one/namespaces/system/n1/unsubscribe` | Cancelar la suscripción a NGINX One. |
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/nginx_service_discoverys` | Crear detección de servicio NGINX. |

| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_service_discoverys` | Listar la detección de servicio NGINX. |
