Distribución de bibliotecas versionadas para integradores de aplicaciones móviles. Las URLs prefirmadas permiten descargas seguras con compilaciones específicas para iOS y Android.

- **Categoría**: Plataforma
- **Complejidad**: simple
- **Rutas**: 9 | **Esquemas**: 17
- **Nivel**: Avanzado
- **Dominios relacionados**: `marketplace`

## Casos de Uso

- Gestionar servicios de almacenamiento de objetos
- Configurar objetos almacenados y buckets
- Gestionar políticas de almacenamiento

## Recursos Principales

- **object_store**: Almacén de objetos para la gestión de almacenamiento de blobs
- **bucket**: Bucket de almacenamiento para almacenamiento de objetos
