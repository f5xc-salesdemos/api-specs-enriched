Distribución de bibliotecas versionadas para integradores de aplicaciones móviles. Las URL prefirmadas permiten descargas seguras con compilaciones específicas por sistema operativo para iOS y Android.

- **Categoría**: Plataforma
- **Complejidad**: simple
- **Rutas**: 9 | **Esquemas**: 17
- **Nivel**: Avanzado
- **Dominios relacionados**: `marketplace`

## Casos de uso

- Gestionar servicios de almacenamiento de objetos
- Configurar objetos almacenados y buckets
- Gestionar políticas de almacenamiento

## Recursos principales

- **object_store**: Almacén de objetos para la gestión de almacenamiento de blobs
- **bucket**: Bucket de almacenamiento para el almacenamiento de objetos
