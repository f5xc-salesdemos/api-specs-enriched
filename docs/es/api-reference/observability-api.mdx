Sondas de disponibilidad HTTP con medición de latencia. Alertas de expiración de certificados y paneles de estado globales para la salud de la infraestructura.

- **Categoría**: Operaciones
- **Complejidad**: moderada
- **Rutas**: 56 | **Esquemas**: 196
- **Nivel**: Estándar
- **Dominios relacionados**: `statistics`, `support`

## Casos de Uso

- Configurar monitoreo sintético
- Definir políticas de monitoreo y pruebas
- Gestionar paneles de observabilidad

## Recursos Principales

- **log_receiver**: Receptor de logs para recopilación centralizada de logs
- **metrics_receiver**: Receptor de métricas para recopilación centralizada de métricas
- **alert_policy**: Política de alertas para monitoreo y notificación
