Sondeos de disponibilidad HTTP con medición de latencia. Alertas de vencimiento de certificados y paneles de estado global para la salud de la infraestructura.

- **Categoría**: Operaciones
- **Complejidad**: moderada
- **Rutas**: 56 | **Esquemas**: 196
- **Nivel**: Estándar
- **Dominios relacionados**: `statistics`, `support`

## Casos de uso

- Configurar monitoreo sintético
- Definir políticas de monitoreo y pruebas
- Gestionar paneles de Observabilidad

## Recursos principales

- **log_receiver**: Receptor de registros para recolección centralizada de logs
- **metrics_receiver**: Receptor de métricas para recolección centralizada de métricas
- **alert_policy**: Política de alertas para monitoreo y notificación
