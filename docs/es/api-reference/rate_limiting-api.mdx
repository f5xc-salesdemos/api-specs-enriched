Aplicación de cuotas basadas en tiempo con ventanas configurables en horas, minutos o segundos. Controles específicos por protocolo para la conformación del tráfico.

- **Categoría**: Redes
- **Complejidad**: simple
- **Rutas**: 12 | **Esquemas**: 70
- **Nivel**: Avanzado
- **Dominios relacionados**: `virtual`, `network_security`

## Casos de uso

- Configurar políticas de limitación de velocidad
- Gestionar configuraciones de control de tráfico
- Controlar el flujo de tráfico y la gestión de colas

## Recursos principales

- **rate_limiter**: Limitador de velocidad para la limitación y protección del tráfico
- **rate_limiter_policy**: Política de limitación de velocidad con reglas de limitación detalladas
- **rate_limit_threshold**: Configuración del umbral de limitación de velocidad para el control del tráfico
