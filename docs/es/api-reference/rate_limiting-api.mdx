Aplicación de cuotas basadas en tiempo con ventanas configurables en horas, minutos o segundos. Controles específicos de protocolo para la conformación de tráfico.

- **Categoría**: Redes
- **Complejidad**: simple
- **Rutas**: 12 | **Esquemas**: 70
- **Nivel**: Avanzado
- **Dominios relacionados**: `virtual`, `network_security`

## Casos de Uso

- Configurar políticas de limitación de tasa
- Gestionar configuraciones de controladores de tráfico
- Controlar el flujo de tráfico y las colas

## Recursos Principales

- **rate_limiter**: Limitador de tasa para la regulación y protección del tráfico
- **rate_limiter_policy**: Política de limitación de tasa con reglas detalladas de regulación
- **rate_limit_threshold**: Configuración de umbral de limitación de tasa para el control de tráfico
