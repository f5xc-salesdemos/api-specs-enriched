Mitigación de usuarios maliciosos con clasificación de nivel de amenaza. Acciones de respuesta automatizada para patrones de comportamiento sospechoso.

- **Categoría**: Seguridad
- **Complejidad**: simple
- **Rutas**: 4 | **Esquemas**: 32
- **Nivel**: Avanzado
- **Dominios relacionados**: `bot_defense`, `virtual`, `network_security`

## Casos de Uso

- Configurar políticas de mitigación de amenazas automatizadas
- Definir reglas para la detección de usuarios maliciosos
- Gestionar flujos de trabajo de respuesta a incidentes
- Aplicar bloqueo o limitación de velocidad a las amenazas

## Recursos Principales

- **mitigation_policy**: Política de mitigación para respuesta a incidentes
- **malicious_user_rule**: Regla de usuario malicioso para mitigación de amenazas

## Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations` | Crear Mitigación de Usuario Malicioso. |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations/&#123;metadata.name&#125;` | Reemplazar Mitigación de Usuario Malicioso. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations` | Listar Mitigación de Usuario Malicioso. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | OBTENER Mitigación de Usuario Malicioso. |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | ELIMINAR Mitigación de Usuario Malicioso. |
