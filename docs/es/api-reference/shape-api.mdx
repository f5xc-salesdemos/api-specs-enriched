Reconocimiento de amenazas con análisis de comportamiento y toma de huellas digitales de dispositivos. Integración de SDK móvil para el blindaje de aplicaciones.

- **Categoría**: Seguridad
- **Complejidad**: avanzada
- **Rutas**: 271 | **Esquemas**: 813
- **Nivel**: Avanzado
- **Dominios relacionados**: `bot_defense`, `virtual`

## Casos de uso

- Configurar políticas de Shape Security
- Gestionar la prevención de bots y amenazas
- Configurar políticas de SafeAP
- Habilitar el reconocimiento de amenazas

## Recursos principales

- **shape_app_firewall**: Firewall de aplicaciones Shape para protección avanzada
- **shape_recognizer**: Reconocedor Shape para análisis de patrones de tráfico
