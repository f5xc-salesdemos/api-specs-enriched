Políticas de alertas con integraciones de receptores. Agregación de registros, vistas de topología y seguimiento del estado de los sitios.

- **Categoría**: Operaciones
- **Complejidad**: avanzada
- **Rutas**: 77 | **Esquemas**: 575
- **Nivel**: Estándar
- **Dominios relacionados**: `observability`, `support`

## Casos de uso

- Acceder a estadísticas y analíticas de flujo
- Gestionar alertas y políticas de alertas
- Ver registros y receptores de registros
- Generar informes y gráficos
- Rastrear topología y descubrimiento de servicios
- Monitorizar el estado en los sitios

## Recursos principales

- **dashboard**: Panel de control para visualización de métricas
- **saved_query**: Consulta guardada para análisis de métricas y registros
