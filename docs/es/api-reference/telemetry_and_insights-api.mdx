Endpoints de recopilación, almacenamiento de métricas y generación de insights para flujos de trabajo de observabilidad.

- **Categoría**: Operaciones
- **Complejidad**: moderada
- **Rutas**: 27 | **Esquemas**: 181
- **Nivel**: Estándar
- **Dominios relacionados**: `observability`, `statistics`

## Casos de Uso

- Recopilar y analizar datos de telemetría
- Generar insights accionables a partir de métricas
- Configurar políticas de recopilación de telemetría

## Recursos Principales

- **telemetry_receiver**: Receptor de telemetría para la recopilación de datos
- **insight_query**: Consulta de insights para analítica
