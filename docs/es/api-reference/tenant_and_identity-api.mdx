Configuraciones de vista de cuenta y canales de alerta de administración. Restablecimientos de contraseña de un solo uso, flujos de aprovisionamiento y monitoreo de conexiones activas.

- **Categoría**: Plataforma
- **Complejidad**: avanzada
- **Rutas**: 183 | **Esquemas**: 511
- **Nivel**: Estándar
- **Dominios relacionados**: `users`, `authentication`, `system`

## Casos de Uso

- Gestionar perfiles de usuario y preferencias de notificación
- Configurar controles de sesión y ajustes de OTP
- Manejar operaciones de gestión de identidad
- Procesar solicitudes de acceso inicial de usuarios

## Recursos Principales

- **user_profile**: Perfil de usuario con configuraciones personales
- **session**: Sesión de usuario para el estado de autenticación
- **otp_policy**: Política de OTP para autenticación multifactor
