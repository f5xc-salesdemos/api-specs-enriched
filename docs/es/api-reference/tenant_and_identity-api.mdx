Configuraciones de vista de cuenta y canales de alerta del administrador. Restablecimientos de contraseñas de un solo uso, flujos de aprovisionamiento y supervisión de conexiones activas.

- **Categoría**: Plataforma
- **Complejidad**: avanzada
- **Rutas**: 183 | **Esquemas**: 511
- **Nivel**: Estándar
- **Dominios relacionados**: `users`, `authentication`, `system`

## Casos de uso

- Gestionar perfiles de usuario y preferencias de notificación
- Configurar controles de sesión y ajustes OTP
- Gestionar operaciones de administración de identidad
- Procesar solicitudes de acceso inicial de usuarios

## Recursos principales

- **user_profile**: Perfil de usuario con configuraciones personales
- **session**: Sesión de usuario para el estado de autenticación
- **otp_policy**: Política OTP para la autenticación multifactor
