Políticas de detección, seguimiento de ataques y generación automatizada de reglas de mitigación para operaciones de seguridad.

- **Categoría**: Seguridad
- **Complejidad**: moderada
- **Rutas**: 1 | **Esquemas**: 161
- **Nivel**: Estándar
- **Dominios relacionados**: `bot_defense`, `ddos`

## Casos de Uso

- Rastrear y analizar campañas de amenazas
- Monitorear amenazas activas y patrones de ataque
- Configurar la integración de inteligencia de amenazas

## Recursos Principales

- **threat_campaign_policy**: Política de detección y mitigación de campañas de amenazas

## Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | Obtener campaña de amenazas por ID. |
