Credenciales de inscripción de sitios con expiración automática. Las claves de taxonomía definen la categorización permitida mientras que las etiquetas derivadas automáticamente se aplican de forma dinámica.

- **Categoría**: Plataforma
- **Complejidad**: simple
- **Rutas**: 13 | **Esquemas**: 50
- **Nivel**: Estándar
- **Dominios relacionados**: `system`, `admin`

## Casos de uso

- Gestionar cuentas de usuario y tokens
- Configurar la identificación de usuarios
- Gestionar configuraciones y preferencias de usuario
- Configurar etiquetas implícitas y conocidas

## Recursos principales

- **user**: Cuenta de usuario para acceso a la Plataforma
- **user_role**: Rol de usuario para la gestión de permisos
- **namespace_role**: Rol con ámbito de espacio de nombres para permisos detallados
