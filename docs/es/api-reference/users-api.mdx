Credenciales de inscripción de sitio con expiración automática. Las claves de taxonomía definen la categorización permitida mientras que las etiquetas derivadas automáticamente se aplican de forma dinámica.

- **Categoría**: Plataforma
- **Complejidad**: simple
- **Rutas**: 13 | **Esquemas**: 50
- **Nivel**: Estándar
- **Dominios relacionados**: `system`, `admin`

## Casos de Uso

- Gestionar cuentas de usuario y tokens
- Configurar la identificación de usuarios
- Gestionar configuraciones y preferencias de usuario
- Configurar etiquetas implícitas y conocidas

## Recursos Principales

- **user**: Cuenta de usuario para acceso a la plataforma
- **user_role**: Rol de usuario para gestión de permisos
- **namespace_role**: Rol con ámbito de namespace para permisos detallados

## Endpoints

|--------|------|-------------|

| POST | `/api/config/namespaces/&#123;namespace&#125;/known_label/create` | Crear |
| POST | `/api/config/namespaces/&#123;namespace&#125;/known_label/delete` | ELIMINAR |
| GET | `/api/config/namespaces/&#123;namespace&#125;/known_labels` | OBTENER |
