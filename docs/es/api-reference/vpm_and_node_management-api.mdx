Control del ciclo de vida, configuración de flotas y políticas de despliegue para la gestión distribuida de nodos.

- **Categoría**: Plataforma
- **Complejidad**: simple
- **Rutas**: 1 | **Esquemas**: 2
- **Nivel**: Estándar
- **Dominios relacionados**: `sites`, `system`

## Casos de Uso

- Gestionar la configuración de la Malla Privada Virtual (VPM)
- Configurar el ciclo de vida y la gestión de nodos
- Monitorear el estado de VPM y nodos

## Recursos Principales

- **node_config**: Configuración de nodos para ajustes de dispositivos de borde
- **vpm_config**: Configuración de VPM para la gestión de plataformas virtuales

## Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/api/data/namespaces/system/upgrade_status` | Estado de Actualización. |
