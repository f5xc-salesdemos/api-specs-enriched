Éléments visuels dont la portée est limitée aux espaces de noms, avec gestion des versions. Déploiement de widgets personnalisés et gestion du catalogue pour les surfaces du portail.

- **Catégorie** : Plateforme
- **Complexité** : simple
- **Chemins** : 2 | **Schémas** : 14
- **Niveau** : Standard
- **Domaines associés** : `admin`, `system`

## Cas d'utilisation

- Gérer les Composants d'interface statiques pour la console d'administration
- Déployer et récupérer des ressources d'interface au sein des espaces de noms
- Configurer les éléments de l'interface de la console
- Gérer les métadonnées des Composants d'interface personnalisés

## Ressources principales

- **ui_component** : Composant d'interface pour la personnalisation de la console
- **static_asset** : Ressource statique pour les ressources d'interface

## Points de terminaison

| Méthode | Chemin | Description |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | Lister le composant statique d'interface UI. |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | Obtenir le composant statique d'interface UI. |
