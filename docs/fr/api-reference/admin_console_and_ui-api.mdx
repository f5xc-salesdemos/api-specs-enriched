Éléments visuels à portée de namespace avec gestion de versions. Déploiement de widgets personnalisés et gestion du catalogue pour les surfaces du portail.

- **Catégorie** : Platform
- **Complexité** : simple
- **Chemins** : 2 | **Schémas** : 14
- **Niveau** : Standard
- **Domaines associés** : `admin`, `system`

## Cas d'utilisation

- Gérer les composants d'interface utilisateur statiques pour la console d'administration
- Déployer et récupérer les ressources d'interface utilisateur au sein des namespaces
- Configurer les éléments d'interface de la console
- Gérer les métadonnées des composants d'interface utilisateur personnalisés

## Ressources principales

- **ui_component** : Composant d'interface utilisateur pour la personnalisation de la console
- **static_asset** : Ressource statique pour les éléments d'interface utilisateur

## Points de terminaison

| Méthode | Chemin | Description |
|---------|--------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | Lister les composants statiques de l'interface utilisateur. |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | Obtenir un composant statique de l'interface utilisateur. |
