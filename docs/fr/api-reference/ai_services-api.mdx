Traitement du langage naturel avec signaux de qualité et surveillance des anomalies. Authentification par jeton pour les abonnements aux flux de données.

- **Catégorie** : IA
- **Complexité** : simple
- **Chemins** : 11 | **Schémas** : 66
- **Niveau** : Avancé


## Cas d'utilisation

- Accéder aux fonctionnalités alimentées par l'IA
- Configurer les politiques de l'assistant IA
- Activer la détection d'anomalies de flux
- Gérer la collecte de données IA

## Ressources principales

- **ai_policy** : Politique IA pour le contrôle du trafic d'IA générative
- **ai_gateway** : Passerelle IA pour la gestion des API LLM

## Points de terminaison

| Méthode | Chemin | Description |
|---------|--------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | Évaluer une requête de l'assistant IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | Évaluer le retour d'une requête de l'assistant IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | Requête de l'assistant IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | Retour d'une requête de l'assistant IA. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | Activer une fonctionnalité. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | Rafraîchir le jeton. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | S'abonner au pipeline BFDP. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | S'abonner au pipeline BFDP. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | Se désabonner du pipeline BFDP. |
| POST | `/api/gia/gia/allocateip` | Allouer une IP via l'allocateur d'IP global. |
| DELETE | `/api/gia/gia/deallocateip` | Désallouer une IP via l'allocateur d'IP global. |
