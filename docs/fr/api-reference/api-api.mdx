Catalogage des ressources avec classification automatique et profilage de sécurité. Les hiérarchies organisationnelles segmentent l'accès par fonction ou politique de protection.

- **Catégorie** : Sécurité
- **Complexité** : avancé
- **Chemins** : 45 | **Schémas** : 279
- **Niveau** : Avancé
- **Domaines associés** : `virtual`, `network_security`

## Cas d'utilisation

- Découvrir et cataloguer les API
- Tester la sécurité et le comportement des API
- Gérer les identifiants des API
- Définir les groupes d'API et les politiques de test

## Ressources principales

- **api_definition** : Définition de schéma d'API pour la découverte et la protection
- **api_endpoint** : Configuration et protection d'un point de terminaison d'API individuel
- **api_rate_limit** : Configuration de limitation de débit d'API pour le contrôle du trafic
