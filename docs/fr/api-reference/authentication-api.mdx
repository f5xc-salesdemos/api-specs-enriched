Gestion des identités avec intégration de fournisseurs, politiques d'accès et contrôle du cycle de vie des identifiants.

- **Catégorie** : Plateforme
- **Complexité** : simple
- **Chemins** : 14 | **Schémas** : 38
- **Niveau** : Standard
- **Domaines associés** : `system`, `users`

## Cas d'utilisation

- Configurer les mécanismes d'authentification
- Gérer les fournisseurs OIDC et OAuth
- Configurer le provisionnement des utilisateurs SCIM
- Gérer les identifiants API et les accès
- Configurer les politiques d'inscription des comptes

## Ressources principales

- **authentication_policy** : Politique d'authentification pour l'accès utilisateur et API
- **token** : Jeton API pour l'accès programmatique
- **api_credential** : Identifiant API pour l'authentification des services
