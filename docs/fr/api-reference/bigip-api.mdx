Orchestration héritée des équipements avec des scripts iRule et la synchronisation des groupes de données. Liaison aux serveurs virtuels et collecte de métriques.

- **Catégorie** : Plateforme
- **Complexité** : avancée
- **Chemins** : 29 | **Schémas** : 238
- **Niveau** : Avancé
- **Domaines associés** : `marketplace`

## Cas d'utilisation

- Gérer les appliances BigIP F5
- Configurer les scripts iRule
- Gérer les groupes de données
- Intégrer BigIP CNE

## Ressources principales

- **bigip_pool** : Pool BIG-IP pour l'intégration de la répartition de charge
- **bigip_device** : Enregistrement et gestion des équipements BIG-IP
