Classification des menaces avec analyse comportementale et correspondance de signatures. Blocage automatisé des schémas de trafic malveillant.

- **Catégorie** : Sécurité
- **Complexité** : modérée
- **Chemins** : 19 | **Schémas** : 74
- **Niveau** : Avancé
- **Domaines associés** : `bot_defense`, `shape`, `virtual`

## Cas d'utilisation

- Configurer les instances de défense contre les bots par espace de noms
- Gérer les catégories de menaces TPM pour la classification
- Provisionner les clés API pour les systèmes de défense automatisés
- Intégrer les services de renseignement sur les menaces

## Ressources principales

- **bot_defense_instance** : Instance de défense contre les bots pour le déploiement
- **threat_category** : Catégorie de menace pour la classification
