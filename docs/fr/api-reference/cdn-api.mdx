Distribution mondiale avec règles de mise en cache et opérations de purge. Surveillance des performances et analyses.

- **Catégorie** : Réseau
- **Complexité** : avancée
- **Chemins** : 29 | **Schémas** : 592
- **Niveau** : Avancé
- **Domaines associés** : `virtual`

## Cas d'utilisation

- Configurer l'équilibrage de charge CDN
- Gérer les services de réseau de diffusion de contenu
- Configurer les politiques de mise en cache
- Gérer la livraison et la distribution de données

## Ressources principales

- **cdn_loadbalancer** : Équilibreur de charge du réseau de diffusion de contenu pour la mise en cache en périphérie
- **cdn_origin_pool** : Pool de serveurs d'origine pour l'approvisionnement en contenu CDN
