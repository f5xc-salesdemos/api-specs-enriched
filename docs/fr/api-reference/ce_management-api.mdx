Provisionnement basé sur des jetons avec téléchargement d'images et validation avant mise à niveau. Le regroupement en flottes permet des opérations en masse sur des emplacements distribués.

- **Catégorie** : Infrastructure
- **Complexité** : modérée
- **Chemins** : 28 | **Schémas** : 231
- **Niveau** : Standard
- **Domaines associés** : `customer_edge`, `sites`

## Cas d'utilisation

- Gérer le cycle de vie des sites Customer Edge
- Configurer les interfaces réseau et les paramètres de flotte
- Gérer l'enregistrement des sites et les flux de jetons
- Exécuter les mises à niveau de sites avec des vérifications préalables

## Ressources principales

- **site_config** : Configuration du site pour les paramètres du nœud edge
- **fleet_config** : Configuration de la flotte pour la gestion multi-sites
- **registration_token** : Jeton d'enregistrement pour l'intégration des sites
