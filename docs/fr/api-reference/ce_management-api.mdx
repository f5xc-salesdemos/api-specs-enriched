Provisionnement basé sur des jetons avec téléchargement d'images et validation pré-mise à jour. Le regroupement en flottes permet des opérations en masse sur des emplacements distribués.

- **Catégorie** : Infrastructure
- **Complexité** : modérée
- **Chemins** : 28 | **Schémas** : 231
- **Niveau** : Standard
- **Domaines associés** : `customer_edge`, `sites`

## Cas d'utilisation

- Gérer le cycle de vie des sites Customer Edge
- Configurer les interfaces réseau et les paramètres de flotte
- Gérer l'enregistrement des sites et les flux de travail basés sur les jetons
- Exécuter des mises à jour de sites avec des vérifications pré-mise à jour

## Ressources principales

- **site_config** : Configuration de site pour les paramètres des nœuds périphériques
- **fleet_config** : Configuration de flotte pour la gestion multi-sites
- **registration_token** : Jeton d'enregistrement pour l'intégration des sites
