Chaînes de certificats et ensembles d'autorités de certification de confiance. Gestion des listes de révocation et configuration des manifestes pour les opérations PKI.

- **Catégorie** : Sécurité
- **Complexité** : modérée
- **Chemins** : 16 | **Schémas** : 74
- **Niveau** : Standard
- **Domaines associés** : `blindfold`, `system`

## Cas d'utilisation

- Gérer les certificats SSL/TLS
- Configurer les autorités de certification de confiance
- Gérer les listes de révocation de certificats (CRL)
- Configurer les manifestes de certificats

## Ressources principales

- **certificate** : Certificat TLS pour les connexions HTTPS sécurisées
- **ca_certificate** : Certificat d'autorité de certification pour la chaîne de confiance
- **certificate_chain** : Chaîne de certificats pour la vérification complète de la confiance
