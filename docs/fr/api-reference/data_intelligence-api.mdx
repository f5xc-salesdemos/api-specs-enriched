Règles de classification, politiques de ressources et génération d'aperçus pour les flux de travail d'analyse des données.

- **Catégorie** : Opérations
- **Complexité** : simple
- **Chemins** : 15 | **Schémas** : 73
- **Niveau** : Standard
- **Domaines connexes** : `statistics`, `observability`

## Cas d'utilisation

- Analyser les données de sécurité et de trafic
- Générer des aperçus intelligents à partir des journaux
- Configurer les politiques d'analyse des données

## Ressources principales

- **analytics_query** : Requête analytique pour l'analyse des données
- **data_export** : Configuration d'exportation des données pour les rapports
