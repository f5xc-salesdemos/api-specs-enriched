Listes de refus, groupes de règles de pare-feu et protections basées sur les tunnels. Limitation de débit et analyse de modèles pour la sécurité du périmètre réseau.

- **Catégorie** : Sécurité
- **Complexité** : avancée
- **Chemins** : 60 | **Schémas** : 246
- **Niveau** : Avancé
- **Domaines associés** : `network_security`, `virtual`

## Cas d'utilisation

- Configurer les politiques de protection DDoS
- Surveiller et analyser les menaces DDoS
- Configurer la protection de l'infrastructure

## Ressources principales

- **ddos_protection** : Configuration de la protection DDoS pour le nettoyage du trafic
- **ddos_mitigation_rule** : Règle d'atténuation DDoS pour la réponse aux attaques
