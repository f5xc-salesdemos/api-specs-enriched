Provisionnement de tunnels GRE et IPSec tiers avec minuteries DPD. Allocation de ressources partagées entre les espaces de noms avec contrôles de placement des tuiles.

- **Catégorie** : Plateforme
- **Complexité** : modérée
- **Chemins** : 36 | **Schémas** : 183
- **Niveau** : Avancé
- **Domaines associés** : `bigip`, `admin`

## Cas d'utilisation

- Accéder aux intégrations et modules complémentaires tiers
- Gérer les extensions du marketplace
- Configurer Terraform et les intégrations externes
- Gérer les politiques TPM

## Ressources principales

- **marketplace_item** : Élément du marketplace pour le catalogue de services
- **subscription** : Abonnement pour les droits d'accès aux services
