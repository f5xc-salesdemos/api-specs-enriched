Découverte d'instances, intégration WAF et connectivité de maillage de services. Cycle de vie des abonnements et synchronisation de la configuration.

- **Catégorie** : Plateforme
- **Complexité** : simple
- **Chemins** : 13 | **Schémas** : 59
- **Niveau** : Avancé
- **Domaines associés** : `marketplace`

## Cas d'utilisation

- Gérer les intégrations de la plateforme NGINX One
- Configurer les instances NGINX Plus
- Intégrer la gestion de configuration NGINX

## Ressources principales

- **nginx_config** : Configuration NGINX pour les paramètres de proxy
- **nginx_upstream** : NGINX upstream pour les pools de serveurs backend
