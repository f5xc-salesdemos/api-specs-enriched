Découverte d'instances, intégration WAF et connectivité service mesh. Cycle de vie des abonnements et synchronisation des configurations.

- **Catégorie** : Plateforme
- **Complexité** : simple
- **Chemins** : 13 | **Schémas** : 59
- **Niveau** : Avancé
- **Domaines associés** : `marketplace`

## Cas d'utilisation

- Gérer les intégrations de la Plateforme NGINX One
- Configurer les instances NGINX Plus
- Intégrer la gestion de la configuration NGINX

## Ressources principales

- **nginx_config** : Configuration NGINX pour les paramètres proxy
- **nginx_upstream** : Upstream NGINX pour les pools de serveurs backend

## Points de terminaison

| Méthode | Chemin | Description |
|--------|------|-------------|
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_csgs` | Lister la configuration des objets CSG de NGINX One. |
