Distribution de bibliothèques versionnées pour les intégrateurs d'applications mobiles. Les URL présignées permettent des téléchargements sécurisés avec des builds spécifiques à chaque système d'exploitation pour iOS et Android.

- **Catégorie** : Plateforme
- **Complexité** : simple
- **Chemins** : 9 | **Schémas** : 17
- **Niveau** : Avancé
- **Domaines associés** : `marketplace`

## Cas d'utilisation

- Gérer les services de stockage d'objets
- Configurer les objets stockés et les compartiments
- Gérer les politiques de stockage

## Ressources principales

- **object_store** : Magasin d'objets pour la gestion du stockage de blobs
- **bucket** : Compartiment de stockage pour le stockage d'objets
