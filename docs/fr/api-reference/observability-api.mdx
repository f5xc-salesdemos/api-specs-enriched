Sondes de disponibilité HTTP avec mesure de la latence. Alertes d'expiration de certificats et tableaux de bord d'état global pour la santé de l'infrastructure.

- **Catégorie** : Opérations
- **Complexité** : modérée
- **Chemins** : 56 | **Schémas** : 196
- **Niveau** : Standard
- **Domaines associés** : `statistics`, `support`

## Cas d'utilisation

- Configurer la surveillance synthétique
- Définir les politiques de surveillance et de test
- Gérer les tableaux de bord d'observabilité

## Ressources principales

- **log_receiver** : Récepteur de journaux pour la collecte centralisée des journaux
- **metrics_receiver** : Récepteur de métriques pour la collecte centralisée des métriques
- **alert_policy** : Politique d'alerte pour la surveillance et les notifications
