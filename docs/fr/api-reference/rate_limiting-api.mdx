Application des quotas temporels avec des fenêtres configurables en heures, minutes ou secondes. Contrôles spécifiques aux protocoles pour la mise en forme du trafic.

- **Catégorie** : Réseau
- **Complexité** : simple
- **Chemins** : 12 | **Schémas** : 70
- **Niveau** : Avancé
- **Domaines associés** : `virtual`, `network_security`

## Cas d'utilisation

- Configurer les politiques de limitation du débit
- Gérer les configurations de contrôleur de débit
- Contrôler le flux de trafic et la mise en file d'attente

## Ressources principales

- **rate_limiter** : Limiteur de débit pour la régulation et la protection du trafic
- **rate_limiter_policy** : Politique de limitation du débit avec des règles de régulation détaillées
- **rate_limit_threshold** : Configuration du seuil de limitation du débit pour le contrôle du trafic
