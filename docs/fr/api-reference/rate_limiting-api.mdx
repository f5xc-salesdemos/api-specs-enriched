Application de quotas basée sur le temps avec des fenêtres configurables en heures, minutes ou secondes. Contrôles spécifiques aux protocoles pour la mise en forme du trafic.

- **Catégorie** : Réseau
- **Complexité** : simple
- **Chemins** : 12 | **Schémas** : 70
- **Niveau** : Avancé
- **Domaines associés** : `virtual`, `network_security`

## Cas d'utilisation

- Configurer les politiques de limitation de débit
- Gérer les configurations de régulateurs
- Contrôler le flux de trafic et la mise en file d'attente

## Ressources principales

- **rate_limiter** : Limiteur de débit pour la régulation et la protection du trafic
- **rate_limiter_policy** : Politique de limitation de débit avec des règles de régulation détaillées
- **rate_limit_threshold** : Configuration de seuil de limitation de débit pour le contrôle du trafic
