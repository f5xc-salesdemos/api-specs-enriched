Atténuation des utilisateurs malveillants avec classification du niveau de menace. Actions de réponse automatisées pour les comportements suspects.

- **Catégorie** : Sécurité
- **Complexité** : simple
- **Chemins** : 4 | **Schémas** : 32
- **Niveau** : Avancé
- **Domaines associés** : `bot_defense`, `virtual`, `network_security`

## Cas d'utilisation

- Configurer des politiques automatisées d'atténuation des menaces
- Définir des règles pour la détection des utilisateurs malveillants
- Gérer les flux de travail de réponse aux incidents
- Appliquer le blocage ou la limitation de débit aux menaces

## Ressources principales

- **mitigation_policy** : Politique d'atténuation pour la réponse aux incidents
- **malicious_user_rule** : Règle d'utilisateur malveillant pour l'atténuation des menaces

## Points de terminaison

| Méthode | Chemin | Description |
|---------|--------|-------------|
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations` | Créer une atténuation d'utilisateur malveillant. |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations/&#123;metadata.name&#125;` | Remplacer une atténuation d'utilisateur malveillant. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations` | Lister les atténuations d'utilisateur malveillant. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | OBTENIR une atténuation d'utilisateur malveillant. |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | SUPPRIMER une atténuation d'utilisateur malveillant. |
