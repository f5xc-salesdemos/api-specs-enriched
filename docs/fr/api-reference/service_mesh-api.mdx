Définitions de types d'applications avec découverte et analyse de schéma appris. Inférence de modèles de trafic pour un traitement intelligent des requêtes.

- **Catégorie** : Infrastructure
- **Complexité** : avancée
- **Chemins** : 38 | **Schémas** : 247
- **Niveau** : Avancé
- **Domaines associés** : `managed_kubernetes`, `container_services`, `virtual`

## Cas d'utilisation

- Configurer la connectivité du service mesh
- Gérer la découverte des endpoints et le routage
- Configurer les services NFV
- Définir les paramètres et types d'applications

## Ressources principales

- **endpoint** : Endpoint du service mesh pour le routage du trafic
- **origin_pool** : Groupe de serveurs backend pour la distribution du trafic de l'équilibreur de charge
- **service_discovery** : Configuration de la découverte de services pour les endpoints dynamiques
