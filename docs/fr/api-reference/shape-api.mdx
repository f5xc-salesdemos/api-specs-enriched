Reconnaissance des menaces avec analyse comportementale et empreinte numérique des appareils. Intégration du SDK mobile pour le blindage des applications.

- **Catégorie** : Sécurité
- **Complexité** : avancée
- **Chemins** : 271 | **Schémas** : 813
- **Niveau** : Avancé
- **Domaines associés** : `bot_defense`, `virtual`

## Cas d'utilisation

- Configurer les politiques Shape Security
- Gérer la prévention des bots et des menaces
- Configurer les politiques SafeAP
- Activer la reconnaissance des menaces

## Ressources principales

- **shape_app_firewall** : Pare-feu applicatif Shape pour une protection avancée
- **shape_recognizer** : Reconnaisseur Shape pour l'analyse des modèles de trafic
