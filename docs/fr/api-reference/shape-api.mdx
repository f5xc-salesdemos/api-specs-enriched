Reconnaissance des menaces avec analyse comportementale et empreinte digitale des appareils. Intégration du SDK mobile pour la protection des applications.

- **Catégorie** : Sécurité
- **Complexité** : avancée
- **Chemins** : 271 | **Schémas** : 813
- **Niveau** : Avancé
- **Domaines associés** : `bot_defense`, `virtual`

## Cas d'utilisation

- Configurer les politiques de sécurité Shape
- Gérer la prévention des bots et des menaces
- Configurer les politiques SafeAP
- Activer la reconnaissance des menaces

## Ressources principales

- **shape_app_firewall** : Pare-feu applicatif Shape pour une protection avancée
- **shape_recognizer** : Analyseur Shape pour l'analyse des schémas de trafic
