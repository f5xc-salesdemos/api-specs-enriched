Points de terminaison de collecte, stockage de métriques et génération d'informations pour les workflows d'Observabilité.

- **Catégorie** : Opérations
- **Complexité** : modérée
- **Chemins** : 27 | **Schémas** : 181
- **Niveau** : Standard
- **Domaines connexes** : `observability`, `statistics`

## Cas d'utilisation

- Collecter et analyser des données de télémétrie
- Générer des informations exploitables à partir de métriques
- Configurer des politiques de collecte de télémétrie

## Ressources principales

- **telemetry_receiver** : Récepteur de télémétrie pour la collecte de données
- **insight_query** : Requête d'analyse pour l'analytique
