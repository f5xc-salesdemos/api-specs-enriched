Configurations de vue de compte et canaux d'alerte administrateur. Réinitialisations de mots de passe à usage unique, flux de provisionnement et surveillance des connexions actives.

- **Catégorie** : Plateforme
- **Complexité** : avancée
- **Chemins** : 183 | **Schémas** : 511
- **Niveau** : Standard
- **Domaines associés** : `users`, `authentication`, `system`

## Cas d'utilisation

- Gérer les profils utilisateurs et les préférences de notification
- Configurer les contrôles de session et les paramètres OTP
- Gérer les opérations de gestion des identités
- Traiter les demandes d'accès initiaux des utilisateurs

## Ressources principales

- **user_profile** : Profil utilisateur avec paramètres personnels
- **session** : Session utilisateur pour l'état d'authentification
- **otp_policy** : Politique OTP pour l'authentification multi-facteurs
