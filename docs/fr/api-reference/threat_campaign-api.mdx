Politiques de détection, suivi des attaques et génération automatisée de règles d'atténuation pour les opérations de sécurité.

- **Catégorie** : Sécurité
- **Complexité** : modérée
- **Chemins** : 1 | **Schémas** : 161
- **Niveau** : Standard
- **Domaines associés** : `bot_defense`, `ddos`

## Cas d'utilisation

- Suivre et analyser les campagnes de menaces
- Surveiller les menaces actives et les schémas d'attaque
- Configurer l'intégration de renseignements sur les menaces

## Ressources principales

- **threat_campaign_policy** : Politique de détection et d'atténuation des campagnes de menaces

## Points de terminaison

| Méthode | Chemin | Description |
|---------|--------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | Obtenir une campagne de menaces par ID. |
