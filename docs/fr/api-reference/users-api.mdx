Identifiants d'inscription au site avec expiration automatique. Les clés de taxonomie définissent la catégorisation autorisée tandis que les balises auto-dérivées s'appliquent dynamiquement.

- **Catégorie** : Plateforme
- **Complexité** : simple
- **Chemins** : 13 | **Schémas** : 50
- **Niveau** : Standard
- **Domaines associés** : `system`, `admin`

## Cas d'utilisation

- Gérer les comptes utilisateurs et les jetons
- Configurer l'identification des utilisateurs
- Gérer les paramètres et préférences des utilisateurs
- Configurer les étiquettes implicites et connues

## Ressources principales

- **user** : Compte utilisateur pour l'accès à la Plateforme
- **user_role** : Rôle utilisateur pour la gestion des permissions
- **namespace_role** : Rôle à portée de namespace pour des permissions affinées
