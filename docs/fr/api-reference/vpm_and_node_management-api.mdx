Contrôle du cycle de vie, configuration de la flotte et politiques de déploiement pour la gestion distribuée des nœuds.

- **Catégorie** : Plateforme
- **Complexité** : simple
- **Chemins** : 1 | **Schémas** : 2
- **Niveau** : Standard
- **Domaines associés** : `sites`, `system`

## Cas d'utilisation

- Gérer la configuration du Virtual Private Mesh (VPM)
- Configurer le cycle de vie et la gestion des nœuds
- Surveiller l'état du VPM et des nœuds

## Ressources principales

- **node_config** : Configuration des nœuds pour les paramètres des dispositifs en périphérie
- **vpm_config** : Configuration VPM pour la gestion de la plateforme virtuelle

## Points de terminaison

| Méthode | Chemin | Description |
|---------|--------|-------------|
| GET | `/api/data/namespaces/system/upgrade_status` | État de la mise à niveau. |
