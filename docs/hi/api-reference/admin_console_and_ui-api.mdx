संस्करण प्रबंधन के साथ नेमस्पेस-स्कोप्ड दृश्य तत्व। पोर्टल सतहों के लिए कस्टम विजेट परिनियोजन और कैटलॉग प्रबंधन।

- **श्रेणी**: Platform
- **जटिलता**: simple
- **पथ**: 2 | **स्कीमा**: 14
- **स्तर**: Standard
- **संबंधित डोमेन**: `admin`, `system`

## उपयोग के मामले

- एडमिन कंसोल के लिए स्थैतिक UI घटकों का प्रबंधन करें
- नेमस्पेस के भीतर UI संसाधनों को परिनियोजित और पुनर्प्राप्त करें
- कंसोल इंटरफ़ेस तत्वों को कॉन्फ़िगर करें
- कस्टम UI घटक मेटाडेटा का प्रबंधन करें

## प्राथमिक संसाधन

- **ui_component**: कंसोल अनुकूलन के लिए UI घटक
- **static_asset**: UI संसाधनों के लिए स्थैतिक संसाधन

## एंडपॉइंट

| विधि | पथ | विवरण |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | UI स्थैतिक घटकों की सूची। |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | UI स्थैतिक घटक प्राप्त करें। |
