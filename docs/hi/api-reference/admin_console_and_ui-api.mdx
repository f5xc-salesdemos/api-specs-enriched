वर्शनिंग के साथ नेमस्पेस-स्कोप्ड विज़ुअल एलिमेंट। पोर्टल सरफ़ेस के लिए कस्टम विजेट डिप्लॉयमेंट और कैटलॉग प्रबंधन।

- **श्रेणी**: Platform
- **जटिलता**: सरल
- **पथ**: 2 | **स्कीमा**: 14
- **टियर**: Standard
- **संबंधित डोमेन**: `admin`, `system`

## उपयोग के मामले

- एडमिन कंसोल के लिए स्टैटिक UI कंपोनेंट प्रबंधित करें
- नेमस्पेस के भीतर UI एसेट डिप्लॉय और प्राप्त करें
- कंसोल इंटरफ़ेस एलिमेंट कॉन्फ़िगर करें
- कस्टम UI कंपोनेंट मेटाडेटा प्रबंधित करें

## प्राथमिक संसाधन

- **ui_component**: कंसोल कस्टमाइज़ेशन के लिए UI कंपोनेंट
- **static_asset**: UI संसाधनों के लिए स्टैटिक एसेट

## एंडपॉइंट

| विधि | पथ | विवरण |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | UI स्टैटिक कंपोनेंट सूचीबद्ध करें। |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | UI स्टैटिक कंपोनेंट प्राप्त करें। |
