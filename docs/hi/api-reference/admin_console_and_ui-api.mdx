वर्शनिंग के साथ नेमस्पेस-स्कोप्ड विज़ुअल एलिमेंट। पोर्टल सतहों के लिए कस्टम विजेट डिप्लॉयमेंट और कैटलॉग प्रबंधन।

- **श्रेणी**: Platform
- **जटिलता**: सरल
- **पथ**: 2 | **स्कीमा**: 14
- **टियर**: Standard
- **संबंधित डोमेन**: `admin`, `system`

## उपयोग के मामले

- एडमिन कंसोल के लिए स्थिर UI घटकों का प्रबंधन करें
- नेमस्पेस के भीतर UI संपत्तियों को डिप्लॉय और पुनर्प्राप्त करें
- कंसोल इंटरफ़ेस एलिमेंट कॉन्फ़िगर करें
- कस्टम UI घटक मेटाडेटा का प्रबंधन करें

## प्राथमिक संसाधन

- **ui_component**: कंसोल कस्टमाइज़ेशन के लिए UI घटक
- **static_asset**: UI संसाधनों के लिए स्थिर संपत्ति

## एंडपॉइंट

| विधि | पथ | विवरण |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | UI स्थिर घटकों की सूची। |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | UI स्थिर घटक प्राप्त करें। |
