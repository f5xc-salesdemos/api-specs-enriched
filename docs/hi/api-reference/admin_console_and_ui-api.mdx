नेमस्पेस-स्कोप्ड विज़ुअल एलिमेंट्स वर्शनिंग के साथ। पोर्टल सतहों के लिए कस्टम विजेट डिप्लॉयमेंट और कैटलॉग प्रबंधन।

- **श्रेणी**: प्लेटफ़ॉर्म
- **जटिलता**: सरल
- **पाथ**: 2 | **स्कीमा**: 14
- **स्तर**: मानक
- **संबंधित डोमेन**: `admin`, `system`

## उपयोग के मामले

- एडमिन कंसोल के लिए स्टैटिक UI घटकों का प्रबंधन करें
- नेमस्पेस के भीतर UI एसेट डिप्लॉय और प्राप्त करें
- कंसोल इंटरफेस एलिमेंट्स कॉन्फ़िगर करें
- कस्टम UI घटक मेटाडेटा का प्रबंधन करें

## प्राथमिक संसाधन

- **ui_component**: कंसोल कस्टमाइज़ेशन के लिए UI घटक
- **static_asset**: UI संसाधनों के लिए स्टैटिक एसेट

## एंडपॉइंट

| विधि | पाथ | विवरण |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | UI स्टैटिक घटक की सूची बनाएं। |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | UI स्टैटिक घटक प्राप्त करें। |
