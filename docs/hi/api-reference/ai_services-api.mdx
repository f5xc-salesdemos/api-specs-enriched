प्राकृतिक भाषा प्रसंस्करण, गुणवत्ता संकेतों और विसंगति निगरानी के साथ। डेटा स्ट्रीम सब्सक्रिप्शन के लिए टोकन प्रमाणीकरण।

- **श्रेणी**: AI
- **जटिलता**: सरल
- **पाथ**: 11 | **स्कीमा**: 66
- **स्तर**: उन्नत


## उपयोग के मामले

- AI-संचालित सुविधाओं तक पहुँच
- AI असिस्टेंट नीतियाँ कॉन्फ़िगर करें
- फ्लो विसंगति पहचान सक्षम करें
- AI डेटा संग्रह प्रबंधित करें

## प्राथमिक संसाधन

- **ai_policy**: जनरेटिव AI ट्रैफ़िक नियंत्रण के लिए AI नीति
- **ai_gateway**: LLM API प्रबंधन के लिए AI गेटवे

## एंडपॉइंट

| विधि | पाथ | विवरण |
|--------|------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | AI असिस्टेंट क्वेरी का मूल्यांकन करें। |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | AI असिस्टेंट क्वेरी के फ़ीडबैक का मूल्यांकन करें। |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | AI असिस्टेंट क्वेरी। |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | AI असिस्टेंट क्वेरी का फ़ीडबैक। |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | सुविधा सक्षम करें। |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | टोकन रीफ्रेश करें। |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | BFDP पाइपलाइन की सदस्यता लें। |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | BFDP पाइपलाइन की सदस्यता लें। |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | BFDP पाइपलाइन की सदस्यता रद्द करें। |
| POST | `/api/gia/gia/allocateip` | Global IP Allocator के माध्यम से IP आवंटित करें। |
| DELETE | `/api/gia/gia/deallocateip` | Global IP Allocator के माध्यम से IP हटाएँ। |
