गुणवत्ता संकेतों और विसंगति निगरानी के साथ प्राकृतिक भाषा प्रसंस्करण। डेटा स्ट्रीम सदस्यताओं के लिए टोकन प्रमाणीकरण।

- **श्रेणी**: AI
- **जटिलता**: सरल
- **पथ**: 11 | **स्कीमा**: 66
- **स्तर**: उन्नत


## उपयोग के मामले

- AI-संचालित सुविधाओं तक पहुँच
- AI सहायक नीतियाँ कॉन्फ़िगर करें
- प्रवाह विसंगति पहचान सक्षम करें
- AI डेटा संग्रहण प्रबंधित करें

## प्राथमिक संसाधन

- **ai_policy**: जनरेटिव AI ट्रैफ़िक नियंत्रण के लिए AI नीति
- **ai_gateway**: LLM API प्रबंधन के लिए AI गेटवे

## एंडपॉइंट

| विधि | पथ | विवरण |
|--------|------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | AI सहायक क्वेरी का मूल्यांकन। |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | AI सहायक क्वेरी के फीडबैक का मूल्यांकन। |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | AI सहायक क्वेरी। |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | AI सहायक क्वेरी का फीडबैक। |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | सुविधा सक्षम करें। |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | टोकन रिफ्रेश करें। |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | BFDP पाइपलाइन की सदस्यता लें। |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | BFDP पाइपलाइन की सदस्यता लें। |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | BFDP पाइपलाइन की सदस्यता रद्द करें। |
| POST | `/api/gia/gia/allocateip` | ग्लोबल IP एलोकेटर के माध्यम से IP आवंटित करें। |
| DELETE | `/api/gia/gia/deallocateip` | ग्लोबल IP एलोकेटर के माध्यम से IP का आवंटन रद्द करें। |
