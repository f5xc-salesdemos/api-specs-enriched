स्वचालित वर्गीकरण और सुरक्षा प्रोफ़ाइलिंग के साथ संसाधन सूचीकरण। संगठनात्मक पदानुक्रम कार्य या सुरक्षा नीति के अनुसार पहुँच को विभाजित करते हैं।

- **श्रेणी**: सुरक्षा
- **जटिलता**: उन्नत
- **पथ**: 45 | **स्कीमा**: 279
- **स्तर**: उन्नत
- **संबंधित डोमेन**: `virtual`, `network_security`

## उपयोग के मामले

- API की खोज और सूचीकरण करें
- API सुरक्षा और व्यवहार का परीक्षण करें
- API क्रेडेंशियल प्रबंधित करें
- API समूह और परीक्षण नीतियाँ परिभाषित करें

## प्राथमिक संसाधन

- **api_definition**: खोज और सुरक्षा के लिए API स्कीमा परिभाषा
- **api_endpoint**: व्यक्तिगत API एंडपॉइंट कॉन्फ़िगरेशन और सुरक्षा
- **api_rate_limit**: ट्रैफ़िक नियंत्रण के लिए API दर सीमा कॉन्फ़िगरेशन
