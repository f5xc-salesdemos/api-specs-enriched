प्रदाता एकीकरण, एक्सेस नीतियों और क्रेडेंशियल जीवनचक्र नियंत्रण के साथ पहचान प्रबंधन।

- **श्रेणी**: प्लेटफ़ॉर्म
- **जटिलता**: सरल
- **पथ**: 14 | **स्कीमा**: 38
- **स्तर**: मानक
- **संबंधित डोमेन**: `system`, `users`

## उपयोग के मामले

- प्रमाणीकरण तंत्र कॉन्फ़िगर करें
- OIDC और OAuth प्रदाताओं का प्रबंधन करें
- SCIM उपयोगकर्ता प्रोविजनिंग कॉन्फ़िगर करें
- API क्रेडेंशियल और एक्सेस प्रबंधित करें
- खाता साइनअप नीतियाँ कॉन्फ़िगर करें

## प्राथमिक संसाधन

- **authentication_policy**: उपयोगकर्ता और API एक्सेस के लिए प्रमाणीकरण नीति
- **token**: प्रोग्रामेटिक एक्सेस के लिए API टोकन
- **api_credential**: सेवा प्रमाणीकरण के लिए API क्रेडेंशियल
