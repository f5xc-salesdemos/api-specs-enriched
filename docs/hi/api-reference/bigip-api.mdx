iRule स्क्रिप्ट और डेटा ग्रुप सिंक्रोनाइज़ेशन के साथ लीगेसी डिवाइस ऑर्केस्ट्रेशन। वर्चुअल सर्वर बाइंडिंग और मेट्रिक्स संग्रह।

- **श्रेणी**: प्लेटफ़ॉर्म
- **जटिलता**: उन्नत
- **पाथ**: 29 | **स्कीमा**: 238
- **स्तर**: उन्नत
- **संबंधित डोमेन**: `marketplace`

## उपयोग के मामले

- BigIP F5 उपकरणों का प्रबंधन करें
- iRule स्क्रिप्ट कॉन्फ़िगर करें
- डेटा ग्रुप प्रबंधित करें
- BigIP CNE को एकीकृत करें

## प्राथमिक संसाधन

- **bigip_pool**: लोड बैलेंसिंग एकीकरण के लिए BIG-IP पूल
- **bigip_device**: BIG-IP डिवाइस पंजीकरण और प्रबंधन
