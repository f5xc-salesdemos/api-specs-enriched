iRule स्क्रिप्ट और डेटा ग्रुप सिंक्रनाइज़ेशन के साथ लीगेसी डिवाइस ऑर्केस्ट्रेशन। वर्चुअल सर्वर बाइंडिंग और मेट्रिक्स संग्रह।

- **श्रेणी**: Platform
- **जटिलता**: उन्नत
- **पथ**: 29 | **स्कीमा**: 238
- **टियर**: Advanced
- **संबंधित डोमेन**: `marketplace`

## उपयोग के मामले

- BigIP F5 उपकरणों का प्रबंधन करें
- iRule स्क्रिप्ट कॉन्फ़िगर करें
- डेटा ग्रुप का प्रबंधन करें
- BigIP CNE को एकीकृत करें

## प्राथमिक संसाधन

- **bigip_pool**: लोड बैलेंसिंग एकीकरण के लिए BIG-IP पूल
- **bigip_device**: BIG-IP डिवाइस पंजीकरण और प्रबंधन
