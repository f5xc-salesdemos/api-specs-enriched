योजना संक्रमण, चालान, और संसाधन खपत। नेमस्पेस-स्तरीय कोटा सीमाएँ और उपयोग ट्रैकिंग।

- **श्रेणी**: प्लेटफ़ॉर्म
- **जटिलता**: मध्यम
- **पथ**: 25 | **स्कीमा**: 101
- **टियर**: स्टैंडर्ड
- **संबंधित डोमेन**: `system`, `users`

## उपयोग के मामले

- सब्सक्रिप्शन योजनाएँ और बिलिंग संक्रमण प्रबंधित करें
- भुगतान विधियाँ और चालान कॉन्फ़िगर करें
- नेमस्पेस भर में संसाधन कोटा उपयोग ट्रैक करें
- उपयोग सीमाओं और क्षमता की निगरानी करें

## प्राथमिक संसाधन

- **subscription**: सेवा अधिकारों के लिए सब्सक्रिप्शन
- **quota**: उपयोग सीमाओं के लिए संसाधन कोटा
- **usage_report**: खपत ट्रैकिंग के लिए उपयोग रिपोर्ट
