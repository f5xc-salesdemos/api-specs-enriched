व्यवहार विश्लेषण और सिग्नेचर मिलान के साथ खतरा वर्गीकरण। दुर्भावनापूर्ण ट्रैफ़िक पैटर्न के लिए स्वचालित अवरोधन।

- **श्रेणी**: सुरक्षा
- **जटिलता**: मध्यम
- **पाथ**: 19 | **स्कीमा**: 74
- **स्तर**: उन्नत
- **संबंधित डोमेन**: `bot_defense`, `shape`, `virtual`

## उपयोग के मामले

- प्रत्येक नेमस्पेस के लिए bot रक्षा इंस्टेंस कॉन्फ़िगर करें
- वर्गीकरण के लिए TPM खतरा श्रेणियाँ प्रबंधित करें
- स्वचालित रक्षा प्रणालियों के लिए API कुंजियाँ प्रावधानित करें
- खतरा इंटेलिजेंस सेवाएँ एकीकृत करें

## प्राथमिक संसाधन

- **bot_defense_instance**: परिनियोजन के लिए bot रक्षा इंस्टेंस
- **threat_category**: वर्गीकरण के लिए खतरा श्रेणी
