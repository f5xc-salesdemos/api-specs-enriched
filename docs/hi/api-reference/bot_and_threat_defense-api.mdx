व्यवहारात्मक विश्लेषण और सिग्नेचर मैचिंग के साथ थ्रेट वर्गीकरण। दुर्भावनापूर्ण ट्रैफ़िक पैटर्न के लिए स्वचालित ब्लॉकिंग।

- **श्रेणी**: सुरक्षा
- **जटिलता**: मध्यम
- **पथ**: 19 | **स्कीमा**: 74
- **टियर**: उन्नत
- **संबंधित डोमेन**: `bot_defense`, `shape`, `virtual`

## उपयोग के मामले

- प्रति नेमस्पेस बॉट डिफेंस इंस्टेंस कॉन्फ़िगर करें
- वर्गीकरण के लिए TPM थ्रेट श्रेणियाँ प्रबंधित करें
- स्वचालित डिफेंस सिस्टम के लिए API कुंजियाँ प्रोविज़न करें
- थ्रेट इंटेलिजेंस सेवाओं को एकीकृत करें

## प्राथमिक संसाधन

- **bot_defense_instance**: तैनाती के लिए बॉट डिफेंस इंस्टेंस
- **threat_category**: वर्गीकरण के लिए थ्रेट श्रेणी
