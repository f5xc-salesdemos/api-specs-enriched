इमेज डाउनलोड और प्री-अपग्रेड सत्यापन के साथ टोकन-आधारित प्रोविजनिंग। फ्लीट ग्रुपिंग वितरित स्थानों पर बल्क ऑपरेशन सक्षम करता है।

- **श्रेणी**: इन्फ्रास्ट्रक्चर
- **जटिलता**: मध्यम
- **पाथ**: 28 | **स्कीमा**: 231
- **टियर**: स्टैंडर्ड
- **संबंधित डोमेन**: `customer_edge`, `sites`

## उपयोग के मामले

- Customer Edge साइट जीवनचक्र प्रबंधित करें
- नेटवर्क इंटरफेस और फ्लीट सेटिंग्स कॉन्फ़िगर करें
- साइट पंजीकरण और टोकन वर्कफ़्लो संभालें
- प्री-अपग्रेड जांच के साथ साइट अपग्रेड निष्पादित करें

## प्राथमिक संसाधन

- **site_config**: एज नोड सेटिंग्स के लिए साइट कॉन्फ़िगरेशन
- **fleet_config**: मल्टी-साइट प्रबंधन के लिए फ्लीट कॉन्फ़िगरेशन
- **registration_token**: साइट ऑनबोर्डिंग के लिए पंजीकरण टोकन
