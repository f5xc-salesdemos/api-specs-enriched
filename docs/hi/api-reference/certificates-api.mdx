प्रमाणपत्र चेन और विश्वसनीय CA बंडल। PKI संचालन के लिए निरस्तीकरण सूची प्रबंधन और मैनिफ़ेस्ट कॉन्फ़िगरेशन।

- **श्रेणी**: सुरक्षा
- **जटिलता**: मध्यम
- **पाथ**: 16 | **स्कीमा**: 74
- **टियर**: स्टैंडर्ड
- **संबंधित डोमेन**: `blindfold`, `system`

## उपयोग के मामले

- SSL/TLS प्रमाणपत्रों का प्रबंधन करें
- विश्वसनीय CA कॉन्फ़िगर करें
- प्रमाणपत्र निरस्तीकरण सूचियों (CRL) का प्रबंधन करें
- प्रमाणपत्र मैनिफ़ेस्ट कॉन्फ़िगर करें

## प्राथमिक संसाधन

- **certificate**: सुरक्षित HTTPS कनेक्शन के लिए TLS प्रमाणपत्र
- **ca_certificate**: ट्रस्ट चेन के लिए प्रमाणपत्र प्राधिकरण प्रमाणपत्र
- **certificate_chain**: पूर्ण ट्रस्ट सत्यापन के लिए प्रमाणपत्र चेन
