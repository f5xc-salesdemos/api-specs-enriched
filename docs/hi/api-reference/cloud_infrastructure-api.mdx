गेटवे पीयरिंग और नेटवर्क पाथ कॉन्फ़िगरेशन के साथ मल्टी-क्लाउड प्रदाता कनेक्शन। क्रेडेंशियल वॉल्ट एकीकरण और सबनेट एन्यूमरेशन।

- **श्रेणी**: Infrastructure
- **जटिलता**: मध्यम
- **पाथ**: 31 | **स्कीमा**: 208
- **टियर**: Standard
- **संबंधित डोमेन**: `sites`, `customer_edge`

## उपयोग के मामले

- क्लाउड प्रदाताओं (AWS, Azure, GCP) से कनेक्ट करें
- क्लाउड क्रेडेंशियल्स और प्रमाणीकरण प्रबंधित करें
- क्लाउड कनेक्टिविटी और इलास्टिक प्रोविज़निंग कॉन्फ़िगर करें
- क्लाउड रीजन को लिंक और प्रबंधित करें

## प्राथमिक संसाधन

- **aws_vpc_site**: एज नोड प्रोविज़निंग के साथ AWS VPC साइट डिप्लॉयमेंट
- **azure_vnet_site**: एज नोड प्रोविज़निंग के साथ Azure VNet साइट डिप्लॉयमेंट
- **gcp_vpc_site**: एज नोड प्रोविज़निंग के साथ Google Cloud VPC साइट डिप्लॉयमेंट
- **cloud_credentials**: साइट डिप्लॉयमेंट के लिए क्लाउड प्रदाता प्रमाणीकरण क्रेडेंशियल्स
