पूर्ण क्लस्टर जटिलता के बिना Pod ऑर्केस्ट्रेशन। एज साइट निष्पादन, कोटा प्रवर्तन, और वितरित ऐप्स के लिए मानकीकृत कंप्यूट प्रोफाइल।

- **श्रेणी**: इंफ्रास्ट्रक्चर
- **जटिलता**: मध्यम
- **पाथ**: 13 | **स्कीमा**: 159
- **स्तर**: उन्नत
- **संबंधित डोमेन**: `managed_kubernetes`, `sites`, `service_mesh`

## उपयोग के मामले

- मल्टी-टेनेंट वर्कलोड के लिए XCCS (कंटेनर सेवाएं) नेमस्पेस तैनात करें
- सरलीकृत ऑर्केस्ट्रेशन के साथ कंटेनर वर्कलोड प्रबंधित करें
- वितरित एज कंटेनर डिप्लॉयमेंट कॉन्फ़िगर करें
- पूर्ण K8s जटिलता के बिना कंटेनरीकृत एप्लिकेशन चलाएं

## प्राथमिक संसाधन

- **virtual_k8s**: कंटेनर वर्कलोड के लिए वर्चुअल Kubernetes नेमस्पेस
- **workload**: कंटेनर वर्कलोड डिप्लॉयमेंट कॉन्फ़िगरेशन
- **pod_security_policy**: कंटेनर रनटाइम बाधाओं के लिए Pod सुरक्षा नीति
