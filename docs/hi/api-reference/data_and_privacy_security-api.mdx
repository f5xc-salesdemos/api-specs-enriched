कस्टम वर्गीकरण नियमों के साथ संवेदनशील डेटा नीतियाँ। LMA क्षेत्र कॉन्फ़िगरेशन और भौगोलिक-आधारित अनुपालन नियंत्रण।

- **श्रेणी**: सुरक्षा
- **जटिलता**: सरल
- **पथ**: 11 | **स्कीमा**: 65
- **स्तर**: उन्नत
- **संबंधित डोमेन**: `blindfold`, `client_side_defense`

## उपयोग के मामले

- संवेदनशील डेटा डिटेक्शन नीतियाँ कॉन्फ़िगर करें
- PII वर्गीकरण के लिए कस्टम डेटा प्रकार परिभाषित करें
- LMA क्षेत्र कॉन्फ़िगरेशन प्रबंधित करें
- अनुपालन के लिए भौगोलिक-कॉन्फ़िगरेशन एकीकृत करें

## प्राथमिक संसाधन

- **sensitive_data_policy**: PII सुरक्षा के लिए संवेदनशील डेटा नीति
- **data_classification**: सामग्री वर्गीकरण के लिए डेटा वर्गीकरण
