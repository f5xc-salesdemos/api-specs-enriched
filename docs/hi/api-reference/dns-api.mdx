ज़ोन ट्रांसफर और हेल्थ चेक के साथ नाम समाधान। रिकॉर्ड प्रकार और डेलिगेशन सपोर्ट।

- **श्रेणी**: नेटवर्किंग
- **जटिलता**: उन्नत
- **पथ**: 49 | **स्कीमा**: 285
- **टियर**: मानक
- **संबंधित डोमेन**: `virtual`, `network`

## उपयोग के मामले

- DNS लोड बैलेंसिंग कॉन्फ़िगर करें
- DNS ज़ोन और डोमेन प्रबंधित करें
- DNS अनुपालन नीतियाँ कॉन्फ़िगर करें
- रिसोर्स रिकॉर्ड सेट (RRSets) प्रबंधित करें

## प्राथमिक संसाधन

- **dns_zone**: रिकॉर्ड प्रबंधन क्षमताओं के साथ प्राधिकृत DNS ज़ोन
- **dns_domain**: DNS डोमेन डेलिगेशन और कॉन्फ़िगरेशन सेटिंग्स
- **dns_load_balancer**: भौगोलिक रूटिंग और फेलओवर के साथ DNS-आधारित ट्रैफ़िक वितरण
