DPD टाइमर्स के साथ थर्ड-पार्टी GRE और IPSec टनल प्रोविज़निंग। टाइल प्लेसमेंट नियंत्रणों के साथ नेमस्पेस में साझा संसाधन आवंटन।

- **श्रेणी**: प्लेटफ़ॉर्म
- **जटिलता**: मध्यम
- **पथ**: 36 | **स्कीमा**: 183
- **स्तर**: उन्नत
- **संबंधित डोमेन**: `bigip`, `admin`

## उपयोग के मामले

- थर्ड-पार्टी इंटीग्रेशन और ऐड-ऑन तक पहुँचें
- मार्केटप्लेस एक्सटेंशन प्रबंधित करें
- Terraform और बाहरी इंटीग्रेशन कॉन्फ़िगर करें
- TPM नीतियाँ प्रबंधित करें

## प्राथमिक संसाधन

- **marketplace_item**: सेवा कैटलॉग के लिए मार्केटप्लेस आइटम
- **subscription**: सेवा एंटाइटलमेंट के लिए सब्सक्रिप्शन
