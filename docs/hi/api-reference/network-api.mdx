ASN प्रबंधन और ऑटोनॉमस सिस्टम संबंधों के साथ बॉर्डर गेटवे प्रोटोकॉल। एन्क्रिप्टेड चैनलों के माध्यम से डेटासेंटरों को जोड़ने वाला साइट-टू-साइट VPN।

- **श्रेणी**: नेटवर्किंग
- **जटिलता**: उन्नत
- **पथ**: 81 | **स्कीमा**: 449
- **स्तर**: उन्नत
- **संबंधित डोमेन**: `virtual`, `network_security`, `dns`

## उपयोग के मामले

- BGP रूटिंग और ASN प्रबंधन कॉन्फ़िगर करें
- IPsec टनल और IKE चरण प्रबंधित करें
- नेटवर्क कनेक्टर और रूट कॉन्फ़िगर करें
- SRv6 और सबनेटिंग प्रबंधित करें
- सेगमेंट कनेक्शन और पॉलिसीज़ परिभाषित करें
- IP प्रीफ़िक्स सेट कॉन्फ़िगर करें

## प्राथमिक संसाधन

- **virtual_network**: साइट कनेक्टिविटी और सेगमेंटेशन के लिए वर्चुअल नेटवर्क
- **network_connector**: साइट-टू-साइट या क्लाउड कनेक्टिविटी के लिए नेटवर्क कनेक्टर
- **site_mesh_group**: कई साइटों के बीच मेश कनेक्टिविटी कॉन्फ़िगरेशन
