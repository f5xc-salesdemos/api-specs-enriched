इंस्टेंस डिस्कवरी, WAF इंटीग्रेशन, और सर्विस मेश कनेक्टिविटी। सब्सक्रिप्शन जीवनचक्र और कॉन्फ़िगरेशन सिंक्रोनाइज़ेशन।

- **श्रेणी**: Platform
- **जटिलता**: सरल
- **पथ**: 13 | **स्कीमा**: 59
- **टियर**: उन्नत
- **संबंधित डोमेन**: `marketplace`

## उपयोग के मामले

- NGINX One प्लेटफ़ॉर्म इंटीग्रेशन प्रबंधित करें
- NGINX Plus इंस्टेंस कॉन्फ़िगर करें
- NGINX कॉन्फ़िगरेशन प्रबंधन को एकीकृत करें

## प्राथमिक संसाधन

- **nginx_config**: प्रॉक्सी सेटिंग्स के लिए NGINX कॉन्फ़िगरेशन
- **nginx_upstream**: बैकएंड सर्वर पूल के लिए NGINX अपस्ट्रीम
