मोबाइल ऐप इंटीग्रेटर्स के लिए वर्शन्ड लाइब्रेरी वितरण। प्रीसाइंड URLs iOS और Android के लिए OS-विशिष्ट बिल्ड्स के साथ सुरक्षित डाउनलोड सक्षम करते हैं।

- **श्रेणी**: प्लेटफ़ॉर्म
- **जटिलता**: सरल
- **पथ**: 9 | **स्कीमा**: 17
- **टियर**: उन्नत
- **संबंधित डोमेन**: `marketplace`

## उपयोग के मामले

- ऑब्जेक्ट स्टोरेज सेवाओं का प्रबंधन करें
- संग्रहीत ऑब्जेक्ट्स और बकेट्स को कॉन्फ़िगर करें
- स्टोरेज नीतियों का प्रबंधन करें

## प्राथमिक संसाधन

- **object_store**: ब्लॉब स्टोरेज प्रबंधन के लिए ऑब्जेक्ट स्टोर
- **bucket**: ऑब्जेक्ट स्टोरेज के लिए स्टोरेज बकेट
