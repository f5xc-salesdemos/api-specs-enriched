विलंबता माप के साथ HTTP उपलब्धता जांच। प्रमाणपत्र समाप्ति अलर्ट और इंफ्रास्ट्रक्चर स्वास्थ्य के लिए वैश्विक स्थिति डैशबोर्ड।

- **श्रेणी**: संचालन
- **जटिलता**: मध्यम
- **पथ**: 56 | **स्कीमा**: 196
- **स्तर**: मानक
- **संबंधित डोमेन**: `statistics`, `support`

## उपयोग के मामले

- सिंथेटिक मॉनिटरिंग कॉन्फ़िगर करें
- मॉनिटरिंग और परीक्षण नीतियाँ परिभाषित करें
- ऑब्जर्वेबिलिटी डैशबोर्ड प्रबंधित करें

## प्राथमिक संसाधन

- **log_receiver**: केंद्रीकृत लॉग संग्रह के लिए लॉग रिसीवर
- **metrics_receiver**: केंद्रीकृत मेट्रिक्स संग्रह के लिए मेट्रिक्स रिसीवर
- **alert_policy**: मॉनिटरिंग और अधिसूचना के लिए अलर्ट नीति
