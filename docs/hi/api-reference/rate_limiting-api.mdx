घंटों, मिनटों या सेकंड में कॉन्फ़िगर करने योग्य विंडो के साथ समय-आधारित कोटा प्रवर्तन। ट्रैफ़िक शेपिंग के लिए प्रोटोकॉल-विशिष्ट नियंत्रण।

- **श्रेणी**: नेटवर्किंग
- **जटिलता**: सरल
- **पथ**: 12 | **स्कीमा**: 70
- **स्तर**: उन्नत
- **संबंधित डोमेन**: `virtual`, `network_security`

## उपयोग के मामले

- रेट लिमिटर नीतियाँ कॉन्फ़िगर करें
- पॉलिसर कॉन्फ़िगरेशन प्रबंधित करें
- ट्रैफ़िक प्रवाह और कतारबद्धता को नियंत्रित करें

## प्राथमिक संसाधन

- **rate_limiter**: ट्रैफ़िक थ्रॉटलिंग और सुरक्षा के लिए रेट लिमिटर
- **rate_limiter_policy**: विस्तृत थ्रॉटलिंग नियमों सहित रेट लिमिटर नीति
- **rate_limit_threshold**: ट्रैफ़िक नियंत्रण के लिए रेट लिमिट थ्रेशोल्ड कॉन्फ़िगरेशन
