घंटों, मिनटों, या सेकंडों में कॉन्फ़िगर करने योग्य विंडो के साथ समय-आधारित कोटा प्रवर्तन। ट्रैफ़िक शेपिंग के लिए प्रोटोकॉल-विशिष्ट नियंत्रण।

- **श्रेणी**: नेटवर्किंग
- **जटिलता**: सरल
- **पथ**: 12 | **स्कीमा**: 70
- **स्तर**: उन्नत
- **संबंधित डोमेन**: `virtual`, `network_security`

## उपयोग के मामले

- दर सीमक नीतियाँ कॉन्फ़िगर करें
- पॉलिसर कॉन्फ़िगरेशन प्रबंधित करें
- ट्रैफ़िक प्रवाह और क्यूइंग नियंत्रित करें

## प्राथमिक संसाधन

- **rate_limiter**: ट्रैफ़िक थ्रॉटलिंग और सुरक्षा के लिए दर सीमक
- **rate_limiter_policy**: विस्तृत थ्रॉटलिंग नियमों के साथ दर सीमक नीति
- **rate_limit_threshold**: ट्रैफ़िक नियंत्रण के लिए दर सीमा सीमा कॉन्फ़िगरेशन
