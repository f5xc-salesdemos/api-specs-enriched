खतरे के स्तर वर्गीकरण के साथ दुर्भावनापूर्ण उपयोगकर्ता शमन। संदिग्ध व्यवहार पैटर्न के लिए स्वचालित प्रतिक्रिया कार्रवाइयाँ।

- **श्रेणी**: सुरक्षा
- **जटिलता**: सरल
- **पथ**: 4 | **स्कीमा**: 32
- **स्तर**: उन्नत
- **संबंधित डोमेन**: `bot_defense`, `virtual`, `network_security`

## उपयोग के मामले

- स्वचालित खतरा शमन नीतियाँ कॉन्फ़िगर करें
- दुर्भावनापूर्ण उपयोगकर्ता पहचान के लिए नियम परिभाषित करें
- घटना प्रतिक्रिया कार्यप्रवाह प्रबंधित करें
- खतरों पर अवरोधन या दर सीमित करना लागू करें

## प्राथमिक संसाधन

- **mitigation_policy**: घटना प्रतिक्रिया के लिए शमन नीति
- **malicious_user_rule**: खतरा शमन के लिए दुर्भावनापूर्ण उपयोगकर्ता नियम
