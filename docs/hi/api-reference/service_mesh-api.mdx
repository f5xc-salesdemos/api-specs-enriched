डिस्कवरी और लर्न्ड स्कीमा विश्लेषण के साथ एप्लिकेशन टाइप परिभाषाएँ। बुद्धिमान अनुरोध हैंडलिंग के लिए ट्रैफ़िक पैटर्न अनुमान।

- **श्रेणी**: इन्फ्रास्ट्रक्चर
- **जटिलता**: उन्नत
- **पथ**: 38 | **स्कीमा**: 247
- **स्तर**: उन्नत
- **संबंधित डोमेन**: `managed_kubernetes`, `container_services`, `virtual`

## उपयोग के मामले

- सर्विस मेश कनेक्टिविटी कॉन्फ़िगर करें
- एंडपॉइंट डिस्कवरी और रूटिंग प्रबंधित करें
- NFV सेवाएँ कॉन्फ़िगर करें
- एप्लिकेशन सेटिंग्स और टाइप परिभाषित करें

## प्राथमिक संसाधन

- **endpoint**: ट्रैफ़िक रूटिंग के लिए सर्विस मेश एंडपॉइंट
- **origin_pool**: लोड बैलेंसर ट्रैफ़िक वितरण के लिए बैकएंड सर्वर समूह
- **service_discovery**: डायनामिक एंडपॉइंट्स के लिए सर्विस डिस्कवरी कॉन्फ़िगरेशन
