व्यवहार विश्लेषण और डिवाइस फिंगरप्रिंटिंग के साथ खतरा पहचान। एप्लिकेशन शील्डिंग के लिए मोबाइल SDK एकीकरण।

- **श्रेणी**: सुरक्षा
- **जटिलता**: उन्नत
- **पथ**: 271 | **स्कीमा**: 813
- **स्तर**: उन्नत
- **संबंधित डोमेन**: `bot_defense`, `virtual`

## उपयोग के मामले

- Shape Security नीतियाँ कॉन्फ़िगर करें
- बॉट और खतरा रोकथाम प्रबंधित करें
- SafeAP नीतियाँ कॉन्फ़िगर करें
- खतरा पहचान सक्षम करें

## प्राथमिक संसाधन

- **shape_app_firewall**: उन्नत सुरक्षा के लिए Shape एप्लिकेशन फ़ायरवॉल
- **shape_recognizer**: ट्रैफ़िक पैटर्न विश्लेषण के लिए Shape रिकॉग्नाइज़र
