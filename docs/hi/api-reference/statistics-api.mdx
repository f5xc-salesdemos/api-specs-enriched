रिसीवर इंटीग्रेशन के साथ अलर्टिंग नीतियाँ। लॉग एकत्रीकरण, टोपोलॉजी दृश्य, और साइट स्वास्थ्य ट्रैकिंग।

- **श्रेणी**: संचालन
- **जटिलता**: उन्नत
- **पथ**: 77 | **स्कीमा**: 575
- **टियर**: मानक
- **संबंधित डोमेन**: `observability`, `support`

## उपयोग के मामले

- फ्लो सांख्यिकी और एनालिटिक्स एक्सेस करें
- अलर्ट और अलर्टिंग नीतियाँ प्रबंधित करें
- लॉग और लॉग रिसीवर देखें
- रिपोर्ट और ग्राफ जनरेट करें
- टोपोलॉजी और सर्विस डिस्कवरी ट्रैक करें
- साइट पर स्थिति की निगरानी करें

## प्राथमिक संसाधन

- **dashboard**: मेट्रिक्स विज़ुअलाइज़ेशन के लिए डैशबोर्ड
- **saved_query**: मेट्रिक्स और लॉग विश्लेषण के लिए सहेजी गई क्वेरी
