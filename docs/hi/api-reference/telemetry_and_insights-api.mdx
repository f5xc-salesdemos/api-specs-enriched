संग्रह एंडपॉइंट, मेट्रिक्स स्टोरेज, और ऑब्ज़र्वेबिलिटी वर्कफ़्लो के लिए इनसाइट जनरेशन।

- **श्रेणी**: संचालन
- **जटिलता**: मध्यम
- **पथ**: 27 | **स्कीमा**: 181
- **टियर**: मानक
- **संबंधित डोमेन**: `observability`, `statistics`

## उपयोग के मामले

- टेलीमेट्री डेटा एकत्र और विश्लेषण करें
- मेट्रिक्स से कार्रवाई योग्य इनसाइट्स उत्पन्न करें
- टेलीमेट्री संग्रह नीतियाँ कॉन्फ़िगर करें

## प्राथमिक संसाधन

- **telemetry_receiver**: डेटा संग्रह के लिए टेलीमेट्री रिसीवर
- **insight_query**: एनालिटिक्स के लिए इनसाइट क्वेरी
