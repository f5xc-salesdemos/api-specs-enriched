खाता दृश्य कॉन्फ़िगरेशन और व्यवस्थापक अलर्ट चैनल। वन-टाइम पासवर्ड रीसेट, प्रोविज़निंग फ़्लो, और सक्रिय कनेक्शन मॉनिटरिंग।

- **श्रेणी**: Platform
- **जटिलता**: उन्नत
- **पाथ**: 183 | **स्कीमा**: 511
- **टियर**: Standard
- **संबंधित डोमेन**: `users`, `authentication`, `system`

## उपयोग के मामले

- उपयोगकर्ता प्रोफ़ाइल और अधिसूचना प्राथमिकताएँ प्रबंधित करें
- सत्र नियंत्रण और OTP सेटिंग्स कॉन्फ़िगर करें
- पहचान प्रबंधन संचालन संभालें
- प्रारंभिक उपयोगकर्ता पहुँच अनुरोध संसाधित करें

## प्राथमिक संसाधन

- **user_profile**: व्यक्तिगत सेटिंग्स के साथ उपयोगकर्ता प्रोफ़ाइल
- **session**: प्रमाणीकरण स्थिति के लिए उपयोगकर्ता सत्र
- **otp_policy**: मल्टी-फ़ैक्टर प्रमाणीकरण के लिए OTP नीति
