अकाउंट व्यू कॉन्फ़िगरेशन और एडमिन अलर्ट चैनल। एक-बार पासवर्ड रीसेट, प्रोविज़निंग फ्लो और सक्रिय कनेक्शन निगरानी।

- **श्रेणी**: प्लेटफ़ॉर्म
- **जटिलता**: उन्नत
- **पथ**: 183 | **स्कीमा**: 511
- **स्तर**: मानक
- **संबंधित डोमेन**: `users`, `authentication`, `system`

## उपयोग के मामले

- उपयोगकर्ता प्रोफाइल और अधिसूचना प्राथमिकताओं का प्रबंधन करें
- सत्र नियंत्रण और OTP सेटिंग्स कॉन्फ़िगर करें
- पहचान प्रबंधन संचालन संभालें
- प्रारंभिक उपयोगकर्ता एक्सेस अनुरोधों को प्रोसेस करें

## प्राथमिक संसाधन

- **user_profile**: व्यक्तिगत सेटिंग्स के साथ उपयोगकर्ता प्रोफाइल
- **session**: प्रमाणीकरण स्थिति के लिए उपयोगकर्ता सत्र
- **otp_policy**: मल्टी-फैक्टर प्रमाणीकरण के लिए OTP नीति
