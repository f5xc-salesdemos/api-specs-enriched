सुरक्षा संचालन के लिए पहचान नीतियाँ, हमले की ट्रैकिंग, और स्वचालित शमन नियम निर्माण।

- **श्रेणी**: Security
- **जटिलता**: moderate
- **पथ**: 1 | **स्कीमा**: 161
- **स्तर**: Standard
- **संबंधित डोमेन**: `bot_defense`, `ddos`

## उपयोग के मामले

- खतरे के अभियानों को ट्रैक और विश्लेषण करें
- सक्रिय खतरों और हमले के पैटर्न की निगरानी करें
- खतरा इंटेलिजेंस एकीकरण कॉन्फ़िगर करें

## प्राथमिक संसाधन

- **threat_campaign_policy**: खतरा अभियान पहचान और शमन नीति

## एंडपॉइंट

| विधि | पथ | विवरण |
|--------|------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | ID द्वारा Threat Campaign प्राप्त करें। |
