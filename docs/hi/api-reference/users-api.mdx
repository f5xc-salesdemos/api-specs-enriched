स्वचालित समाप्ति के साथ साइट नामांकन क्रेडेंशियल। टैक्सोनॉमी कुंजियाँ अनुमत वर्गीकरण को परिभाषित करती हैं जबकि स्वचालित-व्युत्पन्न टैग गतिशील रूप से लागू होते हैं।

- **श्रेणी**: प्लेटफ़ॉर्म
- **जटिलता**: सरल
- **पथ**: 13 | **स्कीमा**: 50
- **स्तर**: मानक
- **संबंधित डोमेन**: `system`, `admin`

## उपयोग के मामले

- उपयोगकर्ता खातों और टोकन प्रबंधित करें
- उपयोगकर्ता पहचान कॉन्फ़िगर करें
- उपयोगकर्ता सेटिंग्स और प्राथमिकताएँ प्रबंधित करें
- अंतर्निहित और ज्ञात लेबल कॉन्फ़िगर करें

## प्राथमिक संसाधन

- **user**: प्लेटफ़ॉर्म एक्सेस के लिए उपयोगकर्ता खाता
- **user_role**: अनुमति प्रबंधन के लिए उपयोगकर्ता भूमिका
- **namespace_role**: सूक्ष्म अनुमतियों के लिए नेमस्पेस-स्कोप्ड भूमिका
