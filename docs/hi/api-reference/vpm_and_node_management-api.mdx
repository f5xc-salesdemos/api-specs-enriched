वितरित नोड प्रबंधन के लिए जीवनचक्र नियंत्रण, फ्लीट कॉन्फ़िगरेशन, और डिप्लॉयमेंट नीतियाँ।

- **श्रेणी**: प्लेटफ़ॉर्म
- **जटिलता**: सरल
- **पथ**: 1 | **स्कीमा**: 2
- **टियर**: स्टैंडर्ड
- **संबंधित डोमेन**: `sites`, `system`

## उपयोग के मामले

- वर्चुअल प्राइवेट मेश (VPM) कॉन्फ़िगरेशन प्रबंधित करें
- नोड जीवनचक्र और प्रबंधन कॉन्फ़िगर करें
- VPM और नोड स्थिति की निगरानी करें

## प्राथमिक संसाधन

- **node_config**: एज डिवाइस सेटिंग्स के लिए नोड कॉन्फ़िगरेशन
- **vpm_config**: वर्चुअल प्लेटफ़ॉर्म प्रबंधन के लिए VPM कॉन्फ़िगरेशन

## एंडपॉइंट्स

| विधि | पथ | विवरण |
|--------|------|-------------|
| GET | `/api/data/namespaces/system/upgrade_status` | अपग्रेड स्थिति। |
