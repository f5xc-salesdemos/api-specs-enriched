Elementi visivi con ambito namespace e versionamento. Distribuzione di widget personalizzati e gestione del catalogo per le superfici del portale.

- **Categoria**: Piattaforma
- **Complessità**: semplice
- **Percorsi**: 2 | **Schemi**: 14
- **Livello**: Standard
- **Domini correlati**: `admin`, `system`

## Casi d'uso

- Gestire componenti UI statici per la console di amministrazione
- Distribuire e recuperare risorse UI all'interno dei namespace
- Configurare gli elementi dell'interfaccia della console
- Gestire i metadati dei componenti UI personalizzati

## Risorse principali

- **ui_component**: Componente UI per la personalizzazione della console
- **static_asset**: Risorsa statica per le risorse UI

## Endpoint

| Metodo | Percorso | Descrizione |
|--------|----------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | Elenco dei componenti statici UI. |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | Ottenere un componente statico UI. |
