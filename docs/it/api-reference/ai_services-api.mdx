Elaborazione del linguaggio naturale con segnali di qualità e monitoraggio delle anomalie. Autenticazione tramite token per le sottoscrizioni ai flussi di dati.

- **Categoria**: IA
- **Complessità**: semplice
- **Percorsi**: 11 | **Schemi**: 66
- **Livello**: Avanzato


## Casi d'uso

- Accedere alle funzionalità basate sull'IA
- Configurare le policy dell'assistente IA
- Abilitare il rilevamento delle anomalie di flusso
- Gestire la raccolta dati IA

## Risorse principali

- **ai_policy**: Policy IA per il controllo del traffico di IA generativa
- **ai_gateway**: Gateway IA per la gestione delle API LLM

## Endpoint

| Metodo | Percorso | Descrizione |
|--------|----------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | Valutazione query dell'assistente IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | Valutazione feedback della query dell'assistente IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | Query dell'assistente IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | Feedback della query dell'assistente IA. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | Abilitazione funzionalità. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | Aggiornamento token. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | Sottoscrizione alla pipeline BFDP. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | Sottoscrizione alla pipeline BFDP. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | Annullamento sottoscrizione alla pipeline BFDP. |
| POST | `/api/gia/gia/allocateip` | Allocazione IP tramite Global IP Allocator. |
| DELETE | `/api/gia/gia/deallocateip` | Deallocazione IP tramite Global IP Allocator. |
