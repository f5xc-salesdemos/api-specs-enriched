Elaborazione del linguaggio naturale con segnali di qualità e monitoraggio delle anomalie. Autenticazione tramite token per le sottoscrizioni ai flussi di dati.

- **Categoria**: AI
- **Complessità**: semplice
- **Percorsi**: 11 | **Schemi**: 66
- **Livello**: Avanzato


## Casi d'uso

- Accedere alle funzionalità basate sull'AI
- Configurare le policy dell'assistente AI
- Abilitare il rilevamento delle anomalie di flusso
- Gestire la raccolta dati AI

## Risorse principali

- **ai_policy**: Policy AI per il controllo del traffico dell'AI generativa
- **ai_gateway**: Gateway AI per la gestione delle API LLM

## Endpoint

| Metodo | Percorso | Descrizione |
|--------|----------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | Valutazione della query dell'assistente AI. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | Valutazione del feedback della query dell'assistente AI. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | Query dell'assistente AI. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | Feedback della query dell'assistente AI. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | Abilitazione funzionalità. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | Aggiornamento del token. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | Sottoscrizione alla pipeline BFDP. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | Sottoscrizione alla pipeline BFDP. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | Annullamento della sottoscrizione alla pipeline BFDP. |
| POST | `/api/gia/gia/allocateip` | Allocazione IP tramite Global IP Allocator. |
| DELETE | `/api/gia/gia/deallocateip` | Deallocazione IP tramite Global IP Allocator. |
