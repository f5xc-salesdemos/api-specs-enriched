Catalogazione delle risorse con classificazione automatica e profilazione della sicurezza. Le gerarchie organizzative segmentano l'accesso per funzione o politica di protezione.

- **Categoria**: Sicurezza
- **Complessità**: avanzata
- **Percorsi**: 45 | **Schemi**: 279
- **Livello**: Avanzato
- **Domini correlati**: `virtual`, `network_security`

## Casi d'uso

- Scoprire e catalogare le API
- Testare la sicurezza e il comportamento delle API
- Gestire le credenziali delle API
- Definire gruppi di API e politiche di testing

## Risorse principali

- **api_definition**: Definizione dello schema API per la scoperta e la protezione
- **api_endpoint**: Configurazione e protezione del singolo endpoint API
- **api_rate_limit**: Configurazione del rate limiting delle API per il controllo del traffico
