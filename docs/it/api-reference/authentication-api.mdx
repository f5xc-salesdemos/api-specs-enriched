Gestione delle identità con integrazione dei provider, policy di accesso e controllo del ciclo di vita delle credenziali.

- **Categoria**: Piattaforma
- **Complessità**: semplice
- **Percorsi**: 14 | **Schemi**: 38
- **Livello**: Standard
- **Domini correlati**: `system`, `users`

## Casi d'uso

- Configurare i meccanismi di autenticazione
- Gestire i provider OIDC e OAuth
- Configurare il provisioning utenti SCIM
- Gestire le credenziali API e gli accessi
- Configurare le policy di registrazione account

## Risorse principali

- **authentication_policy**: Policy di autenticazione per l'accesso utente e API
- **token**: Token API per l'accesso programmatico
- **api_credential**: Credenziale API per l'autenticazione del servizio
