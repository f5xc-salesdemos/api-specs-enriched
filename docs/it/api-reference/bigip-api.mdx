Orchestrazione legacy dei dispositivi con script iRule e sincronizzazione dei data group. Binding dei virtual server e raccolta delle metriche.

- **Categoria**: Piattaforma
- **Complessità**: avanzata
- **Percorsi**: 29 | **Schemi**: 238
- **Livello**: Avanzato
- **Domini correlati**: `marketplace`

## Casi d'uso

- Gestire appliance BigIP F5
- Configurare script iRule
- Gestire data group
- Integrare BigIP CNE

## Risorse principali

- **bigip_pool**: Pool BIG-IP per l'integrazione del bilanciamento del carico
- **bigip_device**: Registrazione e gestione dei dispositivi BIG-IP
