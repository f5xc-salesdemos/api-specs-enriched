Orchestrazione di dispositivi legacy con script iRule e sincronizzazione dei gruppi di dati. Binding dei server virtuali e raccolta di metriche.

- **Categoria**: Piattaforma
- **Complessità**: avanzata
- **Percorsi**: 29 | **Schemi**: 238
- **Livello**: Avanzato
- **Domini correlati**: `marketplace`

## Casi d'uso

- Gestire gli appliance BigIP F5
- Configurare gli script iRule
- Gestire i gruppi di dati
- Integrare BigIP CNE

## Risorse primarie

- **bigip_pool**: Pool BIG-IP per l'integrazione del bilanciamento del carico
- **bigip_device**: Registrazione e gestione dei dispositivi BIG-IP
