Gestione delle chiavi di cifratura con controlli di accesso basati su criteri. Registrazione degli audit e protezione sicura dei dati.

- **Categoria**: Sicurezza
- **Complessità**: moderata
- **Percorsi**: 27 | **Schemi**: 163
- **Livello**: Avanzato
- **Domini correlati**: `client_side_defense`, `certificates`

## Casi d'uso

- Configurare criteri segreti per la cifratura
- Gestire la cifratura dei dati sensibili
- Applicare criteri di protezione dei dati

## Risorse primarie

- **blindfold_secret**: Segreto cifrato tramite cifratura Blindfold
- **secret_policy**: Configurazione dei criteri di gestione dei segreti
- **policy_document**: Documento di criteri per le definizioni del controllo degli accessi
