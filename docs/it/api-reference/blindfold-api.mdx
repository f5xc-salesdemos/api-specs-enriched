Gestione delle chiavi di crittografia con controlli di accesso basati su policy. Registrazione audit e protezione sicura dei dati.

- **Categoria**: Sicurezza
- **Complessità**: moderata
- **Percorsi**: 27 | **Schemi**: 163
- **Livello**: Avanzato
- **Domini correlati**: `client_side_defense`, `certificates`

## Casi d'uso

- Configurare policy dei segreti per la crittografia
- Gestire la crittografia dei dati sensibili
- Applicare policy di protezione dei dati

## Risorse principali

- **blindfold_secret**: Segreto crittografato utilizzando la crittografia Blindfold
- **secret_policy**: Configurazione della policy di gestione dei segreti
- **policy_document**: Documento di policy per le definizioni del controllo degli accessi
