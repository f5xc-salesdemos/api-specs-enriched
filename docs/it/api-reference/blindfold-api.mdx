Gestione delle chiavi di crittografia con controlli di accesso basati su criteri. Registrazione degli audit e protezione sicura dei dati.

- **Categoria**: Sicurezza
- **Complessità**: moderata
- **Percorsi**: 27 | **Schemi**: 163
- **Livello**: Avanzato
- **Domini correlati**: `client_side_defense`, `certificates`

## Casi d'uso

- Configurare criteri dei segreti per la crittografia
- Gestire la crittografia dei dati sensibili
- Applicare criteri di protezione dei dati

## Risorse principali

- **blindfold_secret**: Segreto crittografato tramite crittografia Blindfold
- **secret_policy**: Configurazione dei criteri di gestione dei segreti
- **policy_document**: Documento dei criteri per le definizioni del controllo degli accessi
