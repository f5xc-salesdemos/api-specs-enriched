Classificazione delle minacce con analisi comportamentale e corrispondenza delle firme. Blocco automatizzato per pattern di traffico malevolo.

- **Categoria**: Sicurezza
- **Complessità**: moderata
- **Percorsi**: 19 | **Schemi**: 74
- **Livello**: Avanzato
- **Domini correlati**: `bot_defense`, `shape`, `virtual`

## Casi d'uso

- Configurare istanze di difesa bot per namespace
- Gestire le categorie di minacce TPM per la classificazione
- Provisioning di chiavi API per sistemi di difesa automatizzati
- Integrare servizi di threat intelligence

## Risorse principali

- **bot_defense_instance**: Istanza di difesa bot per il deployment
- **threat_category**: Categoria di minaccia per la classificazione
