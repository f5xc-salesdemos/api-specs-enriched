Distribuzione globale con regole di cache e operazioni di eliminazione. Monitoraggio delle prestazioni e analisi.

- **Categoria**: Rete
- **Complessità**: avanzata
- **Percorsi**: 29 | **Schemi**: 592
- **Livello**: Avanzato
- **Domini correlati**: `virtual`

## Casi d'uso

- Configurare il bilanciamento del carico CDN
- Gestire i servizi di distribuzione dei contenuti
- Configurare le policy di caching
- Gestire la distribuzione e la consegna dei dati

## Risorse primarie

- **cdn_loadbalancer**: Bilanciatore di carico della rete di distribuzione dei contenuti per il caching edge
- **cdn_origin_pool**: Pool di server di origine per l'approvvigionamento dei contenuti CDN
