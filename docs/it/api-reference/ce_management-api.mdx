Provisioning basato su token con download di immagini e validazione pre-aggiornamento. Il raggruppamento in fleet consente operazioni in blocco su sedi distribuite.

- **Categoria**: Infrastruttura
- **Complessità**: moderata
- **Percorsi**: 28 | **Schemi**: 231
- **Livello**: Standard
- **Domini correlati**: `customer_edge`, `sites`

## Casi d'Uso

- Gestire il ciclo di vita dei siti Customer Edge
- Configurare le interfacce di rete e le impostazioni dei fleet
- Gestire la registrazione dei siti e i flussi di lavoro dei token
- Eseguire aggiornamenti dei siti con controlli pre-aggiornamento

## Risorse Principali

- **site_config**: Configurazione del sito per le impostazioni dei nodi edge
- **fleet_config**: Configurazione del fleet per la gestione multi-sito
- **registration_token**: Token di registrazione per l'onboarding dei siti
