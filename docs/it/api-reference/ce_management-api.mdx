Provisioning basato su token con download di immagini e validazione pre-aggiornamento. Il raggruppamento in flotte consente operazioni in blocco su sedi distribuite.

- **Categoria**: Infrastruttura
- **Complessità**: moderata
- **Percorsi**: 28 | **Schemi**: 231
- **Livello**: Standard
- **Domini correlati**: `customer_edge`, `sites`

## Casi d'uso

- Gestire il ciclo di vita del sito Customer Edge
- Configurare le interfacce di rete e le impostazioni della flotta
- Gestire la registrazione del sito e i flussi di lavoro dei token
- Eseguire aggiornamenti del sito con verifiche pre-aggiornamento

## Risorse principali

- **site_config**: Configurazione del sito per le impostazioni del nodo edge
- **fleet_config**: Configurazione della flotta per la gestione multi-sito
- **registration_token**: Token di registrazione per l'onboarding del sito
