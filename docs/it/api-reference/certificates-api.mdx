Catene di certificati e bundle CA attendibili. Gestione degli elenchi di revoca e configurazione del manifest per operazioni PKI.

- **Categoria**: Sicurezza
- **Complessità**: moderata
- **Percorsi**: 16 | **Schemi**: 74
- **Livello**: Standard
- **Domini correlati**: `blindfold`, `system`

## Casi d'uso

- Gestire i certificati SSL/TLS
- Configurare le CA attendibili
- Gestire gli elenchi di revoca dei certificati (CRL)
- Configurare i manifest dei certificati

## Risorse principali

- **certificate**: Certificato TLS per connessioni HTTPS sicure
- **ca_certificate**: Certificato dell'autorità di certificazione per la catena di fiducia
- **certificate_chain**: Catena di certificati per la verifica completa della fiducia
