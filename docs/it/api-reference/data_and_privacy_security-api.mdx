Policy per i dati sensibili con regole di classificazione personalizzate. Configurazione della regione LMA e controlli di conformità basati sulla geolocalizzazione.

- **Categoria**: Sicurezza
- **Complessità**: semplice
- **Percorsi**: 11 | **Schemi**: 65
- **Livello**: Avanzato
- **Domini correlati**: `blindfold`, `client_side_defense`

## Casi d'uso

- Configurare policy di rilevamento dei dati sensibili
- Definire tipi di dati personalizzati per la classificazione PII
- Gestire le configurazioni della regione LMA
- Integrare geo-configurazioni per la conformità

## Risorse principali

- **sensitive_data_policy**: Policy per i dati sensibili per la protezione PII
- **data_classification**: Classificazione dei dati per la categorizzazione dei contenuti
