Regole di classificazione, policy sulle risorse e generazione di approfondimenti per i flussi di lavoro di analisi dei dati.

- **Categoria**: Operations
- **Complessità**: semplice
- **Percorsi**: 15 | **Schemi**: 73
- **Livello**: Standard
- **Domini correlati**: `statistics`, `observability`

## Casi d'uso

- Analizzare dati di sicurezza e traffico
- Generare approfondimenti intelligenti dai log
- Configurare policy di analisi dei dati

## Risorse principali

- **analytics_query**: Query di analisi per l'analisi dei dati
- **data_export**: Configurazione di esportazione dati per la reportistica
