Liste di negazione, gruppi di regole firewall e protezioni basate su tunnel. Limitazione della frequenza e analisi dei pattern per la sicurezza del perimetro di rete.

- **Categoria**: Sicurezza
- **Complessità**: avanzata
- **Percorsi**: 60 | **Schemi**: 246
- **Livello**: Avanzato
- **Domini correlati**: `network_security`, `virtual`

## Casi d'uso

- Configurare le policy di Protezione DDoS
- Monitorare e analizzare le minacce DDoS
- Configurare la protezione dell'infrastruttura

## Risorse principali

- **ddos_protection**: Configurazione della Protezione DDoS per la pulizia del traffico
- **ddos_mitigation_rule**: Regola di mitigazione DDoS per la risposta agli attacchi
