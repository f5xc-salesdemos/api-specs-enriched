Risoluzione dei nomi con trasferimenti di zona e controlli di integrità. Tipi di record e supporto alla delega.

- **Categoria**: Networking
- **Complessità**: avanzata
- **Percorsi**: 49 | **Schemi**: 285
- **Livello**: Standard
- **Domini correlati**: `virtual`, `network`

## Casi d'uso

- Configurare il bilanciamento del carico DNS
- Gestire zone e domini DNS
- Configurare le policy di conformità DNS
- Gestire i set di record di risorse (RRSet)

## Risorse principali

- **dns_zone**: Zona DNS autoritativa con funzionalità di gestione dei record
- **dns_domain**: Delega del dominio DNS e impostazioni di configurazione
- **dns_load_balancer**: Distribuzione del traffico basata su DNS con routing geografico e failover
