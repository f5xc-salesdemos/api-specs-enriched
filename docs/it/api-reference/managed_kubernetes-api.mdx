Binding dei ruoli Kubernetes e policy di ammissione. Integrazione dei registri per workload EKS, AKS e GKE.

- **Categoria**: Infrastruttura
- **Complessità**: moderata
- **Percorsi**: 20 | **Schemi**: 95
- **Livello**: Avanzato
- **Domini correlati**: `container_services`, `sites`, `service_mesh`

## Casi d'uso

- Gestire il RBAC e la sicurezza dei cluster XCKS (Managed Kubernetes)
- Configurare le policy di sicurezza dei pod e i controller di ammissione
- Gestire i registri dei container per distribuzioni enterprise
- Integrazione con cluster Kubernetes esterni (EKS, AKS, GKE)

## Risorse principali

- **mk8s_cluster**: Configurazione del cluster Managed Kubernetes
- **k8s_cluster_role**: Configurazione del ruolo RBAC del cluster Kubernetes
- **container_registry**: Registro delle immagini container per le immagini dei workload
