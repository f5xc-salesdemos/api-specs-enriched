Binding dei ruoli e politiche di ammissione Kubernetes. Integrazione dei registri per i workload EKS, AKS e GKE.

- **Categoria**: Infrastruttura
- **Complessità**: moderata
- **Percorsi**: 20 | **Schemi**: 95
- **Livello**: Avanzato
- **Domini correlati**: `container_services`, `sites`, `service_mesh`

## Casi d'uso

- Gestire RBAC e sicurezza dei cluster XCKS (Kubernetes Gestito)
- Configurare le politiche di sicurezza dei pod e i controller di ammissione
- Gestire i registri di container per le distribuzioni enterprise
- Integrarsi con cluster Kubernetes esterni (EKS, AKS, GKE)

## Risorse principali

- **mk8s_cluster**: Configurazione del cluster Kubernetes Gestito
- **k8s_cluster_role**: Configurazione del ruolo RBAC del cluster Kubernetes
- **container_registry**: Registro delle immagini container per i workload
