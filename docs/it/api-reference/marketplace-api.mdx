Provisioning di tunnel GRE e IPSec di terze parti con timer DPD. Allocazione delle risorse condivise tra namespace con controlli di posizionamento dei riquadri.

- **Categoria**: Piattaforma
- **Complessità**: moderata
- **Percorsi**: 36 | **Schemi**: 183
- **Livello**: Avanzato
- **Domini correlati**: `bigip`, `admin`

## Casi d'uso

- Accedere a integrazioni e componenti aggiuntivi di terze parti
- Gestire le estensioni del marketplace
- Configurare Terraform e le integrazioni esterne
- Gestire le policy TPM

## Risorse primarie

- **marketplace_item**: Elemento del marketplace per il catalogo dei servizi
- **subscription**: Sottoscrizione per i diritti di servizio
