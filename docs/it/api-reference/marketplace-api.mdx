Provisioning di tunnel GRE e IPSec di terze parti con timer DPD. Allocazione di risorse condivise tra namespace con controlli di posizionamento dei tile.

- **Categoria**: Piattaforma
- **Complessità**: moderata
- **Percorsi**: 36 | **Schemi**: 183
- **Livello**: Avanzato
- **Domini correlati**: `bigip`, `admin`

## Casi d'uso

- Accesso a integrazioni e componenti aggiuntivi di terze parti
- Gestione delle estensioni del marketplace
- Configurazione di Terraform e integrazioni esterne
- Gestione delle policy TPM

## Risorse principali

- **marketplace_item**: Elemento del marketplace per il catalogo dei servizi
- **subscription**: Sottoscrizione per i diritti di servizio
