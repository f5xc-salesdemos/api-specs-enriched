Regole firewall con decisioni di routing basate su origine, destinazione o protocollo. La segmentazione isola i carichi di lavoro, mentre i proxy in uscita governano l'accesso.

- **Categoria**: Sicurezza
- **Complessità**: avanzata
- **Percorsi**: 62 | **Schemi**: 386
- **Livello**: Avanzato
- **Domini correlati**: `virtual`, `api`, `network`

## Casi d'uso

- Configurare politiche di firewall di rete e ACL
- Gestire politiche NAT e port forwarding
- Configurare il routing basato su policy
- Definire segmenti di rete e politiche
- Configurare politiche di proxy in avanti

## Risorse principali

- **network_policy**: Politica di sicurezza di rete per il filtraggio del traffico
- **forward_proxy_policy**: Politica di proxy in avanti per il controllo del traffico in uscita
- **network_firewall**: Firewall di rete per la protezione del traffico di livello 3/4
