Rilevamento delle istanze, integrazione WAF e connettività service mesh. Ciclo di vita delle sottoscrizioni e sincronizzazione della configurazione.

- **Categoria**: Piattaforma
- **Complessità**: semplice
- **Percorsi**: 13 | **Schemi**: 59
- **Livello**: Avanzato
- **Domini correlati**: `marketplace`

## Casi d'uso

- Gestire le integrazioni della Piattaforma NGINX One
- Configurare le istanze NGINX Plus
- Integrare la gestione della configurazione NGINX

## Risorse principali

- **nginx_config**: configurazione NGINX per le impostazioni proxy
- **nginx_upstream**: upstream NGINX per i pool di server backend

## Endpoint

| Metodo | Percorso | Descrizione |
|--------|----------|-------------|
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_csgs` | Elenca la configurazione degli oggetti CSG di NGINX One. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_csgs/&#123;name&#125;` | Richiesta GET. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_instances` | Elenca la configurazione degli oggetti istanza di NGINX One. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_instances/&#123;name&#125;` | Richiesta GET. |
| POST | `/api/config/namespaces/&#123;namespace&#125;/nginx_dataplane_servers` | Ottieni i server dataplane di NGINX One. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_servers` | Elenca le API server di NGINX One. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_servers/&#123;name&#125;` | Richiesta GET. |
| POST | `/api/nginx/one/namespaces/system/n1/subscribe` | Sottoscrivi NGINX One. |
