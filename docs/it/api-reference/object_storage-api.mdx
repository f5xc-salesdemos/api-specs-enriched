Distribuzione versionata delle librerie per gli integratori di app mobile. Gli URL prefirmati consentono download sicuri con build specifiche per sistema operativo per iOS e Android.

- **Categoria**: Piattaforma
- **Complessità**: semplice
- **Percorsi**: 9 | **Schemi**: 17
- **Livello**: Avanzato
- **Domini correlati**: `marketplace`

## Casi d'uso

- Gestire i servizi di object storage
- Configurare oggetti archiviati e bucket
- Gestire le policy di archiviazione

## Risorse principali

- **object_store**: Object store per la gestione dell'archiviazione blob
- **bucket**: Bucket di archiviazione per l'object storage
