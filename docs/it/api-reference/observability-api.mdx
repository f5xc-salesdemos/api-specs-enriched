Sonde di disponibilità HTTP con misurazione della latenza. Avvisi di scadenza dei certificati e dashboard di stato globale per il monitoraggio dell'integrità dell'infrastruttura.

- **Categoria**: Operazioni
- **Complessità**: moderata
- **Percorsi**: 56 | **Schemi**: 196
- **Livello**: Standard
- **Domini correlati**: `statistics`, `support`

## Casi d'uso

- Configurare il monitoraggio sintetico
- Definire politiche di monitoraggio e test
- Gestire le dashboard di Osservabilità

## Risorse principali

- **log_receiver**: Ricevitore di log per la raccolta centralizzata dei log
- **metrics_receiver**: Ricevitore di metriche per la raccolta centralizzata delle metriche
- **alert_policy**: Policy di avviso per il monitoraggio e le notifiche
