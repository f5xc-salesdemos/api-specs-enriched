Applicazione di quote basate sul tempo con finestre configurabili in ore, minuti o secondi. Controlli specifici per protocollo per la modellazione del traffico.

- **Categoria**: Rete
- **Complessità**: semplice
- **Percorsi**: 12 | **Schemi**: 70
- **Livello**: Avanzato
- **Domini correlati**: `virtual`, `network_security`

## Casi d'uso

- Configurare le policy di limitazione della velocità
- Gestire le configurazioni policer
- Controllare il flusso del traffico e le code

## Risorse primarie

- **rate_limiter**: Limitatore di velocità per la limitazione e la protezione del traffico
- **rate_limiter_policy**: Policy del limitatore di velocità con regole di limitazione dettagliate
- **rate_limit_threshold**: Configurazione della soglia di limitazione della velocità per il controllo del traffico
