Mitigazione degli utenti malevoli con classificazione del livello di minaccia. Azioni di risposta automatizzate per modelli di comportamento sospetti.

- **Categoria**: Sicurezza
- **Complessità**: semplice
- **Percorsi**: 4 | **Schemi**: 32
- **Livello**: Avanzato
- **Domini correlati**: `bot_defense`, `virtual`, `network_security`

## Casi d'uso

- Configurare politiche di mitigazione automatizzata delle minacce
- Definire regole per il rilevamento di utenti malevoli
- Gestire i flussi di lavoro di risposta agli incidenti
- Applicare blocchi o limitazioni di frequenza alle minacce

## Risorse principali

- **mitigation_policy**: Politica di mitigazione per la risposta agli incidenti
- **malicious_user_rule**: Regola per utenti malevoli per la mitigazione delle minacce

## Endpoint

| Metodo | Percorso | Descrizione |
|--------|------|-------------|
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations` | Crea Mitigazione Utente Malevolo. |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations/&#123;metadata.name&#125;` | Sostituisci Mitigazione Utente Malevolo. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations` | Elenca Mitigazione Utente Malevolo. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | Ottieni Mitigazione Utente Malevolo. |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | Elimina Mitigazione Utente Malevolo. |
