Mitigazione degli utenti malintenzionati con classificazione del livello di minaccia. Azioni di risposta automatizzate per schemi di comportamento sospetti.

- **Categoria**: Sicurezza
- **Complessità**: semplice
- **Percorsi**: 4 | **Schemi**: 32
- **Livello**: Avanzato
- **Domini correlati**: `bot_defense`, `virtual`, `network_security`

## Casi d'uso

- Configurare policy di mitigazione automatizzata delle minacce
- Definire regole per il rilevamento di utenti malintenzionati
- Gestire i flussi di lavoro di risposta agli incidenti
- Applicare il blocco o la limitazione della velocità alle minacce

## Risorse primarie

- **mitigation_policy**: Policy di mitigazione per la risposta agli incidenti
- **malicious_user_rule**: Regola per utenti malintenzionati per la mitigazione delle minacce
