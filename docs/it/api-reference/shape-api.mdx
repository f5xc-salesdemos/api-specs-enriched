Riconoscimento delle minacce con analisi comportamentale e fingerprinting dei dispositivi. Integrazione SDK mobile per la protezione delle applicazioni.

- **Categoria**: Sicurezza
- **Complessità**: avanzata
- **Percorsi**: 271 | **Schemi**: 813
- **Livello**: Avanzato
- **Domini correlati**: `bot_defense`, `virtual`

## Casi d'uso

- Configurare le policy di Shape Security
- Gestire la prevenzione di bot e minacce
- Configurare le policy SafeAP
- Abilitare il riconoscimento delle minacce

## Risorse principali

- **shape_app_firewall**: Firewall applicativo Shape per protezione avanzata
- **shape_recognizer**: Riconoscitore Shape per l'analisi dei pattern di traffico
