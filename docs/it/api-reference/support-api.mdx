Ciclo di vita delle segnalazioni con commenti, modifiche di severità e tracciamento della risoluzione. Cattura dei pacchetti per l'analisi delle connessioni.

- **Categoria**: Operations
- **Complessità**: moderata
- **Percorsi**: 53 | **Schemi**: 167
- **Livello**: Standard
- **Domini correlati**: `statistics`, `observability`

## Casi d'uso

- Inviare e gestire ticket di supporto
- Tracciare le richieste di supporto dei clienti
- Accedere alla documentazione di supporto operativo

## Risorse principali

- **support_case**: Caso di supporto per il tracciamento e la risoluzione delle segnalazioni
- **alert**: Avviso di sistema per le notifiche operative
- **audit_log**: Log di audit per la conformità e il tracciamento della sicurezza
