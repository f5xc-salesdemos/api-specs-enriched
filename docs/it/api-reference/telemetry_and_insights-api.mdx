Endpoint di raccolta, archiviazione delle metriche e generazione di informazioni per i flussi di lavoro di osservabilitĂ .

- **Categoria**: Operazioni
- **ComplessitĂ **: moderata
- **Percorsi**: 27 | **Schemi**: 181
- **Livello**: Standard
- **Domini correlati**: `observability`, `statistics`

## Casi d'uso

- Raccogliere e analizzare dati di telemetria
- Generare informazioni operative dalle metriche
- Configurare le policy di raccolta della telemetria

## Risorse principali

- **telemetry_receiver**: Ricevitore di telemetria per la raccolta dati
- **insight_query**: Query di insight per l'analisi
