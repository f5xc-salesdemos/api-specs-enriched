Endpoint di raccolta, archiviazione delle metriche e generazione di approfondimenti per i flussi di lavoro di Osservabilità.

- **Categoria**: Operazioni
- **Complessità**: moderata
- **Percorsi**: 27 | **Schemi**: 181
- **Livello**: Standard
- **Domini correlati**: `observability`, `statistics`

## Casi d'uso

- Raccogliere e analizzare dati di telemetria
- Generare approfondimenti operativi dalle metriche
- Configurare le politiche di raccolta della telemetria

## Risorse primarie

- **telemetry_receiver**: Ricevitore di telemetria per la raccolta dati
- **insight_query**: Query di approfondimento per l'analisi
