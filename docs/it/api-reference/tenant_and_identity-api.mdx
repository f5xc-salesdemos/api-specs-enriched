Configurazioni della vista account e canali di avviso per amministratori. Reset di password monouso, flussi di provisioning e monitoraggio delle connessioni attive.

- **Categoria**: Piattaforma
- **Complessità**: avanzata
- **Percorsi**: 183 | **Schemi**: 511
- **Livello**: Standard
- **Domini correlati**: `users`, `authentication`, `system`

## Casi d'uso

- Gestire profili utente e preferenze di notifica
- Configurare i controlli di sessione e le impostazioni OTP
- Gestire le operazioni di gestione dell'identità
- Elaborare le richieste di accesso iniziale degli utenti

## Risorse principali

- **user_profile**: Profilo utente con impostazioni personali
- **session**: Sessione utente per lo stato di autenticazione
- **otp_policy**: Policy OTP per l'autenticazione multi-fattore
