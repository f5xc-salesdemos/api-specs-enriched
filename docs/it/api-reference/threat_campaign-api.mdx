Politiche di rilevamento, tracciamento degli attacchi e generazione automatizzata di regole di mitigazione per le operazioni di sicurezza.

- **Categoria**: Sicurezza
- **Complessità**: moderata
- **Percorsi**: 1 | **Schemi**: 161
- **Livello**: Standard
- **Domini correlati**: `bot_defense`, `ddos`

## Casi d'Uso

- Tracciare e analizzare le campagne di minacce
- Monitorare le minacce attive e i modelli di attacco
- Configurare l'integrazione dell'intelligence sulle minacce

## Risorse Principali

- **threat_campaign_policy**: Politica di rilevamento e mitigazione delle campagne di minacce

## Endpoint

| Metodo | Percorso | Descrizione |
|--------|----------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | Ottieni Campagna di Minacce per ID. |
