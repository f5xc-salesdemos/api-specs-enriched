バージョニングを備えた名前空間スコープのビジュアル要素。ポータルサーフェス向けのカスタムウィジェットデプロイメントとカタログ管理。

- **カテゴリ**: Platform
- **複雑度**: simple
- **パス数**: 2 | **スキーマ数**: 14
- **ティア**: Standard
- **関連ドメイン**: `admin`, `system`

## ユースケース

- 管理コンソール用の静的UIコンポーネントを管理する
- 名前空間内でUIアセットをデプロイおよび取得する
- コンソールインターフェース要素を設定する
- カスタムUIコンポーネントのメタデータを管理する

## 主要リソース

- **ui_component**: コンソールカスタマイズ用のUIコンポーネント
- **static_asset**: UIリソース用の静的アセット

## エンドポイント

| メソッド | パス | 説明 |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | UI静的コンポーネントの一覧取得。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | UI静的コンポーネントの取得。 |
