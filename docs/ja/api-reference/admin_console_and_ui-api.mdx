バージョニング機能を備えた名前空間スコープのビジュアル要素。ポータルサーフェス向けのカスタムウィジェットデプロイメントおよびカタログ管理。

- **カテゴリ**: プラットフォーム
- **複雑度**: シンプル
- **パス数**: 2 | **スキーマ数**: 14
- **ティア**: スタンダード
- **関連ドメイン**: `admin`, `system`

## ユースケース

- 管理コンソール用の静的UIコンポーネントの管理
- 名前空間内でのUIアセットのデプロイと取得
- コンソールインターフェース要素の設定
- カスタムUIコンポーネントメタデータの管理

## 主要リソース

- **ui_component**: コンソールカスタマイズ用UIコンポーネント
- **static_asset**: UIリソース用静的アセット

## エンドポイント

| メソッド | パス | 説明 |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | UI静的コンポーネントの一覧取得。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | UI静的コンポーネントの取得。 |
