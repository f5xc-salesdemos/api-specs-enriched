品質シグナルと異常監視を備えた自然言語処理。データストリームサブスクリプション向けのトークン認証。

- **カテゴリ**: AI
- **複雑度**: シンプル
- **パス数**: 11 | **スキーマ数**: 66
- **ティア**: 高度

## ユースケース

- AI 搭載機能へのアクセス
- AI アシスタントポリシーの設定
- フロー異常検知の有効化
- AI データ収集の管理

## 主要リソース

- **ai_policy**: 生成 AI トラフィック制御のための AI ポリシー
- **ai_gateway**: LLM API 管理のための AI ゲートウェイ

## エンドポイント

| メソッド | パス | 説明 |
|--------|------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | AI アシスタントクエリの評価。 |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | AI アシスタントクエリのフィードバック評価。 |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | AI アシスタントクエリ。 |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | AI アシスタントクエリのフィードバック。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | 機能の有効化。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | トークンの更新。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | BFDP パイプラインへのサブスクライブ。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | BFDP パイプラインへのサブスクライブ。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | BFDP パイプラインのサブスクライブ解除。 |
