品質シグナルと異常監視を備えた自然言語処理。データストリームサブスクリプションのためのトークン認証。

- **カテゴリ**: AI
- **複雑さ**: シンプル
- **パス数**: 11 | **スキーマ数**: 66
- **ティア**: アドバンスド


## ユースケース

- AI搭載機能へのアクセス
- AIアシスタントポリシーの設定
- フロー異常検出の有効化
- AIデータ収集の管理

## 主要リソース

- **ai_policy**: 生成AIトラフィック制御のためのAIポリシー
- **ai_gateway**: LLM API管理のためのAIゲートウェイ
