自動分類およびセキュリティプロファイリングによるリソースカタログ化。組織階層により、機能または保護ポリシーに基づいてアクセスをセグメント化します。

- **カテゴリ**: セキュリティ
- **複雑度**: 上級
- **パス数**: 45 | **スキーマ数**: 279
- **ティア**: Advanced
- **関連ドメイン**: `virtual`, `network_security`

## ユースケース

- APIの検出とカタログ化
- APIのセキュリティと動作のテスト
- API資格情報の管理
- APIグループとテストポリシーの定義

## 主要リソース

- **api_definition**: 検出と保護のためのAPIスキーマ定義
- **api_endpoint**: 個別のAPIエンドポイント設定と保護
- **api_rate_limit**: トラフィック制御のためのAPIレート制限設定
