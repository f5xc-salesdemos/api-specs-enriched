iRule スクリプトとデータグループ同期を使用したレガシーデバイスオーケストレーション。仮想サーバーバインディングおよびメトリクス収集。

- **カテゴリ**: プラットフォーム
- **複雑度**: 上級
- **パス数**: 29 | **スキーマ数**: 238
- **ティア**: 上級
- **関連ドメイン**: `marketplace`

## ユースケース

- BigIP F5 アプライアンスの管理
- iRule スクリプトの設定
- データグループの管理
- BigIP CNE の統合

## 主要リソース

- **bigip_pool**: ロードバランシング統合用の BIG-IP プール
- **bigip_device**: BIG-IP デバイスの登録と管理
