iRuleスクリプトとデータグループ同期によるレガシーデバイスオーケストレーション。仮想サーバーバインディングとメトリクス収集。

- **カテゴリ**: プラットフォーム
- **複雑度**: 上級
- **パス数**: 29 | **スキーマ数**: 238
- **ティア**: Advanced
- **関連ドメイン**: `marketplace`

## ユースケース

- BigIP F5アプライアンスの管理
- iRuleスクリプトの設定
- データグループの管理
- BigIP CNEの統合

## 主要リソース

- **bigip_pool**: ロードバランシング統合用のBIG-IPプール
- **bigip_device**: BIG-IPデバイスの登録と管理
