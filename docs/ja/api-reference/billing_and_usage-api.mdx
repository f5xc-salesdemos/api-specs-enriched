プラン移行、請求書発行、リソース消費。ネームスペースレベルのクォータ制限と使用状況の追跡。

- **カテゴリ**: プラットフォーム
- **複雑度**: 中程度
- **パス数**: 25 | **スキーマ数**: 101
- **ティア**: Standard
- **関連ドメイン**: `system`, `users`

## ユースケース

- サブスクリプションプランと課金移行の管理
- 支払い方法と請求書の設定
- ネームスペース全体のリソースクォータ使用状況の追跡
- 使用制限とキャパシティの監視

## 主要リソース

- **subscription**: サービスエンタイトルメントのサブスクリプション
- **quota**: 使用制限のリソースクォータ
- **usage_report**: 消費追跡の使用状況レポート
