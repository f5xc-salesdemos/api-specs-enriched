プランの移行、請求書発行、およびリソース消費。ネームスペースレベルのクォータ制限と使用量の追跡。

- **カテゴリ**: プラットフォーム
- **複雑度**: 中程度
- **パス数**: 25 | **スキーマ数**: 101
- **ティア**: Standard
- **関連ドメイン**: `system`, `users`

## ユースケース

- サブスクリプションプランと請求の移行を管理する
- 支払い方法と請求書を構成する
- ネームスペース全体のリソースクォータ使用量を追跡する
- 使用量の制限と容量を監視する

## 主要リソース

- **subscription**: サービスエンタイトルメントのサブスクリプション
- **quota**: 使用量制限のためのリソースクォータ
- **usage_report**: 消費量追跡のための使用量レポート
