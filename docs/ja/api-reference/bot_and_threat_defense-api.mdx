行動分析とシグネチャマッチングによる脅威分類。悪意のあるトラフィックパターンの自動ブロック。

- **カテゴリ**: セキュリティ
- **複雑度**: 中程度
- **パス数**: 19 | **スキーマ数**: 74
- **ティア**: アドバンスド
- **関連ドメイン**: `bot_defense`, `shape`, `virtual`

## ユースケース

- ネームスペースごとのボット防御インスタンスの設定
- 分類のための TPM 脅威カテゴリの管理
- 自動防御システム用 API キーのプロビジョニング
- 脅威インテリジェンスサービスの統合

## 主要リソース

- **bot_defense_instance**: デプロイ用ボット防御インスタンス
- **threat_category**: 分類用脅威カテゴリ
