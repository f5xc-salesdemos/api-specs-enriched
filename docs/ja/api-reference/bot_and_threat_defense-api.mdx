動作分析とシグネチャマッチングによる脅威分類。悪意のあるトラフィックパターンに対する自動ブロッキング。

- **カテゴリ**: セキュリティ
- **複雑度**: 中程度
- **パス数**: 19 | **スキーマ数**: 74
- **ティア**: Advanced
- **関連ドメイン**: `bot_defense`, `shape`, `virtual`

## ユースケース

- ネームスペースごとにボット防御インスタンスを設定する
- 分類のための TPM 脅威カテゴリを管理する
- 自動化された防御システム向けに API キーをプロビジョニングする
- 脅威インテリジェンスサービスを統合する

## 主要リソース

- **bot_defense_instance**: デプロイメント用ボット防御インスタンス
- **threat_category**: 分類用脅威カテゴリ
