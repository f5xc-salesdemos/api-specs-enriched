行動分析とシグネチャマッチングによる脅威分類。悪意のあるトラフィックパターンに対する自動ブロック。

- **カテゴリ**: セキュリティ
- **複雑度**: 中程度
- **パス数**: 19 | **スキーマ数**: 74
- **ティア**: Advanced
- **関連ドメイン**: `bot_defense`, `shape`, `virtual`

## ユースケース

- ネームスペースごとの Bot 防御インスタンスの設定
- 分類のための TPM 脅威カテゴリの管理
- 自動防御システム向け API キーのプロビジョニング
- 脅威インテリジェンスサービスの統合

## 主要リソース

- **bot_defense_instance**: デプロイ用 Bot 防御インスタンス
- **threat_category**: 分類用脅威カテゴリ
