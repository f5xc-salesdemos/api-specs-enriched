行動分析とシグネチャマッチングによる脅威分類。悪意のあるトラフィックパターンに対する自動ブロック機能。

- **カテゴリ**: セキュリティ
- **複雑度**: 中程度
- **パス数**: 19 | **スキーマ数**: 74
- **ティア**: 上級
- **関連ドメイン**: `bot_defense`, `shape`, `virtual`

## ユースケース

- ネームスペースごとの Bot 防御インスタンスを設定する
- 分類のための TPM 脅威カテゴリを管理する
- 自動防御システム用の API キーをプロビジョニングする
- 脅威インテリジェンスサービスを統合する

## 主要リソース

- **bot_defense_instance**: デプロイメント用の Bot 防御インスタンス
- **threat_category**: 分類のための脅威カテゴリ
