行動分析とシグネチャマッチングによる脅威の分類。悪意のあるトラフィックパターンに対する自動ブロッキング。

- **カテゴリ**: セキュリティ
- **複雑度**: 中程度
- **パス数**: 19 | **スキーマ数**: 74
- **ティア**: Advanced
- **関連ドメイン**: `bot_defense`, `shape`, `virtual`

## ユースケース

- 名前空間ごとにBot防御インスタンスを設定する
- 分類のためにTPM脅威カテゴリを管理する
- 自動防御システム向けにAPIキーをプロビジョニングする
- 脅威インテリジェンスサービスを統合する

## 主要リソース

- **bot_defense_instance**: デプロイ用のBot防御インスタンス
- **threat_category**: 分類のための脅威カテゴリ
