キャッシュルールとパージ操作によるグローバル配信。パフォーマンス監視と分析。

- **カテゴリ**: ネットワーキング
- **複雑度**: 上級
- **パス**: 29 | **スキーマ**: 592
- **ティア**: Advanced
- **関連ドメイン**: `virtual`

## ユースケース

- CDNロードバランシングの設定
- コンテンツ配信ネットワークサービスの管理
- キャッシングポリシーの設定
- データ配信および配布の管理

## 主要リソース

- **cdn_loadbalancer**: エッジキャッシング用コンテンツ配信ネットワークロードバランサー
- **cdn_origin_pool**: CDNコンテンツソーシング用オリジンサーバープール
