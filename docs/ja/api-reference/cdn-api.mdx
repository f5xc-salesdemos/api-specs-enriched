キャッシュルールとパージ操作によるグローバル配信。パフォーマンスモニタリングと分析。

- **カテゴリ**: ネットワーク
- **複雑度**: 高度
- **パス数**: 29 | **スキーマ数**: 592
- **ティア**: 高度
- **関連ドメイン**: `virtual`

## ユースケース

- CDN ロードバランシングの設定
- コンテンツ配信ネットワークサービスの管理
- キャッシュポリシーの設定
- データ配信と流通の管理

## 主要リソース

- **cdn_loadbalancer**: エッジキャッシュ用コンテンツ配信ネットワークロードバランサー
- **cdn_origin_pool**: CDN コンテンツソーシング用オリジンサーバープール
