イメージのダウンロードとアップグレード前の検証を含むトークンベースのプロビジョニング。フリートのグループ化により、分散したロケーション全体での一括操作が可能になります。

- **カテゴリ**: インフラストラクチャ
- **複雑度**: 中程度
- **パス数**: 28 | **スキーマ数**: 231
- **ティア**: スタンダード
- **関連ドメイン**: `customer_edge`, `sites`

## ユースケース

- Customer Edge サイトのライフサイクルを管理する
- ネットワークインターフェースとフリート設定を構成する
- サイト登録とトークンワークフローを処理する
- アップグレード前チェックを含むサイトアップグレードを実行する

## 主要リソース

- **site_config**: エッジノード設定のためのサイト構成
- **fleet_config**: マルチサイト管理のためのフリート構成
- **registration_token**: サイトオンボーディングのための登録トークン
