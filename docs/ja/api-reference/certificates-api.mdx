証明書チェーンと信頼済み CA バンドル。PKI 操作のための失効リスト管理とマニフェスト設定。

- **カテゴリ**: セキュリティ
- **複雑度**: 中程度
- **パス数**: 16 | **スキーマ数**: 74
- **ティア**: スタンダード
- **関連ドメイン**: `blindfold`, `system`

## ユースケース

- SSL/TLS 証明書の管理
- 信頼済み CA の設定
- 証明書失効リスト (CRL) の管理
- 証明書マニフェストの設定

## 主要リソース

- **certificate**: セキュアな HTTPS 接続のための TLS 証明書
- **ca_certificate**: トラストチェーン用の認証局証明書
- **certificate_chain**: 完全なトラスト検証のための証明書チェーン
