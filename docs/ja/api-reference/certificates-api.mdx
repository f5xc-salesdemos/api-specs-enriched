証明書チェーンと信頼済みCAバンドル。PKI操作のための失効リスト管理とマニフェスト設定。

- **カテゴリ**: セキュリティ
- **複雑度**: 中程度
- **パス数**: 16 | **スキーマ数**: 74
- **ティア**: Standard
- **関連ドメイン**: `blindfold`, `system`

## ユースケース

- SSL/TLS証明書の管理
- 信頼済みCAの設定
- 証明書失効リスト（CRL）の管理
- 証明書マニフェストの設定

## 主要リソース

- **certificate**: セキュアなHTTPS接続用のTLS証明書
- **ca_certificate**: トラストチェーン用の認証局証明書
- **certificate_chain**: 完全な信頼検証用の証明書チェーン
