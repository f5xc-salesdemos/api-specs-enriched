ゲートウェイピアリングとネットワークパス構成によるマルチクラウドプロバイダー接続。認証情報ボールト統合とサブネット列挙。

- **カテゴリ**: インフラストラクチャ
- **複雑さ**: 中程度
- **パス**: 31 | **スキーマ**: 208
- **ティア**: Standard
- **関連ドメイン**: `sites`, `customer_edge`

## ユースケース

- クラウドプロバイダー（AWS、Azure、GCP）への接続
- クラウド認証情報と認証の管理
- クラウド接続性とエラスティックプロビジョニングの構成
- クラウドリージョンのリンクと管理

## 主要リソース

- **aws_vpc_site**: エッジノードプロビジョニングを伴う AWS VPC サイトデプロイメント
- **azure_vnet_site**: エッジノードプロビジョニングを伴う Azure VNet サイトデプロイメント
- **gcp_vpc_site**: エッジノードプロビジョニングを伴う Google Cloud VPC サイトデプロイメント
- **cloud_credentials**: サイトデプロイメント用のクラウドプロバイダー認証資格情報
