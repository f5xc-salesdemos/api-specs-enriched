ゲートウェイピアリングおよびネットワークパス設定を含むマルチクラウドプロバイダー接続。認証情報ボールト統合およびサブネット列挙。

- **カテゴリ**: インフラストラクチャ
- **複雑度**: 中程度
- **パス数**: 31 | **スキーマ数**: 208
- **ティア**: スタンダード
- **関連ドメイン**: `sites`、`customer_edge`

## ユースケース

- クラウドプロバイダー（AWS、Azure、GCP）への接続
- クラウド認証情報および認証の管理
- クラウド接続とエラスティックプロビジョニングの設定
- クラウドリージョンのリンクおよび管理

## 主要リソース

- **aws_vpc_site**: エッジノードプロビジョニングを含む AWS VPC サイトデプロイ
- **azure_vnet_site**: エッジノードプロビジョニングを含む Azure VNet サイトデプロイ
- **gcp_vpc_site**: エッジノードプロビジョニングを含む Google Cloud VPC サイトデプロイ
- **cloud_credentials**: サイトデプロイ用クラウドプロバイダー認証情報
