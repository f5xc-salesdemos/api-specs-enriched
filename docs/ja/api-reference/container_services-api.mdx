フルクラスターの複雑さを伴わない Pod オーケストレーション。エッジサイトでの実行、クォータの適用、分散アプリケーション向けの標準化されたコンピュートプロファイル。

- **カテゴリ**: インフラストラクチャ
- **複雑度**: 中程度
- **パス数**: 13 | **スキーマ数**: 159
- **ティア**: Advanced
- **関連ドメイン**: `managed_kubernetes`, `sites`, `service_mesh`

## ユースケース

- マルチテナントワークロード用の XCCS（コンテナサービス）名前空間をデプロイする
- 簡素化されたオーケストレーションでコンテナワークロードを管理する
- 分散エッジコンテナデプロイメントを構成する
- フル K8s の複雑さなしにコンテナ化されたアプリケーションを実行する

## 主要リソース

- **virtual_k8s**: コンテナワークロード用の仮想 Kubernetes 名前空間
- **workload**: コンテナワークロードのデプロイメント構成
- **pod_security_policy**: コンテナランタイム制約のための Pod セキュリティポリシー
