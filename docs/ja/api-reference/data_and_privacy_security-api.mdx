カスタム分類ルールを使用した機密データポリシー。LMA リージョン設定および地理ベースのコンプライアンス制御。

- **カテゴリ**: セキュリティ
- **複雑度**: シンプル
- **パス数**: 11 | **スキーマ数**: 65
- **ティア**: 高度
- **関連ドメイン**: `blindfold`, `client_side_defense`

## ユースケース

- 機密データ検出ポリシーの設定
- PII 分類のためのカスタムデータタイプの定義
- LMA リージョン設定の管理
- コンプライアンスのための地理設定の統合

## 主要リソース

- **sensitive_data_policy**: PII 保護のための機密データポリシー
- **data_classification**: コンテンツ分類のためのデータ分類
