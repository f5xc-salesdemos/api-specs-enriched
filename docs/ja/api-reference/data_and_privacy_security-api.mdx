カスタム分類ルールを備えた機密データポリシー。LMAリージョン設定および地理ベースのコンプライアンス制御。

- **カテゴリ**: セキュリティ
- **複雑度**: シンプル
- **パス**: 11 | **スキーマ**: 65
- **ティア**: アドバンスド
- **関連ドメイン**: `blindfold`, `client_side_defense`

## ユースケース

- 機密データ検出ポリシーの設定
- PII分類のためのカスタムデータタイプの定義
- LMAリージョン設定の管理
- コンプライアンスのための地理設定の統合

## 主要リソース

- **sensitive_data_policy**: PII保護のための機密データポリシー
- **data_classification**: コンテンツ分類のためのデータ分類
