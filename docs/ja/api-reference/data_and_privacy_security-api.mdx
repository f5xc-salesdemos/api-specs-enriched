カスタム分類ルールを使用した機密データポリシー。LMAリージョン設定および地理ベースのコンプライアンス制御。

- **カテゴリ**: セキュリティ
- **複雑さ**: シンプル
- **パス**: 11 | **スキーマ**: 65
- **ティア**: アドバンスド
- **関連ドメイン**: `blindfold`, `client_side_defense`

## ユースケース

- 機密データ検出ポリシーの設定
- PII分類のためのカスタムデータ型の定義
- LMAリージョン設定の管理
- コンプライアンスのためのジオ設定の統合

## 主要リソース

- **sensitive_data_policy**: PII保護のための機密データポリシー
- **data_classification**: コンテンツ分類のためのデータ分類
