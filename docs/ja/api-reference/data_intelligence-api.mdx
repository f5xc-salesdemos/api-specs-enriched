データ分析ワークフローのための分類ルール、リソースポリシー、およびインサイト生成。

- **カテゴリ**: オペレーション
- **複雑度**: シンプル
- **パス数**: 15 | **スキーマ数**: 73
- **ティア**: スタンダード
- **関連ドメイン**: `statistics`, `observability`

## ユースケース

- セキュリティおよびトラフィックデータの分析
- ログからのインテリジェントなインサイトの生成
- データ分析ポリシーの設定

## 主要リソース

- **analytics_query**: データ分析のためのアナリティクスクエリ
- **data_export**: レポート用データエクスポート設定
