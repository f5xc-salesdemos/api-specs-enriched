データ分析ワークフローのための分類ルール、リソースポリシー、およびインサイト生成。

- **カテゴリ**: Operations
- **複雑度**: simple
- **パス**: 15 | **スキーマ**: 73
- **ティア**: Standard
- **関連ドメイン**: `statistics`, `observability`

## ユースケース

- セキュリティおよびトラフィックデータの分析
- ログからのインテリジェントなインサイトの生成
- データ分析ポリシーの設定

## 主要リソース

- **analytics_query**: データ分析用の分析クエリ
- **data_export**: レポート用のデータエクスポート設定
