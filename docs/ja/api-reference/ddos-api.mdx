拒否リスト、ファイアウォールルールグループ、およびトンネルベースのセーフガード。ネットワーク境界セキュリティのためのレート制限とパターン分析。

- **カテゴリ**: セキュリティ
- **複雑度**: 高度
- **パス数**: 60 | **スキーマ数**: 246
- **ティア**: Advanced
- **関連ドメイン**: `network_security`, `virtual`

## ユースケース

- DDoS 対策ポリシーの設定
- DDoS 脅威の監視と分析
- インフラ保護の設定

## 主要リソース

- **ddos_protection**: トラフィックスクラビングの DDoS 対策設定
- **ddos_mitigation_rule**: 攻撃対応のための DDoS 緩和ルール
