拒否リスト、ファイアウォールルールグループ、およびトンネルベースのセーフガード。ネットワーク境界セキュリティのためのレート制限とパターン分析。

- **カテゴリ**: セキュリティ
- **複雑度**: 上級
- **パス数**: 60 | **スキーマ数**: 246
- **ティア**: アドバンスド
- **関連ドメイン**: `network_security`, `virtual`

## ユースケース

- DDoS防御ポリシーの設定
- DDoS脅威の監視と分析
- インフラストラクチャ保護の設定

## 主要リソース

- **ddos_protection**: トラフィックスクラビングのためのDDoS防御設定
- **ddos_mitigation_rule**: 攻撃対応のためのDDoS緩和ルール
