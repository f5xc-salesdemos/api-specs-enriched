ゾーン転送とヘルスチェックによる名前解決。レコードタイプと委任のサポート。

- **カテゴリ**: ネットワーキング
- **複雑度**: 上級
- **パス数**: 49 | **スキーマ数**: 285
- **ティア**: Standard
- **関連ドメイン**: `virtual`, `network`

## ユースケース

- DNSロードバランシングの設定
- DNSゾーンとドメインの管理
- DNSコンプライアンスポリシーの設定
- リソースレコードセット（RRSets）の管理

## 主要リソース

- **dns_zone**: レコード管理機能を備えた権威DNSゾーン
- **dns_domain**: DNSドメイン委任と設定
- **dns_load_balancer**: 地理的ルーティングとフェイルオーバーによるDNSベースのトラフィック分散
