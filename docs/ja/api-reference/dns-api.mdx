ゾーン転送とヘルスチェックによる名前解決。レコードタイプとデリゲーションのサポート。

- **カテゴリ**: ネットワーク
- **複雑度**: 上級
- **パス数**: 49 | **スキーマ数**: 285
- **ティア**: Standard
- **関連ドメイン**: `virtual`, `network`

## ユースケース

- DNS ロードバランシングの設定
- DNS ゾーンとドメインの管理
- DNS コンプライアンスポリシーの設定
- リソースレコードセット (RRSets) の管理

## 主要リソース

- **dns_zone**: レコード管理機能を持つ権威 DNS ゾーン
- **dns_domain**: DNS ドメインのデリゲーションと設定
- **dns_load_balancer**: 地理的ルーティングとフェイルオーバーを含む DNS ベースのトラフィック分散
