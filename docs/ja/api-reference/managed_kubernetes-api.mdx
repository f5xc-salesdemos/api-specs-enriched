Kubernetes ロールバインディングとアドミッションポリシー。EKS、AKS、GKE ワークロード向けのレジストリ統合。

- **カテゴリ**: インフラストラクチャ
- **複雑度**: 中程度
- **パス数**: 20 | **スキーマ数**: 95
- **ティア**: アドバンスド
- **関連ドメイン**: `container_services`, `sites`, `service_mesh`

## ユースケース

- XCKS（マネージド Kubernetes）クラスターの RBAC とセキュリティを管理
- Pod セキュリティポリシーとアドミッションコントローラーを構成
- エンタープライズデプロイメント向けのコンテナレジストリを管理
- 外部 Kubernetes クラスター（EKS、AKS、GKE）と統合

## 主要リソース

- **mk8s_cluster**: マネージド Kubernetes クラスター構成
- **k8s_cluster_role**: Kubernetes クラスター RBAC ロール構成
- **container_registry**: ワークロードイメージ用のコンテナイメージレジストリ
