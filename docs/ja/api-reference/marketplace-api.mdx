DPD タイマーを使用したサードパーティの GRE および IPSec トンネルプロビジョニング。タイル配置制御によるネームスペース間の共有リソース割り当て。

- **カテゴリ**: プラットフォーム
- **複雑度**: 中程度
- **パス数**: 36 | **スキーマ数**: 183
- **ティア**: アドバンスド
- **関連ドメイン**: `bigip`, `admin`

## ユースケース

- サードパーティの統合およびアドオンへのアクセス
- マーケットプレイス拡張機能の管理
- Terraform および外部統合の設定
- TPM ポリシーの管理

## 主要リソース

- **marketplace_item**: サービスカタログのマーケットプレイスアイテム
- **subscription**: サービスエンタイトルメントのサブスクリプション
