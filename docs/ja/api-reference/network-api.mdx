ASN 管理と自律システム関係を備えたボーダーゲートウェイプロトコル。暗号化チャネルを通じてデータセンターを接続するサイト間 VPN。

- **カテゴリ**: ネットワーク
- **複雑度**: 高度
- **パス数**: 81 | **スキーマ数**: 449
- **ティア**: Advanced
- **関連ドメイン**: `virtual`, `network_security`, `dns`

## ユースケース

- BGP ルーティングおよび ASN 管理の設定
- IPsec トンネルおよび IKE フェーズの管理
- ネットワークコネクターおよびルートの設定
- SRv6 およびサブネッティングの管理
- セグメント接続とポリシーの定義
- IP プレフィックスセットの設定

## 主要リソース

- **virtual_network**: サイト接続とセグメンテーションのための仮想ネットワーク
- **network_connector**: サイト間またはクラウド接続のためのネットワークコネクター
- **site_mesh_group**: 複数サイト間のメッシュ接続設定
