ASN管理と自律システム関係を持つボーダーゲートウェイプロトコル。暗号化されたチャネルを通じてデータセンターを接続するサイト間VPN。

- **カテゴリ**: ネットワーキング
- **複雑さ**: 上級
- **パス数**: 81 | **スキーマ数**: 449
- **ティア**: Advanced
- **関連ドメイン**: `virtual`, `network_security`, `dns`

## ユースケース

- BGPルーティングとASN管理の設定
- IPsecトンネルとIKEフェーズの管理
- ネットワークコネクタとルートの設定
- SRv6とサブネッティングの管理
- セグメント接続とポリシーの定義
- IPプレフィックスセットの設定

## 主要リソース

- **virtual_network**: サイト接続とセグメンテーションのための仮想ネットワーク
- **network_connector**: サイト間またはクラウド接続のためのネットワークコネクタ
- **site_mesh_group**: 複数サイト間のメッシュ接続設定
