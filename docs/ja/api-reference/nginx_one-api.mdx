インスタンス検出、WAF 統合、およびサービスメッシュ接続。サブスクリプションのライフサイクルと設定の同期。

- **カテゴリー**: プラットフォーム
- **複雑さ**: シンプル
- **パス数**: 13 | **スキーマ数**: 59
- **ティア**: 上級
- **関連ドメイン**: `marketplace`

## ユースケース

- NGINX One プラットフォームの統合を管理する
- NGINX Plus インスタンスを設定する
- NGINX 設定管理を統合する

## 主要リソース

- **nginx_config**: プロキシ設定のための NGINX 設定
- **nginx_upstream**: バックエンドサーバープール用の NGINX アップストリーム
