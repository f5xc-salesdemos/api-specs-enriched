インスタンスディスカバリー、WAF統合、およびサービスメッシュ接続。サブスクリプションライフサイクルと構成の同期。

- **カテゴリ**: プラットフォーム
- **複雑度**: シンプル
- **パス数**: 13 | **スキーマ数**: 59
- **ティア**: アドバンスド
- **関連ドメイン**: `marketplace`

## ユースケース

- NGINX Oneプラットフォーム統合の管理
- NGINX Plusインスタンスの構成
- NGINX構成管理の統合

## 主要リソース

- **nginx_config**: プロキシ設定用のNGINX構成
- **nginx_upstream**: バックエンドサーバープール用のNGINXアップストリーム
