モバイルアプリインテグレーター向けのバージョン管理されたライブラリ配布。署名付きURLにより、iOSおよびAndroid向けのOS固有ビルドの安全なダウンロードが可能です。

- **カテゴリ**: Platform
- **複雑度**: simple
- **パス数**: 9 | **スキーマ数**: 17
- **ティア**: Advanced
- **関連ドメイン**: `marketplace`

## ユースケース

- オブジェクトストレージサービスの管理
- 保存オブジェクトとバケットの構成
- ストレージポリシーの管理

## 主要リソース

- **object_store**: Blobストレージ管理用のオブジェクトストア
- **bucket**: オブジェクトストレージ用のストレージバケット
