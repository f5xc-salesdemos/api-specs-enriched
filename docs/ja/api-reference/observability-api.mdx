レイテンシー測定を伴うHTTP可用性プローブ。証明書有効期限アラートおよびインフラストラクチャの健全性を監視するグローバルステータスダッシュボード。

- **カテゴリ**: オペレーション
- **複雑度**: 中程度
- **パス数**: 56 | **スキーマ数**: 196
- **ティア**: Standard
- **関連ドメイン**: `statistics`, `support`

## ユースケース

- 合成モニタリングの設定
- モニタリングおよびテストポリシーの定義
- オブザーバビリティダッシュボードの管理

## 主要リソース

- **log_receiver**: 集中ログ収集のためのログレシーバー
- **metrics_receiver**: 集中メトリクス収集のためのメトリクスレシーバー
- **alert_policy**: モニタリングおよび通知のためのアラートポリシー
