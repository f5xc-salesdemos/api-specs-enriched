時間ベースのクォータ適用。時間、分、秒単位で設定可能なウィンドウに対応。トラフィックシェーピングのためのプロトコル固有の制御。

- **カテゴリ**: ネットワーキング
- **複雑度**: シンプル
- **パス数**: 12 | **スキーマ数**: 70
- **ティア**: アドバンスド
- **関連ドメイン**: `virtual`, `network_security`

## ユースケース

- レートリミッターポリシーの設定
- ポリサー設定の管理
- トラフィックフローとキューイングの制御

## 主要リソース

- **rate_limiter**: トラフィックスロットリングおよび保護のためのレートリミッター
- **rate_limiter_policy**: 詳細なスロットリングルールを持つレートリミッターポリシー
- **rate_limit_threshold**: トラフィック制御のためのレート制限しきい値設定
