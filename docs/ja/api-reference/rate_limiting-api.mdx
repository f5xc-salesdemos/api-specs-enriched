時間ベースのクォータ適用。時間、分、または秒単位で設定可能なウィンドウを使用します。トラフィックシェーピングのためのプロトコル固有の制御。

- **カテゴリ**: ネットワーク
- **複雑度**: シンプル
- **パス数**: 12 | **スキーマ数**: 70
- **ティア**: アドバンスド
- **関連ドメイン**: `virtual`、`network_security`

## ユースケース

- レートリミッターポリシーの設定
- ポリサー設定の管理
- トラフィックフローとキューイングの制御

## 主要リソース

- **rate_limiter**: トラフィックのスロットリングと保護のためのレートリミッター
- **rate_limiter_policy**: 詳細なスロットリングルールを含むレートリミッターポリシー
- **rate_limit_threshold**: トラフィック制御のためのレート制限しきい値設定
