脅威レベル分類による悪意のあるユーザーの軽減。不審な動作パターンに対する自動応答アクション。

- **カテゴリー**: セキュリティ
- **複雑度**: シンプル
- **パス数**: 4 | **スキーマ数**: 32
- **ティア**: アドバンスト
- **関連ドメイン**: `bot_defense`, `virtual`, `network_security`

## ユースケース

- 自動脅威軽減ポリシーの設定
- 悪意のあるユーザー検出ルールの定義
- インシデントレスポンスワークフローの管理
- 脅威へのブロッキングまたはレート制限の適用

## 主要リソース

- **mitigation_policy**: インシデントレスポンス用の軽減ポリシー
- **malicious_user_rule**: 脅威軽減用の悪意のあるユーザールール

## エンドポイント

| メソッド | パス | 説明 |
|--------|------|-------------|
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations` | 悪意のあるユーザーの軽減を作成する。 |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations/&#123;metadata.name&#125;` | 悪意のあるユーザーの軽減を置き換える。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations` | 悪意のあるユーザーの軽減を一覧表示する。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | 悪意のあるユーザーの軽減を取得する。 |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | 悪意のあるユーザーの軽減を削除する。 |
