ディスカバリーと学習済みスキーマ分析によるアプリケーションタイプ定義。インテリジェントなリクエスト処理のためのトラフィックパターン推論。

- **カテゴリ**: インフラストラクチャ
- **複雑度**: 上級
- **パス**: 38 | **スキーマ**: 247
- **ティア**: アドバンスド
- **関連ドメイン**: `managed_kubernetes`, `container_services`, `virtual`

## ユースケース

- サービスメッシュの接続を構成する
- エンドポイントのディスカバリーとルーティングを管理する
- NFV サービスを構成する
- アプリケーション設定とタイプを定義する

## 主要リソース

- **endpoint**: トラフィックルーティングのためのサービスメッシュエンドポイント
- **origin_pool**: ロードバランサーのトラフィック分散のためのバックエンドサーバーグループ
- **service_discovery**: 動的エンドポイントのためのサービスディスカバリー設定
