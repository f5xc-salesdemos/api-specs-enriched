ディスカバリーと学習済みスキーマ分析を備えたアプリケーションタイプ定義。インテリジェントなリクエスト処理のためのトラフィックパターン推論。

- **カテゴリ**: インフラストラクチャ
- **複雑度**: 上級
- **パス数**: 38 | **スキーマ数**: 247
- **ティア**: 上級
- **関連ドメイン**: `managed_kubernetes`, `container_services`, `virtual`

## ユースケース

- サービスメッシュの接続性を構成する
- エンドポイントのディスカバリーとルーティングを管理する
- NFVサービスを構成する
- アプリケーション設定とタイプを定義する

## 主要リソース

- **endpoint**: トラフィックルーティング用のサービスメッシュエンドポイント
- **origin_pool**: ロードバランサーのトラフィック分散用バックエンドサーバーグループ
- **service_discovery**: 動的エンドポイント用のサービスディスカバリー構成
