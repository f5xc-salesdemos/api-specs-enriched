行動分析とデバイスフィンガープリンティングによる脅威認識。アプリケーションシールドのためのモバイルSDK統合。

- **カテゴリ**: セキュリティ
- **複雑度**: 上級
- **パス数**: 271 | **スキーマ数**: 813
- **ティア**: Advanced
- **関連ドメイン**: `bot_defense`, `virtual`

## ユースケース

- Shape Securityポリシーの設定
- ボットおよび脅威防止の管理
- SafeAPポリシーの設定
- 脅威認識の有効化

## 主要リソース

- **shape_app_firewall**: 高度な保護のためのShapeアプリケーションファイアウォール
- **shape_recognizer**: トラフィックパターン分析のためのShape認識エンジン
