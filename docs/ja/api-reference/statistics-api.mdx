レシーバー統合を含むアラートポリシー。ログ集約、トポロジービュー、およびサイトヘルストラッキング。

- **カテゴリ**: オペレーション
- **複雑度**: 上級
- **パス数**: 77 | **スキーマ数**: 575
- **ティア**: スタンダード
- **関連ドメイン**: `observability`, `support`

## ユースケース

- フロー統計とアナリティクスへのアクセス
- アラートとアラートポリシーの管理
- ログとログレシーバーの表示
- レポートとグラフの生成
- トポロジーとサービスディスカバリーの追跡
- サイトでのステータス監視

## 主要リソース

- **dashboard**: メトリクス可視化のためのダッシュボード
- **saved_query**: メトリクスおよびログ分析のための保存済みクエリ
