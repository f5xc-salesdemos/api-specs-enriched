レシーバー統合を備えたアラートポリシー。ログ集約、トポロジービュー、およびサイト健全性の追跡。

- **カテゴリ**: オペレーション
- **複雑さ**: 上級
- **パス数**: 77 | **スキーマ数**: 575
- **ティア**: スタンダード
- **関連ドメイン**: `observability`, `support`

## ユースケース

- フロー統計および分析へのアクセス
- アラートおよびアラートポリシーの管理
- ログおよびログレシーバーの表示
- レポートおよびグラフの生成
- トポロジーおよびサービスディスカバリーの追跡
- サイトにおけるステータスの監視

## 主要リソース

- **dashboard**: メトリクス可視化のためのダッシュボード
- **saved_query**: メトリクスおよびログ分析のための保存済みクエリ
