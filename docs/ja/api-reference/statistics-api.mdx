レシーバー統合を含むアラートポリシー。ログ集約、トポロジービュー、およびサイトヘルストラッキング。

- **カテゴリ**: オペレーション
- **複雑さ**: 上級
- **パス**: 77 | **スキーマ**: 575
- **ティア**: Standard
- **関連ドメイン**: `observability`, `support`

## ユースケース

- フロー統計と分析へのアクセス
- アラートとアラートポリシーの管理
- ログとログレシーバーの表示
- レポートとグラフの生成
- トポロジーとサービスディスカバリーの追跡
- サイトのステータス監視

## 主要リソース

- **dashboard**: メトリクス可視化のためのダッシュボード
- **saved_query**: メトリクスとログ分析のための保存済みクエリ
