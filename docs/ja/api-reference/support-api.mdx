コメント、重大度の変更、および解決状況の追跡を含むイシューのライフサイクル管理。接続分析のためのパケットキャプチャ。

- **カテゴリ**: オペレーション
- **複雑度**: 中程度
- **パス**: 53 | **スキーマ**: 167
- **ティア**: スタンダード
- **関連ドメイン**: `statistics`, `observability`

## ユースケース

- サポートチケットの送信と管理
- カスタマーサポートリクエストの追跡
- 運用サポートドキュメントへのアクセス

## 主要リソース

- **support_case**: イシューの追跡と解決のためのサポートケース
- **alert**: 運用通知のためのシステムアラート
- **audit_log**: コンプライアンスおよびセキュリティ追跡のための監査ログ
