コメント、重大度変更、および解決追跡を含むイシューライフサイクル。接続分析のためのパケットキャプチャ。

- **カテゴリ**: オペレーション
- **複雑度**: 中程度
- **パス数**: 53 | **スキーマ数**: 167
- **ティア**: Standard
- **関連ドメイン**: `statistics`, `observability`

## ユースケース

- サポートチケットの送信と管理
- カスタマーサポートリクエストの追跡
- 運用サポートドキュメントへのアクセス

## 主要リソース

- **support_case**: イシュー追跡と解決のためのサポートケース
- **alert**: 運用通知のためのシステムアラート
- **audit_log**: コンプライアンスとセキュリティ追跡のための監査ログ
