オブザーバビリティワークフローのための収集エンドポイント、メトリクスストレージ、およびインサイト生成。

- **カテゴリ**: オペレーション
- **複雑度**: 中程度
- **パス数**: 27 | **スキーマ数**: 181
- **ティア**: スタンダード
- **関連ドメイン**: `observability`, `statistics`

## ユースケース

- テレメトリデータの収集と分析
- メトリクスからの実用的なインサイトの生成
- テレメトリ収集ポリシーの設定

## 主要リソース

- **telemetry_receiver**: データ収集用テレメトリレシーバー
- **insight_query**: アナリティクス用インサイトクエリ
