アカウントビュー設定と管理者アラートチャネル。ワンタイムパスワードのリセット、プロビジョニングフロー、およびアクティブ接続モニタリング。

- **カテゴリ**: プラットフォーム
- **複雑度**: 上級
- **パス数**: 183 | **スキーマ数**: 511
- **ティア**: スタンダード
- **関連ドメイン**: `users`, `authentication`, `system`

## ユースケース

- ユーザープロファイルと通知設定の管理
- セッション制御と OTP 設定の構成
- アイデンティティ管理操作の処理
- 初期ユーザーアクセス要求の処理

## 主要リソース

- **user_profile**: 個人設定を含むユーザープロファイル
- **session**: 認証状態のユーザーセッション
- **otp_policy**: 多要素認証の OTP ポリシー
