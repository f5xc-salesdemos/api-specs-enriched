アカウントビュー設定および管理者アラートチャネル。ワンタイムパスワードのリセット、プロビジョニングフロー、およびアクティブな接続の監視。

- **カテゴリ**: プラットフォーム
- **複雑度**: 上級
- **パス数**: 183 | **スキーマ数**: 511
- **ティア**: スタンダード
- **関連ドメイン**: `users`, `authentication`, `system`

## ユースケース

- ユーザープロファイルと通知設定の管理
- セッション制御とOTP設定の構成
- ID管理操作の処理
- 初回ユーザーアクセスリクエストの処理

## 主要リソース

- **user_profile**: 個人設定を含むユーザープロファイル
- **session**: 認証状態のためのユーザーセッション
- **otp_policy**: 多要素認証のためのOTPポリシー
