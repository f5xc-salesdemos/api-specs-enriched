セキュリティオペレーションのための検知ポリシー、攻撃追跡、および自動緩和ルール生成。

- **カテゴリ**: セキュリティ
- **複雑度**: 中程度
- **パス数**: 1 | **スキーマ数**: 161
- **ティア**: スタンダード
- **関連ドメイン**: `bot_defense`, `ddos`

## ユースケース

- 脅威キャンペーンの追跡と分析
- アクティブな脅威と攻撃パターンの監視
- 脅威インテリジェンス統合の設定

## 主要リソース

- **threat_campaign_policy**: 脅威キャンペーンの検知および緩和ポリシー

## エンドポイント

| メソッド | パス | 説明 |
|--------|------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | ID による脅威キャンペーンの取得。 |
