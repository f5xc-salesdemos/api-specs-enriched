自動有効期限付きのサイト登録資格情報。タクソノミーキーは許可されるカテゴリ分類を定義し、自動派生タグは動的に適用されます。

- **カテゴリ**: プラットフォーム
- **複雑度**: シンプル
- **パス**: 13 | **スキーマ**: 50
- **ティア**: スタンダード
- **関連ドメイン**: `system`, `admin`

## ユースケース

- ユーザーアカウントとトークンの管理
- ユーザー識別の設定
- ユーザー設定とプリファレンスの管理
- 暗黙的ラベルと既知ラベルの設定

## 主要リソース

- **user**: プラットフォームアクセス用のユーザーアカウント
- **user_role**: 権限管理用のユーザーロール
- **namespace_role**: きめ細かな権限制御のためのネームスペーススコープロール
