分散ノード管理のためのライフサイクル制御、フリート構成、およびデプロイメントポリシー。

- **カテゴリ**: Platform
- **複雑度**: simple
- **パス数**: 1 | **スキーマ数**: 2
- **ティア**: Standard
- **関連ドメイン**: `sites`、`system`

## ユースケース

- Virtual Private Mesh（VPM）構成の管理
- ノードのライフサイクルと管理の設定
- VPMおよびノードのステータス監視

## 主要リソース

- **node_config**: エッジデバイス設定のためのノード構成
- **vpm_config**: 仮想プラットフォーム管理のためのVPM構成

## エンドポイント

| メソッド | パス | 説明 |
|--------|------|-------------|
| GET | `/api/data/namespaces/system/upgrade_status` | アップグレードステータス。 |
