分散ノード管理のためのライフサイクル制御、フリート設定、およびデプロイポリシー。

- **カテゴリ**: プラットフォーム
- **複雑度**: シンプル
- **パス数**: 1 | **スキーマ数**: 2
- **ティア**: スタンダード
- **関連ドメイン**: `sites`、`system`

## ユースケース

- Virtual Private Mesh (VPM) 設定の管理
- ノードのライフサイクルと管理の設定
- VPM およびノードステータスの監視

## 主要リソース

- **node_config**: エッジデバイス設定のためのノード設定
- **vpm_config**: 仮想プラットフォーム管理のための VPM 設定

## エンドポイント

| メソッド | パス | 説明 |
|--------|------|-------------|
| GET | `/api/data/namespaces/system/upgrade_status` | アップグレードステータス。 |
