버전 관리가 포함된 네임스페이스 범위의 시각적 요소. 포털 화면을 위한 사용자 정의 위젯 배포 및 카탈로그 관리.

- **카테고리**: 플랫폼
- **복잡성**: 단순
- **경로**: 2 | **스키마**: 14
- **티어**: 표준
- **관련 도메인**: `admin`, `system`

## 사용 사례

- 관리 콘솔을 위한 정적 UI 구성 요소 관리
- 네임스페이스 내 UI 자산 배포 및 검색
- 콘솔 인터페이스 요소 구성
- 사용자 정의 UI 구성 요소 메타데이터 관리

## 기본 리소스

- **ui_component**: 콘솔 사용자 정의를 위한 UI 구성 요소
- **static_asset**: UI 리소스를 위한 정적 자산

## 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | UI 정적 구성 요소 목록 조회. |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | UI 정적 구성 요소 GET. |
