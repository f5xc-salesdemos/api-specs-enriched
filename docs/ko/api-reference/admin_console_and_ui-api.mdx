버전 관리를 지원하는 네임스페이스 범위의 시각적 요소입니다. 포털 화면을 위한 커스텀 위젯 배포 및 카탈로그 관리를 제공합니다.

- **카테고리**: Platform
- **복잡도**: simple
- **경로**: 2 | **스키마**: 14
- **티어**: Standard
- **관련 도메인**: `admin`, `system`

## 사용 사례

- 관리 콘솔용 정적 UI 컴포넌트 관리
- 네임스페이스 내에서 UI 자산 배포 및 조회
- 콘솔 인터페이스 요소 구성
- 커스텀 UI 컴포넌트 메타데이터 관리

## 주요 리소스

- **ui_component**: 콘솔 커스터마이징을 위한 UI 컴포넌트
- **static_asset**: UI 리소스를 위한 정적 자산

## 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | UI 정적 컴포넌트 목록 조회. |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | UI 정적 컴포넌트 조회. |
