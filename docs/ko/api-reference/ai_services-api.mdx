품질 신호 및 이상 징후 모니터링을 통한 자연어 처리. 데이터 스트림 구독을 위한 토큰 인증.

- **카테고리**: AI
- **복잡도**: 단순
- **경로**: 11 | **스키마**: 66
- **티어**: 고급


## 사용 사례

- AI 기반 기능 접근
- AI 어시스턴트 정책 구성
- 플로우 이상 징후 탐지 활성화
- AI 데이터 수집 관리

## 주요 리소스

- **ai_policy**: 생성형 AI 트래픽 제어를 위한 AI 정책
- **ai_gateway**: LLM API 관리를 위한 AI 게이트웨이

## 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | AI 어시스턴트 쿼리 평가. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | AI 어시스턴트 쿼리 평가 피드백. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | AI 어시스턴트 쿼리. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | AI 어시스턴트 쿼리 피드백. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | 기능 활성화. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | 토큰 갱신. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | BFDP 파이프라인 구독. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | BFDP 파이프라인 구독. |
