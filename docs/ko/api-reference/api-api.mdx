자동 분류 및 보안 프로파일링을 통한 리소스 카탈로그 작성. 조직 계층 구조가 기능 또는 보호 정책별로 액세스를 구분합니다.

- **카테고리**: 보안
- **복잡도**: 고급
- **경로**: 45 | **스키마**: 279
- **티어**: Advanced
- **관련 도메인**: `virtual`, `network_security`

## 사용 사례

- API 검색 및 카탈로그 작성
- API 보안 및 동작 테스트
- API 자격 증명 관리
- API 그룹 및 테스트 정책 정의

## 주요 리소스

- **api_definition**: 검색 및 보호를 위한 API 스키마 정의
- **api_endpoint**: 개별 API 엔드포인트 구성 및 보호
- **api_rate_limit**: 트래픽 제어를 위한 API 속도 제한 구성
