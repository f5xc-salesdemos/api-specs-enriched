공급자 통합, 액세스 정책 및 자격 증명 수명 주기 관리를 포함한 ID 관리.

- **카테고리**: 플랫폼
- **복잡도**: 단순
- **경로**: 14 | **스키마**: 38
- **계층**: 표준
- **관련 도메인**: `system`, `users`

## 사용 사례

- 인증 메커니즘 구성
- OIDC 및 OAuth 공급자 관리
- SCIM 사용자 프로비저닝 구성
- API 자격 증명 및 액세스 관리
- 계정 가입 정책 구성

## 기본 리소스

- **authentication_policy**: 사용자 및 API 액세스를 위한 인증 정책
- **token**: 프로그래밍 방식 액세스를 위한 API 토큰
- **api_credential**: 서비스 인증을 위한 API 자격 증명
