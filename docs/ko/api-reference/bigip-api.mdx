iRule 스크립트 및 데이터 그룹 동기화를 통한 레거시 장치 오케스트레이션. 가상 서버 바인딩 및 메트릭 수집.

- **카테고리**: Platform
- **복잡도**: advanced
- **경로**: 29 | **스키마**: 238
- **티어**: Advanced
- **관련 도메인**: `marketplace`

## 사용 사례

- BigIP F5 어플라이언스 관리
- iRule 스크립트 구성
- 데이터 그룹 관리
- BigIP CNE 통합

## 주요 리소스

- **bigip_pool**: 로드 밸런싱 통합을 위한 BIG-IP 풀
- **bigip_device**: BIG-IP 장치 등록 및 관리
