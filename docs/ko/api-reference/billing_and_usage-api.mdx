플랜 전환, 인보이스 발행 및 리소스 소비. 네임스페이스 수준의 할당량 제한 및 사용량 추적.

- **카테고리**: Platform
- **복잡도**: moderate
- **경로**: 25 | **스키마**: 101
- **티어**: Standard
- **관련 도메인**: `system`, `users`

## 사용 사례

- 구독 플랜 및 청구 전환 관리
- 결제 수단 및 인보이스 구성
- 네임스페이스 전반에 걸친 리소스 할당량 사용 추적
- 사용량 제한 및 용량 모니터링

## 주요 리소스

- **subscription**: 서비스 자격을 위한 구독
- **quota**: 사용량 제한을 위한 리소스 할당량
- **usage_report**: 소비 추적을 위한 사용량 보고서
