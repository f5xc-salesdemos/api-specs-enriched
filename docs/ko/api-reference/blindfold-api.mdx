정책 기반 접근 제어를 통한 암호화 키 관리. 감사 로깅 및 안전한 데이터 보호.

- **카테고리**: 보안
- **복잡도**: 보통
- **경로**: 27 | **스키마**: 163
- **등급**: 고급
- **관련 도메인**: `client_side_defense`, `certificates`

## 사용 사례

- 암호화를 위한 비밀 정책 구성
- 민감한 데이터 암호화 관리
- 데이터 보호 정책 적용

## 기본 리소스

- **blindfold_secret**: Blindfold 암호화를 사용하는 암호화된 비밀
- **secret_policy**: 비밀 관리 정책 구성
- **policy_document**: 접근 제어 정의를 위한 정책 문서
