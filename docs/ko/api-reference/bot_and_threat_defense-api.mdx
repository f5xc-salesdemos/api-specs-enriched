행동 분석 및 시그니처 매칭을 통한 위협 분류. 악성 트래픽 패턴에 대한 자동 차단.

- **카테고리**: 보안
- **복잡도**: 보통
- **경로**: 19 | **스키마**: 74
- **등급**: 고급
- **관련 도메인**: `bot_defense`, `shape`, `virtual`

## 사용 사례

- 네임스페이스별 봇 방어 인스턴스 구성
- 분류를 위한 TPM 위협 카테고리 관리
- 자동화된 방어 시스템을 위한 API 키 프로비저닝
- 위협 인텔리전스 서비스 통합

## 주요 리소스

- **bot_defense_instance**: 배포를 위한 봇 방어 인스턴스
- **threat_category**: 분류를 위한 위협 카테고리
