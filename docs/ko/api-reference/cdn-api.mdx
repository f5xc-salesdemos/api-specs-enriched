캐시 규칙과 퍼지 작업을 통한 글로벌 배포. 성능 모니터링 및 분석.

- **카테고리**: 네트워킹
- **복잡도**: 고급
- **경로**: 29 | **스키마**: 592
- **티어**: 고급
- **관련 도메인**: `virtual`

## 사용 사례

- CDN 로드 밸런싱 구성
- 콘텐츠 전송 네트워크 서비스 관리
- 캐싱 정책 구성
- 데이터 전송 및 배포 관리

## 주요 리소스

- **cdn_loadbalancer**: 엣지 캐싱을 위한 콘텐츠 전송 네트워크 로드 밸런서
- **cdn_origin_pool**: CDN 콘텐츠 소싱을 위한 오리진 서버 풀
