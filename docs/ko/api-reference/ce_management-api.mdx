이미지 다운로드 및 사전 업그레이드 검증을 포함한 토큰 기반 프로비저닝. 플릿 그룹화를 통해 분산된 위치에서 대량 작업을 수행할 수 있습니다.

- **카테고리**: 인프라
- **복잡도**: 보통
- **경로**: 28 | **스키마**: 231
- **티어**: Standard
- **관련 도메인**: `customer_edge`, `sites`

## 사용 사례

- Customer Edge 사이트 수명주기 관리
- 네트워크 인터페이스 및 플릿 설정 구성
- 사이트 등록 및 토큰 워크플로우 처리
- 사전 업그레이드 검사를 통한 사이트 업그레이드 실행

## 주요 리소스

- **site_config**: 에지 노드 설정을 위한 사이트 구성
- **fleet_config**: 다중 사이트 관리를 위한 플릿 구성
- **registration_token**: 사이트 온보딩을 위한 등록 토큰
