인증서 체인 및 신뢰할 수 있는 CA 번들. PKI 운영을 위한 해지 목록 관리 및 매니페스트 구성.

- **카테고리**: 보안
- **복잡도**: 보통
- **경로**: 16 | **스키마**: 74
- **티어**: 표준
- **관련 도메인**: `blindfold`, `system`

## 사용 사례

- SSL/TLS 인증서 관리
- 신뢰할 수 있는 CA 구성
- 인증서 해지 목록(CRL) 관리
- 인증서 매니페스트 구성

## 주요 리소스

- **certificate**: 안전한 HTTPS 연결을 위한 TLS 인증서
- **ca_certificate**: 신뢰 체인을 위한 인증 기관 인증서
- **certificate_chain**: 완전한 신뢰 검증을 위한 인증서 체인
