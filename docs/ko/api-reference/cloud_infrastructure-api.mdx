게이트웨이 피어링 및 네트워크 경로 구성을 통한 멀티 클라우드 제공업체 연결. 자격 증명 볼트 통합 및 서브넷 열거.

- **카테고리**: 인프라
- **복잡도**: 보통
- **경로**: 31 | **스키마**: 208
- **티어**: Standard
- **관련 도메인**: `sites`, `customer_edge`

## 사용 사례

- 클라우드 제공업체(AWS, Azure, GCP)에 연결
- 클라우드 자격 증명 및 인증 관리
- 클라우드 연결 및 탄력적 프로비저닝 구성
- 클라우드 리전 연결 및 관리

## 주요 리소스

- **aws_vpc_site**: 엣지 노드 프로비저닝을 포함한 AWS VPC 사이트 배포
- **azure_vnet_site**: 엣지 노드 프로비저닝을 포함한 Azure VNet 사이트 배포
- **gcp_vpc_site**: 엣지 노드 프로비저닝을 포함한 Google Cloud VPC 사이트 배포
- **cloud_credentials**: 사이트 배포를 위한 클라우드 제공업체 인증 자격 증명
