전체 클러스터 복잡성 없이 Pod를 오케스트레이션합니다. 엣지 사이트 실행, 할당량 적용, 분산 앱을 위한 표준화된 컴퓨팅 프로필을 제공합니다.

- **카테고리**: 인프라
- **복잡도**: 중간
- **경로**: 13 | **스키마**: 159
- **티어**: 고급
- **관련 도메인**: `managed_kubernetes`, `sites`, `service_mesh`

## 사용 사례

- 멀티 테넌트 워크로드를 위한 XCCS(컨테이너 서비스) 네임스페이스 배포
- 간소화된 오케스트레이션으로 컨테이너 워크로드 관리
- 분산 엣지 컨테이너 배포 구성
- 전체 K8s 복잡성 없이 컨테이너화된 애플리케이션 실행

## 기본 리소스

- **virtual_k8s**: 컨테이너 워크로드를 위한 가상 Kubernetes 네임스페이스
- **workload**: 컨테이너 워크로드 배포 구성
- **pod_security_policy**: 컨테이너 런타임 제약을 위한 Pod 보안 정책
