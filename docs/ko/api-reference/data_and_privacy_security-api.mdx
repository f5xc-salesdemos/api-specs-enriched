사용자 정의 분류 규칙을 갖춘 민감 데이터 정책. LMA 리전 구성 및 지역 기반 규정 준수 제어.

- **카테고리**: 보안
- **복잡도**: 단순
- **경로**: 11 | **스키마**: 65
- **티어**: 고급
- **관련 도메인**: `blindfold`, `client_side_defense`

## 사용 사례

- 민감 데이터 탐지 정책 구성
- PII 분류를 위한 사용자 정의 데이터 유형 정의
- LMA 리전 구성 관리
- 규정 준수를 위한 지역 구성 통합

## 주요 리소스

- **sensitive_data_policy**: PII 보호를 위한 민감 데이터 정책
- **data_classification**: 콘텐츠 분류를 위한 데이터 분류
