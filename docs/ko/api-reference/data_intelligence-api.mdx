데이터 분석 워크플로우를 위한 분류 규칙, 리소스 정책 및 인사이트 생성.

- **카테고리**: 운영
- **복잡도**: 단순
- **경로**: 15 | **스키마**: 73
- **티어**: 표준
- **관련 도메인**: `statistics`, `observability`

## 사용 사례

- 보안 및 트래픽 데이터 분석
- 로그에서 지능형 인사이트 생성
- 데이터 분석 정책 구성

## 주요 리소스

- **analytics_query**: 데이터 분석을 위한 분석 쿼리
- **data_export**: 보고를 위한 데이터 내보내기 구성
