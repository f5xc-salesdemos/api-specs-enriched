거부 목록, 방화벽 규칙 그룹 및 터널 기반 보호 장치. 네트워크 경계 보안을 위한 속도 제한 및 패턴 분석.

- **카테고리**: 보안
- **복잡도**: 고급
- **경로**: 60 | **스키마**: 246
- **티어**: 고급
- **관련 도메인**: `network_security`, `virtual`

## 사용 사례

- DDoS 보호 정책 구성
- DDoS 위협 모니터링 및 분석
- 인프라 보호 구성

## 주요 리소스

- **ddos_protection**: 트래픽 스크러빙을 위한 DDoS 보호 구성
- **ddos_mitigation_rule**: 공격 대응을 위한 DDoS 완화 규칙
