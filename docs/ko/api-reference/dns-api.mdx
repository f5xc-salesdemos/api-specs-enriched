영역 전송 및 상태 확인을 통한 이름 확인. 레코드 유형 및 위임 지원.

- **카테고리**: 네트워킹
- **복잡성**: 고급
- **경로**: 49 | **스키마**: 285
- **티어**: Standard
- **관련 도메인**: `virtual`, `network`

## 사용 사례

- DNS 부하 분산 구성
- DNS 영역 및 도메인 관리
- DNS 컴플라이언스 정책 구성
- 리소스 레코드 세트(RRSets) 관리

## 주요 리소스

- **dns_zone**: 레코드 관리 기능을 갖춘 권한 있는 DNS 영역
- **dns_domain**: DNS 도메인 위임 및 구성 설정
- **dns_load_balancer**: 지리적 라우팅 및 페일오버를 통한 DNS 기반 트래픽 분산
