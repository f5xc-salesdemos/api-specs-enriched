DPD 타이머를 사용한 서드파티 GRE 및 IPSec 터널 프로비저닝. 타일 배치 제어를 통한 네임스페이스 간 공유 리소스 할당.

- **카테고리**: Platform
- **복잡도**: moderate
- **경로**: 36 | **스키마**: 183
- **티어**: Advanced
- **관련 도메인**: `bigip`, `admin`

## 사용 사례

- 서드파티 통합 및 부가 서비스 접근
- 마켓플레이스 확장 기능 관리
- Terraform 및 외부 통합 구성
- TPM 정책 관리

## 주요 리소스

- **marketplace_item**: 서비스 카탈로그용 마켓플레이스 항목
- **subscription**: 서비스 권한에 대한 구독
