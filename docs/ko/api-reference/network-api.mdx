ASN 관리 및 자율 시스템 관계를 포함하는 보더 게이트웨이 프로토콜. 암호화된 채널을 통해 데이터센터를 연결하는 사이트 간 VPN.

- **카테고리**: 네트워킹
- **복잡도**: 고급
- **경로**: 81 | **스키마**: 449
- **티어**: 고급
- **관련 도메인**: `virtual`, `network_security`, `dns`

## 사용 사례

- BGP 라우팅 및 ASN 관리 구성
- IPsec 터널 및 IKE 단계 관리
- 네트워크 커넥터 및 라우트 구성
- SRv6 및 서브네팅 관리
- 세그먼트 연결 및 정책 정의
- IP 프리픽스 세트 구성

## 주요 리소스

- **virtual_network**: 사이트 연결 및 세그먼테이션을 위한 가상 네트워크
- **network_connector**: 사이트 간 또는 클라우드 연결을 위한 네트워크 커넥터
- **site_mesh_group**: 다중 사이트 간 메시 연결 구성
