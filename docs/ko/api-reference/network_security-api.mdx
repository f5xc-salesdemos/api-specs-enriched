소스, 대상 또는 프로토콜을 기반으로 라우팅 결정을 내리는 방화벽 규칙입니다. 세그멘테이션은 워크로드를 격리하고 아웃바운드 프록시는 액세스를 관리합니다.

- **카테고리**: 보안
- **복잡도**: 고급
- **경로**: 62 | **스키마**: 386
- **티어**: 고급
- **관련 도메인**: `virtual`, `api`, `network`

## 사용 사례

- 네트워크 방화벽 및 ACL 정책 구성
- NAT 정책 및 포트 포워딩 관리
- 정책 기반 라우팅 구성
- 네트워크 세그먼트 및 정책 정의
- 포워드 프록시 정책 구성

## 주요 리소스

- **network_policy**: 트래픽 필터링을 위한 네트워크 보안 정책
- **forward_proxy_policy**: 아웃바운드 트래픽 제어를 위한 포워드 프록시 정책
- **network_firewall**: 레이어 3/4 트래픽 보호를 위한 네트워크 방화벽
