인스턴스 검색, WAF 통합 및 서비스 메시 연결. 구독 수명 주기 및 구성 동기화.

- **카테고리**: 플랫폼
- **복잡도**: 단순
- **경로**: 13 | **스키마**: 59
- **티어**: 고급
- **관련 도메인**: `marketplace`

## 사용 사례

- NGINX One 플랫폼 통합 관리
- NGINX Plus 인스턴스 구성
- NGINX 구성 관리 통합

## 주요 리소스

- **nginx_config**: 프록시 설정을 위한 NGINX 구성
- **nginx_upstream**: 백엔드 서버 풀을 위한 NGINX 업스트림
