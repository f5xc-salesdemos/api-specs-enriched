모바일 앱 통합을 위한 버전 관리된 라이브러리 배포. 사전 서명된 URL을 통해 iOS 및 Android용 OS별 빌드를 안전하게 다운로드할 수 있습니다.

- **카테고리**: 플랫폼
- **복잡도**: 단순
- **경로**: 9 | **스키마**: 17
- **등급**: 고급
- **관련 도메인**: `marketplace`

## 사용 사례

- 오브젝트 스토리지 서비스 관리
- 저장된 오브젝트 및 버킷 구성
- 스토리지 정책 관리

## 주요 리소스

- **object_store**: 블롭 스토리지 관리를 위한 오브젝트 스토어
- **bucket**: 오브젝트 스토리지용 스토리지 버킷
