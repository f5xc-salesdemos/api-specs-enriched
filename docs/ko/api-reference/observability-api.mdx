지연 시간 측정이 포함된 HTTP 가용성 프로브. 인증서 만료 알림 및 인프라 상태를 위한 글로벌 상태 대시보드.

- **카테고리**: 운영
- **복잡도**: 보통
- **경로**: 56 | **스키마**: 196
- **티어**: 표준
- **관련 도메인**: `statistics`, `support`

## 사용 사례

- 합성 모니터링 구성
- 모니터링 및 테스트 정책 정의
- 관측성 대시보드 관리

## 주요 리소스

- **log_receiver**: 중앙 집중식 로그 수집을 위한 로그 수신기
- **metrics_receiver**: 중앙 집중식 메트릭 수집을 위한 메트릭 수신기
- **alert_policy**: 모니터링 및 알림을 위한 알림 정책
