시간, 분, 초 단위의 구성 가능한 윈도우를 사용한 시간 기반 할당량 적용. 트래픽 셰이핑을 위한 프로토콜별 제어.

- **카테고리**: 네트워킹
- **복잡도**: 단순
- **경로**: 12 | **스키마**: 70
- **티어**: 고급
- **관련 도메인**: `virtual`, `network_security`

## 사용 사례

- 속도 제한 정책 구성
- 폴리서 구성 관리
- 트래픽 흐름 및 큐잉 제어

## 주요 리소스

- **rate_limiter**: 트래픽 스로틀링 및 보호를 위한 속도 제한기
- **rate_limiter_policy**: 상세한 스로틀링 규칙이 포함된 속도 제한 정책
- **rate_limit_threshold**: 트래픽 제어를 위한 속도 제한 임계값 구성
