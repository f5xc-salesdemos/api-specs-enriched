검색 및 학습된 스키마 분석을 포함한 애플리케이션 유형 정의. 지능형 요청 처리를 위한 트래픽 패턴 추론.

- **카테고리**: 인프라
- **복잡도**: 고급
- **경로**: 38 | **스키마**: 247
- **티어**: 고급
- **관련 도메인**: `managed_kubernetes`, `container_services`, `virtual`

## 사용 사례

- 서비스 메시 연결 구성
- 엔드포인트 검색 및 라우팅 관리
- NFV 서비스 구성
- 애플리케이션 설정 및 유형 정의

## 주요 리소스

- **endpoint**: 트래픽 라우팅을 위한 서비스 메시 엔드포인트
- **origin_pool**: 로드 밸런서 트래픽 분배를 위한 백엔드 서버 그룹
- **service_discovery**: 동적 엔드포인트를 위한 서비스 디스커버리 구성
