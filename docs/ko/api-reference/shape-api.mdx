행동 분석 및 디바이스 핑거프린팅을 통한 위협 인식. 애플리케이션 보호를 위한 모바일 SDK 통합.

- **카테고리**: 보안
- **복잡성**: 고급
- **경로**: 271 | **스키마**: 813
- **티어**: 고급
- **관련 도메인**: `bot_defense`, `virtual`

## 사용 사례

- Shape 보안 정책 구성
- 봇 및 위협 방지 관리
- SafeAP 정책 구성
- 위협 인식 활성화

## 주요 리소스

- **shape_app_firewall**: 고급 보호를 위한 Shape 애플리케이션 방화벽
- **shape_recognizer**: 트래픽 패턴 분석을 위한 Shape 인식기
