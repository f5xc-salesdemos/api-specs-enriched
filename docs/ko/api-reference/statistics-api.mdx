수신기 통합을 통한 알림 정책. 로그 집계, 토폴로지 보기 및 사이트 상태 추적.

- **카테고리**: 운영
- **복잡도**: 고급
- **경로**: 77 | **스키마**: 575
- **티어**: 표준
- **관련 도메인**: `observability`, `support`

## 사용 사례

- 플로우 통계 및 분석에 액세스
- 알림 및 알림 정책 관리
- 로그 및 로그 수신기 보기
- 보고서 및 그래프 생성
- 토폴로지 및 서비스 탐색 추적
- 사이트의 상태 모니터링

## 주요 리소스

- **dashboard**: 메트릭 시각화를 위한 대시보드
- **saved_query**: 메트릭 및 로그 분석을 위한 저장된 쿼리
