코멘트, 심각도 변경 및 해결 추적을 포함한 이슈 수명 주기. 연결 분석을 위한 패킷 캡처.

- **카테고리**: 운영
- **복잡도**: 보통
- **경로**: 53 | **스키마**: 167
- **티어**: Standard
- **관련 도메인**: `statistics`, `observability`

## 사용 사례

- 지원 티켓 제출 및 관리
- 고객 지원 요청 추적
- 운영 지원 문서 접근

## 주요 리소스

- **support_case**: 이슈 추적 및 해결을 위한 지원 케이스
- **alert**: 운영 알림을 위한 시스템 경고
- **audit_log**: 규정 준수 및 보안 추적을 위한 감사 로그
