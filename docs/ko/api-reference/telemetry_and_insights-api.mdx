관찰 가능성 워크플로를 위한 수집 엔드포인트, 메트릭 저장소 및 인사이트 생성.

- **카테고리**: 운영
- **복잡도**: 보통
- **경로**: 27 | **스키마**: 181
- **등급**: 표준
- **관련 도메인**: `observability`, `statistics`

## 사용 사례

- 텔레메트리 데이터 수집 및 분석
- 메트릭에서 실행 가능한 인사이트 생성
- 텔레메트리 수집 정책 구성

## 주요 리소스

- **telemetry_receiver**: 데이터 수집을 위한 텔레메트리 수신기
- **insight_query**: 분석을 위한 인사이트 쿼리
