계정 보기 구성 및 관리자 알림 채널. 일회용 비밀번호 재설정, 프로비저닝 흐름, 활성 연결 모니터링.

- **카테고리**: 플랫폼
- **복잡도**: 고급
- **경로**: 183 | **스키마**: 511
- **티어**: 스탠다드
- **관련 도메인**: `users`, `authentication`, `system`

## 사용 사례

- 사용자 프로필 및 알림 기본 설정 관리
- 세션 제어 및 OTP 설정 구성
- ID 관리 작업 처리
- 초기 사용자 접근 요청 처리

## 주요 리소스

- **user_profile**: 개인 설정이 포함된 사용자 프로필
- **session**: 인증 상태를 위한 사용자 세션
- **otp_policy**: 다중 인증을 위한 OTP 정책
