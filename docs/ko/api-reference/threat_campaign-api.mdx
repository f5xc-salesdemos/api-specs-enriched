보안 운영을 위한 탐지 정책, 공격 추적 및 자동화된 완화 규칙 생성.

- **카테고리**: 보안
- **복잡도**: 보통
- **경로**: 1 | **스키마**: 161
- **등급**: 표준
- **관련 도메인**: `bot_defense`, `ddos`

## 사용 사례

- 위협 캠페인 추적 및 분석
- 활성 위협 및 공격 패턴 모니터링
- 위협 인텔리전스 통합 구성

## 기본 리소스

- **threat_campaign_policy**: 위협 캠페인 탐지 및 완화 정책

## 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | ID로 위협 캠페인 가져오기. |
