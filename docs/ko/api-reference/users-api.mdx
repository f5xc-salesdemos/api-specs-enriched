자동 만료 기능이 포함된 사이트 등록 자격 증명입니다. 분류 키는 허용된 분류 체계를 정의하며, 자동 파생 태그는 동적으로 적용됩니다.

- **카테고리**: 플랫폼
- **복잡도**: 단순
- **경로**: 13 | **스키마**: 50
- **티어**: 표준
- **관련 도메인**: `system`, `admin`

## 사용 사례

- 사용자 계정 및 토큰 관리
- 사용자 식별 구성
- 사용자 설정 및 환경설정 관리
- 암시적 및 알려진 레이블 구성

## 주요 리소스

- **user**: 플랫폼 접근을 위한 사용자 계정
- **user_role**: 권한 관리를 위한 사용자 역할
- **namespace_role**: 세분화된 권한을 위한 네임스페이스 범위 역할

| 메서드 | 경로 | 설명 |

| POST | `/api/config/namespaces/&#123;namespace&#125;/known_label/create` | 생성 |
| POST | `/api/config/namespaces/&#123;namespace&#125;/known_label/delete` | 삭제 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/known_labels` | 조회 |

| GET | `/api/config/namespaces/&#123;namespace&#125;/known_label_keys` | 조회 |
