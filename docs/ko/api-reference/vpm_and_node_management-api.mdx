분산 노드 관리를 위한 수명 주기 제어, 플릿 구성 및 배포 정책.

- **카테고리**: 플랫폼
- **복잡도**: 단순
- **경로**: 1 | **스키마**: 2
- **티어**: Standard
- **관련 도메인**: `sites`, `system`

## 사용 사례

- Virtual Private Mesh (VPM) 구성 관리
- 노드 수명 주기 및 관리 구성
- VPM 및 노드 상태 모니터링

## 기본 리소스

- **node_config**: 엣지 장치 설정을 위한 노드 구성
- **vpm_config**: 가상 플랫폼 관리를 위한 VPM 구성

## 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|-------------|
| GET | `/api/data/namespaces/system/upgrade_status` | 업그레이드 상태. |
