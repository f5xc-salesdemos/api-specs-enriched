Elementos visuais com escopo de namespace e versionamento. Implantação de widgets personalizados e gerenciamento de catálogo para superfícies de portal.

- **Categoria**: Plataforma
- **Complexidade**: simples
- **Caminhos**: 2 | **Schemas**: 14
- **Nível**: Standard
- **Domínios relacionados**: `admin`, `system`

## Casos de Uso

- Gerenciar componentes de UI estáticos para o console administrativo
- Implantar e recuperar ativos de UI dentro de namespaces
- Configurar elementos de interface do console
- Gerenciar metadados de componentes de UI personalizados

## Recursos Principais

- **ui_component**: Componente de UI para personalização do console
- **static_asset**: Ativo estático para recursos de UI

## Endpoints

| Método | Caminho | Descrição |
|--------|---------|-----------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | Listar componentes estáticos de UI. |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | Obter componente estático de UI. |
