Processamento de linguagem natural com sinais de qualidade e monitoramento de anomalias. Autenticação por token para assinaturas de fluxo de dados.

- **Categoria**: IA
- **Complexidade**: simples
- **Caminhos**: 11 | **Esquemas**: 66
- **Nível**: Avançado


## Casos de Uso

- Acessar recursos com tecnologia de IA
- Configurar políticas do assistente de IA
- Habilitar detecção de anomalias em fluxos
- Gerenciar coleta de dados de IA

## Recursos Primários

- **ai_policy**: Política de IA para controle de tráfego de IA generativa
- **ai_gateway**: Gateway de IA para gerenciamento de API de LLM

## Endpoints

| Método | Caminho | Descrição |
|--------|---------|-----------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | Avaliar Consulta do Assistente de IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | Avaliar Feedback da Consulta do Assistente de IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | Consulta do Assistente de IA. |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | Feedback da Consulta do Assistente de IA. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | Habilitar recurso. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | Atualizar Token. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | Assinar pipeline BFDP. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | Assinar pipeline BFDP. |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | Cancelar assinatura do pipeline BFDP. |
| POST | `/api/gia/gia/allocateip` | Alocar IP por meio do Alocador Global de IP. |
| DELETE | `/api/gia/gia/deallocateip` | Desalocar IP por meio do Alocador Global de IP. |
