Catalogação de recursos com classificação automática e perfil de segurança. Hierarquias organizacionais segmentam o acesso por função ou política de proteção.

- **Categoria**: Segurança
- **Complexidade**: avançada
- **Caminhos**: 45 | **Esquemas**: 279
- **Nível**: Avançado
- **Domínios relacionados**: `virtual`, `network_security`

## Casos de Uso

- Descobrir e catalogar APIs
- Testar segurança e comportamento de APIs
- Gerenciar credenciais de APIs
- Definir grupos de APIs e políticas de teste

## Recursos Principais

- **api_definition**: Definição de esquema de API para descoberta e proteção
- **api_endpoint**: Configuração e proteção de endpoint individual de API
- **api_rate_limit**: Configuração de limitação de taxa de API para controle de tráfego
