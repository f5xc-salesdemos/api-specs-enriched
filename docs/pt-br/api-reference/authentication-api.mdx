Gerenciamento de identidade com integração de provedores, políticas de acesso e controle do ciclo de vida de credenciais.

- **Categoria**: Plataforma
- **Complexidade**: simples
- **Caminhos**: 14 | **Schemas**: 38
- **Nível**: Standard
- **Domínios relacionados**: `system`, `users`

## Casos de Uso

- Configurar mecanismos de autenticação
- Gerenciar provedores OIDC e OAuth
- Configurar provisionamento de usuários SCIM
- Gerenciar credenciais de API e acesso
- Configurar políticas de cadastro de contas

## Recursos Principais

- **authentication_policy**: Política de autenticação para acesso de usuários e API
- **token**: Token de API para acesso programático
- **api_credential**: Credencial de API para autenticação de serviços
