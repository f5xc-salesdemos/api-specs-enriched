Orquestração legada de dispositivos com scripts iRule e sincronização de data groups. Vinculações de servidores virtuais e coleta de métricas.

- **Categoria**: Plataforma
- **Complexidade**: avançada
- **Caminhos**: 29 | **Schemas**: 238
- **Nível**: Avançado
- **Domínios relacionados**: `marketplace`

## Casos de Uso

- Gerenciar appliances BigIP F5
- Configurar scripts iRule
- Gerenciar data groups
- Integrar BigIP CNE

## Recursos Principais

- **bigip_pool**: Pool BIG-IP para integração de balanceamento de carga
- **bigip_device**: Registro e gerenciamento de dispositivos BIG-IP
