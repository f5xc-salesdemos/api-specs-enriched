Transições de plano, faturamento e consumo de recursos. Limites de cota em nível de namespace e rastreamento de uso.

- **Categoria**: Plataforma
- **Complexidade**: moderada
- **Caminhos**: 25 | **Schemas**: 101
- **Nível**: Standard
- **Domínios relacionados**: `system`, `users`

## Casos de Uso

- Gerenciar planos de assinatura e transições de faturamento
- Configurar métodos de pagamento e faturas
- Rastrear o uso de cotas de recursos entre namespaces
- Monitorar limites de uso e capacidade

## Recursos Principais

- **subscription**: Assinatura para direitos de serviço
- **quota**: Cota de recursos para limites de uso
- **usage_report**: Relatório de uso para rastreamento de consumo
