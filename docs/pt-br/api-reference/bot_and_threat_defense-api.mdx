Classificação de ameaças com análise comportamental e correspondência de assinaturas. Bloqueio automatizado para padrões de tráfego malicioso.

- **Categoria**: Segurança
- **Complexidade**: moderada
- **Caminhos**: 19 | **Esquemas**: 74
- **Nível**: Avançado
- **Domínios relacionados**: `bot_defense`, `shape`, `virtual`

## Casos de Uso

- Configurar instâncias de defesa contra bots por namespace
- Gerenciar categorias de ameaças TPM para classificação
- Provisionar chaves de API para sistemas de defesa automatizados
- Integrar serviços de inteligência contra ameaças

## Recursos Principais

- **bot_defense_instance**: Instância de defesa contra bots para implantação
- **threat_category**: Categoria de ameaça para classificação
