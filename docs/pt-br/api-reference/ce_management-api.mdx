Provisionamento baseado em token com downloads de imagens e validação pré-atualização. O agrupamento de frotas permite operações em massa em locais distribuídos.

- **Categoria**: Infraestrutura
- **Complexidade**: moderada
- **Caminhos**: 28 | **Esquemas**: 231
- **Nível**: Standard
- **Domínios relacionados**: `customer_edge`, `sites`

## Casos de Uso

- Gerenciar o ciclo de vida do site Customer Edge
- Configurar interfaces de rede e configurações de frota
- Gerenciar registro de sites e fluxos de trabalho de tokens
- Executar atualizações de sites com verificações pré-atualização

## Recursos Principais

- **site_config**: Configuração de site para definições de nó de borda
- **fleet_config**: Configuração de frota para gerenciamento de múltiplos sites
- **registration_token**: Token de registro para integração de sites
