Provisionamento baseado em tokens com downloads de imagem e validação pré-upgrade. O agrupamento em frotas permite operações em massa em locais distribuídos.

- **Categoria**: Infraestrutura
- **Complexidade**: moderada
- **Caminhos**: 28 | **Schemas**: 231
- **Nível**: Standard
- **Domínios relacionados**: `customer_edge`, `sites`

## Casos de Uso

- Gerenciar o ciclo de vida de sites Customer Edge
- Configurar interfaces de rede e configurações de frota
- Gerenciar registro de sites e fluxos de trabalho com tokens
- Executar upgrades de sites com verificações pré-upgrade

## Recursos Principais

- **site_config**: Configuração de site para definições de nós de borda
- **fleet_config**: Configuração de frota para gerenciamento de múltiplos sites
- **registration_token**: Token de registro para integração de sites
