Cadeias de certificados e pacotes de CA confiáveis. Gerenciamento de listas de revogação e configuração de manifesto para operações PKI.

- **Categoria**: Segurança
- **Complexidade**: moderada
- **Caminhos**: 16 | **Schemas**: 74
- **Nível**: Standard
- **Domínios relacionados**: `blindfold`, `system`

## Casos de Uso

- Gerenciar certificados SSL/TLS
- Configurar CAs confiáveis
- Gerenciar listas de revogação de certificados (CRL)
- Configurar manifestos de certificados

## Recursos Principais

- **certificate**: Certificado TLS para conexões HTTPS seguras
- **ca_certificate**: Certificado de autoridade certificadora para cadeia de confiança
- **certificate_chain**: Cadeia de certificados para verificação completa de confiança
