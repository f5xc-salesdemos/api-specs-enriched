Políticas de dados sensíveis com regras de classificação personalizadas. Configuração de região LMA e controles de conformidade baseados em geolocalização.

- **Categoria**: Segurança
- **Complexidade**: simples
- **Caminhos**: 11 | **Esquemas**: 65
- **Nível**: Avançado
- **Domínios relacionados**: `blindfold`, `client_side_defense`

## Casos de Uso

- Configurar políticas de detecção de dados sensíveis
- Definir tipos de dados personalizados para classificação de PII
- Gerenciar configurações de região LMA
- Integrar geoconfigurações para conformidade

## Recursos Principais

- **sensitive_data_policy**: Política de dados sensíveis para proteção de PII
- **data_classification**: Classificação de dados para categorização de conteúdo
