Regras de classificação, políticas de recursos e geração de insights para fluxos de trabalho de análise de dados.

- **Categoria**: Operações
- **Complexidade**: simples
- **Caminhos**: 15 | **Esquemas**: 73
- **Nível**: Padrão
- **Domínios relacionados**: `statistics`, `observability`

## Casos de Uso

- Analisar dados de segurança e tráfego
- Gerar insights inteligentes a partir de logs
- Configurar políticas de análise de dados

## Recursos Principais

- **analytics_query**: Consulta analítica para análise de dados
- **data_export**: Configuração de exportação de dados para relatórios
