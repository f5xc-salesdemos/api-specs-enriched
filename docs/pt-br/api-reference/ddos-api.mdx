Listas de negação, grupos de regras de firewall e proteções baseadas em túnel. Limitação de taxa e análise de padrões para segurança de perímetro de rede.

- **Categoria**: Segurança
- **Complexidade**: avançada
- **Paths**: 60 | **Schemas**: 246
- **Nível**: Avançado
- **Domínios relacionados**: `network_security`, `virtual`

## Casos de Uso

- Configurar políticas de proteção contra DDoS
- Monitorar e analisar ameaças DDoS
- Configurar proteção de infraestrutura

## Recursos Principais

- **ddos_protection**: Configuração de proteção DDoS para limpeza de tráfego
- **ddos_mitigation_rule**: Regra de mitigação DDoS para resposta a ataques
