Resolução de nomes com transferências de zona e verificações de saúde. Tipos de registro e suporte a delegação.

- **Categoria**: Rede
- **Complexidade**: avançada
- **Caminhos**: 49 | **Esquemas**: 285
- **Nível**: Standard
- **Domínios relacionados**: `virtual`, `network`

## Casos de Uso

- Configurar balanceamento de carga DNS
- Gerenciar zonas e domínios DNS
- Configurar políticas de conformidade DNS
- Gerenciar conjuntos de registros de recursos (RRSets)

## Recursos Principais

- **dns_zone**: Zona DNS autoritativa com capacidades de gerenciamento de registros
- **dns_domain**: Delegação de domínio DNS e configurações
- **dns_load_balancer**: Distribuição de tráfego baseada em DNS com roteamento geográfico e failover
