Vinculações de funções do Kubernetes e políticas de admissão. Integração de registro para cargas de trabalho EKS, AKS e GKE.

- **Categoria**: Infraestrutura
- **Complexidade**: moderada
- **Caminhos**: 20 | **Esquemas**: 95
- **Nível**: Avançado
- **Domínios relacionados**: `container_services`, `sites`, `service_mesh`

## Casos de Uso

- Gerenciar RBAC e segurança de clusters XCKS (Kubernetes Gerenciado)
- Configurar políticas de segurança de pods e controladores de admissão
- Gerenciar registros de contêineres para implantações empresariais
- Integrar com clusters Kubernetes externos (EKS, AKS, GKE)

## Recursos Principais

- **mk8s_cluster**: Configuração de cluster Kubernetes gerenciado
- **k8s_cluster_role**: Configuração de função RBAC de cluster Kubernetes
- **container_registry**: Registro de imagens de contêiner para imagens de carga de trabalho
