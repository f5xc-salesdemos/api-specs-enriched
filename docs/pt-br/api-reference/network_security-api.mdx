Regras de firewall com decisões de roteamento baseadas em origem, destino ou protocolo. A segmentação isola cargas de trabalho enquanto proxies de saída governam o acesso.

- **Categoria**: Segurança
- **Complexidade**: avançada
- **Paths**: 62 | **Schemas**: 386
- **Nível**: Avançado
- **Domínios relacionados**: `virtual`, `api`, `network`

## Casos de Uso

- Configurar políticas de firewall de rede e ACL
- Gerenciar políticas NAT e encaminhamento de portas
- Configurar roteamento baseado em políticas
- Definir segmentos de rede e políticas
- Configurar políticas de proxy de encaminhamento

## Recursos Principais

- **network_policy**: Política de segurança de rede para filtragem de tráfego
- **forward_proxy_policy**: Política de proxy de encaminhamento para controle de tráfego de saída
- **network_firewall**: Firewall de rede para proteção de tráfego nas camadas 3/4
