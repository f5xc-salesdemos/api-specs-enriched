Descoberta de instâncias, integração com WAF e conectividade de malha de serviços. Ciclo de vida de assinatura e sincronização de configuração.

- **Categoria**: Plataforma
- **Complexidade**: simples
- **Caminhos**: 13 | **Esquemas**: 59
- **Nível**: Avançado
- **Domínios relacionados**: `marketplace`

## Casos de Uso

- Gerenciar integrações da plataforma NGINX One
- Configurar instâncias NGINX Plus
- Integrar o gerenciamento de configuração do NGINX

## Recursos Primários

- **nginx_config**: Configuração NGINX para definições de proxy
- **nginx_upstream**: Upstream NGINX para pools de servidores de backend
