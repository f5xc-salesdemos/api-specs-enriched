Descoberta de instâncias, integração com WAF e conectividade de service mesh. Ciclo de vida de assinaturas e sincronização de configuração.

- **Categoria**: Plataforma
- **Complexidade**: simples
- **Paths**: 13 | **Schemas**: 59
- **Nível**: Avançado
- **Domínios relacionados**: `marketplace`

## Casos de Uso

- Gerenciar integrações da plataforma NGINX One
- Configurar instâncias NGINX Plus
- Integrar gerenciamento de configuração NGINX

## Recursos Principais

- **nginx_config**: Configuração NGINX para definições de proxy
- **nginx_upstream**: NGINX upstream para pools de servidores backend
