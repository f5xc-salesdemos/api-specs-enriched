Distribuição de bibliotecas versionadas para integradores de aplicativos móveis. URLs pré-assinadas permitem downloads seguros com builds específicos por sistema operacional para iOS e Android.

- **Categoria**: Plataforma
- **Complexidade**: simples
- **Caminhos**: 9 | **Schemas**: 17
- **Nível**: Avançado
- **Domínios relacionados**: `marketplace`

## Casos de Uso

- Gerenciar serviços de armazenamento de objetos
- Configurar objetos armazenados e buckets
- Gerenciar políticas de armazenamento

## Recursos Principais

- **object_store**: Armazenamento de objetos para gerenciamento de armazenamento de blobs
- **bucket**: Bucket de armazenamento para armazenamento de objetos
