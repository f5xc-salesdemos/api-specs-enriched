Distribuição versionada de bibliotecas para integradores de aplicativos móveis. URLs pré-assinadas permitem downloads seguros com builds específicos por sistema operacional para iOS e Android.

- **Categoria**: Plataforma
- **Complexidade**: simples
- **Caminhos**: 9 | **Esquemas**: 17
- **Nível**: Avançado
- **Domínios relacionados**: `marketplace`

## Casos de Uso

- Gerenciar serviços de armazenamento de objetos
- Configurar objetos armazenados e buckets
- Gerenciar políticas de armazenamento

## Recursos Principais

- **object_store**: Armazenamento de objetos para gerenciamento de blob storage
- **bucket**: Bucket de armazenamento para armazenamento de objetos
