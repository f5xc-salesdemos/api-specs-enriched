Sondas de disponibilidade HTTP com medição de latência. Alertas de expiração de certificados e dashboards de status global para integridade da infraestrutura.

- **Categoria**: Operações
- **Complexidade**: moderada
- **Caminhos**: 56 | **Schemas**: 196
- **Nível**: Standard
- **Domínios relacionados**: `statistics`, `support`

## Casos de Uso

- Configurar monitoramento sintético
- Definir políticas de monitoramento e testes
- Gerenciar dashboards de observabilidade

## Recursos Principais

- **log_receiver**: Receptor de logs para coleta centralizada de logs
- **metrics_receiver**: Receptor de métricas para coleta centralizada de métricas
- **alert_policy**: Política de alertas para monitoramento e notificação
