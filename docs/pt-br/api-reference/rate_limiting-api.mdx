Aplicação de cotas baseadas em tempo com janelas configuráveis em horas, minutos ou segundos. Controles específicos de protocolo para modelagem de tráfego.

- **Categoria**: Rede
- **Complexidade**: simples
- **Caminhos**: 12 | **Esquemas**: 70
- **Nível**: Avançado
- **Domínios relacionados**: `virtual`, `network_security`

## Casos de Uso

- Configurar políticas de limitação de taxa
- Gerenciar configurações de policer
- Controlar fluxo de tráfego e enfileiramento

## Recursos Principais

- **rate_limiter**: Limitador de taxa para limitação e proteção de tráfego
- **rate_limiter_policy**: Política de limitador de taxa com regras detalhadas de limitação
- **rate_limit_threshold**: Configuração de limite de taxa para controle de tráfego
