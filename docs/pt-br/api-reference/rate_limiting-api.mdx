Aplicação de cotas baseadas em tempo com janelas configuráveis em horas, minutos ou segundos. Controles específicos por protocolo para modelagem de tráfego.

- **Categoria**: Redes
- **Complexidade**: simples
- **Caminhos**: 12 | **Schemas**: 70
- **Nível**: Avançado
- **Domínios relacionados**: `virtual`, `network_security`

## Casos de Uso

- Configurar políticas de limitação de taxa
- Gerenciar configurações de policiamento
- Controlar fluxo de tráfego e enfileiramento

## Recursos Primários

- **rate_limiter**: Limitador de taxa para controle e proteção de tráfego
- **rate_limiter_policy**: Política de limitação de taxa com regras detalhadas de throttling
- **rate_limit_threshold**: Configuração de limite de taxa para controle de tráfego
