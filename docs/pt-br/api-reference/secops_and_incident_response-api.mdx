Mitigação de usuários maliciosos com classificação de nível de ameaça. Ações de resposta automatizadas para padrões de comportamento suspeitos.

- **Categoria**: Segurança
- **Complexidade**: simples
- **Caminhos**: 4 | **Esquemas**: 32
- **Nível**: Avançado
- **Domínios relacionados**: `bot_defense`, `virtual`, `network_security`

## Casos de Uso

- Configurar políticas automatizadas de mitigação de ameaças
- Definir regras para detecção de usuários maliciosos
- Gerenciar fluxos de trabalho de resposta a incidentes
- Aplicar bloqueio ou limitação de taxa a ameaças

## Recursos Primários

- **mitigation_policy**: Política de mitigação para resposta a incidentes
- **malicious_user_rule**: Regra de usuário malicioso para mitigação de ameaças

## Endpoints

| Método | Caminho | Descrição |
|--------|---------|-----------|
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations` | Criar Mitigação de Usuário Malicioso. |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations/&#123;metadata.name&#125;` | Substituir Mitigação de Usuário Malicioso. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations` | Listar Mitigação de Usuário Malicioso. |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | OBTER Mitigação de Usuário Malicioso. |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | EXCLUIR Mitigação de Usuário Malicioso. |
