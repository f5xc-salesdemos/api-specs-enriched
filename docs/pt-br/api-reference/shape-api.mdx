Reconhecimento de ameaças com análise comportamental e fingerprinting de dispositivos. Integração com SDK móvel para proteção de aplicações.

- **Categoria**: Segurança
- **Complexidade**: avançada
- **Paths**: 271 | **Schemas**: 813
- **Nível**: Avançado
- **Domínios relacionados**: `bot_defense`, `virtual`

## Casos de Uso

- Configurar políticas de segurança Shape
- Gerenciar prevenção de bots e ameaças
- Configurar políticas SafeAP
- Habilitar reconhecimento de ameaças

## Recursos Principais

- **shape_app_firewall**: Firewall de aplicação Shape para proteção avançada
- **shape_recognizer**: Reconhecedor Shape para análise de padrões de tráfego
