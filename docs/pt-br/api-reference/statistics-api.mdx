Políticas de alertas com integrações de receptores. Agregação de logs, visualizações de topologia e rastreamento de saúde do site.

- **Categoria**: Operações
- **Complexidade**: avançada
- **Caminhos**: 77 | **Esquemas**: 575
- **Nível**: Standard
- **Domínios relacionados**: `observability`, `support`

## Casos de Uso

- Acessar estatísticas e análises de fluxo
- Gerenciar alertas e políticas de alertas
- Visualizar logs e receptores de logs
- Gerar relatórios e gráficos
- Rastrear topologia e descoberta de serviços
- Monitorar status nos sites

## Recursos Principais

- **dashboard**: Dashboard para visualização de métricas
- **saved_query**: Consulta salva para análise de métricas e logs
