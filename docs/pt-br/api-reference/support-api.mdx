Ciclo de vida de issues com comentários, alterações de severidade e rastreamento de resolução. Captura de pacotes para análise de conexão.

- **Categoria**: Operações
- **Complexidade**: moderada
- **Caminhos**: 53 | **Schemas**: 167
- **Nível**: Standard
- **Domínios relacionados**: `statistics`, `observability`

## Casos de Uso

- Enviar e gerenciar tickets de suporte
- Rastrear solicitações de suporte ao cliente
- Acessar documentação de suporte operacional

## Recursos Principais

- **support_case**: Caso de suporte para rastreamento e resolução de issues
- **alert**: Alerta do sistema para notificações operacionais
- **audit_log**: Log de auditoria para rastreamento de conformidade e segurança
