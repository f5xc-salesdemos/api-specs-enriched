Endpoints de coleta, armazenamento de métricas e geração de insights para fluxos de trabalho de observabilidade.

- **Categoria**: Operações
- **Complexidade**: moderada
- **Caminhos**: 27 | **Schemas**: 181
- **Nível**: Standard
- **Domínios relacionados**: `observability`, `statistics`

## Casos de Uso

- Coletar e analisar dados de telemetria
- Gerar insights acionáveis a partir de métricas
- Configurar políticas de coleta de telemetria

## Recursos Primários

- **telemetry_receiver**: Receptor de telemetria para coleta de dados
- **insight_query**: Consulta de insight para análise
