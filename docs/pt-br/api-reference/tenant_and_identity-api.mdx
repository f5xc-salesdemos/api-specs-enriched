Configurações de visualização de conta e canais de alerta de administração. Redefinições de senhas únicas, fluxos de provisionamento e monitoramento de conexões ativas.

- **Categoria**: Plataforma
- **Complexidade**: avançada
- **Caminhos**: 183 | **Schemas**: 511
- **Nível**: Standard
- **Domínios relacionados**: `users`, `authentication`, `system`

## Casos de Uso

- Gerenciar perfis de usuários e preferências de notificação
- Configurar controles de sessão e configurações de OTP
- Realizar operações de gerenciamento de identidade
- Processar solicitações de acesso inicial de usuários

## Recursos Primários

- **user_profile**: Perfil de usuário com configurações pessoais
- **session**: Sessão de usuário para estado de autenticação
- **otp_policy**: Política de OTP para autenticação multifator
