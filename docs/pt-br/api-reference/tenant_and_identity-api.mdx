Configurações de visualização de conta e canais de alerta administrativo. Redefinições de senha única, fluxos de provisionamento e monitoramento de conexões ativas.

- **Categoria**: Plataforma
- **Complexidade**: avançada
- **Caminhos**: 183 | **Esquemas**: 511
- **Nível**: Standard
- **Domínios relacionados**: `users`, `authentication`, `system`

## Casos de Uso

- Gerenciar perfis de usuário e preferências de notificação
- Configurar controles de sessão e configurações de OTP
- Lidar com operações de gerenciamento de identidade
- Processar solicitações de acesso inicial de usuário

## Recursos Principais

- **user_profile**: Perfil de usuário com configurações pessoais
- **session**: Sessão de usuário para estado de autenticação
- **otp_policy**: Política de OTP para autenticação multifator
