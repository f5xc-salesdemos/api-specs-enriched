Políticas de detecção, rastreamento de ataques e geração automatizada de regras de mitigação para operações de segurança.

- **Categoria**: Segurança
- **Complexidade**: moderada
- **Caminhos**: 1 | **Esquemas**: 161
- **Nível**: Standard
- **Domínios relacionados**: `bot_defense`, `ddos`

## Casos de Uso

- Rastrear e analisar campanhas de ameaças
- Monitorar ameaças ativas e padrões de ataque
- Configurar integração de inteligência de ameaças

## Recursos Principais

- **threat_campaign_policy**: Política de detecção e mitigação de campanhas de ameaças

## Endpoints

| Método | Caminho | Descrição |
|--------|---------|-----------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | Obter Campanha de Ameaças por ID. |
