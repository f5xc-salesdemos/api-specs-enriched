Credenciais de registro de site com expiração automática. Chaves de taxonomia definem a categorização permitida enquanto tags derivadas automaticamente são aplicadas dinamicamente.

- **Categoria**: Plataforma
- **Complexidade**: simples
- **Caminhos**: 13 | **Esquemas**: 50
- **Nível**: Padrão
- **Domínios relacionados**: `system`, `admin`

## Casos de Uso

- Gerenciar contas de usuários e tokens
- Configurar identificação de usuários
- Gerenciar configurações e preferências de usuários
- Configurar rótulos implícitos e conhecidos

## Recursos Principais

- **user**: Conta de usuário para acesso à plataforma
- **user_role**: Função de usuário para gerenciamento de permissões
- **namespace_role**: Função com escopo de namespace para permissões detalhadas
