Controle de ciclo de vida, configuração de frota e políticas de implantação para gerenciamento distribuído de nós.

- **Categoria**: Plataforma
- **Complexidade**: simples
- **Caminhos**: 1 | **Schemas**: 2
- **Nível**: Standard
- **Domínios relacionados**: `sites`, `system`

## Casos de Uso

- Gerenciar configuração de Malha Privada Virtual (VPM)
- Configurar ciclo de vida e gerenciamento de nós
- Monitorar status do VPM e dos nós

## Recursos Principais

- **node_config**: Configuração de nó para definições de dispositivos de borda
- **vpm_config**: Configuração de VPM para gerenciamento de plataforma virtual

## Endpoints

| Método | Caminho | Descrição |
|--------|---------|-----------|
| GET | `/api/data/namespaces/system/upgrade_status` | Status de Atualização. |
