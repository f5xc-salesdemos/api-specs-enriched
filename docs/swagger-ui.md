## About Swagger UI

Swagger UI is the industry-standard tool for interactive API documentation. It provides:

- **Interactive Testing** - Try API endpoints directly from the browser
- **Request Examples** - See example requests and responses
- **Schema Exploration** - Browse data models and schemas
- **Authentication** - Configure and test with your API credentials

## Features

- Try It Out functionality for all endpoints
- Request/Response examples with syntax highlighting
- Schema definitions with example values
- Authentication configuration
- Downloadable OpenAPI specifications
