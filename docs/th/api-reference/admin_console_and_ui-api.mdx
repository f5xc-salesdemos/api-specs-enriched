องค์ประกอบภาพที่อยู่ในขอบเขตของ Namespace พร้อมการกำหนดเวอร์ชัน การปรับใช้วิดเจ็ตแบบกำหนดเองและการจัดการแคตตาล็อกสำหรับพื้นผิวพอร์ทัล

- **หมวดหมู่**: Platform
- **ความซับซ้อน**: simple
- **เส้นทาง**: 2 | **สคีมา**: 14
- **ระดับ**: Standard
- **โดเมนที่เกี่ยวข้อง**: `admin`, `system`

## กรณีการใช้งาน

- จัดการคอมโพเนนต์ UI แบบสแตติกสำหรับคอนโซลผู้ดูแลระบบ
- ปรับใช้และดึงข้อมูลแอสเซท UI ภายใน Namespace
- กำหนดค่าองค์ประกอบอินเทอร์เฟซของคอนโซล
- จัดการเมตาดาต้าของคอมโพเนนต์ UI แบบกำหนดเอง

## ทรัพยากรหลัก

- **ui_component**: คอมโพเนนต์ UI สำหรับการปรับแต่งคอนโซล
- **static_asset**: แอสเซทแบบสแตติกสำหรับทรัพยากร UI

## เอนด์พอยต์

| เมธอด | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | แสดงรายการคอมโพเนนต์แบบสแตติกของ UI |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | ดึงข้อมูลคอมโพเนนต์แบบสแตติกของ UI |
