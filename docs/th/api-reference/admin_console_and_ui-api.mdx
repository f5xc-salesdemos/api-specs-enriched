องค์ประกอบภาพที่กำหนดขอบเขตตาม namespace พร้อมการจัดการเวอร์ชัน การปรับใช้วิดเจ็ตแบบกำหนดเองและการจัดการแคตตาล็อกสำหรับพื้นผิวพอร์ทัล

- **หมวดหมู่**: Platform
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 2 | **สคีมา**: 14
- **ระดับ**: Standard
- **โดเมนที่เกี่ยวข้อง**: `admin`, `system`

## กรณีการใช้งาน

- จัดการคอมโพเนนต์ UI แบบ static สำหรับคอนโซลผู้ดูแลระบบ
- ปรับใช้และดึงทรัพยากร UI ภายใน namespace
- กำหนดค่าองค์ประกอบอินเทอร์เฟซคอนโซล
- จัดการเมตาดาต้าคอมโพเนนต์ UI แบบกำหนดเอง

## ทรัพยากรหลัก

- **ui_component**: คอมโพเนนต์ UI สำหรับการปรับแต่งคอนโซล
- **static_asset**: ทรัพยากร static สำหรับทรัพยากร UI

## เอนด์พอยต์

| เมธอด | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | แสดงรายการคอมโพเนนต์ Static ของ UI |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | ดึงคอมโพเนนต์ static ของ UI |
