องค์ประกอบภาพที่กำหนดขอบเขตตาม Namespace พร้อมการกำหนดเวอร์ชัน การปรับใช้วิดเจ็ตแบบกำหนดเองและการจัดการแค็ตตาล็อกสำหรับพื้นผิวพอร์ทัล

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 2 | **สคีมา**: 14
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `admin`, `system`

## กรณีการใช้งาน

- จัดการส่วนประกอบ UI แบบสถิตสำหรับคอนโซลผู้ดูแลระบบ
- ปรับใช้และดึงข้อมูลทรัพย์สิน UI ภายใน Namespace
- กำหนดค่าองค์ประกอบอินเทอร์เฟซคอนโซล
- จัดการข้อมูลเมตาของส่วนประกอบ UI แบบกำหนดเอง

## ทรัพยากรหลัก

- **ui_component**: ส่วนประกอบ UI สำหรับการปรับแต่งคอนโซล
- **static_asset**: ทรัพย์สินแบบสถิตสำหรับทรัพยากร UI

## Endpoints

| วิธีการ | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | แสดงรายการส่วนประกอบ UI แบบสถิต |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | ดึงข้อมูลส่วนประกอบ UI แบบสถิต |
