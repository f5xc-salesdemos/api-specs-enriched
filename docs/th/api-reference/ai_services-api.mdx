การประมวลผลภาษาธรรมชาติพร้อมสัญญาณคุณภาพและการตรวจสอบความผิดปกติ การยืนยันตัวตนด้วย Token สำหรับการสมัครสมาชิกสตรีมข้อมูล

- **หมวดหมู่**: AI
- **ความซับซ้อน**: เรียบง่าย
- **เส้นทาง**: 11 | **สคีมา**: 66
- **ระดับ**: ขั้นสูง


## กรณีการใช้งาน

- เข้าถึงฟีเจอร์ที่ขับเคลื่อนด้วย AI
- กำหนดค่านโยบาย AI assistant
- เปิดใช้งานการตรวจจับความผิดปกติของโฟลว์
- จัดการการเก็บรวบรวมข้อมูล AI

## ทรัพยากรหลัก

- **ai_policy**: นโยบาย AI สำหรับควบคุมทราฟฟิก Generative AI
- **ai_gateway**: AI gateway สำหรับจัดการ LLM API

## Endpoints

| วิธีการ | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | ประเมินการสืบค้น AI Assistant |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | ประเมินความคิดเห็นของการสืบค้น AI Assistant |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | การสืบค้น AI Assistant |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | ความคิดเห็นของการสืบค้น AI Assistant |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | เปิดใช้งานฟีเจอร์ |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | รีเฟรช Token |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | สมัครสมาชิก BFDP pipeline |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | สมัครสมาชิก BFDP pipeline |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | ยกเลิกการสมัครสมาชิก BFDP pipeline |
| POST | `/api/gia/gia/allocateip` | จัดสรร IP ผ่าน Global IP Allocator |
| DELETE | `/api/gia/gia/deallocateip` | ยกเลิกการจัดสรร IP ผ่าน Global IP Allocator |
