การประมวลผลภาษาธรรมชาติพร้อมสัญญาณคุณภาพและการตรวจสอบความผิดปกติ การยืนยันตัวตนด้วย Token สำหรับการสมัครรับข้อมูลสตรีม

- **หมวดหมู่**: AI
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 11 | **สคีมา**: 66
- **ระดับ**: ขั้นสูง


## กรณีการใช้งาน

- เข้าถึงฟีเจอร์ที่ขับเคลื่อนด้วย AI
- กำหนดค่านโยบาย AI Assistant
- เปิดใช้งานการตรวจจับความผิดปกติของ Flow
- จัดการการเก็บรวบรวมข้อมูล AI

## ทรัพยากรหลัก

- **ai_policy**: นโยบาย AI สำหรับการควบคุมทราฟฟิก Generative AI
- **ai_gateway**: AI gateway สำหรับการจัดการ LLM API

## Endpoints

| วิธีการ | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | ประเมินการค้นหา AI Assistant |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | ประเมินข้อเสนอแนะของการค้นหา AI Assistant |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | การค้นหา AI Assistant |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | ข้อเสนอแนะของการค้นหา AI Assistant |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | เปิดใช้งานฟีเจอร์ |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | รีเฟรช Token |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | สมัครรับข้อมูล BFDP pipeline |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | สมัครรับข้อมูล BFDP pipeline |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | ยกเลิกการสมัครรับข้อมูล BFDP pipeline |
| POST | `/api/gia/gia/allocateip` | จัดสรร IP ผ่าน Global IP Allocator |
| DELETE | `/api/gia/gia/deallocateip` | ยกเลิกการจัดสรร IP ผ่าน Global IP Allocator |
