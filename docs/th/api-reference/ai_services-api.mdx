การประมวลผลภาษาธรรมชาติพร้อมสัญญาณคุณภาพและการตรวจสอบความผิดปกติ การยืนยันตัวตนด้วยโทเค็นสำหรับการสมัครรับข้อมูลสตรีม

- **หมวดหมู่**: AI
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 11 | **สคีมา**: 66
- **ระดับ**: ขั้นสูง


## กรณีการใช้งาน

- เข้าถึงฟีเจอร์ที่ขับเคลื่อนด้วย AI
- กำหนดค่านโยบายผู้ช่วย AI
- เปิดใช้งานการตรวจจับความผิดปกติของโฟลว์
- จัดการการรวบรวมข้อมูล AI

## ทรัพยากรหลัก

- **ai_policy**: นโยบาย AI สำหรับการควบคุมทราฟฟิก generative AI
- **ai_gateway**: เกตเวย์ AI สำหรับการจัดการ LLM API

## เอนด์พอยต์

| เมธอด | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | ประเมินการสืบค้นผู้ช่วย AI |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | ประเมินความคิดเห็นของการสืบค้นผู้ช่วย AI |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | การสืบค้นผู้ช่วย AI |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | ความคิดเห็นของการสืบค้นผู้ช่วย AI |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | เปิดใช้งานฟีเจอร์ |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | รีเฟรชโทเค็น |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | สมัครรับไปป์ไลน์ BFDP |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | สมัครรับไปป์ไลน์ BFDP |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | ยกเลิกการสมัครรับไปป์ไลน์ BFDP |
| POST | `/api/gia/gia/allocateip` | จัดสรร IP ผ่าน Global IP Allocator |
| DELETE | `/api/gia/gia/deallocateip` | ยกเลิกการจัดสรร IP ผ่าน Global IP Allocator |
