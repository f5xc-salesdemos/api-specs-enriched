การประมวลผลภาษาธรรมชาติพร้อมสัญญาณคุณภาพและการตรวจสอบความผิดปกติ การยืนยันตัวตนด้วยโทเค็นสำหรับการสมัครสมาชิกสตรีมข้อมูล

- **หมวดหมู่**: AI
- **ความซับซ้อน**: เรียบง่าย
- **เส้นทาง**: 11 | **สคีมา**: 66
- **ระดับ**: ขั้นสูง


## กรณีการใช้งาน

- เข้าถึงฟีเจอร์ที่ขับเคลื่อนด้วย AI
- กำหนดค่านโยบายผู้ช่วย AI
- เปิดใช้งานการตรวจจับความผิดปกติของโฟลว์
- จัดการการรวบรวมข้อมูล AI

## ทรัพยากรหลัก

- **ai_policy**: นโยบาย AI สำหรับการควบคุมทราฟฟิก generative AI
- **ai_gateway**: AI gateway สำหรับการจัดการ LLM API

## จุดปลายทาง

| เมธอด | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | ประเมินคำถามผู้ช่วย AI |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | ประเมินข้อเสนอแนะของคำถามผู้ช่วย AI |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | คำถามผู้ช่วย AI |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | ข้อเสนอแนะของคำถามผู้ช่วย AI |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | เปิดใช้งานฟีเจอร์ |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | รีเฟรชโทเค็น |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | สมัครสมาชิก BFDP pipeline |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | สมัครสมาชิก BFDP pipeline |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | ยกเลิกการสมัครสมาชิก BFDP pipeline |
| POST | `/api/gia/gia/allocateip` | จัดสรร IP ผ่าน Global IP Allocator |
| DELETE | `/api/gia/gia/deallocateip` | ยกเลิกการจัดสรร IP ผ่าน Global IP Allocator |
