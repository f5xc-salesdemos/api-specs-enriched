การจัดการอุปกรณ์แบบเดิมด้วยสคริปต์ iRule และการซิงโครไนซ์ data group การผูก virtual server และการเก็บรวบรวมเมตริก

- **หมวดหมู่**: Platform
- **ความซับซ้อน**: ขั้นสูง
- **Paths**: 29 | **Schemas**: 238
- **ระดับ**: Advanced
- **โดเมนที่เกี่ยวข้อง**: `marketplace`

## กรณีการใช้งาน

- จัดการอุปกรณ์ BigIP F5
- กำหนดค่าสคริปต์ iRule
- จัดการ data groups
- ผสานรวม BigIP CNE

## ทรัพยากรหลัก

- **bigip_pool**: พูล BIG-IP สำหรับการผสานรวมการกระจายโหลด
- **bigip_device**: การลงทะเบียนและจัดการอุปกรณ์ BIG-IP
