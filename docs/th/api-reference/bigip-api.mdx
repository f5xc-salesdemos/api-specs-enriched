การจัดการอุปกรณ์แบบดั้งเดิมด้วยสคริปต์ iRule และการซิงโครไนซ์กลุ่มข้อมูล การผูก Virtual Server และการเก็บรวบรวมเมตริก

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 29 | **สกีมา**: 238
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `marketplace`

## กรณีการใช้งาน

- จัดการอุปกรณ์ BigIP F5
- กำหนดค่าสคริปต์ iRule
- จัดการกลุ่มข้อมูล
- ผสานรวม BigIP CNE

## ทรัพยากรหลัก

- **bigip_pool**: BIG-IP pool สำหรับการผสานรวมการกระจายโหลด
- **bigip_device**: การลงทะเบียนและการจัดการอุปกรณ์ BIG-IP
