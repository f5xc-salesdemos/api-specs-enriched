การจัดการอุปกรณ์แบบเดิมด้วยสคริปต์ iRule และการซิงโครไนซ์ data group การผูก virtual server และการรวบรวมเมตริก

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 29 | **สคีมา**: 238
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `marketplace`

## กรณีการใช้งาน

- จัดการอุปกรณ์ BigIP F5
- กำหนดค่าสคริปต์ iRule
- จัดการ data groups
- รวม BigIP CNE

## ทรัพยากรหลัก

- **bigip_pool**: พูล BIG-IP สำหรับการรวมการทำ load balancing
- **bigip_device**: การลงทะเบียนและจัดการอุปกรณ์ BIG-IP
