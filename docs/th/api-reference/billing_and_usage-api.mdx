การเปลี่ยนแปลงแผน การออกใบแจ้งหนี้ และการใช้งานทรัพยากร ขีดจำกัดโควต้าระดับ Namespace และการติดตามการใช้งาน

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 25 | **สคีมา**: 101
- **ระดับ**: Standard
- **โดเมนที่เกี่ยวข้อง**: `system`, `users`

## กรณีการใช้งาน

- จัดการแผนการสมัครใช้บริการและการเปลี่ยนแปลงการเรียกเก็บเงิน
- กำหนดค่าวิธีการชำระเงินและใบแจ้งหนี้
- ติดตามการใช้งานโควต้าทรัพยากรในแต่ละ Namespace
- ตรวจสอบขีดจำกัดการใช้งานและความจุ

## ทรัพยากรหลัก

- **subscription**: การสมัครใช้บริการสำหรับสิทธิ์การใช้งาน
- **quota**: โควต้าทรัพยากรสำหรับขีดจำกัดการใช้งาน
- **usage_report**: รายงานการใช้งานสำหรับการติดตามการบริโภค
