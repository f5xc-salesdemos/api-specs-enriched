การเปลี่ยนแผน การออกใบแจ้งหนี้ และการใช้ทรัพยากร ขีดจำกัดโควตาระดับ Namespace และการติดตามการใช้งาน

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 25 | **สคีมา**: 101
- **ระดับ**: Standard
- **โดเมนที่เกี่ยวข้อง**: `system`, `users`

## กรณีการใช้งาน

- จัดการแผนการสมัครสมาชิกและการเปลี่ยนแผนการเรียกเก็บเงิน
- กำหนดค่าวิธีการชำระเงินและใบแจ้งหนี้
- ติดตามการใช้โควตาทรัพยากรในแต่ละ Namespace
- ตรวจสอบขีดจำกัดการใช้งานและความจุ

## ทรัพยากรหลัก

- **subscription**: การสมัครสมาชิกสำหรับสิทธิ์การใช้บริการ
- **quota**: โควตาทรัพยากรสำหรับขีดจำกัดการใช้งาน
- **usage_report**: รายงานการใช้งานสำหรับการติดตามการบริโภค
