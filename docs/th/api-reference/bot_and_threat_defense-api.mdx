การจำแนกประเภทภัยคุกคามด้วยการวิเคราะห์พฤติกรรมและการจับคู่ลายเซ็น การบล็อกอัตโนมัติสำหรับรูปแบบทราฟฟิกที่เป็นอันตราย

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 19 | **สกีมา**: 74
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `bot_defense`, `shape`, `virtual`

## กรณีการใช้งาน

- กำหนดค่าอินสแตนซ์การป้องกันบอทต่อเนมสเปซ
- จัดการหมวดหมู่ภัยคุกคาม TPM สำหรับการจำแนกประเภท
- จัดเตรียม API Key สำหรับระบบป้องกันอัตโนมัติ
- ผสานรวมบริการข่าวกรองภัยคุกคาม

## ทรัพยากรหลัก

- **bot_defense_instance**: อินสแตนซ์การป้องกันบอทสำหรับการปรับใช้
- **threat_category**: หมวดหมู่ภัยคุกคามสำหรับการจำแนกประเภท
