การจำแนกภัยคุกคามด้วยการวิเคราะห์พฤติกรรมและการจับคู่ลายเซ็น การบล็อกอัตโนมัติสำหรับรูปแบบทราฟฟิกที่เป็นอันตราย

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 19 | **สคีมา**: 74
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `bot_defense`, `shape`, `virtual`

## กรณีการใช้งาน

- กำหนดค่าอินสแตนซ์การป้องกันบอตต่อเนมสเปซ
- จัดการหมวดหมู่ภัยคุกคาม TPM สำหรับการจำแนกประเภท
- จัดเตรียม API key สำหรับระบบการป้องกันอัตโนมัติ
- ผสานรวมบริการข่าวกรองภัยคุกคาม

## ทรัพยากรหลัก

- **bot_defense_instance**: อินสแตนซ์การป้องกันบอตสำหรับการปรับใช้งาน
- **threat_category**: หมวดหมู่ภัยคุกคามสำหรับการจำแนกประเภท
