การกระจายทั่วโลกพร้อมกฎการแคชและการดำเนินการล้างแคช การตรวจสอบประสิทธิภาพและการวิเคราะห์

- **หมวดหมู่**: Networking
- **ความซับซ้อน**: ขั้นสูง
- **Paths**: 29 | **Schemas**: 592
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `virtual`

## กรณีการใช้งาน

- กำหนดค่าการทำ CDN load balancing
- จัดการบริการเครือข่ายจัดส่งเนื้อหา
- กำหนดค่านโยบายการแคช
- จัดการการจัดส่งและการกระจายข้อมูล

## ทรัพยากรหลัก

- **cdn_loadbalancer**: ตัวกระจายโหลดเครือข่ายจัดส่งเนื้อหาสำหรับการแคชที่ขอบเครือข่าย
- **cdn_origin_pool**: กลุ่มเซิร์ฟเวอร์ต้นทางสำหรับแหล่งเนื้อหา CDN
