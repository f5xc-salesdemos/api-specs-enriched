การกระจายทั่วโลกพร้อมกฎแคชและการดำเนินการล้างแคช การตรวจสอบประสิทธิภาพและการวิเคราะห์

- **หมวดหมู่**: เครือข่าย
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 29 | **สคีมา**: 592
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `virtual`

## กรณีการใช้งาน

- กำหนดค่าการกระจายโหลด CDN
- จัดการบริการเครือข่ายการจัดส่งเนื้อหา
- กำหนดค่านโยบายแคช
- จัดการการจัดส่งและกระจายข้อมูล

## ทรัพยากรหลัก

- **cdn_loadbalancer**: ตัวกระจายโหลดเครือข่ายการจัดส่งเนื้อหาสำหรับแคชที่ขอบเครือข่าย
- **cdn_origin_pool**: พูลเซิร์ฟเวอร์ต้นทางสำหรับการดึงเนื้อหา CDN
