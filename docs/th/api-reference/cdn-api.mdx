การกระจายระดับโลกพร้อมกฎการแคชและการดำเนินการล้างแคช การตรวจสอบประสิทธิภาพและการวิเคราะห์

- **หมวดหมู่**: Networking
- **ความซับซ้อน**: ขั้นสูง
- **Paths**: 29 | **Schemas**: 592
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `virtual`

## กรณีการใช้งาน

- กำหนดค่าการทำโหลดบาลานซ์ CDN
- จัดการบริการเครือข่ายจัดส่งเนื้อหา
- กำหนดค่านโยบายการแคช
- จัดการการจัดส่งและการกระจายข้อมูล

## ทรัพยากรหลัก

- **cdn_loadbalancer**: โหลดบาลานเซอร์เครือข่ายจัดส่งเนื้อหาสำหรับการแคชที่ Edge
- **cdn_origin_pool**: พูลเซิร์ฟเวอร์ต้นทางสำหรับการดึงเนื้อหา CDN
