การจัดเตรียมแบบใช้โทเค็นพร้อมการดาวน์โหลดอิมเมจและการตรวจสอบก่อนอัปเกรด การจัดกลุ่มฟลีตช่วยให้สามารถดำเนินการจำนวนมากข้ามสถานที่ที่กระจายตัวได้

- **หมวดหมู่**: โครงสร้างพื้นฐาน
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 28 | **สคีมา**: 231
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `customer_edge`, `sites`

## กรณีการใช้งาน

- จัดการวงจรชีวิตของไซต์ Customer Edge
- กำหนดค่าอินเทอร์เฟซเครือข่ายและการตั้งค่าฟลีต
- จัดการการลงทะเบียนไซต์และเวิร์กโฟลว์โทเค็น
- ดำเนินการอัปเกรดไซต์พร้อมการตรวจสอบก่อนอัปเกรด

## ทรัพยากรหลัก

- **site_config**: การกำหนดค่าไซต์สำหรับการตั้งค่าโหนดเอดจ์
- **fleet_config**: การกำหนดค่าฟลีตสำหรับการจัดการหลายไซต์
- **registration_token**: โทเค็นการลงทะเบียนสำหรับการเริ่มต้นใช้งานไซต์
