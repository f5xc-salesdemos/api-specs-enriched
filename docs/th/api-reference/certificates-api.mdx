ห่วงโซ่ใบรับรองและชุด CA ที่เชื่อถือได้ การจัดการรายการเพิกถอนและการกำหนดค่าไฟล์แมนิเฟสต์สำหรับการดำเนินการ PKI

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 16 | **สคีมา**: 74
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `blindfold`, `system`

## กรณีการใช้งาน

- จัดการใบรับรอง SSL/TLS
- กำหนดค่า CA ที่เชื่อถือได้
- จัดการรายการเพิกถอนใบรับรอง (CRL)
- กำหนดค่าไฟล์แมนิเฟสต์ใบรับรอง

## ทรัพยากรหลัก

- **certificate**: ใบรับรอง TLS สำหรับการเชื่อมต่อ HTTPS ที่ปลอดภัย
- **ca_certificate**: ใบรับรองจากผู้ออกใบรับรอง (CA) สำหรับห่วงโซ่ความเชื่อถือ
- **certificate_chain**: ห่วงโซ่ใบรับรองสำหรับการตรวจสอบความเชื่อถือแบบสมบูรณ์
