เชนใบรับรองและชุด CA ที่เชื่อถือได้ การจัดการรายการเพิกถอนและการกำหนดค่า manifest สำหรับการดำเนินงาน PKI

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 16 | **สคีมา**: 74
- **ระดับ**: Standard
- **โดเมนที่เกี่ยวข้อง**: `blindfold`, `system`

## กรณีการใช้งาน

- จัดการใบรับรอง SSL/TLS
- กำหนดค่า CA ที่เชื่อถือได้
- จัดการรายการเพิกถอนใบรับรอง (CRL)
- กำหนดค่า manifest ใบรับรอง

## ทรัพยากรหลัก

- **certificate**: ใบรับรอง TLS สำหรับการเชื่อมต่อ HTTPS ที่ปลอดภัย
- **ca_certificate**: ใบรับรองผู้ออกใบรับรอง (CA) สำหรับเชนความเชื่อถือ
- **certificate_chain**: เชนใบรับรองสำหรับการตรวจสอบความเชื่อถือแบบสมบูรณ์
