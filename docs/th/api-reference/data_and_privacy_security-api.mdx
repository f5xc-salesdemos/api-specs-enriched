นโยบายข้อมูลที่ละเอียดอ่อนพร้อมกฎการจำแนกประเภทแบบกำหนดเอง การกำหนดค่าภูมิภาค LMA และการควบคุมการปฏิบัติตามกฎระเบียบตามพื้นที่ทางภูมิศาสตร์

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 11 | **สคีมา**: 65
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `blindfold`, `client_side_defense`

## กรณีการใช้งาน

- กำหนดค่านโยบายการตรวจจับข้อมูลที่ละเอียดอ่อน
- กำหนดประเภทข้อมูลแบบกำหนดเองสำหรับการจำแนกประเภท PII
- จัดการการกำหนดค่าภูมิภาค LMA
- รวม geo-configurations สำหรับการปฏิบัติตามกฎระเบียบ

## ทรัพยากรหลัก

- **sensitive_data_policy**: นโยบายข้อมูลที่ละเอียดอ่อนสำหรับการป้องกัน PII
- **data_classification**: การจำแนกประเภทข้อมูลสำหรับการจัดหมวดหมู่เนื้อหา
