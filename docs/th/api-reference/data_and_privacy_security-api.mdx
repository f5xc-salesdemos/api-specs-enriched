นโยบายข้อมูลที่ละเอียดอ่อนพร้อมกฎการจำแนกประเภทแบบกำหนดเอง การกำหนดค่าภูมิภาค LMA และการควบคุมการปฏิบัติตามข้อกำหนดตามพื้นที่ทางภูมิศาสตร์

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: simple
- **เส้นทาง**: 11 | **สคีมา**: 65
- **ระดับ**: Advanced
- **โดเมนที่เกี่ยวข้อง**: `blindfold`, `client_side_defense`

## กรณีการใช้งาน

- กำหนดค่านโยบายการตรวจจับข้อมูลที่ละเอียดอ่อน
- กำหนดประเภทข้อมูลแบบกำหนดเองสำหรับการจำแนกประเภท PII
- จัดการการกำหนดค่าภูมิภาค LMA
- รวมการกำหนดค่าทางภูมิศาสตร์เพื่อการปฏิบัติตามข้อกำหนด

## ทรัพยากรหลัก

- **sensitive_data_policy**: นโยบายข้อมูลที่ละเอียดอ่อนสำหรับการปกป้อง PII
- **data_classification**: การจำแนกประเภทข้อมูลสำหรับการแบ่งหมวดหมู่เนื้อหา
