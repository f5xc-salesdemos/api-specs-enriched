กฎการจำแนกประเภท นโยบายทรัพยากร และการสร้างข้อมูลเชิงลึกสำหรับเวิร์กโฟลว์การวิเคราะห์ข้อมูล

- **หมวดหมู่**: Operations
- **ความซับซ้อน**: simple
- **Paths**: 15 | **Schemas**: 73
- **ระดับ**: Standard
- **โดเมนที่เกี่ยวข้อง**: `statistics`, `observability`

## กรณีการใช้งาน

- วิเคราะห์ข้อมูลความปลอดภัยและทราฟฟิก
- สร้างข้อมูลเชิงลึกอัจฉริยะจากล็อก
- กำหนดค่านโยบายการวิเคราะห์ข้อมูล

## ทรัพยากรหลัก

- **analytics_query**: คิวรีการวิเคราะห์สำหรับการวิเคราะห์ข้อมูล
- **data_export**: การกำหนดค่าการส่งออกข้อมูลสำหรับการรายงาน
