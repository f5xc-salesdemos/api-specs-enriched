รายการปฏิเสธ กลุ่มกฎไฟร์วอลล์ และมาตรการป้องกันที่ใช้อุโมงค์ การจำกัดอัตราและการวิเคราะห์รูปแบบสำหรับความปลอดภัยของขอบเขตเครือข่าย

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 60 | **สคีมา**: 246
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `network_security`, `virtual`

## กรณีการใช้งาน

- กำหนดค่านโยบายการป้องกัน DDoS
- ติดตามและวิเคราะห์ภัยคุกคาม DDoS
- กำหนดค่าการป้องกันโครงสร้างพื้นฐาน

## ทรัพยากรหลัก

- **ddos_protection**: การกำหนดค่าการป้องกัน DDoS สำหรับการทำความสะอาดทราฟฟิก
- **ddos_mitigation_rule**: กฎการบรรเทา DDoS สำหรับการตอบสนองต่อการโจมตี
