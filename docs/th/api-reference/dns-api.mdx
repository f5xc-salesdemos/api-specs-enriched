การแก้ไขชื่อพร้อมการถ่ายโอนโซนและการตรวจสอบสุขภาพ รองรับประเภทระเบียนและการมอบหมาย

- **หมวดหมู่**: เครือข่าย
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 49 | **สคีมา**: 285
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `virtual`, `network`

## กรณีการใช้งาน

- กำหนดค่าการกระจายโหลด DNS
- จัดการโซนและโดเมน DNS
- กำหนดค่านโยบายการปฏิบัติตามข้อกำหนด DNS
- จัดการชุดระเบียนทรัพยากร (RRSets)

## ทรัพยากรหลัก

- **dns_zone**: โซน DNS ที่มีอำนาจพร้อมความสามารถในการจัดการระเบียน
- **dns_domain**: การมอบหมายโดเมน DNS และการตั้งค่าการกำหนดค่า
- **dns_load_balancer**: การกระจายทราฟฟิกผ่าน DNS พร้อมการกำหนดเส้นทางตามภูมิศาสตร์และการสำรอง
