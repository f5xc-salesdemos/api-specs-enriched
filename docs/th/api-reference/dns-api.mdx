การแก้ไขชื่อด้วยการถ่ายโอนโซนและการตรวจสอบสถานะ รองรับประเภทเรคคอร์ดและการมอบหมาย

- **หมวดหมู่**: เครือข่าย
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 49 | **สคีมา**: 285
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `virtual`, `network`

## กรณีการใช้งาน

- กำหนดค่าการกระจายโหลด DNS
- จัดการโซนและโดเมน DNS
- กำหนดค่านโยบายความสอดคล้อง DNS
- จัดการชุดเรคคอร์ดทรัพยากร (RRSets)

## ทรัพยากรหลัก

- **dns_zone**: โซน DNS ที่มีอำนาจพร้อมความสามารถในการจัดการเรคคอร์ด
- **dns_domain**: การมอบหมายโดเมน DNS และการตั้งค่าการกำหนดค่า
- **dns_load_balancer**: การกระจายทราฟฟิกผ่าน DNS พร้อมการกำหนดเส้นทางตามภูมิศาสตร์และการสำรอง
