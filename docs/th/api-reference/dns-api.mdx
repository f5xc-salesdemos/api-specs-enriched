การแก้ไขชื่อพร้อมการถ่ายโอนโซนและการตรวจสอบสุขภาพ ประเภทระเบียนและการรองรับการมอบหมาย

- **หมวดหมู่**: เครือข่าย
- **ความซับซ้อน**: ขั้นสูง
- **Paths**: 49 | **Schemas**: 285
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `virtual`, `network`

## กรณีการใช้งาน

- กำหนดค่าการกระจายโหลด DNS
- จัดการโซน DNS และโดเมน
- กำหนดค่านโยบายการปฏิบัติตาม DNS
- จัดการชุดระเบียนทรัพยากร (RRSets)

## ทรัพยากรหลัก

- **dns_zone**: โซน DNS ที่มีสิทธิ์พร้อมความสามารถในการจัดการระเบียน
- **dns_domain**: การมอบหมายโดเมน DNS และการตั้งค่าการกำหนดค่า
- **dns_load_balancer**: การกระจายทราฟฟิกแบบ DNS พร้อมการกำหนดเส้นทางตามภูมิศาสตร์และการสำรอง
