การจัดเตรียม GRE และ IPSec tunnel ของบุคคลที่สามพร้อมตัวจับเวลา DPD การจัดสรรทรัพยากรที่แชร์ข้ามเนมสเปซพร้อมการควบคุมการวางไทล์

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 36 | **สคีมา**: 183
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `bigip`, `admin`

## กรณีการใช้งาน

- เข้าถึงการผสานรวมและส่วนเสริมของบุคคลที่สาม
- จัดการส่วนขยายตลาดกลาง
- กำหนดค่าการผสานรวม Terraform และภายนอก
- จัดการนโยบาย TPM

## ทรัพยากรหลัก

- **marketplace_item**: รายการตลาดกลางสำหรับแคตาล็อกบริการ
- **subscription**: การสมัครสมาชิกสำหรับสิทธิ์การใช้บริการ
