การจัดเตรียมอุโมงค์ GRE และ IPSec ของบุคคลที่สามพร้อมตัวจับเวลา DPD การจัดสรรทรัพยากรร่วมข้าม namespace พร้อมการควบคุมการวางตำแหน่ง tile

- **หมวดหมู่**: Platform
- **ความซับซ้อน**: ปานกลาง
- **Paths**: 36 | **Schemas**: 183
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `bigip`, `admin`

## กรณีการใช้งาน

- เข้าถึงการผสานรวมและส่วนเสริมของบุคคลที่สาม
- จัดการส่วนขยาย marketplace
- กำหนดค่า Terraform และการผสานรวมภายนอก
- จัดการนโยบาย TPM

## ทรัพยากรหลัก

- **marketplace_item**: รายการ marketplace สำหรับแคตตาล็อกบริการ
- **subscription**: การสมัครสมาชิกสำหรับสิทธิ์การใช้บริการ
