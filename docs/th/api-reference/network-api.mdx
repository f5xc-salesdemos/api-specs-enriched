โปรโตคอล Border Gateway พร้อมการจัดการ ASN และความสัมพันธ์ของระบบอัตโนมัส VPN แบบ Site-to-Site เชื่อมต่อศูนย์ข้อมูลผ่านช่องทางที่เข้ารหัส

- **หมวดหมู่**: เครือข่าย
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 81 | **สคีมา**: 449
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `virtual`, `network_security`, `dns`

## กรณีการใช้งาน

- กำหนดค่าการเราต์ BGP และการจัดการ ASN
- จัดการอุโมงค์ IPsec และเฟส IKE
- กำหนดค่าตัวเชื่อมต่อเครือข่ายและเส้นทาง
- จัดการ SRv6 และการแบ่งซับเน็ต
- กำหนดการเชื่อมต่อเซกเมนต์และนโยบาย
- กำหนดค่าชุดคำนำหน้า IP

## ทรัพยากรหลัก

- **virtual_network**: เครือข่ายเสมือนสำหรับการเชื่อมต่อไซต์และการแบ่งส่วน
- **network_connector**: ตัวเชื่อมต่อเครือข่ายสำหรับการเชื่อมต่อแบบ Site-to-Site หรือคลาวด์
- **site_mesh_group**: การกำหนดค่าการเชื่อมต่อแบบ Mesh ระหว่างหลายไซต์
