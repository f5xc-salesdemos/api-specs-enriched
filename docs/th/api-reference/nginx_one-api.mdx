การค้นหาอินสแตนซ์ การผสานรวม WAF และการเชื่อมต่อ service mesh การจัดการวงจรชีวิตการสมัครสมาชิกและการซิงโครไนซ์การกำหนดค่า

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 13 | **สคีมา**: 59
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `marketplace`

## กรณีการใช้งาน

- จัดการการผสานรวมแพลตฟอร์ม NGINX One
- กำหนดค่าอินสแตนซ์ NGINX Plus
- ผสานรวมการจัดการการกำหนดค่า NGINX

## ทรัพยากรหลัก

- **nginx_config**: การกำหนดค่า NGINX สำหรับการตั้งค่าพร็อกซี
- **nginx_upstream**: NGINX upstream สำหรับพูลเซิร์ฟเวอร์แบ็กเอนด์
