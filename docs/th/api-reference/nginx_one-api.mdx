การค้นหาอินสแตนซ์, การรวม WAF และการเชื่อมต่อ service mesh การจัดการวงจรชีวิตการสมัครสมาชิกและการซิงโครไนซ์การกำหนดค่า

- **หมวดหมู่**: Platform
- **ความซับซ้อน**: ง่าย
- **Paths**: 13 | **Schemas**: 59
- **ระดับ**: Advanced
- **โดเมนที่เกี่ยวข้อง**: `marketplace`

## กรณีการใช้งาน

- จัดการการรวม NGINX One platform
- กำหนดค่าอินสแตนซ์ NGINX Plus
- รวมการจัดการการกำหนดค่า NGINX

## ทรัพยากรหลัก

- **nginx_config**: การกำหนดค่า NGINX สำหรับการตั้งค่าพร็อกซี
- **nginx_upstream**: NGINX upstream สำหรับกลุ่มเซิร์ฟเวอร์แบ็กเอนด์
