การค้นพบอินสแตนซ์ การผสานรวม WAF และการเชื่อมต่อเซอร์วิสเมช วงจรชีวิตการสมัครสมาชิกและการซิงโครไนซ์คอนฟิกูเรชัน

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 13 | **สกีมา**: 59
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `marketplace`

## กรณีการใช้งาน

- จัดการการผสานรวมแพลตฟอร์ม NGINX One
- กำหนดค่าอินสแตนซ์ NGINX Plus
- ผสานรวมการจัดการคอนฟิกูเรชัน NGINX

## ทรัพยากรหลัก

- **nginx_config**: คอนฟิกูเรชัน NGINX สำหรับการตั้งค่าพร็อกซี
- **nginx_upstream**: NGINX upstream สำหรับพูลเซิร์ฟเวอร์แบ็กเอนด์

## เอนด์พอยต์

| เมธอด | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_csgs` | แสดงรายการคอนฟิกูเรชันออบเจกต์ NGINX One CSG |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_csgs/&#123;name&#125;` | คำร้อง GET |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_instances` | แสดงรายการคอนฟิกูเรชันออบเจกต์อินสแตนซ์ NGINX One |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_instances/&#123;name&#125;` | คำร้อง GET |
| POST | `/api/config/namespaces/&#123;namespace&#125;/nginx_dataplane_servers` | ดึงข้อมูลเซิร์ฟเวอร์ดาต้าเพลน NGINX One |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_servers` | แสดงรายการ API เซิร์ฟเวอร์ NGINX One |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_servers/&#123;name&#125;` | คำร้อง GET |
| POST | `/api/nginx/one/namespaces/system/n1/subscribe` | สมัครสมาชิก NGINX One |
| POST | `/api/nginx/one/namespaces/system/n1/unsubscribe` | ยกเลิกการสมัครสมาชิก NGINX One |
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/nginx_service_discoverys` | สร้าง NGINX Service Discovery |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/nginx_service_discoverys/&#123;metadata.name&#125;` | แทนที่ NGINX Service Discovery |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_service_discoverys` | แสดงรายการ NGINX Service Discovery |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_service_discoverys/&#123;name&#125;` | ดึงข้อมูล NGINX Service Discovery |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/nginx_service_discoverys/&#123;name&#125;` | ลบ NGINX Service Discovery |
