การกระจายไลบรารีที่มีการกำหนดเวอร์ชันสำหรับผู้รวมระบบแอปมือถือ Presigned URLs ช่วยให้สามารถดาวน์โหลดได้อย่างปลอดภัยพร้อมบิลด์เฉพาะระบบปฏิบัติการสำหรับ iOS และ Android

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 9 | **สคีมา**: 17
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `marketplace`

## กรณีการใช้งาน

- จัดการบริการ Object Storage
- กำหนดค่าออบเจกต์และบัคเก็ตที่จัดเก็บ
- จัดการนโยบายการจัดเก็บ

## ทรัพยากรหลัก

- **object_store**: Object Store สำหรับการจัดการ Blob Storage
- **bucket**: Storage Bucket สำหรับ Object Storage
