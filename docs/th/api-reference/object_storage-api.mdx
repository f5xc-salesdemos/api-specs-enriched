การแจกจ่ายไลบรารีแบบเวอร์ชันสำหรับผู้ผนวกรวมแอปมือถือ Presigned URL ช่วยให้สามารถดาวน์โหลดได้อย่างปลอดภัยพร้อมบิลด์เฉพาะระบบปฏิบัติการสำหรับ iOS และ Android

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 9 | **สกีมา**: 17
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `marketplace`

## กรณีการใช้งาน

- จัดการบริการพื้นที่จัดเก็บอ็อบเจกต์
- กำหนดค่าอ็อบเจกต์และบัคเก็ตที่จัดเก็บ
- จัดการนโยบายการจัดเก็บ

## ทรัพยากรหลัก

- **object_store**: พื้นที่จัดเก็บอ็อบเจกต์สำหรับจัดการ blob storage
- **bucket**: บัคเก็ตจัดเก็บสำหรับพื้นที่จัดเก็บอ็อบเจกต์
