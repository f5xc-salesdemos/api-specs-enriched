การแจกจ่ายไลบรารีที่มีเวอร์ชันสำหรับผู้รวมระบบแอปมือถือ Presigned URLs ช่วยให้สามารถดาวน์โหลดอย่างปลอดภัยพร้อมบิลด์เฉพาะระบบปฏิบัติการสำหรับ iOS และ Android

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 9 | **สคีมา**: 17
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `marketplace`

## กรณีการใช้งาน

- จัดการบริการจัดเก็บวัตถุ
- กำหนดค่าวัตถุที่จัดเก็บและบัคเก็ต
- จัดการนโยบายการจัดเก็บ

## ทรัพยากรหลัก

- **object_store**: ที่จัดเก็บวัตถุสำหรับการจัดการ blob storage
- **bucket**: บัคเก็ตจัดเก็บสำหรับ object storage
