โพรบความพร้อมใช้งาน HTTP พร้อมการวัดค่าเวลาแฝง การแจ้งเตือนการหมดอายุของใบรับรอง และแดชบอร์ดสถานะระดับโกลบอลสำหรับสุขภาพโครงสร้างพื้นฐาน

- **หมวดหมู่**: การดำเนินงาน
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 56 | **สคีมา**: 196
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `statistics`, `support`

## กรณีการใช้งาน

- กำหนดค่าการตรวจสอบแบบสังเคราะห์
- กำหนดนโยบายการตรวจสอบและการทดสอบ
- จัดการแดชบอร์ดการสังเกตการณ์

## ทรัพยากรหลัก

- **log_receiver**: ตัวรับล็อกสำหรับการรวบรวมล็อกแบบรวมศูนย์
- **metrics_receiver**: ตัวรับเมตริกสำหรับการรวบรวมเมตริกแบบรวมศูนย์
- **alert_policy**: นโยบายการแจ้งเตือนสำหรับการตรวจสอบและการแจ้งเตือน
