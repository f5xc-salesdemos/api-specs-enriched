โพรบความพร้อมใช้งาน HTTP พร้อมการวัดค่าความหน่วง การแจ้งเตือนวันหมดอายุของใบรับรอง และแดชบอร์ดสถานะทั่วโลกสำหรับสุขภาพโครงสร้างพื้นฐาน

- **หมวดหมู่**: การดำเนินการ
- **ความซับซ้อน**: ปานกลาง
- **Paths**: 56 | **Schemas**: 196
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `statistics`, `support`

## กรณีการใช้งาน

- กำหนดค่าการตรวจสอบแบบ Synthetic
- กำหนดนโยบายการตรวจสอบและการทดสอบ
- จัดการแดชบอร์ดการสังเกตการณ์

## ทรัพยากรหลัก

- **log_receiver**: ตัวรับบันทึกสำหรับการรวบรวมบันทึกแบบรวมศูนย์
- **metrics_receiver**: ตัวรับเมตริกสำหรับการรวบรวมเมตริกแบบรวมศูนย์
- **alert_policy**: นโยบายการแจ้งเตือนสำหรับการตรวจสอบและการแจ้งเตือน
