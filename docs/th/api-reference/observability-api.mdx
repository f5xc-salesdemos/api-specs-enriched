HTTP availability probes พร้อมการวัดค่าความหน่วงเวลา การแจ้งเตือนการหมดอายุของใบรับรอง และแดชบอร์ดสถานะทั่วโลกสำหรับการตรวจสอบสุขภาพโครงสร้างพื้นฐาน

- **Category**: Operations
- **Complexity**: moderate
- **Paths**: 56 | **Schemas**: 196
- **Tier**: Standard
- **Related domains**: `statistics`, `support`

## กรณีการใช้งาน

- กำหนดค่าการตรวจสอบแบบสังเคราะห์
- กำหนดนโยบายการตรวจสอบและการทดสอบ
- จัดการแดชบอร์ดการสังเกตการณ์

## ทรัพยากรหลัก

- **log_receiver**: ตัวรับล็อกสำหรับการรวบรวมล็อกแบบรวมศูนย์
- **metrics_receiver**: ตัวรับเมตริกสำหรับการรวบรวมเมตริกแบบรวมศูนย์
- **alert_policy**: นโยบายการแจ้งเตือนสำหรับการตรวจสอบและการแจ้งเตือน
