การตรวจสอบความพร้อมใช้งาน HTTP พร้อมการวัดค่าความหน่วง การแจ้งเตือนการหมดอายุของใบรับรอง และแดชบอร์ดสถานะแบบรวมสำหรับสุขภาพของโครงสร้างพื้นฐาน

- **หมวดหมู่**: การดำเนินงาน
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 56 | **สคีมา**: 196
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `statistics`, `support`

## กรณีการใช้งาน

- กำหนดค่าการตรวจสอบแบบสังเคราะห์
- กำหนดนโยบายการตรวจสอบและการทดสอบ
- จัดการแดชบอร์ดการสังเกตการณ์ระบบ

## ทรัพยากรหลัก

- **log_receiver**: ตัวรับบันทึกสำหรับการรวบรวมบันทึกแบบรวมศูนย์
- **metrics_receiver**: ตัวรับเมตริกสำหรับการรวบรวมเมตริกแบบรวมศูนย์
- **alert_policy**: นโยบายการแจ้งเตือนสำหรับการตรวจสอบและการแจ้งเตือน
