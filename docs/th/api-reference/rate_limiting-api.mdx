การบังคับใช้โควต้าตามเวลาพร้อมช่วงเวลาที่กำหนดค่าได้เป็นชั่วโมง นาที หรือวินาที การควบคุมเฉพาะโปรโตคอลสำหรับการจัดรูปแบบทราฟฟิก

- **หมวดหมู่**: เครือข่าย
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 12 | **สคีมา**: 70
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `virtual`, `network_security`

## กรณีการใช้งาน

- กำหนดค่านโยบาย rate limiter
- จัดการการกำหนดค่า policer
- ควบคุมการไหลของทราฟฟิกและการจัดคิว

## ทรัพยากรหลัก

- **rate_limiter**: Rate limiter สำหรับการควบคุมและป้องกันทราฟฟิก
- **rate_limiter_policy**: นโยบาย rate limiter พร้อมกฎการควบคุมอัตราการรับส่งข้อมูลโดยละเอียด
- **rate_limit_threshold**: การกำหนดค่าเกณฑ์ rate limit สำหรับการควบคุมทราฟฟิก
