การบังคับใช้โควตาตามเวลาพร้อมหน้าต่างเวลาที่กำหนดค่าได้เป็นชั่วโมง นาที หรือวินาที การควบคุมเฉพาะโปรโตคอลสำหรับการจัดรูปแบบทราฟฟิก

- **หมวดหมู่**: เครือข่าย
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 12 | **สคีมา**: 70
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `virtual`, `network_security`

## กรณีการใช้งาน

- กำหนดค่านโยบายตัวจำกัดอัตรา
- จัดการการกำหนดค่า policer
- ควบคุมการไหลของทราฟฟิกและการจัดคิว

## ทรัพยากรหลัก

- **rate_limiter**: ตัวจำกัดอัตราสำหรับการควบคุมทราฟฟิกและการป้องกัน
- **rate_limiter_policy**: นโยบายตัวจำกัดอัตราพร้อมกฎการควบคุมโดยละเอียด
- **rate_limit_threshold**: การกำหนดค่าเกณฑ์การจำกัดอัตราสำหรับการควบคุมทราฟฟิก
