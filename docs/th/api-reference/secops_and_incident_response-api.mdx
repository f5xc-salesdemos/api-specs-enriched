การบรรเทาผลกระทบจากผู้ใช้ที่เป็นอันตรายด้วยการจำแนกระดับภัยคุกคาม การดำเนินการตอบสนองอัตโนมัติสำหรับรูปแบบพฤติกรรมที่น่าสงสัย

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 4 | **สคีมา**: 32
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `bot_defense`, `virtual`, `network_security`

## กรณีการใช้งาน

- กำหนดค่านโยบายการบรรเทาภัยคุกคามอัตโนมัติ
- กำหนดกฎสำหรับการตรวจจับผู้ใช้ที่เป็นอันตราย
- จัดการเวิร์กโฟลว์การตอบสนองต่อเหตุการณ์
- ใช้การบล็อกหรือการจำกัดอัตราต่อภัยคุกคาม

## ทรัพยากรหลัก

- **mitigation_policy**: นโยบายการบรรเทาผลกระทบสำหรับการตอบสนองต่อเหตุการณ์
- **malicious_user_rule**: กฎผู้ใช้ที่เป็นอันตรายสำหรับการบรรเทาภัยคุกคาม

| DELETE | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | ลบ Malicious User Mitigation |
