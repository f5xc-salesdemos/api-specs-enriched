การลดความเสี่ยงจากผู้ใช้ที่เป็นอันตรายด้วยการจำแนกระดับภัยคุกคาม การดำเนินการตอบสนองอัตโนมัติสำหรับรูปแบบพฤติกรรมที่น่าสงสัย

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 4 | **สคีมา**: 32
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `bot_defense`, `virtual`, `network_security`

## กรณีการใช้งาน

- กำหนดนโยบายการลดความเสี่ยงจากภัยคุกคามอัตโนมัติ
- กำหนดกฎสำหรับการตรวจจับผู้ใช้ที่เป็นอันตราย
- จัดการขั้นตอนการตอบสนองต่อเหตุการณ์
- ใช้การบล็อกหรือการจำกัดอัตราสำหรับภัยคุกคาม

## ทรัพยากรหลัก

- **mitigation_policy**: นโยบายการลดความเสี่ยงสำหรับการตอบสนองต่อเหตุการณ์
- **malicious_user_rule**: กฎผู้ใช้ที่เป็นอันตรายสำหรับการลดความเสี่ยงจากภัยคุกคาม

## ปลายทาง

| วิธีการ | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations` | สร้างการลดความเสี่ยงจากผู้ใช้ที่เป็นอันตราย |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations/&#123;metadata.name&#125;` | แทนที่การลดความเสี่ยงจากผู้ใช้ที่เป็นอันตราย |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations` | แสดงรายการการลดความเสี่ยงจากผู้ใช้ที่เป็นอันตราย |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | ดึงข้อมูลการลดความเสี่ยงจากผู้ใช้ที่เป็นอันตราย |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | ลบการลดความเสี่ยงจากผู้ใช้ที่เป็นอันตราย |
