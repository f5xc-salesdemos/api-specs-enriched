คำจำกัดความประเภทแอปพลิเคชันพร้อมการค้นพบและการวิเคราะห์สคีมาที่เรียนรู้ การอนุมานรูปแบบการรับส่งข้อมูลสำหรับการจัดการคำขออัจฉริยะ

- **หมวดหมู่**: โครงสร้างพื้นฐาน
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 38 | **สคีมา**: 247
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `managed_kubernetes`, `container_services`, `virtual`

## กรณีการใช้งาน

- กำหนดค่าการเชื่อมต่อ service mesh
- จัดการการค้นพบ endpoint และการกำหนดเส้นทาง
- กำหนดค่าบริการ NFV
- กำหนดการตั้งค่าและประเภทแอปพลิเคชัน

## ทรัพยากรหลัก

- **endpoint**: endpoint ของ service mesh สำหรับการกำหนดเส้นทางการรับส่งข้อมูล
- **origin_pool**: กลุ่มเซิร์ฟเวอร์แบ็กเอนด์สำหรับการกระจายการรับส่งข้อมูลของ load balancer
- **service_discovery**: การกำหนดค่าการค้นพบบริการสำหรับ endpoint แบบไดนามิก
