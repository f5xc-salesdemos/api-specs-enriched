การจดจำภัยคุกคามด้วยการวิเคราะห์พฤติกรรมและการพิมพ์ลายนิ้วมืออุปกรณ์ การรวม Mobile SDK สำหรับการป้องกันแอปพลิเคชัน

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 271 | **สคีมา**: 813
- **ระดับ**: ขั้นสูง
- **โดเมนที่เกี่ยวข้อง**: `bot_defense`, `virtual`

## กรณีการใช้งาน

- กำหนดค่านโยบาย Shape Security
- จัดการการป้องกันบอทและภัยคุกคาม
- กำหนดค่านโยบาย SafeAP
- เปิดใช้งานการจดจำภัยคุกคาม

## ทรัพยากรหลัก

- **shape_app_firewall**: ไฟร์วอลล์แอปพลิเคชัน Shape สำหรับการป้องกันขั้นสูง
- **shape_recognizer**: ตัวจดจำ Shape สำหรับการวิเคราะห์รูปแบบทราฟฟิก
