นโยบายการแจ้งเตือนพร้อมการเชื่อมต่อกับตัวรับ การรวบรวมบันทึก มุมมองโทโพโลยี และการติดตามสุขภาพไซต์

- **หมวดหมู่**: การดำเนินงาน
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 77 | **สคีมา**: 575
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `observability`, `support`

## กรณีการใช้งาน

- เข้าถึงสถิติโฟลว์และการวิเคราะห์
- จัดการการแจ้งเตือนและนโยบายการแจ้งเตือน
- ดูบันทึกและตัวรับบันทึก
- สร้างรายงานและกราฟ
- ติดตามโทโพโลยีและการค้นพบบริการ
- ตรวจสอบสถานะที่ไซต์

## ทรัพยากรหลัก

- **dashboard**: แดชบอร์ดสำหรับการแสดงผลเมตริก
- **saved_query**: คิวรีที่บันทึกไว้สำหรับการวิเคราะห์เมตริกและบันทึก
