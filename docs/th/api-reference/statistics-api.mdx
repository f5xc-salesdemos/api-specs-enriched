นโยบายการแจ้งเตือนพร้อมการผสานรวมกับผู้รับ การรวบรวมบันทึก มุมมองโทโปโลยี และการติดตามสุขภาพของไซต์

- **Category**: Operations
- **Complexity**: advanced
- **Paths**: 77 | **Schemas**: 575
- **Tier**: Standard
- **Related domains**: `observability`, `support`

## กรณีการใช้งาน

- เข้าถึงสถิติการไหลของข้อมูลและการวิเคราะห์
- จัดการการแจ้งเตือนและนโยบายการแจ้งเตือน
- ดูบันทึกและผู้รับบันทึก
- สร้างรายงานและกราฟ
- ติดตามโทโปโลยีและการค้นพบบริการ
- ตรวจสอบสถานะที่ไซต์ต่างๆ

## ทรัพยากรหลัก

- **dashboard**: แดชบอร์ดสำหรับการแสดงภาพเมตริก
- **saved_query**: คิวรีที่บันทึกไว้สำหรับการวิเคราะห์เมตริกและบันทึก
