นโยบายการแจ้งเตือนพร้อมการผสานรวมกับตัวรับ การรวบรวมบันทึก มุมมองโทโพโลยี และการติดตามสุขภาพของไซต์

- **หมวดหมู่**: การปฏิบัติการ
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 77 | **สคีมา**: 575
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `observability`, `support`

## กรณีการใช้งาน

- เข้าถึงสถิติการไหลของข้อมูลและการวิเคราะห์
- จัดการการแจ้งเตือนและนโยบายการแจ้งเตือน
- ดูบันทึกและตัวรับบันทึก
- สร้างรายงานและกราฟ
- ติดตามโทโพโลยีและการค้นพบบริการ
- ตรวจสอบสถานะที่ไซต์

## ทรัพยากรหลัก

- **dashboard**: แดชบอร์ดสำหรับการแสดงภาพข้อมูลเมตริก
- **saved_query**: คิวรีที่บันทึกไว้สำหรับการวิเคราะห์เมตริกและบันทึก
