นโยบายการแจ้งเตือนพร้อมการผสานรวมตัวรับ การรวมบันทึก มุมมองโทโพโลยี และการติดตามความสมบูรณ์ของไซต์

- **หมวดหมู่**: การดำเนินงาน
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 77 | **สคีมา**: 575
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `observability`, `support`

## กรณีการใช้งาน

- เข้าถึงสถิติและการวิเคราะห์การไหลของข้อมูล
- จัดการการแจ้งเตือนและนโยบายการแจ้งเตือน
- ดูบันทึกและตัวรับบันทึก
- สร้างรายงานและกราฟ
- ติดตามโทโพโลยีและการค้นหาบริการ
- ตรวจสอบสถานะที่ไซต์

## ทรัพยากรหลัก

- **dashboard**: แดชบอร์ดสำหรับการแสดงผลเมตริก
- **saved_query**: คิวรีที่บันทึกไว้สำหรับการวิเคราะห์เมตริกและบันทึก
