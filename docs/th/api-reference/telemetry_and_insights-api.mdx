endpoint สำหรับการรวบรวมข้อมูล การจัดเก็บเมตริก และการสร้างข้อมูลเชิงลึกสำหรับเวิร์กโฟลว์การสังเกตการณ์

- **หมวดหมู่**: การดำเนินงาน
- **ความซับซ้อน**: ปานกลาง
- **พาธ**: 27 | **สคีมา**: 181
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `observability`, `statistics`

## กรณีการใช้งาน

- รวบรวมและวิเคราะห์ข้อมูล telemetry
- สร้างข้อมูลเชิงลึกที่นำไปปฏิบัติได้จากเมตริก
- กำหนดค่านโยบายการรวบรวม telemetry

## ทรัพยากรหลัก

- **telemetry_receiver**: ตัวรับ telemetry สำหรับการรวบรวมข้อมูล
- **insight_query**: คิวรีข้อมูลเชิงลึกสำหรับการวิเคราะห์
