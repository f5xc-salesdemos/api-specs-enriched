Endpoints สำหรับการเก็บรวบรวมข้อมูล การจัดเก็บเมตริก และการสร้างข้อมูลเชิงลึกสำหรับเวิร์กโฟลว์การสังเกตการณ์

- **หมวดหมู่**: การดำเนินงาน
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 27 | **สคีมา**: 181
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `observability`, `statistics`

## กรณีการใช้งาน

- เก็บรวบรวมและวิเคราะห์ข้อมูล telemetry
- สร้างข้อมูลเชิงลึกที่นำไปใช้ได้จากเมตริก
- กำหนดค่านโยบายการเก็บรวบรวม telemetry

## ทรัพยากรหลัก

- **telemetry_receiver**: ตัวรับ telemetry สำหรับการเก็บรวบรวมข้อมูล
- **insight_query**: คำสั่งค้นหาข้อมูลเชิงลึกสำหรับการวิเคราะห์
