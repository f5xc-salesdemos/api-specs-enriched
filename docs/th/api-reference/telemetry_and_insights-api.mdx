จุดปลายทางการรวบรวมข้อมูล การจัดเก็บเมตริก และการสร้างข้อมูลเชิงลึกสำหรับเวิร์กโฟลว์การสังเกตการณ์

- **หมวดหมู่**: การดำเนินงาน
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 27 | **สคีมา**: 181
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `observability`, `statistics`

## กรณีการใช้งาน

- รวบรวมและวิเคราะห์ข้อมูลเทเลเมทรี
- สร้างข้อมูลเชิงลึกที่นำไปปฏิบัติได้จากเมตริก
- กำหนดค่านโยบายการเก็บรวบรวมเทเลเมทรี

## ทรัพยากรหลัก

- **telemetry_receiver**: ตัวรับเทเลเมทรีสำหรับการรวบรวมข้อมูล
- **insight_query**: คิวรีข้อมูลเชิงลึกสำหรับการวิเคราะห์
