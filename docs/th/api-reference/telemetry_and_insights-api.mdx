เอนด์พอยต์สำหรับการเก็บรวบรวม การจัดเก็บเมตริก และการสร้างข้อมูลเชิงลึกสำหรับเวิร์กโฟลว์การสังเกตการณ์ระบบ

- **หมวดหมู่**: Operations
- **ความซับซ้อน**: ปานกลาง
- **Paths**: 27 | **Schemas**: 181
- **ระดับ**: Standard
- **โดเมนที่เกี่ยวข้อง**: `observability`, `statistics`

## กรณีการใช้งาน

- เก็บรวบรวมและวิเคราะห์ข้อมูลเทเลเมทรี
- สร้างข้อมูลเชิงลึกที่นำไปปฏิบัติได้จากเมตริก
- กำหนดค่านโยบายการเก็บรวบรวมเทเลเมทรี

## ทรัพยากรหลัก

- **telemetry_receiver**: ตัวรับเทเลเมทรีสำหรับการเก็บรวบรวมข้อมูล
- **insight_query**: คิวรีข้อมูลเชิงลึกสำหรับการวิเคราะห์
