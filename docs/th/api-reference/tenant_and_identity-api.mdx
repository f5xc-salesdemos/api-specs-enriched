การกำหนดค่ามุมมองบัญชีและช่องทางการแจ้งเตือนของผู้ดูแลระบบ การรีเซ็ตรหัสผ่านแบบใช้ครั้งเดียว ขั้นตอนการจัดเตรียม และการตรวจสอบการเชื่อมต่อที่ใช้งานอยู่

- **หมวดหมู่**: Platform
- **ความซับซ้อน**: ขั้นสูง
- **เส้นทาง**: 183 | **สคีมา**: 511
- **ระดับ**: Standard
- **โดเมนที่เกี่ยวข้อง**: `users`, `authentication`, `system`

## กรณีการใช้งาน

- จัดการโปรไฟล์ผู้ใช้และการตั้งค่าการแจ้งเตือน
- กำหนดค่าการควบคุมเซสชันและการตั้งค่า OTP
- จัดการการดำเนินงานด้านการระบุตัวตน
- ดำเนินการคำขอเข้าถึงเริ่มต้นของผู้ใช้

## ทรัพยากรหลัก

- **user_profile**: โปรไฟล์ผู้ใช้พร้อมการตั้งค่าส่วนบุคคล
- **session**: เซสชันผู้ใช้สำหรับสถานะการยืนยันตัวตน
- **otp_policy**: นโยบาย OTP สำหรับการยืนยันตัวตนแบบหลายปัจจัย
