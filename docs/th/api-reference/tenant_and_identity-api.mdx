การกำหนดค่าการแสดงผลบัญชีและช่องทางการแจ้งเตือนผู้ดูแลระบบ การรีเซ็ตรหัสผ่านแบบใช้ครั้งเดียว ขั้นตอนการจัดเตรียมระบบ และการตรวจสอบการเชื่อมต่อที่ใช้งานอยู่

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ขั้นสูง
- **พาธ**: 183 | **สคีมา**: 511
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `users`, `authentication`, `system`

## กรณีการใช้งาน

- จัดการโปรไฟล์ผู้ใช้และการตั้งค่าการแจ้งเตือน
- กำหนดค่าการควบคุมเซสชันและการตั้งค่า OTP
- ดำเนินการจัดการข้อมูลประจำตัว
- ประมวลผลคำขอการเข้าถึงครั้งแรกของผู้ใช้

## ทรัพยากรหลัก

- **user_profile**: โปรไฟล์ผู้ใช้พร้อมการตั้งค่าส่วนตัว
- **session**: เซสชันผู้ใช้สำหรับสถานะการยืนยันตัวตน
- **otp_policy**: นโยบาย OTP สำหรับการยืนยันตัวตนแบบหลายปัจจัย
