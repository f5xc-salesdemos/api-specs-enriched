การกำหนดค่าการแสดงผลบัญชีและช่องทางการแจ้งเตือนสำหรับผู้ดูแลระบบ การรีเซ็ตรหัสผ่านแบบใช้ครั้งเดียว ขั้นตอนการจัดเตรียม และการตรวจสอบการเชื่อมต่อที่ใช้งานอยู่

- **Category**: แพลตฟอร์ม
- **Complexity**: advanced
- **Paths**: 183 | **Schemas**: 511
- **Tier**: Standard
- **Related domains**: `users`, `authentication`, `system`

## กรณีการใช้งาน

- จัดการโปรไฟล์ผู้ใช้และการตั้งค่าการแจ้งเตือน
- กำหนดค่าการควบคุมเซสชันและการตั้งค่า OTP
- ดำเนินการจัดการข้อมูลประจำตัว
- ประมวลผลคำขอการเข้าถึงผู้ใช้ครั้งแรก

## ทรัพยากรหลัก

- **user_profile**: โปรไฟล์ผู้ใช้พร้อมการตั้งค่าส่วนตัว
- **session**: เซสชันผู้ใช้สำหรับสถานะการยืนยันตัวตน
- **otp_policy**: นโยบาย OTP สำหรับการยืนยันตัวตนแบบหลายปัจจัย
