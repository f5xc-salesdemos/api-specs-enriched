นโยบายการตรวจจับ การติดตามการโจมตี และการสร้างกฎการบรรเทาผลกระทบอัตโนมัติสำหรับการดำเนินงานด้านความปลอดภัย

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 1 | **สคีมา**: 161
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `bot_defense`, `ddos`

## กรณีการใช้งาน

- ติดตามและวิเคราะห์แคมเปญภัยคุกคาม
- ตรวจสอบภัยคุกคามที่ใช้งานอยู่และรูปแบบการโจมตี
- กำหนดค่าการผสานรวมข้อมูลภัยคุกคาม

## ทรัพยากรหลัก

- **threat_campaign_policy**: นโยบายการตรวจจับและบรรเทาผลกระทบของแคมเปญภัยคุกคาม

## Endpoints

| วิธีการ | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | GET Threat Campaign ตาม ID |
