นโยบายการตรวจจับ การติดตามการโจมตี และการสร้างกฎบรรเทาผลกระทบอัตโนมัติสำหรับการปฏิบัติการด้านความปลอดภัย

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 1 | **สคีมา**: 161
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `bot_defense`, `ddos`

## กรณีการใช้งาน

- ติดตามและวิเคราะห์แคมเปญภัยคุกคาม
- ตรวจสอบภัยคุกคามที่กำลังดำเนินอยู่และรูปแบบการโจมตี
- กำหนดค่าการรวมข้อมูลข่าวกรองภัยคุกคาม

## ทรัพยากรหลัก

- **threat_campaign_policy**: นโยบายการตรวจจับและบรรเทาผลกระทบจากแคมเปญภัยคุกคาม

## เอนด์พอยต์

| เมธอด | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | ดึงข้อมูล Threat Campaign ตาม ID |
