นโยบายการตรวจจับ การติดตามการโจมตี และการสร้างกฎลดความเสี่ยงอัตโนมัติสำหรับการดำเนินงานด้านความปลอดภัย

- **หมวดหมู่**: ความปลอดภัย
- **ความซับซ้อน**: ปานกลาง
- **เส้นทาง**: 1 | **สคีมา**: 161
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `bot_defense`, `ddos`

## กรณีการใช้งาน

- ติดตามและวิเคราะห์แคมเปญภัยคุกคาม
- ตรวจสอบภัยคุกคามที่ใช้งานอยู่และรูปแบบการโจมตี
- กำหนดค่าการผสานรวมข่าวกรองภัยคุกคาม

## ทรัพยากรหลัก

- **threat_campaign_policy**: นโยบายการตรวจจับและลดความเสี่ยงของแคมเปญภัยคุกคาม

## จุดปลายทาง

| วิธีการ | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | GET แคมเปญภัยคุกคามตาม ID |
