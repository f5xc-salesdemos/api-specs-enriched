ข้อมูลประจำตัวสำหรับการลงทะเบียนไซต์พร้อมการหมดอายุอัตโนมัติ คีย์อนุกรมวิธานกำหนดการจัดหมวดหมู่ที่อนุญาต ในขณะที่แท็กที่ได้รับมาอัตโนมัติจะถูกนำไปใช้แบบไดนามิก

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 13 | **สคีมา**: 50
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `system`, `admin`

## กรณีการใช้งาน

- จัดการบัญชีผู้ใช้และโทเค็น
- กำหนดค่าการระบุตัวตนผู้ใช้
- จัดการการตั้งค่าและค่ากำหนดของผู้ใช้
- กำหนดค่าป้ายกำกับโดยนัยและที่รู้จัก

## ทรัพยากรหลัก

- **user**: บัญชีผู้ใช้สำหรับการเข้าถึงแพลตฟอร์ม
- **user_role**: บทบาทผู้ใช้สำหรับการจัดการสิทธิ์
- **namespace_role**: บทบาทที่กำหนดขอบเขตตาม namespace สำหรับสิทธิ์ที่ละเอียดยิ่งขึ้น
