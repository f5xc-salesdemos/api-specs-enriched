ข้อมูลรับรองสำหรับการลงทะเบียนไซต์พร้อมการหมดอายุอัตโนมัติ คีย์อนุกรมวิธานกำหนดการจัดหมวดหมู่ที่อนุญาต ในขณะที่แท็กที่ได้มาโดยอัตโนมัติจะถูกนำไปใช้แบบไดนามิก

- **หมวดหมู่**: Platform
- **ความซับซ้อน**: simple
- **เส้นทาง**: 13 | **สคีมา**: 50
- **ระดับ**: Standard
- **โดเมนที่เกี่ยวข้อง**: `system`, `admin`

## กรณีการใช้งาน

- จัดการบัญชีผู้ใช้และโทเค็น
- กำหนดค่าการระบุตัวตนผู้ใช้
- จัดการการตั้งค่าและค่ากำหนดของผู้ใช้
- กำหนดค่าป้ายกำกับโดยนัยและป้ายกำกับที่รู้จัก

## ทรัพยากรหลัก

- **user**: บัญชีผู้ใช้สำหรับการเข้าถึงแพลตฟอร์ม
- **user_role**: บทบาทผู้ใช้สำหรับการจัดการสิทธิ์
- **namespace_role**: บทบาทระดับ namespace สำหรับสิทธิ์แบบละเอียด
