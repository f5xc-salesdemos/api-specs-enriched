การควบคุมวงจรชีวิต การกำหนดค่าฟลีต และนโยบายการปรับใช้สำหรับการจัดการโหนดแบบกระจาย

- **หมวดหมู่**: แพลตฟอร์ม
- **ความซับซ้อน**: ง่าย
- **เส้นทาง**: 1 | **สคีมา**: 2
- **ระดับ**: มาตรฐาน
- **โดเมนที่เกี่ยวข้อง**: `sites`, `system`

## กรณีการใช้งาน

- จัดการการกำหนดค่า Virtual Private Mesh (VPM)
- กำหนดค่าวงจรชีวิตและการจัดการโหนด
- ตรวจสอบสถานะ VPM และโหนด

## ทรัพยากรหลัก

- **node_config**: การกำหนดค่าโหนดสำหรับการตั้งค่าอุปกรณ์เอดจ์
- **vpm_config**: การกำหนดค่า VPM สำหรับการจัดการแพลตฟอร์มเสมือน

## เอนด์พอยต์

| เมธอด | เส้นทาง | คำอธิบาย |
|--------|------|-------------|
| GET | `/api/data/namespaces/system/upgrade_status` | สถานะการอัปเกรด |
