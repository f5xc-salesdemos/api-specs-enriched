具有版本控制的命名空间级别可视化元素。门户界面的自定义小组件部署和目录管理。

- **类别**：Platform
- **复杂度**：simple
- **路径数**：2 | **模式数**：14
- **层级**：Standard
- **相关领域**：`admin`、`system`

## 使用场景

- 管理管理控制台的静态 UI 组件
- 在命名空间内部署和检索 UI 资源
- 配置控制台界面元素
- 管理自定义 UI 组件元数据

## 主要资源

- **ui_component**：用于控制台自定义的 UI 组件
- **static_asset**：UI 资源的静态资源

## 端点

| 方法 | 路径 | 描述 |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | 列出 UI 静态组件。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | 获取 UI 静态组件。 |
