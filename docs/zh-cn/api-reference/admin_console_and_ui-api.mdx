支持版本控制的命名空间范围可视化元素。用于门户界面的自定义组件部署和目录管理。

- **类别**：平台
- **复杂度**：简单
- **路径数**：2 | **Schema 数**：14
- **层级**：标准
- **相关域**：`admin`、`system`

## 使用场景

- 管理管理控制台的静态 UI 组件
- 在命名空间内部署和检索 UI 资源
- 配置控制台界面元素
- 管理自定义 UI 组件元数据

## 主要资源

- **ui_component**：用于控制台自定义的 UI 组件
- **static_asset**：用于 UI 资源的静态资源

## 端点

| 方法 | 路径 | 描述 |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | 列出 UI 静态组件。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | 获取 UI 静态组件。 |
