自然语言处理，具备质量信号与异常监控功能。支持通过令牌认证进行数据流订阅。

- **类别**：AI
- **复杂度**：简单
- **路径数**：11 | **模式数**：66
- **层级**：高级


## 使用场景

- 访问 AI 驱动的功能
- 配置 AI 助手策略
- 启用流量异常检测
- 管理 AI 数据采集

## 主要资源

- **ai_policy**：用于生成式 AI 流量控制的 AI 策略
- **ai_gateway**：用于 LLM API 管理的 AI 网关

## 端点

| 方法 | 路径 | 描述 |
|--------|------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | 评估 AI 助手查询。 |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | 评估 AI 助手查询的反馈。 |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | AI 助手查询。 |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | AI 助手查询的反馈。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | 启用功能。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | 刷新令牌。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | 订阅 BFDP 管道。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | 订阅 BFDP 管道。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | 取消订阅 BFDP 管道。 |
| POST | `/api/gia/gia/allocateip` | 通过全局 IP 分配器分配 IP。 |
| DELETE | `/api/gia/gia/deallocateip` | 通过全局 IP 分配器释放 IP。 |
