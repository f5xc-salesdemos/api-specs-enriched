具有自动分类和安全配置的资源编目。组织层次结构按功能或保护策略划分访问权限。

- **类别**: 安全
- **复杂度**: 高级
- **路径**: 45 | **模式**: 279
- **层级**: 高级
- **相关领域**: `virtual`, `network_security`

## 使用场景

- 发现和编目 API
- 测试 API 安全性和行为
- 管理 API 凭据
- 定义 API 分组和测试策略

## 主要资源

- **api_definition**: 用于发现和保护的 API 模式定义
- **api_endpoint**: 单个 API 端点配置和保护
- **api_rate_limit**: 用于流量控制的 API 速率限制配置
