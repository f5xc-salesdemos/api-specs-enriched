身份管理，包括提供商集成、访问策略和凭证生命周期控制。

- **类别**：平台
- **复杂度**：简单
- **路径**：14 | **模式**：38
- **层级**：标准
- **相关领域**：`system`、`users`

## 使用场景

- 配置认证机制
- 管理 OIDC 和 OAuth 提供商
- 配置 SCIM 用户预配
- 管理 API 凭证和访问
- 配置账户注册策略

## 主要资源

- **authentication_policy**：用户和 API 访问的认证策略
- **token**：用于编程访问的 API 令牌
- **api_credential**：用于服务认证的 API 凭证

## 端点

| 方法 | 路径 | 描述 |
|--------|------|-------------|
| POST | `/api/web/namespaces/system/bulk_revoke/api_credentials` | 批量撤销 API 凭证。 |
| POST | `/api/web/namespaces/system/bulk_revoke/service_credentials` | 批量撤销服务凭证。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/activate/api_credentials` | 激活 API 凭证。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/activate/service_credentials` | 激活 API 服务凭证。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/api_credentials` | 列出 API 凭证。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/api_credentials` | 创建 API 凭证。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/api_credentials/&#123;name&#125;` | 获取 API 凭证。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/renew/api_credentials` | 续期 API 凭证。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/renew/service_credentials` | 续期 API 服务凭证。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/revoke/api_credentials` | 撤销 API 凭证。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/revoke/scim_token` | 撤销 SCIM API 凭证。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/revoke/service_credentials` | 撤销服务凭证。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/scim_token` | 获取 SCIM 令牌。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/scim_token` | 创建/重新创建 SCIM API 令牌。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/service_credentials` | 列出服务凭证。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/service_credentials` | 创建服务凭证。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/service_credentials/&#123;name&#125;` | 获取服务凭证。 |
| PUT | `/api/web/namespaces/&#123;namespace&#125;/service_credentials/&#123;name&#125;` | 替换服务凭证。 |
