使用 iRule 脚本和数据组同步进行传统设备编排。虚拟服务器绑定和指标收集。

- **类别**：平台
- **复杂度**：高级
- **路径**：29 | **模式**：238
- **层级**：高级
- **相关领域**：`marketplace`

## 使用场景

- 管理 BigIP F5 设备
- 配置 iRule 脚本
- 管理数据组
- 集成 BigIP CNE

## 主要资源

- **bigip_pool**：用于负载均衡集成的 BIG-IP 池
- **bigip_device**：BIG-IP 设备注册和管理
