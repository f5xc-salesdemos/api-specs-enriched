计划变更、发票开具和资源消耗。命名空间级别的配额限制和用量跟踪。

- **类别**：平台
- **复杂度**：中等
- **路径数**：25 | **模式数**：101
- **层级**：标准
- **相关领域**：`system`、`users`

## 使用场景

- 管理订阅计划和计费变更
- 配置支付方式和发票
- 跨命名空间跟踪资源配额用量
- 监控使用限制和容量

## 主要资源

- **subscription**：用于服务权限的订阅
- **quota**：用于使用限制的资源配额
- **usage_report**：用于消耗跟踪的用量报告
