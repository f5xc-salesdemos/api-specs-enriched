基于策略访问控制的加密密钥管理。审计日志记录与安全数据保护。

- **类别**：安全
- **复杂度**：中等
- **路径数**：27 | **Schema 数**：163
- **层级**：高级
- **相关域**：`client_side_defense`、`certificates`

## 使用场景

- 为加密配置密钥策略
- 管理敏感数据加密
- 执行数据保护策略

## 主要资源

- **blindfold_secret**：使用 Blindfold 加密的加密密钥
- **secret_policy**：密钥管理策略配置
- **policy_document**：用于访问控制定义的策略文档
