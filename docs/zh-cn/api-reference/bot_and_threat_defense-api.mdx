基于行为分析和签名匹配的威胁分类。针对恶意流量模式的自动化拦截。

- **类别**：安全
- **复杂度**：中等
- **路径数**：19 | **模式数**：74
- **层级**：高级
- **相关领域**：`bot_defense`、`shape`、`virtual`

## 使用场景

- 按命名空间配置机器人防御实例
- 管理 TPM 威胁分类以进行分类识别
- 为自动化防御系统配置 API 密钥
- 集成威胁情报服务

## 主要资源

- **bot_defense_instance**：用于部署的机器人防御实例
- **threat_category**：用于分类的威胁类别
