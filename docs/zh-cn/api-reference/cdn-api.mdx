全球分发，支持缓存规则和清除操作。性能监控与分析。

- **类别**：网络
- **复杂度**：高级
- **路径数**：29 | **Schema 数**：592
- **层级**：高级
- **相关领域**：`virtual`

## 使用场景

- 配置 CDN 负载均衡
- 管理内容分发网络服务
- 配置缓存策略
- 管理数据分发与交付

## 主要资源

- **cdn_loadbalancer**：用于边缘缓存的内容分发网络负载均衡器
- **cdn_origin_pool**：用于 CDN 内容源获取的源服务器池
