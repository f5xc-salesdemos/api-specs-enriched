基于令牌的预配置，支持镜像下载和升级前验证。机群分组可跨分布式位置执行批量操作。

- **类别**：基础设施
- **复杂度**：中等
- **路径数**：28 | **模式数**：231
- **层级**：标准
- **相关域**：`customer_edge`、`sites`

## 使用场景

- 管理 Customer Edge 站点生命周期
- 配置网络接口与机群设置
- 处理站点注册及令牌工作流
- 执行带升级前检查的站点升级

## 主要资源

- **site_config**：边缘节点设置的站点配置
- **fleet_config**：多站点管理的机群配置
- **registration_token**：站点入驻的注册令牌
