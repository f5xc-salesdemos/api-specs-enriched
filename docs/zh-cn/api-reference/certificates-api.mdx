证书链与受信任 CA 包。吊销列表管理及 PKI 操作的清单配置。

- **类别**：安全
- **复杂度**：中等
- **路径数**：16 | **Schema 数**：74
- **层级**：标准
- **相关域**：`blindfold`、`system`

## 使用场景

- 管理 SSL/TLS 证书
- 配置受信任 CA
- 管理证书吊销列表（CRL）
- 配置证书清单

## 主要资源

- **certificate**：用于安全 HTTPS 连接的 TLS 证书
- **ca_certificate**：用于信任链的证书颁发机构证书
- **certificate_chain**：用于完整信任验证的证书链
