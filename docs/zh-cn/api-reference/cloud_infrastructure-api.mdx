多云提供商连接，支持网关对等和网络路径配置。凭据保管库集成和子网枚举。

- **类别**：基础设施
- **复杂度**：中等
- **路径**：31 | **模式**：208
- **层级**：标准
- **相关领域**：`sites`、`customer_edge`

## 使用场景

- 连接到云提供商（AWS、Azure、GCP）
- 管理云凭据和身份验证
- 配置云连接和弹性配置
- 链接和管理云区域

## 主要资源

- **aws_vpc_site**：AWS VPC 站点部署及边缘节点配置
- **azure_vnet_site**：Azure VNet 站点部署及边缘节点配置
- **gcp_vpc_site**：Google Cloud VPC 站点部署及边缘节点配置
- **cloud_credentials**：用于站点部署的云提供商身份验证凭据
