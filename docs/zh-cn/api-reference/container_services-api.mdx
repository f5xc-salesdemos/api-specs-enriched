无需完整集群复杂性的 Pod 编排。边缘站点执行、配额管理以及分布式应用的标准化计算配置文件。

- **类别**：基础设施
- **复杂度**：中等
- **路径数**：13 | **Schema 数**：159
- **层级**：高级
- **相关领域**：`managed_kubernetes`、`sites`、`service_mesh`

## 使用场景

- 部署 XCCS（容器服务）命名空间以支持多租户工作负载
- 通过简化的编排管理容器工作负载
- 配置分布式边缘容器部署
- 无需完整 K8s 复杂性即可运行容器化应用

## 主要资源

- **virtual_k8s**：用于容器工作负载的虚拟 Kubernetes 命名空间
- **workload**：容器工作负载部署配置
- **pod_security_policy**：用于容器运行时约束的 Pod 安全策略
