用于数据分析工作流的分类规则、资源策略和洞察生成。

- **类别**：运营
- **复杂度**：简单
- **路径**：15 | **模式**：73
- **层级**：标准
- **相关领域**：`statistics`、`observability`

## 使用场景

- 分析安全和流量数据
- 从日志中生成智能洞察
- 配置数据分析策略

## 主要资源

- **analytics_query**：用于数据分析的分析查询
- **data_export**：用于报告的数据导出配置

## 端点

| 方法 | 路径 | 描述 |
|--------|------|-------------|
| POST | `/api/data-intelligence/namespaces/&#123;metadata.namespace&#125;/receivers` | 创建数据传递。 |
| PUT | `/api/data-intelligence/namespaces/&#123;metadata.namespace&#125;/receivers/&#123;metadata.name&#125;` | 替换数据传递。 |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers` | 列出数据传递。 |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers/&#123;name&#125;` | 获取数据传递。 |
| DELETE | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers/&#123;name&#125;` | 删除数据传递。 |
| GET | `/api/data-intelligence/namespaces/system/dataSets` | 获取数据集。 |
| GET | `/api/data-intelligence/namespaces/system/datadictionary/&#123;dataset&#125;` | 获取数据字典。 |
| POST | `/api/data-intelligence/namespaces/system/init-request` | 启用数据智能。 |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/data-dictionary/datasets` | 列出数据集。 |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/di/flowlabels` | 列出流标签。 |
| POST | `/api/data-intelligence/namespaces/&#123;namespace&#125;/di/summary` | 加载执行摘要。 |
| POST | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers/&#123;id&#125;/status` | 更新接收器状态。 |
| POST | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers/&#123;id&#125;/test` | 测试接收器。 |
