拒绝列表、防火墙规则组和基于隧道的安全防护。用于网络边界安全的速率限制和模式分析。

- **类别**：安全
- **复杂度**：高级
- **路径数**：60 | **模式数**：246
- **层级**：高级
- **相关领域**：`network_security`、`virtual`

## 使用场景

- 配置 DDoS 防护策略
- 监控和分析 DDoS 威胁
- 配置基础设施防护

## 主要资源

- **ddos_protection**：用于流量清洗的 DDoS 防护配置
- **ddos_mitigation_rule**：用于攻击响应的 DDoS 缓解规则
