Kubernetes 角色绑定和准入策略。适用于 EKS、AKS 和 GKE 工作负载的镜像仓库集成。

- **类别**：基础设施
- **复杂度**：中等
- **路径数**：20 | **Schema 数**：95
- **层级**：高级
- **相关领域**：`container_services`、`sites`、`service_mesh`

## 使用场景

- 管理 XCKS（托管 Kubernetes）集群 RBAC 和安全性
- 配置 Pod 安全策略和准入控制器
- 管理企业部署的容器镜像仓库
- 与外部 Kubernetes 集群（EKS、AKS、GKE）集成

## 主要资源

- **mk8s_cluster**：托管 Kubernetes 集群配置
- **k8s_cluster_role**：Kubernetes 集群 RBAC 角色配置
- **container_registry**：用于工作负载镜像的容器镜像仓库
