具有 DPD 计时器的第三方 GRE 和 IPSec 隧道配置。跨命名空间的共享资源分配与磁贴放置控制。

- **类别**：平台
- **复杂度**：中等
- **路径数**：36 | **架构数**：183
- **层级**：高级
- **相关域**：`bigip`、`admin`

## 使用场景

- 访问第三方集成和附加组件
- 管理市场扩展
- 配置 Terraform 和外部集成
- 管理 TPM 策略

## 主要资源

- **marketplace_item**：服务目录的市场条目
- **subscription**：服务权利的订阅
