边界网关协议（BGP）与 ASN 管理及自治系统关系。通过加密通道连接数据中心的站点间 VPN。

- **类别**：网络
- **复杂度**：高级
- **路径数**：81 | **模式数**：449
- **层级**：高级
- **相关领域**：`virtual`、`network_security`、`dns`

## 使用场景

- 配置 BGP 路由和 ASN 管理
- 管理 IPsec 隧道和 IKE 阶段
- 配置网络连接器和路由
- 管理 SRv6 和子网划分
- 定义分段连接和策略
- 配置 IP 前缀集

## 主要资源

- **virtual_network**：用于站点连接和分段的虚拟网络
- **network_connector**：用于站点间或云连接的网络连接器
- **site_mesh_group**：多站点之间的网格连接配置
