基于源、目标或协议做出路由决策的防火墙规则。网段隔离用于隔离工作负载，而出站代理则管控访问权限。

- **类别**：安全
- **复杂度**：高级
- **路径数**：62 | **模式数**：386
- **层级**：高级
- **相关领域**：`virtual`、`api`、`network`

## 用例

- 配置网络防火墙和 ACL 策略
- 管理 NAT 策略和端口转发
- 配置基于策略的路由
- 定义网络分段和策略
- 配置正向代理策略

## 主要资源

- **network_policy**：用于流量过滤的网络安全策略
- **forward_proxy_policy**：用于出站流量控制的正向代理策略
- **network_firewall**：用于第 3/4 层流量保护的网络防火墙
