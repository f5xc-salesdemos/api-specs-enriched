实例发现、WAF 集成和服务网格连接。订阅生命周期和配置同步。

- **类别**：平台
- **复杂度**：简单
- **路径数**：13 | **模式数**：59
- **层级**：高级
- **相关领域**：`marketplace`

## 使用场景

- 管理 NGINX One 平台集成
- 配置 NGINX Plus 实例
- 集成 NGINX 配置管理

## 主要资源

- **nginx_config**：用于代理设置的 NGINX 配置
- **nginx_upstream**：用于后端服务器池的 NGINX 上游配置

## 端点

| 方法 | 路径 | 描述 |
|--------|------|-------------|
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_csgs` | 列出 NGINX One CSG 对象配置。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_csgs/&#123;name&#125;` | GET 请求。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_instances` | 列出 NGINX One 实例对象配置。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_instances/&#123;name&#125;` | GET 请求。 |
| POST | `/api/config/namespaces/&#123;namespace&#125;/nginx_dataplane_servers` | 获取 NGINX One 数据平面服务器。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_servers` | 列出 NGINX One 服务器 API。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_servers/&#123;name&#125;` | GET 请求。 |
| POST | `/api/nginx/one/namespaces/system/n1/subscribe` | 订阅 NGINX One。 |
| POST | `/api/nginx/one/namespaces/system/n1/unsubscribe` | 取消订阅 NGINX One。 |
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/nginx_service_discoverys` | 创建 NGINX 服务发现。 |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/nginx_service_discoverys/&#123;metadata.name&#125;` | 替换 NGINX 服务发现。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_service_discoverys` | 列出 NGINX 服务发现。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_service_discoverys/&#123;name&#125;` | 获取 NGINX 服务发现。 |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/nginx_service_discoverys/&#123;name&#125;` | 删除 NGINX 服务发现。 |
