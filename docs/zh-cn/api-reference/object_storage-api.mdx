面向移动应用集成商的版本化库分发。预签名 URL 支持安全下载，提供针对 iOS 和 Android 的特定操作系统构建。

- **类别**：平台
- **复杂度**：简单
- **路径数**：9 | **模式数**：17
- **层级**：高级
- **相关领域**：`marketplace`

## 使用场景

- 管理对象存储服务
- 配置存储对象和存储桶
- 管理存储策略

## 主要资源

- **object_store**：用于 Blob 存储管理的对象存储
- **bucket**：用于对象存储的存储桶
