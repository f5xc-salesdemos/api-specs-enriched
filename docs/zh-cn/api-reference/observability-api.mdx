具有延迟测量功能的 HTTP 可用性探测。证书过期告警和全局状态仪表板，用于基础设施健康监控。

- **类别**：运维
- **复杂度**：中等
- **路径数**：56 | **模式数**：196
- **层级**：标准
- **相关领域**：`statistics`、`support`

## 使用场景

- 配置合成监控
- 定义监控和测试策略
- 管理可观测性仪表板

## 主要资源

- **log_receiver**：用于集中日志收集的日志接收器
- **metrics_receiver**：用于集中指标收集的指标接收器
- **alert_policy**：用于监控和通知的告警策略
