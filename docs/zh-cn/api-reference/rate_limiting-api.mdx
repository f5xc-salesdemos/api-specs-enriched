基于时间的配额执行，支持以小时、分钟或秒为单位的可配置窗口。用于流量整形的协议专属控制。

- **类别**：网络
- **复杂度**：简单
- **路径数**：12 | **Schema 数**：70
- **版本层级**：高级
- **相关域**：`virtual`、`network_security`

## 使用场景

- 配置限速策略
- 管理流量管控配置
- 控制流量和排队行为

## 主要资源

- **rate_limiter**：用于流量限速与防护的限速器
- **rate_limiter_policy**：包含详细限速规则的限速策略
- **rate_limit_threshold**：用于流量控制的限速阈值配置
