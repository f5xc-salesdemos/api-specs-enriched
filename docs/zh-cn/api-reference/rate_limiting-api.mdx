基于时间的配额实施，支持以小时、分钟或秒为单位的可配置窗口。协议特定的流量整形控制。

- **类别**：网络
- **复杂度**：简单
- **路径数**：12 | **模式数**：70
- **层级**：高级
- **相关领域**：`virtual`、`network_security`

## 使用场景

- 配置速率限制器策略
- 管理策略器配置
- 控制流量和排队

## 主要资源

- **rate_limiter**：用于流量节流和保护的速率限制器
- **rate_limiter_policy**：包含详细节流规则的速率限制器策略
- **rate_limit_threshold**：用于流量控制的速率限制阈值配置
