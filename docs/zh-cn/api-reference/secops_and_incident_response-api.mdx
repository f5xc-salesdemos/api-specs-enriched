通过威胁级别分类进行恶意用户缓解。针对可疑行为模式的自动响应操作。

- **类别**：Security
- **复杂度**：simple
- **路径数**：4 | **模式数**：32
- **层级**：Advanced
- **相关领域**：`bot_defense`、`virtual`、`network_security`

## 使用场景

- 配置自动化威胁缓解策略
- 定义恶意用户检测规则
- 管理事件响应工作流
- 对威胁应用封禁或速率限制

## 主要资源

- **mitigation_policy**：用于事件响应的缓解策略
- **malicious_user_rule**：用于威胁缓解的恶意用户规则

## 端点

| 方法 | 路径 | 描述 |
|--------|------|-------------|
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations` | 创建恶意用户缓解。 |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations/&#123;metadata.name&#125;` | 替换恶意用户缓解。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations` | 列出恶意用户缓解。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | 获取恶意用户缓解。 |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | 删除恶意用户缓解。 |
