应用类型定义，包含发现和学习型模式分析。流量模式推断，用于智能请求处理。

- **类别**：基础设施
- **复杂度**：高级
- **路径数**：38 | **模式数**：247
- **层级**：高级
- **相关领域**：`managed_kubernetes`、`container_services`、`virtual`

## 使用场景

- 配置服务网格连接
- 管理端点发现和路由
- 配置 NFV 服务
- 定义应用设置和类型

## 主要资源

- **endpoint**：用于流量路由的服务网格端点
- **origin_pool**：用于负载均衡器流量分发的后端服务器组
- **service_discovery**：用于动态端点的服务发现配置
