通过行为分析和设备指纹识别进行威胁识别。集成移动 SDK 实现应用程序防护。

- **类别**: 安全
- **复杂度**: 高级
- **路径**: 271 | **模式**: 813
- **层级**: 高级
- **相关领域**: `bot_defense`、`virtual`

## 使用场景

- 配置 Shape Security 策略
- 管理机器人和威胁防护
- 配置 SafeAP 策略
- 启用威胁识别

## 主要资源

- **shape_app_firewall**: Shape 应用防火墙，提供高级防护
- **shape_recognizer**: Shape 识别器，用于流量模式分析
