告警策略与接收器集成。日志聚合、拓扑视图和站点健康状态追踪。

- **类别**：运维
- **复杂度**：高级
- **路径数**：77 | **模式数**：575
- **层级**：标准
- **相关领域**：`observability`、`support`

## 使用场景

- 访问流量统计和分析数据
- 管理告警和告警策略
- 查看日志和日志接收器
- 生成报告和图表
- 追踪拓扑和服务发现
- 监控站点状态

## 主要资源

- **dashboard**：用于指标可视化的仪表板
- **saved_query**：用于指标和日志分析的保存查询
