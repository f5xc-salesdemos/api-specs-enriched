包含评论、严重性变更和解决方案跟踪的问题生命周期管理。用于连接分析的数据包捕获。

- **类别**：运维
- **复杂度**：中等
- **路径数**：53 | **模式数**：167
- **层级**：标准
- **相关领域**：`statistics`、`observability`

## 使用场景

- 提交和管理支持工单
- 跟踪客户支持请求
- 访问运维支持文档

## 主要资源

- **support_case**：用于问题跟踪和解决的支持案例
- **alert**：用于运维通知的系统告警
- **audit_log**：用于合规和安全跟踪的审计日志
