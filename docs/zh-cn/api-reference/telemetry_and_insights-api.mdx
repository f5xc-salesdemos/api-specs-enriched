用于可观测性工作流的收集端点、指标存储和洞察生成。

- **类别**：运维
- **复杂度**：中等
- **路径数**：27 | **模式数**：181
- **层级**：标准
- **相关领域**：`observability`、`statistics`

## 使用场景

- 收集和分析遥测数据
- 从指标中生成可操作的洞察
- 配置遥测收集策略

## 主要资源

- **telemetry_receiver**：用于数据收集的遥测接收器
- **insight_query**：用于分析的洞察查询
