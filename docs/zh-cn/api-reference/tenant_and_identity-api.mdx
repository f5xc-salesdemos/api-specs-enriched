账户视图配置和管理员告警渠道。一次性密码重置、配置流程以及活动连接监控。

- **类别**：平台
- **复杂度**：高级
- **路径数**：183 | **模式数**：511
- **层级**：标准
- **相关域**：`users`、`authentication`、`system`

## 使用场景

- 管理用户配置文件和通知偏好设置
- 配置会话控制和 OTP 设置
- 处理身份管理操作
- 处理初始用户访问请求

## 主要资源

- **user_profile**：包含个人设置的用户配置文件
- **session**：用于身份验证状态的用户会话
- **otp_policy**：多因素身份验证的 OTP 策略
