用于安全运营的检测策略、攻击跟踪和自动化缓解规则生成。

- **类别**：Security
- **复杂度**：moderate
- **路径数**：1 | **模式数**：161
- **层级**：Standard
- **相关领域**：`bot_defense`、`ddos`

## 使用场景

- 跟踪和分析威胁活动
- 监控活跃威胁和攻击模式
- 配置威胁情报集成

## 主要资源

- **threat_campaign_policy**：威胁活动检测和缓解策略

## 端点

| 方法 | 路径 | 描述 |
|--------|------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | 根据 ID 获取威胁活动。 |
