跨区域的流量分配与路由规则。健康检查与故障转移策略。

- **类别**：网络
- **复杂度**：高级
- **路径数**：133 | **模式数**：903
- **层级**：高级
- **相关域**：`dns`、`service_policy`、`network`

## 用例

- 配置 HTTP/TCP/UDP 负载均衡器
- 管理源站池和服务
- 配置虚拟主机和路由
- 定义速率限制器和服务策略
- 管理基于地理位置的路由
- 配置代理和转发策略
- 管理恶意软件防护和威胁活动
- 配置健康检查和端点监控

## 主要资源

- **http_loadbalancer**：第 7 层 HTTP/HTTPS 负载均衡器，用于应用流量分配
- **tcp_loadbalancer**：第 4 层 TCP/UDP 负载均衡器，用于非 HTTP 协议流量
- **origin_pool**：后端服务器组，用于负载均衡器流量分配
- **healthcheck**：源服务器可用性的健康监控配置
- **app_firewall**：用于 HTTP 流量防护的 Web 应用防火墙策略
- **service_policy**：服务级别的访问控制和流量管理规则
- **malicious_user_detection**：恶意用户行为的自动检测和缓解
