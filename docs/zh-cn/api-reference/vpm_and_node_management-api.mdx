分布式节点管理的生命周期控制、集群配置和部署策略。

- **类别**：平台
- **复杂度**：简单
- **路径数**：1 | **模式数**：2
- **层级**：标准
- **相关域**：`sites`、`system`

## 使用场景

- 管理虚拟私有网格（VPM）配置
- 配置节点生命周期与管理
- 监控 VPM 与节点状态

## 主要资源

- **node_config**：用于边缘设备设置的节点配置
- **vpm_config**：用于虚拟平台管理的 VPM 配置

## 端点

| 方法 | 路径 | 描述 |
|--------|------|-------------|
| GET | `/api/data/namespaces/system/upgrade_status` | 升级状态。 |
