具有版本控制的命名空間範圍視覺元素。適用於入口介面的自訂小工具部署與目錄管理。

- **類別**：Platform
- **複雜度**：simple
- **路徑數**：2 | **結構描述數**：14
- **層級**：Standard
- **相關領域**：`admin`、`system`

## 使用案例

- 管理管理主控台的靜態 UI 元件
- 在命名空間內部署和擷取 UI 資源
- 設定主控台介面元素
- 管理自訂 UI 元件中繼資料

## 主要資源

- **ui_component**：用於主控台自訂的 UI 元件
- **static_asset**：用於 UI 資源的靜態資產

## 端點

| 方法 | 路徑 | 說明 |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | 列出 UI 靜態元件。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | 取得 UI 靜態元件。 |
