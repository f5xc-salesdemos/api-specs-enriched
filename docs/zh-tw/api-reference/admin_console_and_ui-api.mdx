具有版本控制的命名空間範疇視覺元素。針對入口網站介面的自訂小工具部署與目錄管理。

- **分類**：平台
- **複雜度**：簡單
- **路徑數**：2 | **結構描述數**：14
- **層級**：標準
- **相關網域**：`admin`、`system`

## 使用案例

- 管理管理控制台的靜態 UI 元件
- 在命名空間內部署與擷取 UI 資產
- 設定控制台介面元素
- 管理自訂 UI 元件中繼資料

## 主要資源

- **ui_component**：用於控制台自訂的 UI 元件
- **static_asset**：UI 資源的靜態資產

## 端點

| 方法 | 路徑 | 描述 |
|--------|------|-------------|
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components` | 列出 UI 靜態元件。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/static_components/&#123;name&#125;` | 取得 UI 靜態元件。 |
