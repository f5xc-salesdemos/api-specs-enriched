自然語言處理，具備品質訊號與異常監控功能。資料串流訂閱的令牌驗證。

- **類別**：AI
- **複雜度**：簡單
- **路徑數**：11 | **結構描述數**：66
- **層級**：進階


## 使用情境

- 存取 AI 驅動的功能
- 設定 AI 助理政策
- 啟用流量異常偵測
- 管理 AI 資料收集

## 主要資源

- **ai_policy**：用於生成式 AI 流量控制的 AI 政策
- **ai_gateway**：用於 LLM API 管理的 AI 閘道

## 端點

| 方法 | 路徑 | 描述 |
|--------|------|-------------|
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query` | 評估 AI 助理查詢。 |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/eval_query_feedback` | 評估 AI 助理查詢的回饋。 |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query` | AI 助理查詢。 |
| POST | `/api/gen-ai/namespaces/&#123;namespace&#125;/query_feedback` | AI 助理查詢的回饋。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/enable_feature` | 啟用功能。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/refresh_token` | 更新令牌。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/gettoken` | 訂閱 BFDP 管線。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/subscribe` | 訂閱 BFDP 管線。 |
| POST | `/api/ai_data/bfdp/namespaces/system/bfdp/unsubscribe` | 取消訂閱 BFDP 管線。 |
| POST | `/api/gia/gia/allocateip` | 透過全域 IP 分配器分配 IP。 |
| DELETE | `/api/gia/gia/deallocateip` | 透過全域 IP 分配器釋放 IP。 |
