具備自動分類與安全性分析的資源目錄。組織層級依功能或保護政策區隔存取權限。

- **類別**：安全性
- **複雜度**：進階
- **路徑數**：45 | **結構描述數**：279
- **層級**：進階
- **相關網域**：`virtual`、`network_security`

## 使用案例

- 探索並建立 API 目錄
- 測試 API 安全性與行為
- 管理 API 憑證
- 定義 API 群組與測試政策

## 主要資源

- **api_definition**：用於探索與保護的 API 結構描述定義
- **api_endpoint**：個別 API 端點設定與保護
- **api_rate_limit**：用於流量控制的 API 速率限制設定
