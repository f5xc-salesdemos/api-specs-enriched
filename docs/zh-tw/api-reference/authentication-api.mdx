身分管理，包含提供者整合、存取策略及憑證生命週期控制。

- **類別**：平台
- **複雜度**：簡單
- **路徑數**：14 | **結構描述數**：38
- **層級**：標準
- **相關領域**：`system`、`users`

## 使用案例

- 設定認證機制
- 管理 OIDC 和 OAuth 提供者
- 設定 SCIM 使用者佈建
- 管理 API 憑證與存取權限
- 設定帳號註冊策略

## 主要資源

- **authentication_policy**：使用者與 API 存取的認證策略
- **token**：用於程式化存取的 API 權杖
- **api_credential**：用於服務認證的 API 憑證

## 端點

| 方法 | 路徑 | 說明 |
|--------|------|-------------|
| POST | `/api/web/namespaces/system/bulk_revoke/api_credentials` | 批量撤銷 API 憑證。 |
| POST | `/api/web/namespaces/system/bulk_revoke/service_credentials` | 批量撤銷服務憑證。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/activate/api_credentials` | 啟用 API 憑證。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/activate/service_credentials` | 啟用 API 服務憑證。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/api_credentials` | 列出 API 憑證。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/api_credentials` | 建立 API 憑證。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/api_credentials/&#123;name&#125;` | 取得 API 憑證。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/renew/api_credentials` | 續期 API 憑證。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/renew/service_credentials` | 續期 API 服務憑證。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/revoke/api_credentials` | 撤銷 API 憑證。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/revoke/scim_token` | 撤銷 SCIM API 憑證。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/revoke/service_credentials` | 撤銷服務憑證。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/scim_token` | 取得 SCIM 權杖。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/scim_token` | 建立/重新建立 SCIM API 權杖。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/service_credentials` | 列出服務憑證。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/service_credentials` | 建立服務憑證。 |
| GET | `/api/web/namespaces/&#123;namespace&#125;/service_credentials/&#123;name&#125;` | 取得服務憑證。 |
| PUT | `/api/web/namespaces/&#123;namespace&#125;/service_credentials/&#123;name&#125;` | 替換服務憑證。 |
