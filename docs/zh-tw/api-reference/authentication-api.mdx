身份管理，包含提供者整合、存取策略與憑證生命週期控制。

- **類別**: Platform
- **複雜度**: simple
- **路徑數**: 14 | **結構描述**: 38
- **層級**: Standard
- **相關領域**: `system`, `users`

## 使用案例

- 配置身份驗證機制
- 管理 OIDC 與 OAuth 提供者
- 配置 SCIM 使用者佈建
- 管理 API 憑證與存取權限
- 配置帳號註冊策略

## 主要資源

- **authentication_policy**: 使用者與 API 存取的身份驗證策略
- **token**: 用於程式化存取的 API 權杖
- **api_credential**: 用於服務身份驗證的 API 憑證

## 端點

| 方法 | 路徑 | 說明 |
|--------|------|-------------|
| POST | `/api/web/namespaces/system/bulk_revoke/api_credentials` | 批次撤銷 API 憑證。 |
| POST | `/api/web/namespaces/system/bulk_revoke/service_credentials` | 批次撤銷服務憑證。 |
| POST | `/api/web/namespaces/&#123;namespace&#125;/activate/api_credentials` | 啟用 API 憑證。 |
