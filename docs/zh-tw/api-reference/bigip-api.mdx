使用 iRule 腳本與資料群組同步的舊式設備協調。虛擬伺服器綁定與指標收集。

- **分類**：平台
- **複雜度**：進階
- **路徑數**：29 | **結構描述數**：238
- **層級**：進階
- **相關網域**：`marketplace`

## 使用案例

- 管理 BigIP F5 設備
- 設定 iRule 腳本
- 管理資料群組
- 整合 BigIP CNE

## 主要資源

- **bigip_pool**：用於負載平衡整合的 BIG-IP 資源池
- **bigip_device**：BIG-IP 設備註冊與管理
