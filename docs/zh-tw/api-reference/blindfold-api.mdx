具有基於政策的存取控制之加密金鑰管理。稽核記錄與安全資料保護。

- **類別**：安全性
- **複雜度**：中等
- **路徑數**：27 | **綱要數**：163
- **層級**：進階
- **相關網域**：`client_side_defense`、`certificates`

## 使用案例

- 設定加密的秘密政策
- 管理敏感資料加密
- 執行資料保護政策

## 主要資源

- **blindfold_secret**：使用 Blindfold 加密的已加密秘密
- **secret_policy**：秘密管理政策設定
- **policy_document**：用於存取控制定義的政策文件
