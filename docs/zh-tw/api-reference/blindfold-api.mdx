具備基於策略的存取控制的加密金鑰管理。稽核日誌記錄與安全資料保護。

- **類別**：安全性
- **複雜度**：中等
- **路徑**：27 | **結構描述**：163
- **層級**：進階
- **相關領域**：`client_side_defense`、`certificates`

## 使用案例

- 設定密鑰的加密策略
- 管理敏感資料加密
- 實施資料保護策略

## 主要資源

- **blindfold_secret**：使用 Blindfold 加密的加密密鑰
- **secret_policy**：密鑰管理策略設定
- **policy_document**：存取控制定義的策略文件
