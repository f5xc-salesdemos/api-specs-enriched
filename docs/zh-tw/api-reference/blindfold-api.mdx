具備基於政策之存取控制的加密金鑰管理。稽核日誌記錄與安全資料保護。

- **類別**：安全性
- **複雜度**：中等
- **路徑數**：27 | **結構描述數**：163
- **層級**：進階
- **相關領域**：`client_side_defense`、`certificates`

## 使用案例

- 為加密配置密鑰政策
- 管理敏感資料加密
- 強制執行資料保護政策

## 主要資源

- **blindfold_secret**：使用 Blindfold 加密的加密密鑰
- **secret_policy**：密鑰管理政策配置
- **policy_document**：用於存取控制定義的政策文件
