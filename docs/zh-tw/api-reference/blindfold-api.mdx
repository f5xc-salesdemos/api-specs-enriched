具有基於原則的存取控制之加密金鑰管理。稽核記錄與安全資料保護。

- **類別**：安全性
- **複雜度**：中等
- **路徑**：27 | **結構描述**：163
- **層級**：進階
- **相關領域**：`client_side_defense`、`certificates`

## 使用案例

- 配置加密的機密原則
- 管理敏感資料加密
- 強制執行資料保護原則

## 主要資源

- **blindfold_secret**：使用 Blindfold 加密的已加密機密
- **secret_policy**：機密管理原則配置
- **policy_document**：用於存取控制定義的原則文件
