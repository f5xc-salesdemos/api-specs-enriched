透過行為分析與特徵比對進行威脅分類。自動封鎖惡意流量模式。

- **類別**：安全性
- **複雜度**：中等
- **路徑**：19 | **結構描述**：74
- **層級**：進階
- **相關領域**：`bot_defense`、`shape`、`virtual`

## 使用案例

- 為每個命名空間配置機器人防禦實例
- 管理 TPM 威脅分類以進行歸類
- 為自動化防禦系統佈建 API 金鑰
- 整合威脅情報服務

## 主要資源

- **bot_defense_instance**：用於部署的機器人防禦實例
- **threat_category**：用於歸類的威脅類別
