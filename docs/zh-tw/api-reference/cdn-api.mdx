全球分佈式快取規則與清除操作。效能監控與分析。

- **類別**: 網路
- **複雜度**: 進階
- **路徑數**: 29 | **結構描述**: 592
- **層級**: 進階
- **相關領域**: `virtual`

## 使用案例

- 設定 CDN 負載平衡
- 管理內容傳遞網路服務
- 設定快取策略
- 管理資料傳遞與分發

## 主要資源

- **cdn_loadbalancer**: 用於邊緣快取的內容傳遞網路負載平衡器
- **cdn_origin_pool**: 用於 CDN 內容來源的原始伺服器池
