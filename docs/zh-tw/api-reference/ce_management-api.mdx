基於令牌的佈建，支援映像檔下載與升級前驗證。機群分組可跨分散式位置執行批量操作。

- **類別**：基礎架構
- **複雜度**：中等
- **路徑數**：28 | **結構描述數**：231
- **層級**：標準
- **相關網域**：`customer_edge`、`sites`

## 使用案例

- 管理 Customer Edge 站點生命週期
- 設定網路介面與機群設定
- 處理站點註冊與令牌工作流程
- 執行含升級前檢查的站點升級

## 主要資源

- **site_config**：邊緣節點設定的站點組態
- **fleet_config**：多站點管理的機群組態
- **registration_token**：站點上線的註冊令牌
