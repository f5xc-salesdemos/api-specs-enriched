憑證鏈和受信任的 CA 套件。撤銷清單管理以及 PKI 操作的清單配置。

- **類別**：安全性
- **複雜度**：中等
- **路徑**：16 | **結構描述**：74
- **層級**：標準
- **相關領域**：`blindfold`、`system`

## 使用案例

- 管理 SSL/TLS 憑證
- 配置受信任的 CA
- 管理憑證撤銷清單 (CRL)
- 配置憑證清單

## 主要資源

- **certificate**：用於安全 HTTPS 連線的 TLS 憑證
- **ca_certificate**：用於信任鏈的憑證授權單位憑證
- **certificate_chain**：用於完整信任驗證的憑證鏈
