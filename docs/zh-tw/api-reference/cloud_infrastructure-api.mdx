多雲供應商連線，支援閘道對等互連與網路路徑配置。憑證保管庫整合與子網路列舉。

- **類別**：基礎架構
- **複雜度**：中等
- **路徑數**：31 | **綱要數**：208
- **層級**：標準
- **相關領域**：`sites`、`customer_edge`

## 使用案例

- 連線至雲端供應商（AWS、Azure、GCP）
- 管理雲端憑證與驗證
- 配置雲端連線與彈性佈建
- 連結與管理雲端區域

## 主要資源

- **aws_vpc_site**：AWS VPC 站點部署，含邊緣節點佈建
- **azure_vnet_site**：Azure VNet 站點部署，含邊緣節點佈建
- **gcp_vpc_site**：Google Cloud VPC 站點部署，含邊緣節點佈建
- **cloud_credentials**：用於站點部署的雲端供應商驗證憑證
