具備自訂分類規則的敏感資料政策。LMA 區域設定與以地理位置為基礎的合規性控制。

- **類別**：安全性
- **複雜度**：簡單
- **路徑數**：11 | **結構描述數**：65
- **層級**：進階
- **相關領域**：`blindfold`、`client_side_defense`

## 使用案例

- 設定敏感資料偵測政策
- 定義 PII 分類的自訂資料類型
- 管理 LMA 區域設定
- 整合地理設定以符合合規性要求

## 主要資源

- **sensitive_data_policy**：用於 PII 保護的敏感資料政策
- **data_classification**：用於內容分類的資料分類
