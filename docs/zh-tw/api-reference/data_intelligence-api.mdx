用於數據分析工作流程的分類規則、資源政策與洞察生成。

- **類別**：操作
- **複雜度**：簡單
- **路徑數**：15 | **結構描述數**：73
- **層級**：標準
- **相關領域**：`statistics`、`observability`

## 使用案例

- 分析安全性與流量數據
- 從日誌生成智能洞察
- 配置數據分析政策

## 主要資源

- **analytics_query**：用於數據分析的分析查詢
- **data_export**：用於報告的數據導出配置

## 端點

| 方法 | 路徑 | 描述 |
|--------|------|-------------|
| POST | `/api/data-intelligence/namespaces/&#123;metadata.namespace&#125;/receivers` | 建立數據傳遞。 |
| PUT | `/api/data-intelligence/namespaces/&#123;metadata.namespace&#125;/receivers/&#123;metadata.name&#125;` | 替換數據傳遞。 |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers` | 列出數據傳遞。 |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers/&#123;name&#125;` | 取得數據傳遞。 |
| DELETE | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers/&#123;name&#125;` | 刪除數據傳遞。 |
| GET | `/api/data-intelligence/namespaces/system/dataSets` | 取得數據集。 |
| GET | `/api/data-intelligence/namespaces/system/datadictionary/&#123;dataset&#125;` | 取得數據字典。 |
| POST | `/api/data-intelligence/namespaces/system/init-request` | 啟用數據智能。 |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/data-dictionary/datasets` | 列出數據集。 |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/di/flowlabels` | 列出流量標籤。 |
| POST | `/api/data-intelligence/namespaces/&#123;namespace&#125;/di/summary` | 載入執行摘要。 |
| POST | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers/&#123;id&#125;/status` | 更新接收器狀態。 |
| POST | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers/&#123;id&#125;/test` | 測試接收器。 |
| POST | `/api/data-intelligence/namespaces/&#123;namespace&#125;/suggest-values` | 建議值。 |
| POST | `/api/data-intelligence/namespaces/system/data-intelligence/addon/subscribe` | 訂閱數據智能。 |
| POST | `/api/data-intelligence/namespaces/system/data-intelligence/addon/unsubscribe` | 取消訂閱用戶端防護。 |
