用於資料分析工作流程的分類規則、資源政策及洞察生成。

- **類別**：Operations
- **複雜度**：simple
- **路徑數**：15 | **結構描述**：73
- **層級**：Standard
- **相關領域**：`statistics`、`observability`

## 使用案例

- 分析安全性及流量資料
- 從日誌生成智慧洞察
- 設定資料分析政策

## 主要資源

- **analytics_query**：用於資料分析的分析查詢
- **data_export**：用於報告的資料匯出設定

## 端點

| 方法 | 路徑 | 說明 |
|--------|------|-------------|
| POST | `/api/data-intelligence/namespaces/&#123;metadata.namespace&#125;/receivers` | 建立資料交付。 |
| PUT | `/api/data-intelligence/namespaces/&#123;metadata.namespace&#125;/receivers/&#123;metadata.name&#125;` | 替換資料交付。 |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers` | 列出資料交付。 |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers/&#123;name&#125;` | 取得資料交付。 |
| DELETE | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers/&#123;name&#125;` | 刪除資料交付。 |
| GET | `/api/data-intelligence/namespaces/system/dataSets` | 取得資料集。 |
| GET | `/api/data-intelligence/namespaces/system/datadictionary/&#123;dataset&#125;` | 取得資料字典。 |
| POST | `/api/data-intelligence/namespaces/system/init-request` | 啟用資料智慧。 |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/data-dictionary/datasets` | 列出資料集。 |
| GET | `/api/data-intelligence/namespaces/&#123;namespace&#125;/di/flowlabels` | 列出流量標籤。 |
| POST | `/api/data-intelligence/namespaces/&#123;namespace&#125;/di/summary` | 載入執行摘要。 |
| POST | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers/&#123;id&#125;/status` | 更新接收器狀態。 |
| POST | `/api/data-intelligence/namespaces/&#123;namespace&#125;/receivers/&#123;id&#125;/test` | 測試接收器。 |
| POST | `/api/data-intelligence/namespaces/&#123;namespace&#125;/suggest-values` | 建議值。 |
| POST | `/api/data-intelligence/namespaces/system/data-intelligence/addon/subscribe` | 訂閱資料智慧。 |
| POST | `/api/data-intelligence/namespaces/system/data-intelligence/addon/unsubscribe` | 取消訂閱用戶端防護。 |
