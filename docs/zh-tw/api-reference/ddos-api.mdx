拒絕清單、防火牆規則群組及基於隧道的防護措施。針對網路邊界安全的速率限制與模式分析。

- **類別**：安全性
- **複雜度**：進階
- **路徑**：60 | **結構描述**：246
- **層級**：進階
- **相關領域**：`network_security`、`virtual`

## 使用案例

- 設定 DDoS 防護策略
- 監控與分析 DDoS 威脅
- 設定基礎架構防護

## 主要資源

- **ddos_protection**：用於流量清洗的 DDoS 防護設定
- **ddos_mitigation_rule**：用於攻擊回應的 DDoS 緩解規則
