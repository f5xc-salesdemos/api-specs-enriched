具備區域傳輸與健康檢查的名稱解析功能。支援記錄類型與委派。

- **類別**：網路
- **複雜度**：進階
- **路徑數**：49 | **結構描述**：285
- **層級**：標準
- **相關領域**：`virtual`、`network`

## 使用案例

- 設定 DNS 負載平衡
- 管理 DNS 區域與網域
- 設定 DNS 合規政策
- 管理資源記錄集 (RRSets)

## 主要資源

- **dns_zone**：具備記錄管理功能的權威 DNS 區域
- **dns_domain**：DNS 網域委派與設定配置
- **dns_load_balancer**：基於 DNS 的流量分配，支援地理路由與故障轉移
