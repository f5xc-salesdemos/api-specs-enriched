Kubernetes 角色繫結與存取許可政策。支援 EKS、AKS 及 GKE 工作負載的登錄檔整合。

- **類別**：基礎設施
- **複雜度**：中等
- **路徑數**：20 | **結構描述數**：95
- **層級**：進階
- **相關領域**：`container_services`、`sites`、`service_mesh`

## 使用案例

- 管理 XCKS（受管理的 Kubernetes）叢集 RBAC 及安全性
- 設定 Pod 安全政策與存取許可控制器
- 管理企業部署的容器登錄檔
- 與外部 Kubernetes 叢集整合（EKS、AKS、GKE）

## 主要資源

- **mk8s_cluster**：受管理的 Kubernetes 叢集設定
- **k8s_cluster_role**：Kubernetes 叢集 RBAC 角色設定
- **container_registry**：工作負載映像的容器映像登錄檔
