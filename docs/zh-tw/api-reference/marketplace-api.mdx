具有 DPD 計時器的第三方 GRE 和 IPSec 隧道佈建。跨命名空間的共享資源分配與磁貼放置控制。

- **類別**：平台
- **複雜度**：中等
- **路徑數**：36 | **結構描述數**：183
- **層級**：進階
- **相關領域**：`bigip`、`admin`

## 使用案例

- 存取第三方整合與附加元件
- 管理市集擴充功能
- 設定 Terraform 及外部整合
- 管理 TPM 政策

## 主要資源

- **marketplace_item**：服務目錄的市集項目
- **subscription**：服務授權的訂閱
