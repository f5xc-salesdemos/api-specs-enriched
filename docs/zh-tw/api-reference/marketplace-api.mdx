第三方 GRE 和 IPSec 隧道佈建，支援 DPD 計時器。跨命名空間的共享資源分配，具備圖塊放置控制功能。

- **類別**：Platform
- **複雜度**：moderate
- **路徑**：36 | **結構描述**：183
- **層級**：Advanced
- **相關領域**：`bigip`、`admin`

## 使用案例

- 存取第三方整合與附加元件
- 管理市集擴充功能
- 設定 Terraform 與外部整合
- 管理 TPM 政策

## 主要資源

- **marketplace_item**：服務目錄的市集項目
- **subscription**：服務權限的訂閱
