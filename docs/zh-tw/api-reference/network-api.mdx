邊界閘道協定，具備 ASN 管理與自治系統關係功能。站點對站點 VPN 透過加密通道連結資料中心。

- **類別**：網路
- **複雜度**：進階
- **路徑數**：81 | **綱要數**：449
- **層級**：進階
- **相關領域**：`virtual`、`network_security`、`dns`

## 使用情境

- 設定 BGP 路由與 ASN 管理
- 管理 IPsec 隧道與 IKE 階段
- 設定網路連接器與路由
- 管理 SRv6 與子網路劃分
- 定義分段連線與策略
- 設定 IP 前綴集合

## 主要資源

- **virtual_network**：用於站點連線與分段的虛擬網路
- **network_connector**：用於站點對站點或雲端連線的網路連接器
- **site_mesh_group**：多站點間的網狀連線配置
