基於來源、目的地或協定進行路由決策的防火牆規則。分段隔離工作負載，而出站代理控制存取。

- **類別**：安全
- **複雜度**：進階
- **路徑**：62 | **結構描述**：386
- **層級**：進階
- **相關領域**：`virtual`、`api`、`network`

## 使用案例

- 設定網路防火牆和 ACL 策略
- 管理 NAT 策略和連接埠轉發
- 設定基於策略的路由
- 定義網路分段和策略
- 設定正向代理策略

## 主要資源

- **network_policy**：用於流量過濾的網路安全策略
- **forward_proxy_policy**：用於出站流量控制的正向代理策略
- **network_firewall**：用於第 3/4 層流量保護的網路防火牆
