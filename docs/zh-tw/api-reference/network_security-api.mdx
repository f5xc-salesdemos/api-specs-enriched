基於來源、目的地或協定進行路由決策的防火牆規則。網段隔離可分離工作負載，而出站代理則管控存取權限。

- **類別**：安全
- **複雜度**：進階
- **路徑**：62 | **結構描述**：386
- **層級**：進階
- **相關領域**：`virtual`、`api`、`network`

## 使用案例

- 設定網路防火牆和 ACL 政策
- 管理 NAT 政策和連接埠轉發
- 設定基於政策的路由
- 定義網路區段和政策
- 設定正向代理政策

## 主要資源

- **network_policy**：用於流量過濾的網路安全政策
- **forward_proxy_policy**：用於出站流量控制的正向代理政策
- **network_firewall**：用於第 3/4 層流量保護的網路防火牆
