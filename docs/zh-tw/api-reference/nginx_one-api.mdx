實例探索、WAF 整合與服務網格連接。訂閱生命週期與組態同步。

- **類別**：平台
- **複雜度**：簡單
- **路徑數**：13 | **結構描述數**：59
- **層級**：進階
- **相關網域**：`marketplace`

## 使用案例

- 管理 NGINX One 平台整合
- 設定 NGINX Plus 實例
- 整合 NGINX 組態管理

## 主要資源

- **nginx_config**：代理設定的 NGINX 組態
- **nginx_upstream**：後端伺服器池的 NGINX 上游

## 端點

| 方法 | 路徑 | 說明 |
|--------|------|-------------|
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_csgs` | 列出 NGINX One CSG 物件組態。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_csgs/&#123;name&#125;` | GET 請求。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_instances` | 列出 NGINX One 實例物件組態。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_instances/&#123;name&#125;` | GET 請求。 |
| POST | `/api/config/namespaces/&#123;namespace&#125;/nginx_dataplane_servers` | 取得 NGINX One 資料平面伺服器。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_servers` | 列出 NGINX One 伺服器 API。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_servers/&#123;name&#125;` | GET 請求。 |
| POST | `/api/nginx/one/namespaces/system/n1/subscribe` | 訂閱 NGINX One。 |
| POST | `/api/nginx/one/namespaces/system/n1/unsubscribe` | 取消訂閱 NGINX One。 |
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/nginx_service_discoverys` | 建立 NGINX 服務探索。 |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/nginx_service_discoverys/&#123;metadata.name&#125;` | 取代 NGINX 服務探索。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_service_discoverys` | 列出 NGINX 服務探索。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/nginx_service_discoverys/&#123;name&#125;` | 取得 NGINX 服務探索。 |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/nginx_service_discoverys/&#123;name&#125;` | 刪除 NGINX 服務探索。 |
