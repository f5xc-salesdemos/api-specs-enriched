為行動應用程式整合者提供的版本化程式庫分發。預簽名 URL 可為 iOS 和 Android 提供安全的作業系統特定建置下載。

- **類別**：平台
- **複雜度**：簡單
- **路徑**：9 | **結構描述**：17
- **層級**：進階
- **相關領域**：`marketplace`

## 使用案例

- 管理物件儲存服務
- 設定儲存物件和儲存桶
- 管理儲存策略

## 主要資源

- **object_store**：用於 Blob 儲存管理的物件儲存
- **bucket**：物件儲存的儲存桶
