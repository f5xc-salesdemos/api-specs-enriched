適用於行動應用程式整合者的版本化函式庫發布。預簽署 URL 可透過 iOS 和 Android 的特定作業系統建置，實現安全下載。

- **類別**：平台
- **複雜度**：簡單
- **路徑數**：9 | **結構描述數**：17
- **層級**：進階
- **相關網域**：`marketplace`

## 使用案例

- 管理物件儲存服務
- 設定已儲存物件與儲存桶
- 管理儲存原則

## 主要資源

- **object_store**：用於 Blob 儲存管理的物件儲存庫
- **bucket**：用於物件儲存的儲存桶
