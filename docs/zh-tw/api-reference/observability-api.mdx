具備延遲測量的 HTTP 可用性探測。憑證到期警報與全域狀態儀表板，用於基礎架構健康監控。

- **類別**：營運
- **複雜度**：中等
- **路徑數**：56 | **結構描述**：196
- **層級**：標準
- **相關領域**：`statistics`、`support`

## 使用案例

- 設定合成監控
- 定義監控與測試策略
- 管理可觀測性儀表板

## 主要資源

- **log_receiver**：用於集中式日誌收集的日誌接收器
- **metrics_receiver**：用於集中式指標收集的指標接收器
- **alert_policy**：用於監控與通知的警報策略
