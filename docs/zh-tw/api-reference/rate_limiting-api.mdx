基於時間的配額執行，支援以小時、分鐘或秒為單位的可配置時間窗口。針對流量整形的協定特定控制。

- **類別**：網路
- **複雜度**：簡單
- **路徑數**：12 | **結構描述數**：70
- **層級**：進階
- **相關領域**：`virtual`、`network_security`

## 使用案例

- 配置速率限制器策略
- 管理策略執行器配置
- 控制流量與佇列

## 主要資源

- **rate_limiter**：用於流量節流與保護的速率限制器
- **rate_limiter_policy**：包含詳細節流規則的速率限制器策略
- **rate_limit_threshold**：用於流量控制的速率限制閾值配置
