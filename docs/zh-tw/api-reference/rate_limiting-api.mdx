基於時間的配額強制執行，支援以小時、分鐘或秒為單位的可配置時間窗口。提供協定專屬的流量整形控制。

- **類別**：網路
- **複雜度**：簡單
- **路徑數**：12 | **結構描述數**：70
- **層級**：進階
- **相關網域**：`virtual`、`network_security`

## 使用情境

- 配置速率限制器策略
- 管理流量管控配置
- 控制流量流向與佇列

## 主要資源

- **rate_limiter**：用於流量節流與防護的速率限制器
- **rate_limiter_policy**：包含詳細節流規則的速率限制器策略
- **rate_limit_threshold**：用於流量控制的速率限制閾值配置
