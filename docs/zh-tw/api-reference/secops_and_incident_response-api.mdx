具備威脅等級分類的惡意使用者緩解機制。針對可疑行為模式的自動回應動作。

- **類別**：安全性
- **複雜度**：簡單
- **路徑數**：4 | **綱要數**：32
- **方案層級**：進階
- **相關網域**：`bot_defense`、`virtual`、`network_security`

## 使用案例

- 設定自動化威脅緩解政策
- 定義惡意使用者偵測規則
- 管理事件回應工作流程
- 對威脅套用封鎖或速率限制

## 主要資源

- **mitigation_policy**：事件回應的緩解政策
- **malicious_user_rule**：威脅緩解的惡意使用者規則

## 端點

| 方法 | 路徑 | 說明 |
|--------|------|-------------|
| POST | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations` | 建立惡意使用者緩解措施。 |
| PUT | `/api/config/namespaces/&#123;metadata.namespace&#125;/malicious_user_mitigations/&#123;metadata.name&#125;` | 取代惡意使用者緩解措施。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations` | 列出惡意使用者緩解措施。 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | 取得惡意使用者緩解措施。 |
| DELETE | `/api/config/namespaces/&#123;namespace&#125;/malicious_user_mitigations/&#123;name&#125;` | 刪除惡意使用者緩解措施。 |
