應用程式類型定義，包含探索與學習式結構描述分析。流量模式推斷，實現智慧請求處理。

- **類別**：基礎架構
- **複雜度**：進階
- **路徑**：38 | **結構描述**：247
- **層級**：進階
- **相關領域**：`managed_kubernetes`、`container_services`、`virtual`

## 使用案例

- 配置服務網格連線
- 管理端點探索與路由
- 配置 NFV 服務
- 定義應用程式設定與類型

## 主要資源

- **endpoint**：用於流量路由的服務網格端點
- **origin_pool**：用於負載平衡器流量分配的後端伺服器群組
- **service_discovery**：用於動態端點的服務探索配置
