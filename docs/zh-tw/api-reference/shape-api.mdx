透過行為分析與裝置指紋識別進行威脅辨識。整合行動 SDK 以實現應用程式防護。

- **類別**：安全性
- **複雜度**：進階
- **路徑**：271 | **結構描述**：813
- **層級**：進階
- **相關領域**：`bot_defense`、`virtual`

## 使用案例

- 設定 Shape Security 策略
- 管理機器人與威脅防護
- 設定 SafeAP 策略
- 啟用威脅辨識

## 主要資源

- **shape_app_firewall**：Shape 應用程式防火牆，用於進階防護
- **shape_recognizer**：Shape 辨識器，用於流量模式分析
