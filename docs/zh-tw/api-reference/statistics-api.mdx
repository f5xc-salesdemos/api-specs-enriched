具備接收器整合的警報策略。日誌聚合、拓撲視圖與站點健康狀態追蹤。

- **類別**：Operations
- **複雜度**：advanced
- **路徑數**：77 | **結構描述數**：575
- **層級**：Standard
- **相關領域**：`observability`、`support`

## 使用案例

- 存取流量統計與分析
- 管理警報及警報策略
- 檢視日誌與日誌接收器
- 產生報告與圖表
- 追蹤拓撲與服務發現
- 監控站點狀態

## 主要資源

- **dashboard**：用於指標視覺化的儀表板
- **saved_query**：用於指標與日誌分析的已儲存查詢
