包含留言、嚴重性變更及解決追蹤的問題生命週期。用於連線分析的封包擷取。

- **類別**：營運
- **複雜度**：中等
- **路徑**：53 | **結構定義**：167
- **層級**：標準
- **相關領域**：`statistics`、`observability`

## 使用案例

- 提交與管理支援工單
- 追蹤客戶支援請求
- 存取營運支援文件

## 主要資源

- **support_case**：用於問題追蹤與解決的支援案例
- **alert**：用於營運通知的系統警報
- **audit_log**：用於合規性與安全性追蹤的稽核日誌
