用於可觀測性工作流程的收集端點、指標儲存和洞察生成。

- **類別**：營運
- **複雜度**：中等
- **路徑**：27 | **結構描述**：181
- **層級**：標準
- **相關領域**：`observability`、`statistics`

## 使用案例

- 收集並分析遙測資料
- 從指標生成可操作的洞察
- 配置遙測收集策略

## 主要資源

- **telemetry_receiver**：用於資料收集的遙測接收器
- **insight_query**：用於分析的洞察查詢
