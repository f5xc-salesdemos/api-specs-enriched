帳戶檢視配置與管理員警示通道。一次性密碼重設、佈建流程與活躍連線監控。

- **類別**: 平台
- **複雜度**: 進階
- **路徑**: 183 | **結構描述**: 511
- **層級**: 標準
- **相關領域**: `users`、`authentication`、`system`

## 使用案例

- 管理使用者個人檔案與通知偏好設定
- 配置工作階段控制與 OTP 設定
- 處理身分識別管理操作
- 處理初始使用者存取請求

## 主要資源

- **user_profile**: 包含個人設定的使用者個人檔案
- **session**: 用於驗證狀態的使用者工作階段
- **otp_policy**: 用於多因素驗證的 OTP 政策
