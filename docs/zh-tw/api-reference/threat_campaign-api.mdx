安全運營的偵測策略、攻擊追蹤及自動化緩解規則生成。

- **類別**：安全性
- **複雜度**：中等
- **路徑數**：1 | **結構描述數**：161
- **層級**：標準
- **相關網域**：`bot_defense`、`ddos`

## 使用案例

- 追蹤並分析威脅活動
- 監控活躍威脅與攻擊模式
- 設定威脅情報整合

## 主要資源

- **threat_campaign_policy**：威脅活動偵測與緩解策略

## 端點

| 方法 | 路徑 | 說明 |
|--------|------|-------------|
| GET | `/api/waf/threat_campaign/&#123;id&#125;` | 依 ID 取得威脅活動。 |
