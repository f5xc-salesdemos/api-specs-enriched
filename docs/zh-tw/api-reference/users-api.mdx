具備自動過期功能的站台註冊憑證。分類法鍵定義允許的分類方式，而自動衍生的標籤則會動態套用。

- **類別**：平台
- **複雜度**：簡單
- **路徑數**：13 | **結構描述數**：50
- **層級**：標準
- **相關網域**：`system`、`admin`

## 使用情境

- 管理使用者帳號與權杖
- 設定使用者識別資訊
- 管理使用者設定與偏好
- 設定隱含標籤與已知標籤

## 主要資源

- **user**：用於存取平台的使用者帳號
- **user_role**：用於權限管理的使用者角色
- **namespace_role**：用於細粒度權限控制的命名空間範圍角色

## 端點

| 方法 | 路徑 | 說明 |
|--------|------|-------------|
| GET | `/api/config/namespaces/system/implicit_labels` | 取得隱含標籤。 |
| POST | `/api/config/namespaces/&#123;namespace&#125;/known_label/create` | 建立 |
| POST | `/api/config/namespaces/&#123;namespace&#125;/known_label/delete` | 刪除 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/known_labels` | 取得 |
| POST | `/api/config/namespaces/&#123;namespace&#125;/known_label_key/create` | 建立 |
| POST | `/api/config/namespaces/&#123;namespace&#125;/known_label_key/delete` | 刪除 |
| GET | `/api/config/namespaces/&#123;namespace&#125;/known_label_keys` | 取得 |
| GET | `/api/register/namespaces/system/get-cloud-init-config` | 取得 Cloud Init 設定。 |
| POST | `/api/register/namespaces/&#123;metadata.namespace&#125;/tokens` | 建立權杖。 |
| PUT | `/api/register/namespaces/&#123;metadata.namespace&#125;/tokens/&#123;metadata.name&#125;` | 取代權杖。 |
| GET | `/api/register/namespaces/&#123;namespace&#125;/tokens` | 列出權杖。 |
| GET | `/api/register/namespaces/&#123;namespace&#125;/tokens/&#123;name&#125;` | 取得權杖 |
| DELETE | `/api/register/namespaces/&#123;namespace&#125;/tokens/&#123;name&#125;` | 刪除權杖。 |
| POST | `/api/register/namespaces/&#123;namespace&#125;/tokens/&#123;name&#125;/state` | 設定權杖狀態。 |
