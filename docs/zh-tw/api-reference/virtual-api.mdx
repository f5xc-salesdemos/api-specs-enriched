跨區域的流量分配與路由規則。健康檢查與故障轉移策略。

- **類別**：網路
- **複雜度**：進階
- **路徑**：133 | **結構描述**：903
- **層級**：進階
- **相關領域**：`dns`、`service_policy`、`network`

## 使用案例

- 設定 HTTP/TCP/UDP 負載平衡器
- 管理來源池與服務
- 設定虛擬主機與路由
- 定義速率限制器與服務策略
- 管理基於地理位置的路由
- 設定代理與轉發策略
- 管理惡意軟體防護與威脅活動
- 設定健康檢查與端點監控

## 主要資源

- **http_loadbalancer**：第 7 層 HTTP/HTTPS 負載平衡器，用於應用程式流量分配
- **tcp_loadbalancer**：第 4 層 TCP/UDP 負載平衡器，用於非 HTTP 協定流量
- **origin_pool**：後端伺服器群組，用於負載平衡器流量分配
- **healthcheck**：來源伺服器可用性的健康監控設定
- **app_firewall**：用於 HTTP 流量防護的 Web 應用程式防火牆策略
- **service_policy**：服務層級的存取控制與流量管理規則
- **malicious_user_detection**：惡意使用者行為的自動偵測與緩解
