分散式節點管理的生命週期控制、叢集配置與部署策略。

- **類別**：Platform
- **複雜度**：simple
- **路徑數**：1 | **結構描述**：2
- **層級**：Standard
- **相關領域**：`sites`、`system`

## 使用案例

- 管理虛擬私有網格（VPM）配置
- 設定節點生命週期與管理
- 監控 VPM 與節點狀態

## 主要資源

- **node_config**：邊緣裝置設定的節點配置
- **vpm_config**：虛擬平台管理的 VPM 配置

## 端點

| 方法 | 路徑 | 說明 |
|--------|------|-------------|
| GET | `/api/data/namespaces/system/upgrade_status` | 升級狀態。 |
