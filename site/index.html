    F5 Distributed Cloud API Documentation¶

    Welcome to the comprehensive API documentation for F5 Distributed Cloud Services (F5 XC).

    Overview¶

    This documentation provides enriched API specifications for F5 XC services including:

    • Load Balancing - HTTP/HTTPS load balancing and traffic management
    • Security - WAF, bot defense, API protection, and threat intelligence
    • Networking - Virtual networks, service mesh, and connectivity
    • Infrastructure - Sites, clusters, and cloud infrastructure management
    • Observability - Monitoring, logging, and analytics

    Features¶

    Enriched Specifications

    All API specifications are enriched with:

    • Acronym normalization (100+ terms)
    • Grammar improvements
    • Branding updates (Volterra → F5 Distributed Cloud)
    • Fixed orphan references
    • Type standardization
    • Validated with Spectral OpenAPI linter

    Multi-Domain Organization

    Specifications are organized by domain for easier navigation:

    • Virtual (HTTP/HTTPS load balancing)
    • WAF (Web Application Firewall)
    • Network (Networking and connectivity)
    • Sites (Infrastructure sites)
    • And 30+ more domains
